MENU : Data - Points

 

 

This page displays the survey points that belongs to the selected station of the active DTM project.

Use the menu functions to add, edit or delete any of the survey points.

 

The header panel

 

       Spot        This is the number of the point relative to all the other points surveyed from this station.

       Pnt                This is the number of the point relative to all the points of all the stations belonging to the active project.

       Code        The code assigned to the point.

       Tag                The tag assigned to the point.

       Prism        The height / length of the prism used in the survey. If the Top and Bottom wire options are selected in the station configuration, this column will hold the Bottom wire value.

               -----         If the Top and Bottom wire options are selected in the station configuration, this column will hold the Top wire value.

               Hd        The horizontal distance column. This column has a dual purpose. If the selected station configuration requires that the slope (direct distance) must be entered, this column receives the value and after the line input has been completed and the survey point reduced, displays the reduced horizontal distance in the same column.

 

 

EXAMPLE:

This is what the screen looks like when a new point is entered and the line not yet reduced. Notice that the slope distance of 100 has been entered in the HD column.

 

This what the screen looks like after the entry has been completed and the co-ordinates of the point calculated. Notice that the original slope value of 100 has been reduced to the horizontal equivalent of 99.587 and is now correctly being displayed in the HD column.

 

Note:

Take care when re-entering data over a previously defined spot. Model Maker saves the distance between a station and the survey spot in the horizontal format. This means that if you have entered the slope distance, the distance gets reduced to the horizontal value when the vertical angle is entered.

 

It is recommended that you completely re-enter the information of a survey point if you want change any one of it's values.

               Va                The vertical angle

               Dz                The relative height difference as calculated, taking the Va into account.

               Z,Y,X                The final co-ordinates of the point.

                         

 

Adding points

Survey points are added by going to the first available blank line and entering the relevant survey data. If the station does not already have any other survey points, use the first line in the editor to enter the data. If the station has other survey points, click the bottom line button and on the last line of the editor, enter the data.

Use the TAB key on the keyboard to move to the next column when entering the data.

If multiple lines of data are added and the bottom of the page is reached, the lines will automatically scroll up and present you with a blank line to continue inputting data.

 

Editing points

To edit an existing point, move the cursor to the line of the spot to be edited. Start in the Code column and enter the new values or accept the existing data.

Remember that the value you see in the HD column of an existing point is the horizontal distance value. If your survey format requires the slope distance to be entered, re-enter the value in the HD column. (See the previous NOTE). Entry is completed when you have tabbed through all the fields and the cursor returns to the code field of the next line. Notice that the data in the edited line is automatically recalculated when the cursor moves on to the next line.

 

Deleting points

To delete a point, place the cursor on the line of the spot to be deleted and press the button. The code of the point will change to *.*, the code used by Model Maker to indicate that the spot is marked for deletion. The points marked with *.* in their code field are not automatically removed from the system but are kept in the data base to maintain triangle numbering system intact.

To physically remove the *.* points from the system use the General Detele option.

Spots marked as deleted may be reactivated if necessary by changing their codes to any other value.

 

Definition of the Code field

 

The code item of DTM points may consist of up to six alphanumeric characters. The characters can be used in any combination except for the following ones which are reserved for special use:

       Any code starting with a '.' (dot) for example .rock will automatically be excluded from the model building process. If you have points in the survey which is not relevant to the topography, points with zero elevations, points on top of building etc., start their codes with the dot character

       The 01 code designates a field spot. The codes (01) of Field spots will not be displayed when the Code option in the plot parameters is selected.

       The *.* code used to indicate spots marked for deletion.

       The characters >, < and @ also have a special meaning which is described in the Triangle Detailsection.

 

 

Except for the above restrictions, any code may be used to identify the points.

       If a complicated site is surveyed and longer descriptions are necessary to name the survey points, give each point in your field book a unique number along with the full description of the spot. When entering these points into Model Maker, enter your point number in the code field. When the points are then plotted on the screen and the code option selected you will see your point numbers which will then provide a quick reference to the full description of each point in your survey fieldbook.

 

Column widths

 

Change the column width of the grid editor. A small width of one character or less allows Model Maker to ignore the column. This is useful entering data manually and if, for example, you do not have a tag to enter, but the program stops in that column every time.
