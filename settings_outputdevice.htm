MENU : Settings - Plotter & Pens

 

 

Set the printer settings for plotted output.

This list box displays the available printers installed by the WINDOWS settings option.

Select the to use as the final printer to produce the plot.

 

 

This button takes you to the colour control menu.

 

 

Use this button to access the WINDOWS setup menu for the selected printer. Change the paper size on this menu. Detail settings of the printer is done via the WINDOWS SETTINGS PRINTERS menus.

 

 

 

Checking the APPLY PEN WIDTHS allows Model Maker to use the width of the pens as set in the colour control menu when generating the drawing. If no pen widths are applied, all the lines will have a .18 line thickness.

The Y&X scale factor allows you handle for calibration problems between the plotter and the final paper drawing. A value of 1 represents a neutral state. A value of 1.05, for instance, applies a 5% enlargement to the drawing before placed on paper. It is seldom necessary to change this option.
