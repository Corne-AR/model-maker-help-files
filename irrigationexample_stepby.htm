Each block is 130m*130m.

 

The 4 blocks are next to each other forming a square of 260m*260 m

 

Elevations are given for corners of all the blocks.

This results in 9 co-ordinates. Four on the corners and 4 midway at each 130m mark and 1 in the center.

Micros operate at 10psi and deliver 2.46 l/h/m. Each emitter at 0.75m spacing would then deliver 2.46*0.75 = 1.85 l/h.

 

Proposed lateral inside diameters (ID) = 10mm and 15.2mm.

Method

We will make use of the 3 systems that Model Maker provides.

 

1) The CAD to draw the layout

2) The DTM to enter the elevations points and to produce the contours

3) Irri Maker to define the blocks, add the drip pipe and do the main line connections

 

Note:

The arrow symbol will be used to indicate actual instruction to be executed.

Note:

In a micro or drip system Irri Maker will simulate the specified emitters (sprinkler) to operate at the design interval.

In this example the discharge along the line is 2.46 l/h/m but the emitters are spaced at 0.75 meters. If we want to do the hydraulics very accurately, we should also specify the emitter spacing as 0.75 meters with each emitter discharging 2.46*0.75=1.85 l/h.

It would also not be too inaccurate to set the emitter spacing at 1m with the discharge of each emitter to be 2.46l/h in which case the values in the node data base must be updated accordingly.

To define new pipe types

 

Click on the Pipes page and then the Database button.

 

Click the ADD button. The Pipe Data box will be cleared for input.

 

Set the pipe units in the bottom left unit boxes.

   

 

In the Pipe Data box enter the pipe Code, Name and type.

 

We are not going to use the Darcy Weisbach formula.

 

 

In the Pipe Group box, enter the pipe name, Id and pressure.

The main line

The main line must be defined by first placing the pump and then connecting the pump and valves by means of the main pipeline.

 

On the right - the Block number list - Select 0 (zero=main line)

 

Select nodes page and Add Single. Choose a pump type.

 

Set the snap mode to Free. Press spacebar and select FREE.

 

Indicate the position of the pump just above point 1.

 

Note :

If the pump falls off the DTM, use edit SINGLE and enter the correct elevation. In this case it could be 107.1.0 meters.

 

The pump can be connected to the valves directly but we will introduce 2 Fitting nodes that will branch to the valves.

Nodes Page. Add single - Select fitting and define the position of fitting 1 (F1) and fitting 2 (F2)

 

Note:

If these fitting fall off the model. Use the Node Edit single function to change their elevations to: F1=107.0 and F2=104.0

 

Now add the main line pipe.

 

Layout with positions of fittings 1 & 2.

 

Note :

The position of valve 1 was manually defined while that of vales 2 to 4 were automatically defined when the blocks were copied.

We will use the same group of pipes that defined the manifolds in each block, to define the main line.

 

Pipes - Add Single.

 

Select the mainline pipe and start at the pump. Then point to F1, F2, V2 and V4. Press escape.

Start at V3 then point to V1 and then F1.
