The Google Earth program can be used for the following actions:

 

·Google Earth Export - The main purpose of the Google Earth Output function in Model Maker is to export dtm points to GE and to show them as markers on the terrain.

 

·Google Earth Utility - The Utility is used to update the elevations of existing dtm points in Model Maker with the elevations from the GE terrain.

 

·Google Earth Kml file - To transfer Markers from Google Earth into Model Maker.

 The KML file format is used to transfer marker data between GE and Model Maker.

 

·Creating DTm points in Model Maker within a Google Earth area.

 To follow a typical example, click here.

 

The settings and functions on this page will export Model Maker entities to Google Earth.

 

 

 

Please note : The functions on this dialog box are only useful if the survey points in Model Maker is on the correct coordinate system.

 

 1 

Check that the Hemisphere, Datum and System is set correctly in the Settings, Configuration dialog box.

 

All Datum, Coordinate, Central Meridian or Zone  changes are now made in the Configuration window, allowing it to be saved along with each project.

 

The markers will not be placed correctly in Google Earth if any of these settings are not correct or the dtm coordinate (Y&X) values are not on the correct system.

 

 2 

Specify how the elevations of the markers (dtm points) in GE must be defined.

 

 3 

Google Earth : GE will open and the Model Maker elements will be shown on the map.

KML file : The Model Maker elements will be saved to a KML file.

 

 4 

Select which of the MM elements must be added to GE or KML file.

Click on the color boxes to define the specific color in which the elements will be drawn in GE

 

 5 

If a constant has to be added to the MM coordinates when they are exported to GE, define the values here.

 

Use the Utility option to apply Google elevations to DTM points.

NOTE : Elevating dtm points based on the Google Earth terrain is not definitive and should only be used for initial planning purposes.

An actual survey of the terrain is necessary before doing the final planning.
