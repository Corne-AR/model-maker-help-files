MENU : Irr - Nodes

 

 

A pump node will always represent the water source in the main line. If the system is gravity fed and the source is a dam on a high elevation, then a pump node will still be used to represent the water source.

 

This dialog box is displayed when a new node is defined or an existing node is edited.

 

New nodes are normally defined by selecting appropriate values from the list boxes. Also, the type of node must be specified.

 

 1 

Item

Select the appropriate radio button for the type of node that you want to use.

Pumps can only be added to the main line. The main line can have either one or two pumps.

 

More about pumps.

 

 2 

Series/Models/Nozzles

Depending on the type of node, selected from the available list boxes, pick the combination of parameters that should make up the node attributes.

 

 3 

Pressure / Radius

Depending on the node type, select the design pressure. For sprinkler nodes, this is the pressure that will be used to calculate the deviation from.
