AREA : Define

 Top  Previous  Next

 

Many functions require you to define an area to work in.

 

This dialog box holds all the options to define, edit, save and load an area.

 

               

 

Edit Node

 

 

Add

Add a new node on a segment between two existing nodes. Use any of the snap modes to define the new point.

 

Move

Move any node on the existing area. Point to the node and move it to the new position.

 

Delete

Delete a node on the current defined area. Point to the node that must be deleted .

 

Files

 

 

Open

Current co-ordinates will be cleared and a new set of area co-ordinates may be loaded from a previously saved file.

 

Save

Co-ordinates defining the current area will be saved to a file. This is an ASCII file and may be edited with any editor program.

 

Select Element

 

Gxt (String)

Existing strings can be used to define an area. Point to the desired string. The string will be followed and the co-ordinates used to define the area. The closed area will be formed by automatically joining the first and last nodes of the string.

 

Irrigation block

If an irrigation project is available, point to the desired block. The boundary of the irrigation block will be used to define the new area. This would not effect the irrigation block in any way.

 

New irregular shape

 

 

Use the Nearline mode to have Model Maker follow a poly CAD line. Change the cursor mode, eg. Free Point, etc., for your application.

Model Maker will always assume that the first 2 points are used to indicate a rectangle. In that case, press Escape after the second point. If the area consists of a polygon, keep adding new nodes to the area after the first two. A rubber band will connect back to the first point to show a complete picture.

Press Escape (or mouse escape button) when done. You do not need to end by indicating the first point again. Model Maker automatically closes the area by joining the last node with the first.

 

New rectangular area

 

 

Clear current co-ordinates and define new co-ordinates. Press define rectangular area. Click to make selection.

 

???

to Cad

Copy the current defined area boundary to CAD lines. The default pen and line type will be used.

 

Move

Moves existing area to a new position.

 

Show

This option will draw the current defined area in green to the graphics screen. This image of the area is erased after a replot. Use this mostly for visual inspection.  

 

 

 

Track poly around curves

 

This check box controls the way Model Maker interprets a poly line when 2 line segments (a&b) are joined by an arc(c).

If Track Poly Around Curves is CHECKED option B will be followed in the above picture. The area function will track around the curve in 0.5m intervals.

If the curve length is shorter than 2.5m (5*0.5), Model Maker will use at least 5 increments around the curve.

If the curve length is longer than 25m (50*0.5), Model Maker will use at least 50 increments around the curve.

 

If the option is not checked, line a & b will be extended to meet at A. The area function will ignore the arc and use point A as a node.

 

Auto New

If this box is checked, Model Maker will automatically start the new area indication mode each time the dialog box is started. Use this mode when the previous defined area is not often re-used.

When a new area is defined and you press Escape without indicating any points, the previous defined area is automatically retained.

 

Area

The size of the current defined area is displayed. Also note that the header of the dialog indicates the number of nodes that are defining the current area.
