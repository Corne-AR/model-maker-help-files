Top 

 

MENU : Cad -  Areas                                                Show me how to get there

 

 

Define geographic areas for Model Maker's mine planning module. Use normal CAD functions to define the initial outlay of the area. Use the Add option to enter new areas. The geo data is saved along with CAD data in the same drawing file. (KCD).

 

 

Add

Adds a new geo area.

 

Edit

Edits a geo area.

 

Delete

Deletes identified/selected geo area.

 

Move

Moves a geo area.

 

Copy

Indicate inside geo area to copy.

 

Rotate

Indicate inside of geo area to rotate.

 

Plot

Plots geo area.

 

cLear

Deletes all geo areas.

 

Id

Identifies geo areas for manipulation.

 

inFo

Indicate inside of geo area to display attributes.

 

Sort

Sorts geo areas from left to right / botton to top / by baseline.

 

cOnsis

Check geo area for consistancy.

 

Xascii

Output geo boundary points in ASCII co-ordinate file.
