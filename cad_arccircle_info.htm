MENU : Cad - Circ - Info                                

 

This function displays the information of an indicated arc or circle.

 

 

Directions

Start                   : The direction from the centre point to the start of the arc.

End                   : The direction from the centre point to the end of the arc.

 

For a full circle, the start direction will always be zero and the end direction will be 360.

 

Results

Arc length       : The length of the circle perimeter measured from the start point to the end point.

                    For a full circle the value would be 2PI*Radius.

Circle area     : The area of the arc taken as a wedge to the center point.

 

Hints:

When any of the arc values are changed, Model Maker will update the other attributes.

 

CHANGE                                                                                        RESULT

Centre co-ordinates                                                                Circle plot at a new position

Directions                                                                        Area and perimeter length will change.

Arc length                                                                        The end direction will change. Radius kept constant.

Circle area                                                                        The radius will change. Directions kept constant.

 

The editor button copies the information of the circle as regular text values to the editor.  From there, you may print it or combine it with information from other data or circles.

 

EXAMPLE OF EDITOR:

 

Use the button to re-calculate the values after making changes.

 

How to select the function for Arc/circle info.

 

1.                Select the CAD in the left hand side menu.

2.                Choose the Circ button in the bottom menu.

3.                From the sub menu select the Info button.
