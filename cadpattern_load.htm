MENU : Cad - Patt - Load

 

Any pattern saved to disk by the patterns save function can be loaded from disk and placed on the drawing by various methods.

There is also an option to select any image and place it as a pattern on the drawing.

 

A)

 

Load : This is the default tab allowing you to select the pattern directory and pattern name you want to place.

 

Alternative folders : On this tab you will find options to specify a forced folder and alternative folder.

                           The purposed of the 2 folders are described on that tab.

 

B)

 

Patterns : Existing patterns can be placed on the drawing by the 10 listed options.

1.Put Single : The pattern will be placed at the indicated cursor position.
2.Fill Area - Retain : You will define an area and pattern spacing. Patterns will be placed in the area.
3.Fill Area - Explode : Same as 2 but the patterns will be explodes as regular cad elements.
4.Fill Area - Grids : Patterns will be placed on the grid points as specified by the Grid Snap mode settings.
5.DTM : Placed on all points with user selected codes.

Identified :

6.DTM : Patterns can be placed automatically on all the IDENTIFIED DTM points. First identify the DTM points and then use this option to place place patterns on the points.

       Rotate and scale the pattern while it is being placing.   .

7.Cad line endpoints : Patterns on all the end points of identified lines.
8.Cad intersection : Patterns on all line intersecting points.
9.Cad line and intersections : Same as 7 and 8 combined.
10. Cad text : On all the identified text coordinates.

 

When loading patterns a size and rotation can be specified.

Size : Value of 1 will place the pattern at the same size it was created. To make the pattern bigger, specify a value>1 and a value less than 1 to place it at a smaller scale.

If you want to place many patterns at 50% of the original size, specify 0.5 and press Default. Next time you load a pattern the size value will be set as 0.5

 

Images : Options to load images as patterns are on this tab. See detail below.

 

C)

 

Single View will show a drawing of the selected pattern.

Multi view will show all the drawings of 6 patterns in a grid.

 

Single Image as pattern

 

A new pattern will be created with the selected image as the only element and the new pattern will be placed in the same folder as the image.

 

a) Scale to fixed mm : With this option the image in the pattern will always be drawn to a height of 7 millimeters (or what the user specifies) no matter the cad drawing scale. The pattern will assume the image name but 2 additional pieces of information is added to the pattern name.

eg. Image 'Road sign.png' that is 200 pixels hight and drawn to 8mm.

    The pattern name will be 'Road sign_x200x_8.pat

The _x200x_ indicates the images in the pattern is 200 pixels in size and the 8 shows it will be drawn to a fixed 8mm in size on the plan.

 

b) Applied factor : The size factor is applied to the pattern and the pattern will scale according to the cad drawing scale. The initial size of the pattern is based on the size of the image in terms of pixels. If the image is 500 pixels high then it will be drawn as 500mm high at a cad scale of 1/1000 and a size factor of 1. The height may be different at a different cad scale or size factor. The pattern will assume the image name and a .Pat extension.

 

 

 

 

Note: Although the _x....x_.. notation is automatically added to the pattern name when single images are loaded with option (a), the user may manually change existing pattern names with this notation if always plotting a pattern to a fixed size is needed.

 

Load the pattern on a drawing and measure it's height in meters. eg 450

Divide the height by 2 and use that in the new name.

Edit the pattern name and add the _x225x_20 to the name.

The pattern with the new name will draw 20mm high.
