Node add

Add nodes to existing strings. These nodes may be added to the start, end or between other nodes. When indicating the string, the cursor placement on the string is it important. To add a new start or end node, place the cursor close to the first or last node on the first or last segment. To insert a node ,place the cursor on the segment joining two existing nodes. To insert the node on the first or last segment, do not place the cursor close to the first or last node. This menu relates only to horizontal nodes. Vertical alignment nodes for alignment strings must be added in the Vert menu.  

 

Node edit

 

Edit the actual values of a node. Change the Y, X and Z value of a general string. For an alignment string, you may change the Y,X co-ordinates and the curve radius and transitional curve values.

Node delete

 

Delete any node of a string. Use the '+' key to undo the last deleted node. A string consisting of only a start and end node must be deleted with the string delete menu.  

 

Node move

 

Move any node of an existing string to a new position. The horizontal curve position of alignment strings will be updated to fit the nodes in their new positions. This may lead to two overlapping curves.

String move

 

Move an existing string to a new position. The cursor will hook onto the closest node of the base string. This node is then  defines the relative new position of the string. Use any cursor mode to define the new position. Use the normal rotation options to swing the string eg. Page up/down , Ctrl page up/down , shift Tab). If the base node is placed on a tacPoint, Line Point or Near Line Model Maker will ask you if it should re-elevate the whole string. If so, all the elevations of the nodes of the string will change by the factor : (new Z=old Z + (new base Z - old base Z).

 

Update string nodes based on values of the closest dtm point.

If OK is pressed, the selected attributes of the string nodes will except the values from the dtm points.

Move / Copy

 

Select move if the original string must be moved to the new position. Select copy if the original must remain as is and a new copy of the string must be placed at the new position.

 

Match

 

Move an existing string from its original position to a new position by indicating two base points and their new positions. The string will move, rotate and scale itself to occupy the new position.

 

(Edit Z) The match function will by default update the plan co-ordinates of all the nodes on the string to fit between the 2 new base points. You can additionally select that the elevations must also be matched between the original and new base positions in which case the elevations of all the nodes on the string will be updated pro-rata.

 

See also cad match

Combine

 

Combine two strings to form one single new string. The new string will be of a general type except when the original strings are both of the alignment type. The vertical alignment of a combined alignment string must be redefined. Strings can only be combined by joining any two of the respective end nodes.

The new string will take on the attributes of the first of the two indicated strings

 

Split 1

 

The split function divides a string into two separate new strings.

The attributes of the new string are identical to the original string.

Split 2

 

This method will not delete a segment as above. Instead, the node as indicated by the proximity of the cursor will be duplicated on both strings.

 

Below Ground

Use this function to trim sections of a string, either below or above the ground surface of the active survey project. This feature is normally used when contructing slimes dams or quarry pits. See this example.

Flip

 

Exchange the X and elevations co-ordinates of all the nodes of an indicated general string. The purpose of the function is to display the string in its vertical mode while still being in the normal plan co-ordinate system of Model Maker.  The Vert View option does the same, but then Model Maker puts you in the temporary vertical system and you cannot access the CAD, Survey and other normal functions. You may manipulate the string with the help of all the CAD functions. Remember you are basically manipulating the elevations of the string. Flip the string back to the normal mode after manipulation.  

 

VERTICAL TO PLAN

 

The co-ordinates of the indicated general string will be manipulated as follows. The Y co-ordinate will stay the same. Model Maker will prompt you for the new elevation of the base node. The elevations of all the other nodes will be calculated with the formula: Z=(Base X)-(Node X)+(Base Elevation). The X co-ordinates will all become equal to the X value of the cursor position. The new string will form a straight line with the correct elevations. Move the string to the correct plan position with the sMove function.  

 

PLAN TO VERTICAL

 

Model Maker will transform the string into a system of chainage and elevations and re orientate as follows. The Y co-ordinates of the nodes will hold the chainages. The X co-ordinates of the nodes will hold the elevations. The current Z co-ordinates will become redundant. The string will thus be displayed from left to right on the screen starting with the 0 chainage at the cursor position. Keep in mind that this should only be a temporary arrangement for designing purposes. Flip the string back to its original format as soon as possible.  

 

Attributes

 

Change the pen and line type of a string. Set the desired parameters by the (P)en menu, then indicate the string. The pen & line type of the string will automatically be set to the default values.
