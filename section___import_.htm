MENU : Sec

 

Cross sections (.xx1) and long sections (.gro) generated by Model Maker or Road Maker can be imported and recorded to the CAD.  

 

After the file name of the section is entered and the file located, Model Maker will display the next dialog box used to enter all the section parameters.

 

 

Setup

 

Enter the horizontal and vertical scales in normal format.   eg. 1/1000  Enter 1000

 

Grid interval specifies the distance between grid lines both horizontally and vertically.

 

General

Plot heading : The description of the plot. Used only when drawing a long section. (profile)

General CAD layer : The detailed layer names are defined in the TEXT SIZE AND LAYER dialog.

Paper size

The relative paper size is important at this stage because Model Maker can divide the sections in groups that will fit in the selected paper area.

Cross-sections : The first section will be placed at the user indicated point. The other sections will be stacked above the first one until paper space runs out. A new column will then be started to the right of the first. The sections will continue to be added above each other in columns until the right edge of the paper area is crossed. A wide spacing will then be placed between the columns of sections and continued to be stacked on the second sheet.

 

The plot length option is available for long sections. For the selected paper size and horizontal scale it shows the maximum length the paper sheet can handle. The section will be divided if the maximum is less than the total length.

 

The default start and end chainages are displayed. If you want to plot a specific part of the section, change the start and/or end accordingly.

 

The User Z value can be used to define a specific lower limit on the plot.

Example: The lowest point on the section is at 110m but you want a bigger open area below the plot. Use the Z datum option and define the value as 80m. An extra 30m will be provided for at the bottom the section.

 

 

Fit maximum : Based on the selected paper size and horizontal scale the program will automatically draw the longest section on each paper sheet.

 

User defined : You may select what the length of each section must be. The user length must be less than the maximum for each sheet.

 

Single continuous : The program will ignore the paper size and draw the whole section as one long entity. This is useful when you want to create a single section to output as a dwg.

 

 

 

EXAMPLE: stacked cross section on 3 sheets

 

Long-sections : The section will be drawn from the bottom left of the sheet and extend to the right until the paper edge is crossed. If necessary, the section will be divided and                         continue on the next page.

         

EXAMPLE: divided long section

                     

 

Section Format

 

A line chart format will connect the points on the section with direct lines. A bar chart will only draw horizontal lines between the grid intervals.

 

EXAMPLE: bar chart and a line chart

 

The actual slope value of each section of the string will be written on the line.

Text sizes and layers

This button opens the dialog that controls the text size parameters.

 

 

SECTION : Distance and elevation text

Specify the row height and number of decimals used in the labeling of the sections. Use the SHOW HELP PICTURE for a description of the parameters.

 

After the parameters have been specified you may display the section only or plot it to CAD.

 

Display Section

Display the sections which are loaded from the XX1 or GRO files on the screen. Use this function to visually inspect the size and positions of the sections and grid lines before converting it to CAD.

Change the display parameters if necessary.

 

Plot to CAD

After indicating the origin base point, the sections will drawn in CAD as regular lines and text. The elements may then be manipulated in the CAD using the regular functions.

 

Parameters

Allows you to reset the parameters.
