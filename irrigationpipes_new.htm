MENU : Irr - Pipes

 

This dialog box is displayed when a new pipe is defined or an existing pipe is edited.

 

New pipes are normally defined by selecting appropriate values from the list boxes. Also, the pipe application must be specified.

 

Code / Name / Type

 

As seen in the pipe data base, pipes are grouped together under a single code. This list box holds the codes of all the various pipe groups. Select an appropriate group. All the classes of that group are displayed on the right hand side of the dialog box.

 

Classes / Pressures

 

All the available classes and diameters as defined for the selected code group are displayed. Select from this range the appropriate pipe.

 

Note:

When Model Maker sizes pipes automatically in the Hydraulics function, the different pipe sizes will be selected from the code group that the original pipe belongs to.

 

Application

 

Pipes are used in three distinct applications. After selecting the pipe group and class, specify the application of the pipe.

 

Data Base

 

This button transfers control to the data base dialog box where you may define new pipes or edit existing ones.

 

Get type

 

Use this button when you want to set all the pipe parameters equal to that of a pipe already defined in the project. The graphics screen will be activated. Point to the pipe that must adopt the default parameters.
