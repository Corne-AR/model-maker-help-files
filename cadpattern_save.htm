MENU : Cad - Patt - Save

 

Draw any shape, identify the elements and save it to disk as a pattern. The patterns are always saved to the subdirectory which is automatically created by Model Maker. These patterns can then be added to any drawing. A pattern is created from all the identified elements. Indicate a base point when the pattern is saved. This base point will be used when the pattern is recalled from disk and placed on the drawing.

 

 

In this example a shape was first drawn on the side of the CAD drawing, identified and saved as a pattern. Then the pattern was loaded and placed at the desired positions on the plan.  Normally one would use the center (little red circle) of the pattern as the indicated base point.

 

 

Special options within a pattern:

The following special CAD layer names can be used in a Pattern to Intelligently pick up the DTM information and convert it to Text.

?DTMZ?         (Elevation of the DTM Point)

?DTMC?        (Dtm Code)

?DTMCV?            (Value of (dtm code) That you can use on its own)

?DTMZ?_?DTMCV?5 (Z+Value of the code)

?DTMT?        (Dtm Tag)

?DTMY?        (The Y coordinate)

?DTMX?        (The X coordinate)

 

 

Example of codes & elevations:

Z=1264.3217

?DTMZ? = 1264.32 (STD 2 decimals)

?DTMZ?3 = 1264.322 (3 decimal places)

C=ABCD-1.4896                                                        

?DTMC? = ABCD-1.4896 (Display all text in code field)                                      

?DTMCV? = -1.4896 (Model Maker try to take code to value)                                              

?DTMCV?2 = -1.49 (Model Maker try to take code to value with specific decimal places)                                                  

C=ABCD,-1.4896 (Comma Added between description and Value)

?DTMC? = ABCD,-1.4896 (Display all text in code field)

?DTMCV? = -1.49 (Only numbers after comma go's to value STD 2 decimals)

?DTMCV?5 = -1.48960 (Only numbers after comma go's to value with specific decimal places)

C=1234,-1.3683                                                        

?DTMC? = 1234,-1.3683 (Display all text in code field)                                    

?DTMCV? = -1.37  (Only numbers after comma go's to value STD 2 decimals)                                                

?DTMCV?5 = -1.36830 (Only numbers after comma go's to value with specific decimal places)

                                                   

?DTMZ?_?DTMCV? = 1264.32 - 1.49 = 1262.83 (Z+Value STD 2 Decimal places)

?DTMZ?_?DTMCV?3 = 1264.3217 - 1.4896 = 1262.832 (Z+Value with specific decimal places)
