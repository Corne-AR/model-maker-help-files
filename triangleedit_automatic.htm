The editing methods described can be applied to a triangle model generated with the NORMAL  or QUICK SEARCH  methods.

 

 

Height

 

 

 

Assume the common leg between the 4 points connects points a and c

 

Because the elevation difference between a and c (110-95=15) is greater than b and d (102-100), the leg of the 2 triangles will change from connecting a and c to that of b and d.

 

ID CAD

 

This method requires that the breaklines  are drawn in CAD as lines. Additionally. these lines must be Identified in the CAD. These identified CAD lines will be used to check against the legs of the triangles. If any leg of a triangle crosses the CAD line, that leg of the triangle will be changed (swung). It is often necessary on complicated jobs to activate this ID CAD  editing function more than once to ensure that all the incorrect triangles are edited.

 

Code

 

This is similar to the  NUMBER option, but now MM edits a leg if it finds that the new position of the leg joins two points having the same code, where the previous position joined different coded points.

 

Number

 

Triangle legs will be edited to join points with sequential numbers. If the point numbers of A and D are not sequential to those of B and C, the leg will be edited.

No additional information is required. MM  searches through the triangles in batches of 2 triangles at a time. If the common leg between the 2 triangles joins 2 points which are not sequential in the database, and the other option of the leg would join 2 sequential points, the leg will be changed to join the sequential points.

 

Line

 

Indicate a line, and all the triangle legs crossing this line will be swung. This may be a continuous line from point to point that represents a detail or breakline.

This method does not require CAD lines. You will be given the opportunity to draw a temporary line between the points making up the breakline. This temporary line will be used in much the same way as an identified CAD line to edit the triangles legs that cross the line. The first example shows the line indicating the triangles to be swung. Valley

 

Apply this function to data that consists of many valleys and ridges and the DTM points are measured along contours. Because contour-generated DTM's do not present evenly distributed points, the results may form horizontal flat surfaces at some specific topographic features.

 

In this EXAMPLE, the dotted contour is incorrect as indicated by the 2 red crosses because the triangles were formed by joining points c,d and e, which are all on the same elevation. Correct the error by editing the triangles to join the points on the dotted contour to the point at b. This can be done manually. The valley function has been designed to inspect the whole model and correct all the occurrences of this type of error.

 

String_ID

This is the same as ID CAD, but the identified strings indicate the breaklines.
