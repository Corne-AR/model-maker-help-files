This function allows you to pick individual cad text elements and build a dtm point from this information.

 

The cad text from which the point will be developed should have the relevant information as separate text entities.

 

In this example the cad text elements describe 2 dtm points. In the dialog box, the attributes of the point called TREE is entered.

 

 

The sequence works as follows.

1.The cad drawings has text elements that describe the code, tag and coordinates of a point.
2.The 'From Cad Text' function allows you to indicate the individual text elements that define the point.

3.    After Picking the elements they are shown in the Value list.

4.Edit the values if needed and then click the ADD button to create a new dtm point from the text values.
