EXAMPLE : Working Example 5 (Terrace toe points)

 Top  Previous  Next

 

MENU : toE

 

Object

 

This example shows you how to determine toe points for a terrace model and then to extend the model to the toe line.

 

Toe points are calculated between the triangle models of 2 projects.

               Project 1 will consist of the points and triangle of the original ground model.

               Project 2 will hold the points and triangles of the design platform.

               The toe points will be calculated from project 2 to the intersection with project 1.

 

Procedure

 

Notice that the procedure to load the ground data is also described in detail in Working Example 6.

 

 

STEP 1 : Load ogl data                (Step1 - Example 6)

 

       For this example the PLATNGL demonstration data files will be used for the original ground model.

Select  the FILE OPEN option and load the original ground data - points and triangles - from the file called PLATNGL into project #1.

 

 

STEP 2 : Create the platform data

 

In this picture the yellow dots and text represent the points of the data loaded into project 1.

The 4 green points were added to project 2 CAD lines drawn between them.

The 4 red circles were only added to indicate the positions of the 4 platform points and are not really required for the exercise.

 

Below is a detailed description, showing how the design platform is generated.

 

Adding 4 corner points

 

               Activate the PNT option on the left menu and from the bottom menu, select ADD option.

               On the ADD dialog box, activate project # 2 and choose the Add Single Format.

               Set the cursor snap mode to FREE and point approximately to the position of points a20.

               On the New Point Attributes, enter the elevation of the new point as 65.

               Continue to enter points at the positions of a21,a22 and a23. Press escape after the last point is defined.

 

Adding Cad lines

 

               On the left menu select the CAD option, on the bottom menu the Line, ADD function.

               Select the Polyline option and set the cursor snap mode to DTM Point .

               Add the 4 lines by pointing to the new points.

 

Creating the triangles of the design platform

 

After the 4 corner points have been added, triangles must be generated to model the surface between them.

               On the left menu select the TRI option.

               On the bottom menu select the CSEARCH function and create the triangles by following the relevant instructions of example 1.

After these steps have been completed, the system is loaded with both surface models and the toe points can be generated.

 

 

STEP 3 : Project relation                (Step3 - Example 6)

 

       Use the DEFAULT button to access the Status dialog box.

 

 

STEP 4: Batter parameters

 

On the left menu select TOE and then PARAM  on the bottom menu. You may now specify the batter slopes in cut and fill.

 

You may also specify a fixed number of toe points or a fixed distance interval along each outside triangle leg. These additional points will only be added if you select  YES (Y)  for the  ADD EVERY TOE POINT  option. If it is set to  NO (N)., toe points will only be generated at a change in ground slope.

 

 

STEP 5 : Determine toe points

 

On the bottom menu select the  ALL  option.

 

You have the option of adding the toe points to the data base or viewing the toe line position.

 

               Add New Toepoints

 

 

               Show toepoints

The position of the toe points will be calculated along all the open sides of the triangles in project 2 and the new points added to the design project.

 

The toe line will be calculated and shown but the points will not be added to any project.

 

 

An example of what the picture looks like after the Show Toepoint option have been selected.

Notice that the toe line in CUT is shown in red. The FILL toe line is drawn in yellow.

 

It is suggested that you use the Show option first before adding the toepoints. By doing this you get a chance to inspect the position of the toe line before accepting the new TOE points.

 

This picture shows the toe points after the Add New Toepoints option is used.

All the new toe points have been added to the design project - in this case project #2 - with their codes = TOE.

Notice that after the points have been created you have the option of adding a string, - if the module is active - or CAD lines between the new points.

After the toe points have been created and added to the design project, extend the triangles to these new points.

 

Option : Different slopes

 

If one side of the model should have different batter slopes than another side, use the  IDSIDE  option to determine the toe points.

First identify (PNT - ID - NEW - ID.) the points connected to the outside legs of the triangles where a specific batter slope must be applied. Then, use the  TOE  option, set the slope parameters and use the  IDSIDE  option to determine toe points.

 

 

STEP 6 : Extend the model

 

Having the toepoints added does not have real meaning unless triangles are build to the new points.

Use the Csearch option on the TOE menu to create the new triangles.

The extended triangles will then model the complete design surface, including the side slopes, so that areas and volumes can be calculated.

 

               Make sure that the design project, in this case project #2 is active.

               On the TOE menu (see step 4 above) select the CSEARCH option. You will be prompted to confirm that triangles must be built to the neutral points, ie., points that indicate the position where the batter changes from a cut to a fill situation. (See the above picture where the toe line changes colour from red to yellow)

         Confirm the search to neutral points.

               After the triangles are built to the neutral points, confirm that triangles must be extended to all the other toe points.

               After all the triangles have been created, select the TRI option on the top menu to draw and visually inspect the new triangles.

 

An example of what the picture should look like after the design triangles have been extended to the toe points.

       Save the new model, consisting of the original points and the additional toe points as either a Total file or as separate        points and triangles.

               To calculate volumes between the final design model and the natural ground model, refer to Working Example 6.

               To merge the new design model with the original ground surface, follow the projects example.
