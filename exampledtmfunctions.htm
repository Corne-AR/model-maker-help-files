MENU : Example - DTM functions

 

Explanation of general Model Maker routines.

 

This example explains to you, with questions and answers, the basics of the DTM part of the program.

Other more detailed examples can also help you to better understand certain procedures.

It is a good idea to also refer back to these examples when reading through the following questions.

 

 

I want to start a project (consisting of two tachy stations) and enter the spots?

 

a) Click on the DATA option on the top menu. Or press ALT D.

b) Use the NEW button to clear the memory from any existing data.

c) Enter the project name.

d) Go to row number one in the grid box.

e) For each station enter the name, Y & X co-ordinate, the elevation, the instrument height and the correction angle if the station was not in orientation.

f)  Click on the row of station number 1 then click on the POINTS tab at the top.

g) The STATION CONFIGURATION MENU will be displayed. Select the method for reducing the survey spots by choosing the applicable options for the height, distance.      Your selection will be highlighted.

h) You are now in the editor where spots can be entered, changed or deleted.

i)  Enter the data by typing the code and other relevant information.

j)  After entering the last spot of station No 1, click the STATIONS tab and then on line 2 (station #2).

k) Click on the POINTS tab again to enter the data of station 2.

l)  After entering all the data click the OK button. You will be returned to the graphics screen. Click on NORMAL to plot the data. The data is now entered but not yet stored.

 

 

How do you save data?

a) Click FILES - SAVE or press F2

b) Use the functions of the SAVE menu.

 

 

How do I know which data is loaded?

a) Click on the STATUS button.

 

 

How do I store or recall data from a different sub-directory or disk drive?

a) There are two ways to define the directory.

    1)  When you enter the file name to be recalled or stored, add the path as well, eg. A:DEMO or C:

    2)  In the FILES MENU, select the correct sub-directory using the normal Windows format

 

 

How do I load data surveyed with electronic instruments into MODEL MAKER?

a) Model Maker reads a wide range of data types.

b) The data must be downloaded from the data logger by an appropriate communications program and stored on files that have the appropriate extensions.

c) Select the UTILITIES option on the MAIN MENU.

d) Choose the ELECTRONIC FIELDBOOK option and select the specific file type. A file containing the station co-ordinates may be loaded before the observation data with the TRIG option.

e) Enter the relevant data and co-ordinates when prompted.

f) The survey observation data will then be reduced.

 

 

How does one make changes to the data?

a) Select the DATA option on the MAIN MENU.

b) Adding, deleting or changing a station is done with option <STATION>.

c) Editing spots are done with the option <POINTS>.

d) General changes to the data like adding constants, changing the codes etc. are done with the GENERAL option.

 

 

I want to add Y, X & Z data to the project, the data currently consists of 2 tachy stations and the surveyed spots?

a) After the <DATA> option - add a station <STATION> - <ADD>.  When asked for the station name, enter the name KOORD and then enter 0 for all other station information.

b) A KOORD station is a dummy station which can hold data consisting of final Y, X and Z co-ordinates.

c) Use <DATA> - <POINTS> to add data to station No. 3.

d) Change the station configuration to final Y and X co-ordinates and final elevation or no elevation.

e) Enter the data.

 

 

The data is entered correctly.  How do I plot it on the screen?

a) From the top menu, select NORMAL.

b) The survey points will be plotted with all the main Model Maker functions arranged around the graphic screen.

c) Refer to the screen layout for a description of all the menu items.

 

 

Where can I change some of the plotting parameters eg. labeling of the spots?

a) All the variables that effect the plotting output are found in the PLOT PARAMETERS MENU which is one of the options on the top menu bar <GOP> or - <PLOT PARAMETERS>.

 

 

How do I create a model and draw contours?

a) A model is created by means of a network of triangles. Please refer to the working example.

 

 

The contours are not following the break lines correctly, how do I change the model?

a) Select <TRI> on the left menu and <EDIT> on the bottom menu. (Example)

b) Point to the leg which you want to swing around for a different interpolation. The contours will be drawn according to the new position of the triangle legs.

 

 

How do I store the triangles?

a) On the MAIN MENU select <FILES> - <SAVE> - <TRIANGLES>.

 

 

I have a mine dump or quarry.  How do I calculate it's volume?

a) Volumes can only be calculated after a triangular model has either been created or loaded from disk.

b) From the left menu of the graphic screen select <VOL>.

c) Different options of calculation is displayed on the bottom menu.

d) Volumes can be calculated by cross-sections or contours.

e) The areas of the sections are calculated by co-ordinates. The volume of the dump is then calculated by end area method.

 

 

What type of surface analysis can be done on a model?

a) Three types of analysis can be done :- SLOPE, HEIGHT and WATERFLOW

b) These options are found on the bottom menu after you selected <ANA> on the left menu.

c) Follow the prompts to do an analysis.

 

 

How do I enquire about the survey data on the graphic screen?

a) Select <PNT> on the left menu.

b) At the bottom menu select <POINT> to enquire about a specific point or edit any of the elements of the point.

c) Select <SUR> on the left menu.

d) The options on the bottom menu may be used to calculate joins, polars, intersection co-ordinates or areas.

 

 

How do I exit from the program?

a) On the MAIN MENU select FILE - EXIT

b) Check the status line to see if the data is stored before you exit the program.
