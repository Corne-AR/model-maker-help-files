MENU : Cad - Line - Special - 2Circle-line                                

 

Add a new line tangent to two circles.

 

 

Normally, there are 4 possible line positions: one on each side of the circles and the 2 lines crossing in the center that join the 2 opposing tangent points. If the circles touch each other, the last 2 options are not possible. If one circle falls completely within the other, no new lines are possible.

 

When indicating the circles to be used, put the cursor as close as possible to the expected co-ordinate positions of the new line.

Use the example to identify by colour where the cursor should be to indicate the different line positions.  Use the + key to delete the last generated line if needed.
