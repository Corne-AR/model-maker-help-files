MENU : Cad - diSp                                

 

Select which layers must be displayed and/or be active. The separate elements like lines, text, circles, patterns, etc. can be put ON or OFF.

 

Hint:

When the drawing is saved to disk (kcd) and some elements are switch off, you will be asked whether all the elements, or only those being displayed, must be stored. The display status also controls the elements that go to a plot file, dxf and other transfer formats.

 

 

Layer names

 

This displays, in alphabetical order, all the layers in the current drawing.

The layer names are created under pen attributes menu.

 

Batch

 

 

This option can be used to save the cad display settings in a batch file.

Imagine a project with 100 different layers. For a specific purpose you may want a certain group of layers switched on and for another view a different group of layers must be displayed. It would be quite a hassle to remember which layers must be on or off for each view.

 

By using the Batch option you can record and recall the settings.

To create a certain batch, set all your display and active parameters for the layers, type a batch name in the name edit box (in this example the name is ROADS ONLY) and then press the SAVE button.

If you want to save another combination of display parameters, set the display options as needed, type the batch name and save it.

 

To recall a saved batch, just select the batch name from the name drop down list. All the settings will be set according to the values in the recalled batch file.

 

NOTE: When saving a batch file, the settings and layers names are recorded based on the layers in the current cad project. If the current project has 10 different layers then the batch will record the settings for the 10 layers.

If another project is loaded with 50 different layers and the previously stored batch is recalled, it can at most only effect the display settings of 10 of the current projects, 50 layers.

Furthermore, the layer names in the batch can only effect the layers of the current project which has the same spelling.

 

Example:

A batch called Trees was previously saved with the settings for 4 layers:

Layer                Display                Active

Tree                disp                active

Rock                noshow                active

Road                noshow                active

Fence                disp                active.

 

The current project has the following 6 layers:

Layer                Display                Active

Tree                noshow                active

Rock                disp                active

TarRoad        disp                active

WoodFence        disp                active.

Building        noshow                active

Track                noshow                active

 

After recalling the batch, the settings are going to look like this:

 

The current project:

Layer                Display                Active

Tree                disp                active

Rock                noshow                active

TarRoad        disp                active

WoodFence        disp                active.

Building        noshow                active

Track                noshow                active

 

Since the layers, TarRoad, WoodFence, Building and Track was present when the batch was saved it will only effect layers Tree and Rock when recalled.

 

 

Display

 

Disp will display the layer. No show means the elements on that layer will not be displayed.

 

Active

 

This column allows you to lock certain layers. Locked layers will be displayed if the display

option is set, but you will not be allowed to make any changes to the elements on that layer.

Endpoints

 

The endpoints (grips) show the origin of the element. Check this box to display the endpoints.

The endpoints will be in different colours for different elements.

The endpoints of an arc are purple, the lines are in green and the text is in blue.

 

                ENDPOINTS ON                                                ENDPOINTS OFF

                       

 

Reset

 

Press this button to set all the layers to either Disp or No show, lock or not locked or in the element display window, to visible or not visible.

 

Replot on exit

 

If this option is checked an automatic replot of the screen is done when the dialog box is exited.
