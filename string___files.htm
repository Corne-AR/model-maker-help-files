MENU : gXt - Files

 

Strings are saved to disk and loaded from disk in this menu. Strings operate separately from CAD lines and are stored in separate files. The file extension is .KCS.

The main files menu may also be used to save the strings.

 

Save identified

Only the strings that have been identified are saved to disk. Strings are identified with the ID option.
