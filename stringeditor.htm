MENU : gXt - Edit - Text Editor

 

A dedicated data editor is used to manipulate the coordinates and other values of strings. Note that the editor will appear differently for general and alignment strings.

 

The string name and starting chainage may be edited. The starting chainage is the chainage value of the first node of the string.

 

The co-ordinates of the string are displayed in the editor with each line related to a node.

The example shows a general string. The Z column will be replaced by the curve radius value for alignment string.

 

 

Use the co-ordinate insert and delete buttons to add a new node between two existing ones and to delete a entry.

The Print button will send the data to a defined printer. Use the Setup button to set the printer and font.

The Files Save button will save the string information to disk in a PID format. The open file will read PID format data and automatically replace the data in the editor. The PID format is the file structure that the horizontalalignment uses to store data

 

Global Editing

This page of the editor allows you to apply changes to the data across the board. Use the appropriate  button for the global change to take affect.

 

 

 

Use this dialog box to change the PI names of all the nodes. Select the PI number that the change must be applied from. If the new name ends on a numeric value like in the example, and the Increment name option is checked, Model Maker will update the node names and also add one to the name each time (eg. Pl1, Pl2, Pl3, etc.).

 

 

Applied a constant value to the Y and X co-ordinate values.

 

 

This constant will be applied to radius of alignment strings and the horizontal alignment and to the Z value of general strings. Add the value to the existing node values or multiply the existing values by the constant. If, for example, you want a random set of elevations to be all equal to 100, enter a constant of zero and multiply. Now enter a constant of 100 and add.

 

DELETE                        : Allows you to remove co-ordinates from the list depending on the following.

NAME                        : All the co-ordinates with the specified name will be deleted.

DUPLICATED                : All the co-ordinates within the specified height difference and radius distance apart will be deleted.
