MENU : Cad - Callout

 

Active

 

 

The Callout functionality allows you to add a Callout box to your project, with three shapes to choose from: Pointer, Rectangle, and Oval.

 

Instructions:

 

1. Callout Text:

Enter the text you want to display in the Callout box.

 

2. List of Callouts:

View a list of all Callouts. Select a Callout from the list to edit it. The selected Callout will have a circle with red dots over it. (Callout 2 from the left image is the selected Callout)

 

3. Add New Callout:

Click 'Add new Callout' and then click anywhere on the canvas to place the Callout. The Callout Dialog box will appear, where you can manage your Callouts.

 

4. Zoom to Selected Callout:

Focus the view on the selected Callout.

 

5. Pick a Callout from the Canvas:

Select a Callout directly from the canvas.

 

6. Move Callouts:

Move any Callout or adjust the pointers of any Callout as needed.

 

Callout Dialog Options:

Add more Callouts.

Delete a selected Callout.

Update the text of a Callout.

 

 

 

Defaults

 

The Defaults Tab allows you to save a selected Callout as a template and give it a new name. Here’s how you can manage your templates:

 

Steps:

 

1. Select Template:

 

Go to the Defaults tab.

Choose the template from the list to set it as the default for new Callouts.

 

2. Select Callout:

 

From the list of Callouts, select the one you wish to save as a template.

 

3. Name and Save:

 

Enter a new name for the selected Callout template.

Save it to make it the active template.
