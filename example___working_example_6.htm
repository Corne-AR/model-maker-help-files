EXAMPLE : Working Example 6 (Terrace cut & fill)

 Top  Previous  Next

 

MENU : File - Open (F3)

 

Object

 

This example shows you how to calculate the cut and fill earthworks volume between DTM surfaces.

 

A DTM surface in Model Maker consists of the survey points and the triangle network created between the points. This example does not show you in depth how to create a DTM surface. The two surfaces used is already on disk and have been copied to disk as demonstration data at the time when Model Maker was installed.

 

Procedure

 

The points and triangles of both surfaces will be loaded from disk. These files called PLATNGL and PLATTOP can be found in the default Model Maker directory at . If you have installed Model Maker into a directory different to the standard folder, use the find file option to locate the files on your computer.

Also note that the data is going to be loaded into different DTM project numbers. The original surface, PLATNGL will be loaded into project #1 and the newer surface, PLATTOP is loaded into project #2.

 

 

STEP 1: Reading ground and triangle data into project 1

 

Use the standard file open routines to load the data into project #1.

Click on the File, Open function.

 

In the file dialog box, make sure that the MMS button is selected and in the folder window, select the mmsfolder.

 

 

 

After the Open Selection button is pressed, the points and triangles from the PLATNGL.MMS file will be loaded into project #1.

Use the Tri and Quick  functions on the top menu to view your data.

You should see a picture similar to this one.

The colours on your screen could look different to this example.

 

 

STEP 2: Reading platform design and triangle data into project 2

 

Click on the File, Open function.

 

In the file dialog box, the MMS button should still be selected and the folder window pointing to the mmsfolder.

Two things must be done.

         Click on the Plattop.MMS file name and

         Select project #2 in the project list box.

 

 

Press the Open Selection button to load the points and triangles of the Plattop.MMS file into project #2.

Use the Tri and Quick  functions on the top menu to view your data.

You should see a picture similar to this one.

The colours on your screen could look different to this example

 

 

STEP 3: Project relationship

To calculate the volumes between project 1 and 2, select the teR option on the left menu and define the relationship (ground/design) between the projects.

 

On the bottom menu, select the deFault button and set the following values in the Status of DTM, CAD & STRINGS dialog box.

 

 

In this menu, check that the (Ground Triangles) are selected as project #1 and that the (Design Triangles) are selected as project #2.

 

 

STEP 4: Calculate the volumes

 

Click the Cut&fill button on the bottom menu.

 

In this dialog box notice that the GROUND PROJECT is set to #1 and the DESIGN PROJECT to #2.   Make sure that the flat surface checkbox is not selected.

 

Click the Grid button to proceed to calculate the volumes.

 

After the Grid button is pressed the following dialog is displayed.

 

In this specific example the Suggested interval is 1.6. For other model sizes the suggested grid interval will be different.

Set the Grid Interval to 2.0.  Again, on other models this size can be different and should be inspected each time.

Click the OK button to start the calculation process.

 

The volume will be calculated and the result displayed in the editor box.

 

 

Hint:

This example uses projects 1 and 2 to hold the two DTM surface models. In real life any of the 21 projects can be used to hold either of the surfaces.
