Printer Setup

 

This dialogue is used to access the Windows setup menu for the selected printer. The physical paper size is changed on this menu. Detail settings for the printer are done via the Windows Settings Printers menu. This example is for the HP LaserJet 4L . Each printer has a different driver. refer to the manufacturers instructions if you cannot work out the settings from the information displayed within the driver.
