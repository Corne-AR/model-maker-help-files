MENU : Sec - X-sections / L-section                                

 

This dialog box controls the format and destination of the extracted cross and long sections from the DTM model.

 

DESCRIPTION OF SECTIONS                : The data entered here will be added to the saved file as a project heading.

INTERPOLATE METHOD                        : Sections can be generated from the DTM model in one of two methods.

 

 

1) From TRIANGLE model.

The triangles that represent the surface of the model will be used. A new offset (distance & elevation) will be generated at each point where the plan position of the section intersects a triangle leg. This option will not be available if no triangles are loaded.

2) From DTM points.

Model Maker will travel along the defined horizontal alignment and determine the DTM points that are within the selected strip width from the centre line. The available points will be used to calculate intersections with the center line as to determine the chainage distance.

An offset will be generated for each selected point

 

 

While the sections are extracted, Model Maker draws a gray line indicating the position of the section as well as a green line depicting the profile of the section. These lines are usually only for visual purposes. Checking the quick copy box will allow Model Maker to copy these lines into CAD as normal elements. This option is only available when the sections are generated from the triangle model.

Cross sections

 

 

 

 

Select the format to save the sections. Use the XX1 format if you intend to import the sections again into Model Maker

Use the Do not save option if you only want to see how the sections will be interpolated.

 

Specify the maximum width that the sections should extend to the left and to the right of the center line. If the sections are close to the edge of the model and it can not be interpolated to the desired width, the section will not be extrapolated and will only extend to the boundary of the triangle model.

Enter a width value of zero if the sections must be extracted right across the whole model.

 

Width defined by 2 user selected string.

 

If you are working with strings and need to control the width of individual cross sections, check this box.

You will be prompted to indicate the left and right controlling strings and the sections will only go to the intersections of indicated strings.

 

Note: The 2 indicated strings must be General string types.

 

Offsets on the section will be generated at all the available changes of slope. This often results in a high density of offsets very close together. You can prevent this by specifying a minimum horizontal distance and vertical height difference between the offsets

Example:                  Horizontal distance 0.01m and height difference 0.1 meter.

 This means that on the following typical section        

offset #2 will be deleted as it is too close and nearly the same elevation as offset#1 offset #5 will not be deleted. Although it is close to #4 the height difference prevents it from be discarded.

 

 

It is often necessary to limit the number of offsets on a single section.

If a limit is imposed, Model Maker will still extract all the sections, and then eliminate offsets until the desired number is reached.

The selection process involves an iteration whereby the offsets that model the fewest slope changes are deleted first. The inaccuracy generated by removing some of the points is kept to a minimum.

On very undulated terrains, this could still result in sections being trimmed to such an extend that it no longer represents an accurate profile of the surface and should always be used with caution.

 

Long sections

Select the format to save the sections. Use the GRO format if you plan to import the sections again into Model Maker

Use the Do not save option if you only want to see how the sections will be interpolated.

 

General

 

 

From triangle model : This option will only be available if the current project is loaded with points and triangles. Model Maker will generate offsets on the sections where ever the section intersects a leg of a triangle.

 

 

The red dots indicate where offsets will be generated on a typical section through triangle model

 

 

From DTM points : All the points will be inspected and if found to be  will be included on the section.

 

This method does not require any triangles to be present.
