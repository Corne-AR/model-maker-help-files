MENU : Cad - gloBal                                                

 

The global CAD function give you quick access to some of the more frequently used identification functions. The identify functions provide extensive control over the global manipulation of CAD. However, it is often a cumbersome method, especially if you are interested in manipulating CAD elements based on a simple selection.

 

The GLOBAL option works on the same principal as the regular identify option, except that it combines the selection and manipulation of elements in one easy step.

 

 

The global method consists of 3 steps:

 

Included elements

Select the CAD elements. Any combination can be specified. For example, to work with only the lines, select .

Check to include all the different elements regardless of their check box status.

 

Select method

Choose a method to select the elements to be manipulated. The relevant window will be opened.

Area

Indicate the area to be used when selecting the elements.  The Quick Area button allows you to define a new area without first going to the formal define area dialog box.

Pen

All the elements in the include list (1) with the specified pen will be manipulated.

Layer

All the elements in the include list (1) on the specified layer will be manipulated.

LineType

All the elements in the include list (1) with the specified line type will be manipulated. (This automatically excludes any text element)

Allx

All the elements in the include list (1) will be manipulated.

 

Global changes

Select  the change that should be applied to the selected elements.

The patterns can only be manipulated by the Move*, Copy*, Delete* and Rotate* functions.
