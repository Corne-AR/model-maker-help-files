Settings - File Units

 

 

When you load a file and the System units and the File units are not the same you get this dialog box. All it asks is that you select whether you want to continue as per the SYSTEM or as per the FILE units.  Normally, if the person who saved the file initially had his Settings correct when saving, the FILE option (OK) should be the correct option.

 

On the top menu you must always setup the settings according to how/what/where you work.

 

SETTINGS - CONFIGURATION

 

Coordinate  Region, Datum, System and the Central Meridian (or Zone for UTM).  This is then referred to as the SYSTEM UNITS on the dialog box on the left.

 

These settings are saved in all files that contain coordinates. This is used because users want to exchange data between each other and want to link with Google Earth.  The saved settings are referred to as the FILE UNITS on the dialog box on the left.
