MENU : Cad - Hatch - Add                                                

 

Define a hatch by pointing to the nodes on the boundary of the area.  The cursor snap mode plays an important role in defining the area. Lines are usually defined by pointing to the line end points.

 

When the area definition option is set to Indicate Each Point the following applies:

 

A circle or arc is defined by using the nEararc cursor mode.

 

         

 

To define an area bounded by 3 lines and an arc between line 1 and 2, use the following procedure:

 

1) With the cursor mode (endLine), point to the first end point of line 1.

2) Use nEararc and point to the arc that meets line 1. Staying close to the side of the arc that meets line 1. The cursor will follow the arc to it's opposite end point which meets line 2.

3) Proceed with endLine and point to the start point of line 3. Press escape to go back to the starting line point. Once the area is defined,  the hatch pattern menu is displayed.

 

Hint:

To hatch a single circle, make sure the circle is defined as a CAD circle. Then, point to the inside of the circle with the cursor mode set to nEararc. The cursor will jump onto the circle, go around it, and wait for the next node to be shown. Press escape if the single circle has no other nodes. The hatch was defined by one node indicating the circle.

 

 

When the area definition option is set to Area definition the following applies:

 

The nearArc snapmode will not track an arc but will only add the coordinate on the arc, to the area nodes.

The tracK Line snapmode will track any element including arcs and circles.
