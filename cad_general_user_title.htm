The user title option lets you define your own title blocks that can be used on the drawings that are sent to the plotter.

 

Apart from the user defined title block, Model Maker also provides an automatic title block option.

The default title block makes provision for the following headings: Drawn by, Drawn for, Project title and some other items.

Select the desired title block option from the plot parameter menu.

 

See how to manage already defined title blocks.

 

If the automatic title block option is not complete enough for your application, define and utilize the user title option.

 EXAMPLE of a title block with the title area along the bottom edge of the drawing.

 

The user defined title block consists of 2 major areas namely the title block and drawing area. When Model Maker sends the plotting information to a printer the title block is activated and drawn first. Then the active area (drawing area) is activated and all the other information eg. contours, CAD and irrigation is drawn while being clipped so that it does run into the title block area.

 

Note:

Use the inverted title block option to define an irregular active plotting area when necessary.

 

Title area

The title area are normally formed along one of the sides of the overall paper rectangle. In this area you will define all your headers, dates, scales and other relevant items. You can also define your logo in this area by means of CAD lines or even a bitmap.

 

         Example of a bitmap that is part of the title area.

 

Drawing area

 

This is the rectangular area that remains of the paper area after the title area has been filled.

 

To define a user title block

 

1) Draw the title block at 1/1000 scale and to actual millimeters (1m=1mm) in size.

    The best way is to draw a rectangular frame with the height and width just smaller than the actual paper size that will be used.

    Click on the CAD button, Lines and ADD. Select the rectangle option. Indicate the one corner point at any position by using the FREEHAND snap mode. Define the opposite corner              

    point by using the DYDX mode.

    Decide upon the side of the title area. If the title must run down the right hand side of the paper rectangle, add the line forming the left side of the title by means of the line Add

    Parallel option. Continue and define any other lines that must form part of the title area.

 

         

 

2) Place text such as names, dates, and descriptions on layers called ?1, ?2, ?3, ?4.

    The ? indicates that it is an item to be edited. The 1, 2 or 3, etc., indicates the order of editing.

    Use ? 1SCALE for Model Maker to put the active scale into that position.

    Use ? 2DATE for Model Maker to put the current date into that position.

 Use ?Project for Model Maker to put the current project description in the text.

 

 Use ?SHEET to have the current paper sheet number in the text.

 Use ?SHEET_NO to have the total number of paper sheets in the text.

 

 

 Use ?EMAIL to place the email address

 Use ?ADDRESS1   ?ADDRESS2   and   ?ADDRESS3 for 3 address lines.

 Use ?CELLNO for the designers cell no.

 

          EXAMPLE of title text elements, and on the right, the layer names associated with each.

 

3) Identify the complete title block.

4) Use the CAD GEN USER TITLE function. Enter the pattern file name. The drawing (the title block) will be saved as a normal pattern.

    Indicate the base point of the pattern. Use the Line point snap mode and indicate the bottom left line point of the outside rectangle.

5) Indicate the bottom left corner of the title block by again clicking on the corner of the outside rectangle.

6) Indicate the active area. This is done by clicking the bottom left (6a) and then the top right (6b) corner of the active area. Each time using the line point snap mode.

 

          Snap positions for items 4,5, 6a and 6b.

 

7) Indicate the active size of the drawing area. (A0, A1 etc). Also, enter the name of the newly defined title block. This name are used to reference the title block by in the plot parameters menu. It is common to use the same name as that of the pattern defined in step No. 4.

 

After the title block has been defined you may activate and use it by selecting the title name from the plot parameters, title option.

 

Please refer to the title block example for a complete description of it's implementation.
