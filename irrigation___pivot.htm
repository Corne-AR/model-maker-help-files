MENU : Irr - Pivot

 

 

1.Pivots are another entity in the irrigation design along with solid set blocks and drip blocks.

 

2.A pivot in it's basic form will have a description, definition of spans, being a full or partial pivot and the center coordinates in the plan.

 

3.A regular valve node entity is used to connect a pivot to the irrigation design. The pressure and flow assigned to the valve will be used in the mainline to calculate the system flow and pipe sizes from the pump to the pivot.
4.The pressure and flow needed in the valve can be entered manually or the can be used to calculate the required flow and pressure of the pivot.

 

5.Note: The mainline irrigation system does not have any direct knowledge of the pivot. The pivot is represented in the mainline system through the valve placed at the center point. After placing the pivot and calculating it's flow and required pressure, the values must be transferred to the valve so the mainline knows what the pivot's demand is.  If the pivot is changed afterwards (pressure or flow) do ensure that the valve values are updated accordingly.

 

Follow these basic steps design and place a center pivot sprinkler.

 

 

 

This are the basic steps needed to create and place a pivot.

 

1) Click on Add Pivot to generate a new pivot. This will add a new entry in the pivot list (2). After a pivot is added, use the following options to set the details, define the spans and the center coordinates of the pivot.

 

2) Pivot headers are displayed in this list as they are added. To select any pivot, click on these lines to set the active pivot.

 

    The detail of the active pivot is then shown in the bottom half of the dialog.

 

 

 

 

 

3) Use these 2 lines to edit the pivot's name and description.

 

 

4) Click on the Add/Edit Spans button to manage the spans. You can then type the lengths of each span or use a linked database to select spans from an OEM table.

 

 

5) After the spans have been defined, use this button to set the center point on the graphics screen.
