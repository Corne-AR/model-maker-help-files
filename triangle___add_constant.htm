Add constant to triangles after data link.

 

 

After reading the introduction about triangles, you will realize that triangles are only meaningful if they exist in harmony with the DTM points.

 

If points are added to the data in memory, a number is reported when files are linked. This is the number of points already in the computer. Write the number down if the triangles modeling the data of  file 2 are to be linked. Due to the fact that triangles are related, not by co-ordinates but by the specific number of a points in the database, triangles must be moved to recognize a new position in the database. The number reported when point files are linked must be added to the triangles of the second data set with this option.

 

Although the above mentioned method is available, you can save both projects as TOTAL data and then add the two TOTAL files together. Or, use the Project Copy function to combine 2 projects.
