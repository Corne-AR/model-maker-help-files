MENU : Data - Points - Popup

 

The survey points in this editor belongs to a specific station. This station's number, name and the number of points surveyed from the station is displayed.

 

This panel shows the methods that are used to reduce the observations to co-ordinates.

 

Configuration button

Press this button to access the station configuration menu.

 

Left and right arrows

 

 

Use the two buttons to increase or decrease the Spot/Pnt column width.

 

Scroll buttons

Use the buttons to go to the first line or last line in the editor. Use the scroll box to traverse the other lines in the editor.

 

Data editor

Use this grid editor to add or edit survey points.

 

Decimal button

 

Use the Decimal button to specify the number of decimals to be used when displaying the information in each column.

When the decimal button is clicked the column name list box is displayed. Select the name of the data field  you want to edit. In this example the vertical angle column is set.

 

The decimal value scroll box is then displayed at the bottom. Change the value in this input box to specify the desired number of decimals for the selected data field.

 

These settings are automatically stored as the new default.

 

               To work accurately it is advisable to keep the distance values to 3 decimal places and the angles to 4 decimal places.

               To hide the column name list box, click the Decimal button again.

 

Options

The input box allows you to jump to the selected point number.

               After the point number is entered, double click in the input box. The editor jump to the survey point with the specified number.

         If the requested survey point belongs to another station, the editor will be updated accordingly.

 

The Code, Tag, Prism, HD, HA and VA check boxes control the behavior of the points editor when new data is entered. When entering data, the value of the previous survey point will be used as default in those fields whose check boxes have been checked.

 

Use this button to add a new station to the project by using the coordinates of the current selected survey point.

 

If this box is selected, the correction factors will be applied when a new point is entered or an existing point is edited in the editor.

 

Note:

This action is only applied to points that are manually being added or edited in the editor. It is not intended to and will not globally adjust the reading of all the survey points in one action.

 

The same formulas are optionally applied to all the surveyed points read and reduced from electronic fieldbook data.
