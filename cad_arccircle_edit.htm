MENU : Cad - Circ - Edit                                

 

Use this option to change the layer, line-type and pen of a circle or arc. First set the new attributes via the (P)en mode button on the right side of the screen. Then select EDIT to change the attributes of an existing circle or arc.

 

Use the "+" key to undo the changes to the last edited circle if necessary.

 

How to select the Edit arc or circle functions.

 

1.                Select the CAD button in the left hand side menu.

2.                Choose the Circ button in the bottom menu.

3.                From the sub menu select the Edit button.

 

Hints:

Use the identify or global options to change the attributes of a group of circles.

You may grab the attributes of an existing circle by pointing to it and pressing the DELETE key.
