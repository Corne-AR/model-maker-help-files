MENU : 3D - GridDtm                                

 

 

To view your model three dimensionally you must generate a grid (DTM) file.

 

 

On the dialog box select a grid size to extract the data on. Choose a grid size with approximately 200 lines in each direction. Smaller grids offer higher resolution but produces more CAD lines.

 

 

Alternatively, Model Maker can extract a gridded points file (ASCII) from the triangle model for other applications.

 

Model size

 

The width and height of the triangle model is displayed for informational purposes. In this example the length is 194 and the height is 134. If a user interval of 2 is selected, the resulting gridded file will have 97 by 67 grid lines or 97*67=6499 points.

 

 

Example sizes

 

The number of lines that would be generated at 1,5 & 10 units intervals are displayed. This is only a suggestion. You may select any grid interval of 1 unit and up.

 

User selected grid interval

 

Enter your selection. Remember that the grid interval has an influence on the number of grid points that will be generated.

 

Select file type

 

Use the "Binary" option to save the generated results as a DTM file for Model Maker to view.

Use the "Ascii (yxz)" option to generate a standard co-ordinate file at regular grid intervals. This grid file can be read back as regular survey points by using the load yxz option.

Select the "Don't save" option to only view the sections. In this mode all the sections will be extracted and displayed on the screen. Nothing will be saved and after refreshing the screen, the images of the sections will be erased.

 

Select CANCEL if nothing must be done.
