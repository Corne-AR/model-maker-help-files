MENU : Output - Old plotters

 

Although Model Maker is able to use the Windows drivers for all plotter/printers, occasionally the driver is not available or unknown. Create a standard plot file to address any HP compatible pen or HPGL2 plotter.

 

 

Output format

Select between a pen plotter or a HPGL2 (Raster) plotter. Specify an A3 or A1-A0 plotter. The text alignment on an A3 plotter is slightly different from bigger plotters.

 

Destination

A plot file may be saved to a user defined filename. Transfer the instructions directly to the plotter through any of the parallel ports. You will not be able to address a plotter via the serial port by using this function. Use the appropriate Windows drivers to communicate with a serial plotter.

 

GL2 codes

Certain entry and exit codes may be specified when a HPGL2 plotter is addressed. This is necessary to place the plotter in the correct mode at the start of the plot and to instruct the plotter to start processing the instructions at the end of the plot. The example uses default codes which apply to most plotters. You may change it if needed for your specific plotter.

 

It is recommended that you first try and use the standard format of plotting with the Windows drivers.
