Irrigation Folders

 

Irri Maker saves the nodes, pipes, parts and clusters in a specific folder. By default, this folder exists at the address  and under normal circumstances does not have to be changed. If you often work with different types of designs or have to work on designs generated on other computers, changing the data base folder names can be useful.

To change the folder names click on any one of the names or click the DEF&CHK page.

 

The first folder name displayed is that of the directory where all the node and pipe data bases are kept.

The parts, clusters and template are now also saved in the same folder.

 

In the past, each element was in a separate DBF file. Now all the elements are saved in a single SQL file with DB extension.

Project Information

: You may enter up to three lines of data describing the project. These header lines will print out with all the reports. The current filename is displayed for information purposes along with the status of the data. This tells you if the data has been changes, saved or loaded.

 

 2 

Project Status

 

: Information regarding the changes made to the existing irrigation project.

 

Block Status

: Information about the status of the current active block. Within the overall picture of the       irrigation project, each block exists as an entity for which the hydraulic calculations should be done separately. The block status is an indication whether the block has been finally calculated and that no nodes or pipe have been changed since then.

 

The last file name used to load or save the irrigation project is displayed.

 

 3 

All the different types of measuring units can be specified with the radio buttons. The unit settings apply to all the calculations in the project and are also saved to disk when the project is stored.

 

Length

: This unit defines the length of pipes, the velocity of the flow of water and the diameter of the spray width of the sprinklers.

Diameter

: This unit defines the pipe diameter.

Pressure

: This unit measures the pressure.

Flow

: This unit measures water flow capacity.

 

Note:

The defined units are applicable to the project data only. Separate default units are specified when entering items into the supplier databases.

Blocks
