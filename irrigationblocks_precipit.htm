MENU : Irr - Blocks

 

 

This dialog helps to globally change the pressure and flow of irrigation blocks that consist only of one valve node.

 

Normally an irrigation block will be defined by the layout of the sprinklers which in turn are all connected to the source (valve) node. One would then use the hydraulics functions to calculate the flow required by the block when all the sprinklers are active.

There is however often a need to define a block with only the source valve as the solitary node. This situation is often encountered when one is estimating a design by assuming a general precipitation rate over each block and then need to know what the size of the main line pipes must be to feed the blocks.

In these situations the source valve of each block will then hold the values for the flow and pressure that is required by the block.

The flow and pressure of these valves can individually be changed by using the node edit functions.

 

This dialog box assists in changing the attributes of the source node of these types of blocks without the user having to edit each block individually.

 

 

The components

1) The list that shows all the blocks that has only one node which is of a valve type.

2) The 3 command buttons that may be used to set the flow, pressure or precipitation in each block.

 

 

The actions

1) Set the flow

2) Set the precipitation

3) Set the pressure

:This action will change the input flow of each block to the set value.

:The block flow will be changed to match the required precipitation within the block area.

:The pressure of all the block valves will be set to the desired value.

                                                 

Note:

This is a global command and all the blocks in the list will automatically be updated when any of the 3 command buttons are used.
