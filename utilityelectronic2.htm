MENU : Utility - Electronic data

 

Project status

The status of the current project is displayed. The number of stations in each project is also displayed. You may change the project to receive the reduced data by clicking on the check box.

 

NEW>OLD

Replace : Any existing data in the active project will be replaced by the new field book data.

Add         : The new field book data will be added to the existing data in the active project.

 

Vertical correction

This constant will be added to all the measured vertical angles. It could be used to compensate for a small vertical angle error of the instrument or a value of 180 degrees may be added to swing the value totally around.

 

Default Tag

Enter a tag value to be used as the default when no tag is defined in the field book. Check the overwrite box if the default tag must be used irrelevant of any other specified.

 

The info button will show you more status information.
