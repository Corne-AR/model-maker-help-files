MENU : Data Types - CAD

 

Model Maker has a comprehensive integrated CAD that is intelligently linked to the DTM.

It can work independently of the DTM, but is designed enhance the DTM.

 

CAD is usually used to draw the detail into a contour plan generated from the surveyed points of the surveyor. There are no limits to the number of elements in CAD and it is easy to operate. You do not need to have DTM data loaded to draw with  Model Maker's CAD.
