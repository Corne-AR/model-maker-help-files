The purpose of the keyplan is to add a very simple legend to your plotted output showing the current paper window in relation to the complete drawing.

This image shows the result of a paper plot window. The keyplan shows the position of the plotted window (red arrow)

 

Setting up a keyplan.

 

 

The boundary of the keyplan is defined by indicating the bottom left and top right corners. This will normally be a rectangle that defines the total area of your cad drawing.

 

For each paper window that you have defined you may specify the key position by selecting the position in this block.

 

The actual reduced size of the keyplan is specified in millimeters.

 

Click this button if you want to redefine the keyplan boundary.

 

From these options select the detail that should be drawn on the keyplan.

 

Lastly, on the output dialog box, check the Draw keyplan option to add the keyplan to the output.
