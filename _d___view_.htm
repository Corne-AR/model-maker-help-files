MENU : 3D - View                                

 

Model Maker will allow you to view a terrain in 3D using 2 different methods:

 

1.  The 3D QUICK uses a method of filling the triangles with different shades.

2. The VIEW method draws the image using sections and shows a wire frame result.

 

 

To prepare a 3D view, first load the points data and model the surface by using the appropriate triangle methods. Check the model by looking at the contours and edit the triangles where necessary. Save the project data to disk. Now use the 3D GRID function to generate the gridded file.

Use the Binary DTM option, which produces a DTM file for viewing. It is advisable to first view the 3D picture using the DISPLAY ONLY function. This function produces a 3D image without copying the lines to CAD. The image is erased from the screen as soon as a Normal or Replot is done. Once the 3D picture is ready to combine it with CAD or contour elements, use the Plot to CAD functions. The lines making up the 3D image will be copied to the CAD as the picture is generated. The lines can then be manipulated using all the CAD functions.

 

To print or plot the 3D image to a plotter use the Plot to CAD function. The image is then part of the CAD and can be plotted and manipulated. Model Maker produces the 3D image by positioning regular 2 dimensional CAD lines in such a way that it appears to have a 3D effect. Once the image is part of the CAD it is treated as any other CAD lines and has no status above the other CAD elements.

Manipulate the image using the regular identify and global functions. When the 3D is created, the lines are placed on new unique layer names. The lines can easily be identified by using the layer names.

 

Both Grid Lines

 

Both sides - only applicable when viewing the model from one of the corner horizontal view sides. Check this box if the model must be generated by drawing both directions of the grid lines. This will produce a gridded mesh image.

 

Hidden Line Removal

 

Hidden line removal - check this box if the lines representing unseen sides of the model must not be drawn. Model Maker will determine if a grid point is visible from the user's view point and skip the invisible ones.

 

Grid Line Colour

 

SIDE              :

SLOPE          :

ELEVATION  :

When using both grid lines, the lines in the one direction will have a different colour than those in the other direction.

Individual slope areas on the model will be drawn with a different colours.

Line colours will be determined by the relative elevation of the grid on the model.

 

Contours On 3D

 

Indicate if contours should be drawn on the grid model. You may also specify the contour interval.
