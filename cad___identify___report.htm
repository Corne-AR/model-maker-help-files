MENU : Cad - Id - Report                                

 

Generate a report of relevant information to the editor.

 

 

This report will inform you of the number of identified element and their layers, as well as measure the length of all the identified CAD lines.

 

Use this report to determine quantities of features such as road curbing, paint markings, irrigation and other structures.
