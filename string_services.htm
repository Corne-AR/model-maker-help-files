MENU : gXt - Utility

 

Use strings  to identify other services like roads, telephone lines etc. that cross the design line. Add general strings on the position of these services. It is advisable that two points be surveyed at both sides of the design line in order to identify the service.

A string may only exist with at least 2 nodes. When adding the service string, use the string name option to describe the service, eg., 'road'. Ensure that the string is correctly elevated by jumping to the tachy point when placing the string node.  Model Maker will pickup all the crossing services and display them at the correct chainages along the design line. A report of all the crossings and elevation differences may be send to the printer or ASCII file. It may also be viewed individually if the screen output is selected.
