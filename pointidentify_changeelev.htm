MENU : Pnt - Id - Man-id - Elevation

 

Edit the elevations of all the identified points to a new elevation. You will be prompted to indicate an identified point to be used as reference and give a new elevation.  

 

The same dialog box is also used when a constant factor must be applied to most of the identified CAD elements. The reference to the elevation should  be read as referring to the attribute to be changed.

 

For example, this dialog box is also used when changing the size of the identified CAD text elements.

 

Options

Set to constant elevation

 

The elevations of all the identified points will set to this value, eg. 1000.

 

Enter new elevation

 

The elevation of the indicated point is displayed. Changing to a new value updates all the identified points by the difference between the old elevation and new elevation.

 

Enter elevation difference

 

All the identified points will be updated by this value.
