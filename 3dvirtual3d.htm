3D : Virtual 3D

 Top  Previous  Next

 

The virtual 3D function shows the selected elements in real time 3D and allows you to rotate around the model at will.

 

 

The virtual 3D image consists of solid surfaces and wire frame elements.

 

The Solid surfaces are generated by the triangles of the dtm projects.

Use the Set Surface Colors to tell the system how the image must be created.

 

The wire elements can be from various sources.

 

DTM Triangles : The wire frame lines of the triangles will be shown.

 

Contours : The contours of all the displayed surfaces will be added to the 3D as lines.

     To show/hide contours on selected surfaces, use the Display Menu.

 

Cad Lines : Displayed cad lines will be shown.

 

GXT Strings : Regular strings will be shown as lines. If pipe strings are present they can be shown as 3D pipes.

 

Irrigation Pipes : Shown as lines. The manifold and main line pipes can be shown as 3D pipes.

 

Use the Pipe Scale Factor to increase the 3D size of the pipes so they are more visible on the drawings.

 

 

 

 

 

1) The list of dtm project numbers. The options, 2,3 and 7 apply individually to each of the dtm projects.

To change the surface properties of a dtm project, select it from list (1) and then change the settings.

 

2) Each dtm project can be displayed in one of 4 formats.

  Format 1) The surface will have a standard brown color.

  Format 2) The project will be shown in it's assigned points color.

  Format 3) Use color list (7) to define a unique surface color.

  Format 4) The surface, according to elevation difference between the lowest and heighest point will be divided in 10 intervals and each band will have a different color.

 

3) Set the transparency factor of the surface. A more transparent setting will allow other surfaces (underneath) to be shown. A more solid setting will hide lower surfaces.

 

4) Click this button if the settings of the current project must be applied to all the other dtm projects.

 

5) If this is checked, any triangle that has a user defined color assigned to it, will be drawn using that color.

 

6) Click this button to change the 3D screen background color.

 

7) The list of user defined colors for each project. Only used when display option (3) is used.
