The functions on the general data editing dialog box are used to do the following:

               Audit the survey data by marking points within a certain radius from each other.

               Delete the points marked with the *.* code.

               Copy points from one station to another.

 

 

 

Audit DTM data

 

Model Maker will not generate triangles automatically if any of the survey points are duplicated. The AUDIT menu is designed to check for and de-activate duplicated points. The function can also be used to filter out some survey points.

Aerial surveyed DTM's often generate massive amounts of unnecessary points not needed to model simple terrains.

To delete some of the points that are very close to each other, use the AUDIT function and specify a radius and elevation limit of 5 meters and 0.2 meters respectively. All the points falling within a distance of 5 meters and having an elevation difference of not more than 0.2 meters will be marked as deleted.

 

The audit function will automatically de-activate points within a specified radius of each other and is commonly used to:

       Weed out survey points that are unnecessarily too close to each other.

       Check for points that are exactly duplicated or virtually on top of each other.

       Remove points from a survey that consists of too many points to be handled practically.

 

Radius

Specify the minimum distance between points.

       Specify a zero radius if only points duplicated exactly must be marked as deleted.

       Specify a distance larger than zero to relate each point to all the others within that radius area.

 

 

Elevation

Specify the maximum allowable height difference between points.

       Specify a zero elevation difference if the points may only have the same elevation.

       Specify a big value like 1000 if the elevation factor is not important.

 

 

Press the Audit button to start.

       It may be necessary to repeat this operation a few times in order to mark all unwanted spots is many points fall within the parameters.

       A point will only be marked for deletion if it is within the given radius and height difference relative to another point.

       The audited points are only marked with the *.* code. Use the delete function to physically remove them.

A function similar to the audit function is available in the point identify menu

 

 

Delete Points

Use the REMOVE button to delete the points marked with code *.*.

 

Copy from

       Enter the station's number, the start number, and the end spot number.

 Note : The spot number refers to the relative number of the point within the station, ie. first spot number 1 and not the combined point number.

See the Spot/Pnt numbering system on the points page.

 

Copy to

Enter the new station number.

 

Press the COPY SPOTS button to do the copying.
