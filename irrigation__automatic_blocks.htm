Using the function to create a irrigation system.

Following these steps should allow you to see the flow intended when designing the function.

You can however use the function in any other way you see fit to allow you to get to the desired result.

The function is used to design a permanent overhead Irrigation systems, together with its Cable layout for CAD and quantities.

 

Step 1

Load your survey and create Contours. This can be done using the Google earth elevations and Image background, or a detailed survey with geo referenced aerial photos as background.

 

Step 2

Activate the "Auto Blocks" function

 

Complete all the relevant fields as required, see descriptions at start of help topic for more information on each input.

Irrigation:

Each Block will be connected from the Block valve to the Manifold and the manifold to the Laterals and sprinklers. Based on the pipes Valves and Sprinklers  you select each block will start with that information as initial Flow Pressure and pipe sizes.

Note: It is important to select the sizes and pressures close to the final result to minimize changes in each block once it is placed.

After placing the blocks the user then still needs to connect the Block valves to the Mainline.

Use the Shortcut keys below to exclude some blocks and select the valve positions.

Shortcut keys:

 

Left Click on Black Dots = change Proposed Valve position for each Block.

Shift + Left Click = Group all the adjacent blocks valves at that position.

The Manifold will move with the valve to the new position.

 

Ctrl + Left Click = Hide or Display Blocks. These blocks will not be added when you create the project.

 

When you place the block you can rotate the block with "Page up" and "Page down" keys on the keyboard.

Ctrl + "Page up" or "Page down" will rotate at 45 deg default.

To change the rotation angle go to: Settings  > Defaults > Cursor.

To rotate a block in the same direction as a existing Cad element, use the "Shift + Tab" key's  while holding your cursor next to the Cad line.

 

Cables:

 

Cables allows you to create a cable and Pole layout  of a Irrigation system fitted on overhead  cables.

There are 3 Patterns available to select. if you define your patterns in the same direction as below they will plot at the correct angle in the design.

Note: The corner and anchor pattern's  can be any shape but should represent the top right Corner and top row relatively  at  rotation = 0°. This will allow the program to rotate the pattern correctly at any angle and corner.

 

4. Selecting this option the poles can also be displayed as different size circles. 3 Circle sizes are used: Small = Mid poles. Med = Anchor poles and Large = Corner poles.

5. Toggle between the patterns or Circles with the selections

6. Display both the patterns or the circles.

7. To disable the Cable function and use the Auto block function just for Irrigation select this option.

8. Enter the Pole Height to calculate the cable length for the Anchor Poles. This option works on a Fixed Anchor angle of 45 Degrees.

In plan view the Anchor Cable will be on its own CAD layer as well as the normal Cables.

 

9. When doing smaller blocks the Mid row of Cable and Poles can be omitted using this option.

Step 3:

Once you placed the Irrigation Blocks the Editor will report the relevant Information.

You can copy this information to CAD using the normal Cad Functions like Cad > Text > Add > Multiple >Editor > Copy to Cad.

Step 4:

The dimensions is done according to the Dimension Default settings at the time you run the Auto Blocks function. This means that you need to make changes before you place the Blocks.

The Block dimension will be the value of the Dimension position B settings away from the block and the Project total dimension will be double the value of B. Make changes and select "Use as default?" to save the settings.
