MENU : Cad - Hatch - Par                                

 

Line Type

 

Indicate the hatch. The Pen mode setting will change the line type defining the hatch.

 

 Pattern

 

 

 

Layer

 

Change the layer if necessary. Use the drop down arrow to view all the current layers in use.

 

Files

 

Use the FILES option to save this combination of parameters as a default for later use or to OPEN previously saved parameters.

 

Default

 

AS DEFAULT makes the current set of parameters the default when other new hatches are defined.

 

Spacings

 

Define the spacing between the hatch lines and the angle of the lines. Enter up to 5 spacing distances in millimeters. The number of hatch lines in an area will depend on the scale of the plot.

 

 

 

 

 

If the Fill option is selected the hatch are will be shaded with the selected pen in a solid fill.

If a bitmap name is entered in the edit box next to the fill option, the hatch area will be filled with the bitmap image rather than the solid colored fill.

Fill Pattern

 

Border:

Indicate if Model Maker should draw a border around the hatch area. Use this area if the hatch area is not already defined by CAD lines.

 

Single lines:

The hatch pattern may be made up of single lines, cross lines, or a full fill. The single line pattern uses the angle as set in ANGLE 1.

 

Cross line pattern uses ANGLE 2.

The hatch line type is also defined by the Pen parameters. Set the correct line type before the hatch is indicated. Use the CHANGE button to change the line type and colour after the hatch has been defined.

 

 

 

Fill:

Define the fill percentage. The example above shows a typical line hatch. If you select fill hatch, define the percentage fill using the last of the spacing input boxes. That box now has the 1.5mm specified. The fill percentage may vary from 0%, producing a white colour, to 100% which is equivalent to the selected line type colour.

 

The line type and pattern pictures are displayed as additional information.

 

-Opaque:

Select this option make the hatch transparent. The level of transparency are set through the Shading percentage tool.

 

 

 

Notice: The image must be a BMP image format and it must reside in the PATDATA folder. In a default installed system you will find the patdata folder in the path C:\program files\mmsystems\mm\patdata

The bitmap filename must not exceed 10 characters and you must not add the .bmp extension.

 

The image selected to do the fill may be any size but in practice an image size of 8x8, up to 100x100 gives the best results.

 

The Spacings panel will also hold the following inputs when filling with a bitmap image.

The 2 spacing input boxes can be used to specify a gap between the bitmap images. In this example there will be 2 open pixels between the left and right of each image and 4 open pixels between the bottom and top of each image.

If the transparent option is checked, any pixels within the image having the same color as that of the bottom left pixel of the bitmap will be transparent.

Examples.

 

 

A hatch with a dotted bitmap image, transparent is selected and no open spaces between the images.

 

The same dotted image is used but with a 4 pixel gap (left to right) and a 10 pixel gap (bottom and top)

 

 

Same bitmap image with a 2 by 2 pixel spacing and transparent not selected. Notice how the background only displays through the hatch in the 2 pixel gaps.
