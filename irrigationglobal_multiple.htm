IRRIGATION : Global - Multiple nodes

 Top  Previous  Next

 

MENU : Irr - Global

 

Use this function to introduce multiple fitting nodes at regular distances along the main line pipes.

 

To use this function, enter the maximum length of the individual sections and click the Divide Pipes button.  All the main line pipes will be divided accordingly.

As a precaution, save your design data first before using this function which would allow you to recall your original data if the result of this function is not desired.

The advantage of having these nodes is:

               The pipe would conform better to the elevations of the surface. If you have a long pipe with no other nodes between then start and end node a straight line will be                assumed between the 2 end points.

               It allows Irri Maker to size the pipes more accurately when doing pipe sizing.

 

Before using this option, load the DTM surface data to allow the correct interpolation of new elevations for the new nodes.

 

Here are some rules regarding this function:

1) The DIVIDE only works on the active block.

2) It will only divide a segment that is longer than the specified divide length. Meaning, if originally you had 2 pipes, lengths 110 and 90 and you divide in 50 meters, pipe 1 will be 50m, 50m, and 10m long and pipe 2 will be 50 and 40m long.

3) If you run the DIVIDE a second time, MM will not divide them again as no segment is longer than the minimum specified length.

4) If you add new pipes that are longer than the divide length and you run divide again, only the new ones will be divided.

5) If the active block is not block 0 (the mainline) only the manifold pipes will be divided according to the above rules.
