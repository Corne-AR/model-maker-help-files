MENU : Irr

 

Model Maker Irrigation

The irrigation module is a comprehensive designing system to develop irrigation projects.

 

Operating within Model Maker, Irri Maker integrates with the powerful CAD and survey functions to provide the most complete software package available.

 

Irri Maker makes the design of irrigation systems a pleasure by dividing the whole process into logical compartments.

The two main components are the nodes and pipes.

The system (blocks or main line) are build by placing nodes on the ground which are then connected by means of pipes.

Blocks are defined in which the sprinkler or micro/drip layout is placed and managed. Each block consists of lateral lines (connecting the emitters), the manifold line (connecting all the laterals) and the valve which connects the block to the main line.

The main line transports the water from the source to the valves of all the blocks. The blocks are easily grouped to operate in different shifts.

 

Comprehensive hydraulic functions are used to size the pipes and to calculate all the pressures and flows.

A complete list of parts and clusters can be defined and automatically placed on the nodes, providing a detailed list of materials with prices. Impressive drawings of the design are produced along with the CAD detail and contours by means of the plotting functions provided by Model Maker.

 

Access the main irrigation dialog box by clicking on the Irr button on the left or by pressing Shift I on the keyboard.

All the irrigation functions are grouped together on the main dialog box. Pages with descriptive tab names will lead you to the individual functions.
