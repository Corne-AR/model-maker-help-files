MENU : File - Export All

 

When you save MAL files on your own pc you normally do not want to include the bitmap source, pattern source, irrigation databases etc. in the MAL file because when loading the MAL file, it is just going to make duplicates of your original data. So typically, when saving MAL files you set your selection to only save the actual data and not the supporting entities.

 

 

When you want to give the MAL file to somebody on another pc you maybe want to include all the supporting images,patterns and databases. In this case you would export the MAL file.

 

 

To make it easy to use either the SAVE and EXPORT function, each option has it's own set of parameters. If the parameters of the SAVE and EXPORT are set the same then there is no difference between the 2 options.
