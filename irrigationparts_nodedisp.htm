Edit Mode

If this box is checked the parts grid editor will allow you to manually type part numbers in the grid blocks. In this mode you can also cut and paste part numbers to various nodes.

If the edit mode is unchecked it allows you to double click in a grid block and have the parts database menu displayed from where you can pick a part number for the specific node.

 

Copy and Paste

Apply the copy function by placing the contents of the selected cell in the grid editor, on the clipboard.

The paste function will take the contents from the clipboard and copy it into the selected cell in the grid editor.

 

Use the cancel button to close the panel without any action taking place.

 

Note:

Although the terms, end piece, reducer, T piece and + piece are loosely used, they do not refer to the actual layout of the node but rather to the number of pipes connected to it.

For example, Irri Maker uses the term T piece when 3 pipes connects to a node and is not concerned if it actually is a T piece or a Y piece or any other orientation.

When adding part numbers to these nodes, graphically inspect the layout of the nodes and based on that, select the correct part number to represent the correct orientation.

 

Click the UPDATE NODES IN LIST button after selecting the desired combination. Only nodes complying to the selection will be displayed in the grid editor.

 

Global editing

                         

 

The options below the grid editor allows for global editing of the part numbers.

 

Edit selected

When a specific part number must be allocated to a range of nodes, select the first row by indicating the part no column.

Keep the right mouse button down and drag it over the other lines. Now click on the Edit Selected button and select the desired part number from the data base. The part number will be allocated to all the nodes selected.

 

Clear Selected

Part numbers allocated to the nodes within the selected rows and only on the selected columns will be cleared. With this option you can clear all the parts on a selected  column and leave the others intact. If you do not want to clear the main part number in the Part No1 column, ensure that the selection only covers the other columns eg. Part No.2, Part No3 and/or Part No.4.

If the ALL option is selected the following will happen:

 

If the main block - block 0 - is currently active, you will be asked if all the part numbers of all the blocks must be cleared. Click Yes to clear all the part numbers of all the blocks. Click No to clear only the part numbers of the main line.

If a regular block is active, only the the part numbers of the selected block will be cleared.

 

Plot

When entering the parts page the first time, all the parts will be listed. If you have changed the Node display options only the displayed nodes in the grid editor will be highlighted. If you have manually selected certain rows in the grid editor and the selected option is checked, only the nodes on the selected lines will be plotted.

Adding part numbers automatically

 

 

 

Template

 

Edit                : Access the template data base to manipulate the values.

Teach        : Takes the information from the grid editor and add any new combinations to the template data base.

 

·The Teach function works as follows.

When you install Irri Maker on your computer the first time, it does not really know what type of designs you will be doing. When you start to use Irri Maker to do the designs, you can then teach the program how to add the parts and clusters to new designs by following the same logic as yourself.

 

After your first design, you would most probably add the part number manually by inspecting each node - pipe combination. By clicking the teach button, all the node combinations and specific parts that you have added to it will be recorded in the template data base.

 

The next time when doing a similar design, click the Fill from Template button. All the combinations in the template data base will be compared to all the nodes of the new design and if exact matches are found, the part numbers will be added automatically to the nodes of the design.

 

Carefully inspect the node combinations and check for any combinations not previously encountered. If new combinations exist in the new design, manually add the applicable part numbers and use the teach option again. The template data base will intelligently ignore the combinations that it already has record of and only the new node - part combination will be added. In this fashion, the program can eventually layout the part numbers on a design with very little user assistance.

 

Fill from template

 

Click the Fill from Template button. Irri Maker will inspect each node and all the pipes connected to it and check this combination against the possible combinations as specified by the user in the template data base. Part numbers will be placed on the nodes that have pipe combinations that complies to the combinations in the template data base.

If All the Blocks is checked, Irri Maker will in return inspect and fill all the blocks, including the main line, with the defined part numbers.

 

 

The part numbers of that node will be displayed in the node info dialog box.

 

Use the information in this dialog box to visually inspect the parts of each node.

 

If you want to edit the node itself, click the right button of the mouse and the node editor will be displayed.

 

When done inspecting the nodes, press escape to return to the irrigation menu.

 

 

Guideline - Adding parts to an irrigation project.

1) From the parts page in the main irrigation menu, click on to access the parts data base.

    In the parts data base, enter the part numbers, descriptions and prices. Also define the clusters if necessary.

 

2) In the node data base, enter the applicable part number for each node type. By doing this, the major part number for each node is already defined at the stage when the node is placed in the system.

 

3) On the parts page, use the grid editor and manually select each part number or use the Fill from Template option to do it automatically.

 

4) Click the teach button to record any new combinations that were found.

 

Note :                

Step 1 is only necessary in the beginning until all the part combinations have been defined.
