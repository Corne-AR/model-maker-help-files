MENU : Cad - Text - Search_ replace.

 

 

Specify up to 100 text strings that often need to be changed from project to project. This is particularly useful when you plot drawings on different scales and dates.

 

This function, for example can quickly search the whole drawing and update all CAD text strings that contain the word 'scale'.

 1 

Text to be searched for

Enter up to 100 text strings to search through the CAD database.

 

 2 

In the SEARCH FOR column, enter the string you want to find.

 

 3 

In the REPLACE WITH column, enter the string you want to replace it with. If the replace with column is left blank, (see line 3 as an example) the new suggested replacement string will be the same as the original string. If something is entered in the replace with column, it will be used as the suggested new value.

 

 4 

In the Y/N column, type Y if the string must be included in the search. Type N if the string must not be included in the search. You may also double click on the specific line to toggle between the Y and N options.

 

 5 

Click the Start button to initiate the search process. (You cannot change the search parameters once the search has        started).

 

Any CAD text element that contains the SEARCHED FOR string will be displayed.

 

If the Exact Only box is checked, only text elements that exactly matches the desired input will be displayed. If it is unchecked, all the text elements that matches the input, either partially or fully, will be reported.

 

 6 

Text found edit box

In this box the actual CAD text string is displayed when a match with the selected text is found.

 

Use the REPLACE WITH THIS edit box when making changes to the original string.
