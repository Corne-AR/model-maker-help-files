MENU : Cad - Dim - Edit                                

 

Edit the attributes of a dimension.

 

 

 

Move Text

Move the text of the dimension to any other position and/or rotation. Use any of the relevant hot keys,  e.g., Shift/Tab, Page up, Page down if necessary.

 

Move Line

Move the dimension line closer to or further from the original position. Check that the cursor mode is set correctly before placing the line.

 

Align with other

Indicate the base dimension from which the alignment must be calculated. The current dimension will move to the alignment position.

Model Maker will automatically align the existing dimension with the previous one if the little check box on the side is checked. Use this option when you are adding multiple dimensions and want them all to be on a single line.

 

Parameters

Change any of the dimension attributes.
