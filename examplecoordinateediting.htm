MENU : Example - Co-ordinate Editing

 

There are several options that can be used to edit DTM points.

 

               The DATA option on the top menu

               The PNT. option on the left menu.

 

 

The Data Menu:

 

This following dialog boxes hold various options.

               Station - Use this to add, edit and delete survey station co-ordinates.

               Points - Use this to access the survey points data base. Individual points can be manipulated.

               General - Functions are available to audit the data, change codes, add constants and to print the survey data.

 

 

Points Editing Functions on the Graphics Menus:

 

Many of the options found in the data menu - and more - are also available on the graphics screen.

Use the PNT option on the left menu  - the point functions are displayed on the bottom menu.

The point function is used to edit a point.

               POINT - allows you to individually inspect and edit selected points.

         The point number, station number, code, tag, Y X and Z co-ordinates are displayed.

         Changes to any of the point attributes are automatically updated in the survey data base.

 

       Be careful when changing the co-ordinates of a point after triangles have been generated to it. The triangles connected to the point also move to the new co-ordinates, possibly causing the triangles to overlap. Changing the elevation of a points does not affect the position of the triangles.

 

Hint:

Inspect and test out all the other functions available on the PNT menu.
