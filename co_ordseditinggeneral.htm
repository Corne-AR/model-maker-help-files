MENU : Co-ords Editing - Gen

 

Change the codes of points.

 

 

Every point can have a six character code. This code is primarily for the user's own use.

 

A few defined codes have special significance to Model Maker.

 

1)                01 :                 Any point that can be classified as an ordinary field shot should be coded as 01. MODEL MAKER will not write the code 01 next to these points when codes are

                                displayed on the graphics screen. 01 is  a blank code.

 

2)                *.* :                 MODEL MAKER uses this code to indicate a point has been marked s being deleted. Because the triangles are

                                referenced to the points via their position in the database, MODEL MAKER will not allow you to delete points randomly. Deleting points on the graphics menu does not

                                remove those points immediately from the database. The codes are just changed to " *.* ". Use the delete option described below to remove these points.

                     

3)                Any point having a code that starts with a ".", eg. .TREE, would automatically be excluded from the triangle search process.
