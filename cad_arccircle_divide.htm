MENU : Cad - Circ - diVide                                                

 

This function can be used to divide an arc/circle into different segments or draw CAD lines from the perimeter of the circle to it's centre or draw co-ordinate lines between the circle segments.

 

 

DIVIDE @

 

Three different methods can be used to position the new segments or CAD lines.

 

·Regular angle        : Starting from the start direction of the arc/circle, new segments will be developed at regular                                  angular intervals around the circle.

 

·Regular distance        : Starting from the start position of the arc/circle, new segments will be developed around the                                  circle with the arc length of the new element equal to the selected distance value.

 

·Number of points        : The full angular value of the arc/circle (end direction - start direction) is calculated and                                      divided by the selected number of points.

 

DIVIDE METHOD

 

Arc segments

 

When this option is selected, the original arc/circle will be cut at the requested positions and each segment will exist as an arc by itself.  For example, the new segments will look different from the original until every second segment is deleted. New endpoints will be created at the arc segments and the user can use them to jump to.

 

Cad lines

 

When the CAD lines option is selected, an additional box will appear (highlighted). With this option the original arc will not be changed, instead CAD lines will be drawn from the circle perimeter to the centre of the circle.

 

       

 

In circle 1, the circle center check box was selected and all the lines were drawn to the centre point.

In circle 2, the check box was off and all the line were drawn towards the centre point for a length of 10mm.

 

 ALSO@ : Check these boxes if lines must also be drawn from the start or end points of the arc/circle.

 

Co-ords

 

With this option the original arc will not be changed, instead, CAD lines will be drawn on the circle perimeter between the segments.
