MINE : Update Surface

 Top  Previous  Next

 

This function will add new dtm points to the surface of the active project.

 

Typical application:

 

·A complete survey of the original ground model exists.
·A list of bore hole data consisting of coordinates and surface elevations is available.

 

This picture (below) shows 2 models:

 

 

·
·Red contours = OGL model.
·
·White contours = model defined by 5 borehole coordinates.  (blue points)

 

The UPDATE SURFACE function can be used to copy the survey points from the OGL model to that of the bore hole model.

 

Before using the function.

 

·Ensure that the active project is the one holding the borehole points.
·In this example, the bore hole points are in project number 7.

 

When the function is run the following happens.

 

·The area of each of the triangles in the bore hole project is inspected and all the OGL points falling within the triangle area, are copied to the bore hole project.
·The bore hole project will end up with the original bore hole points plus all the OGL survey points between the bore holes.

 

 

After the ogl points have been added, you must use the Csearch function to create new triangles on all the points.
