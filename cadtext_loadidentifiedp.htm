MENU : Cad - Text - Load - loadIdpoint

 

Draw the code and co-ordinates of the Identified points of the active project into CAD using the current text parameters.

 

 

You will be prompted to indicate the base position. The co-ordinates will be sorted by code and the values of the first point will be placed at the base point. The rest of the points will be placed in lines below the first. It is important to have the default text parameters set correctly before the operation. It is also good practice to set the drawing scale correctly beforehand.
