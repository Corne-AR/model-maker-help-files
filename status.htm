MENU : Status                                (Alt+T)

 

 

How can this topic help me ?

This information box can be useful when you think you have data loaded and cannot find it.  Everything open in memory is displayed here.  It also tells you if data has been saved or changed.

Project # Box

 

Click on the desired project number in the Project # list box to display the relevant information of the required project.  This box also includes the number of points next to the project number.
