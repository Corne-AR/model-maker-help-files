A pump node will always represent the water source in the main line (block 0).

 

If the system is gravity fed and the water source is a dam on a high elevation, then a pump node will still be used to represent the water source. The starting pressure of the system will then be set to the depth of the water in the dam.

 

Note : The starting pressure of a gravity system cannot be zero. There has to be some pressure in the dam to start the hydraulic calculation of the system .

If the mainline is an open system (no looped pipes) then only one pump can be used.

 

A looped mainline system can have one or more pumps.

 

If your design consists of 2 or more separate pipe networks, each with it's own water source and no interconnected pipes, then each networks will be done in it's own project.
