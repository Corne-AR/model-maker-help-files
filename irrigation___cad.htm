MENU : Irr - Cad

 

Because Irri Maker works within Model Maker, all the CAD functions are available to the irrigation designer. This also enables Irri Maker to copy some of the design information to the CAD environment where the powerful CAD function can be used to improve the drawing.

 

Two types of information can be copied to CAD.

 

 

Block information

 

Check the information element to be copied, then press the Copy information button.

Indicate the position where the data must be placed on the drawing. The information will be transferred to CAD as regular CAD text. You may edit the text by using the functions in the CAD Text menu.

This function only copies the information of the active project. To produce a report of block area, rate and pressure of all the blocks use the global report options.

 

Copy All to CAD

 

This function will copy all the information of all the blocks to CAD. Everything that is displayed by the irrigation part of Model Maker, nodes, shapes, pipes, text on pipes, block boundaries etc. will be copied to CAD as a complete replica of the irrigation image.

This feature can be used to combine multiple irrigation designs onto a single plotted plan. Note that the replica in CAD consist of lines and text and has no further relation with the irrigation components after the copy process is completed. Therefore, be careful not to make changes to the irrigation design after this function has been used as the CAD image will not be updated accordingly. Rather delete the CAD elements and copy the irrigation image to CAD again to reflect the final design.

 

Pipe Information

 

Copy information (all pipes)

This function will copy all the pipe information as text to CAD. The lines representing the pipes will not be copied. Once the text information is in CAD, you can move, delete and rescale the text using the normal CAD text functions.

The text to be copied depends on the settings on the pipe and default page.

 

Diameter - indicate pipes

Use this function to copy the information of selected pipes to CAD. The all pipes function, described above, copies the information of all the pipes to CAD, often resulting in a cluttered picture. By copying the information of selected pipes to CAD one can control the distribution of the text, maintain a clean plot and still describe the irrigation design sufficiently.

 

Note:

If you intend to copy data to CAD, only do it after the layout is completed, the pipes sized and the hydraulics calculated. Once the data is copied to CAD, refrain from changing the irrigation design again as the text in CAD is removed from the irrigation design and will not be updated automatically.
