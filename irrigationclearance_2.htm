MENU : Irr - Pivot - Pivot Parameters

 

Use the Clearance function on the Irrigation menu to check pivot span heights over the terrain.

 

To access the clearance function, click the Clearance after the Pivot has been placed.

 

The clearance check dialog box requires the following values.

 

 1 

Sprinkler height. This is the height of the span where it is attached to the towers.

 

 2 

Clearance values. A color map will be produced depicting the areas where the height above ground, along any position on the spans are within the set values.

 

 3 

Check the Copy to Cad option if the color map must be retained as a bitmap image on the drawing.
