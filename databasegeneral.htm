MENU : dB - data base-Gen

 

 

The functions on this menu can be used to do the following:

               Print the DTM (survey) data.

               Add constant values to the coordinates of the DTM, strings or CAD.

               Check for duplicate DTM points and delete the *.* codes points.

 

Print Data

Use the options on this panel to print the survey co-ordinate information to either the printer or editor.

 

               Print station data

         Select this option to print only the information of the stations. (Example)

 

               Print station and points

         This option will print the complete survey field book. (Example)
