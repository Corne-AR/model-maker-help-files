MENU : Hor

 

View and print more detailed information about the alignment.

 

Each PI is inspected individually. This header box indicates the number and the name of the PI that is under view. Each PI has a radius and, optionally, transition lengths can be defined. This header box also indicates what type of curve it is.

 

The general section shows information about the direction and distances between the current PI and the ones before and after it.

 

The circular section holds useful information if an actual radius value is entered.

 

The transitional box shows the entered lengths and the calculated tangents if applicable.

 

Use this scroll bar to navigate the alignment data base.

 

This page tab controls the printing of the alignment curve data.

 

Enter the range numbers of the curves to print if you do not want to print the whole alignment.

 

Direct the result of the printout to the editor or to the printer.

 

Select a printout format.
