MENU : Cad - Line - Edit                                

 

Edit Line

 

Indicate the line to change. The line will take on all the attributes of the default pen & line type.

 

Edit all

 

The Pen menu  changes the attributes of all the lines. Use the Identify functions to change selected groups of CAD lines.
