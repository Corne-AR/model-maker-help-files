MENU : Utility - Digitize data

 

 

Data can be digitized from existing plans.

The plan to be digitized must have two points with known co-ordinates diagonally across. This may be on the national grid or a local grid drawn on the plan.

The digitizer devices supported are: HP plotters and compatibles and most other digitizers.

Digitized points are added to the data of the current active project.

 

Before the digitizing can start all the settings on the setup page must be completed.

 

Click the Open button to set the communications serial port to the digitizer is. The actual digitizing buttons will only be available after the port is successfully opened.

 

 

After the port has been opened, set the digitizer type. MM distinguish between the HP plotter (old pen type) and digitizers.

Most of the digitizers operate on the same principles by sending a string, binary or ASCII, back to the computer each time a button on the digitizer stylus is pressed. MM will analyze the return string and try to convert that into actual digitizer coordinates. Check the format. The way that MM will interpret the return string is displayed. Click on the defaults page to make changes to the transfer string format.

 

 

Check the sound check box if a beep sound must be emitted on the computer each time a signal from the digitizer is received. This is very useful as you will be able to determine if the computer has recognized each signal after a stylus press.

 

 

After the comms port is opened and the digitizer type selected, use the paper orientation functions to setup the correct coordinate system.

Two points on the plan must be digitized and their plan coordinates entered. Model Maker will relate these plan (survey) coordinates to the corresponding digitizer coordinates and use a formula to translate the digitized points to the correct survey coordinates.

 

 

Click this button. MM will allow you to digitize the 2 known points. The best accuracy is obtained by selecting the two orientation points at the opposite corners of the plan.

 

This panel is displayed when MM has placed the system in digitizing mode. Digitize the desired points. To abort the digitizing action, click on the panel with the mouse.

 

 

After the two co-ordinates have been digitized, enter their plan co-ordinate values in the appropriate boxes.

 

 

Click on this button to signal that the co-ordinates have been entered and the system should activate the actual digitizing functions.

 

 

After the co-ordinates of the 2 orientation points have been entered the PlotP1,P2 button will become active. Click on this button if you want a rectangle to be drawn between the 2 points. This rectangle will consist of regular CAD lines and can be used to orientate the graphic display area to that of the area being digitized.

 

 

Click on the Manplot button to temporary deactivate the digitizing dialog box and allow access to the regular plot manipulation function. After manipulating the graphics screen, press Escape to return to the digitizing menu.

 

 

 

Before digitizing of data, set the correct string format. When the stylus on the digitizer is pressed a predefined string is outputted to the computer. MM has to be instructed how the string must be analyzed.

 

 

Note:

Change the settings on your digitizer to output data in the following format.

         Point mode as oppose to continuous mode.

         In this format a single string will be transmitted when the stylus is pressed.

         ASCII mode as oppose to binary mode.

         In this mode the return string will be in a format that you can read, interpret and instruct MM on how to extract the 2 coordinates from it.

 

 

The setting on the digitizer is normally change by means of the dip switches or by accessing a certain menu on the digitizer. Consult the manual of the digitizer for additional information.

MM uses the following method to extract the desired co-ordinate values from the return string.

For your specific instrument, indicate the Y and X co-ordinate positions by placing x and y characters in the positions in the string where the numbers must be picked from.

 

 

EXAMPLES:

         199000 50505                The return string.

         _yyyyy_xxxxx                The matching y and x filter.

or

         Y99000 X50505                The return string.

         _yyyyy__xxxxx                The matching y and x filter.

 

 

In the first example, the starting character is not part of the co-ordinate values but indicates the number of the stylus button that was pressed.

199000 50505                To ignore the character, place a underscore '_' in that position.

_                                                Filter string.

199000 50505                The following characters, '99000', are the y co-ordinates that must be filtered.

_yyyyy                                Filter string.

199000 50505                There is a space between the 2 sets of co-ordinates. Place an underscore there.

_yyyyy_                                Filter string.

199000 50505                The last part is the x co-ordinate values.

_yyyyy_xxxxx                Filter string.

 

In the second example the 'Y', space and 'X' characters must be skipped. For that reason the filter string will start with the underscore and have 2 underscore between the y and x co-ordinate values.

 

 

Digitizing actual points or lines are done by using the functions on the digitize page. Note that the digitize page is only activated after the setup has been completed successfully.

 

 

The digitize function is designed to capture the co-ordinates of indicated points on the plan.

Enter the desired Code, Tag and elevation values that should be assigned to each digitized point.

 

 

When digitizing contours you often need to raise of lower the elevation by a constant amount between different contours.

Use the up and down buttons to easily add the value in the constant input box to the default Z values of points to be digitized.

 

 

Check this box if continuous CAD lines must be added between the digitized points. The lines will automatically have a break each time the elevation of the digitized point is changed.

 

 

Use this function to change the line parameters while digitizing.

 

 

Click on the digitize button to start digitizing.

 

 

The panel will be displayed. Click on this panel stop the digitizing when necessary.

 

 

Check this box if you want to remain in the digitize mode with out returning to the menu between each new point. This is useful when digitizing contours.
