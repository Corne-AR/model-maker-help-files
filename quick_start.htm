Quick overview of an Irrigation project.

 

 

 

 

 

A) To start a new project:

 

 

1) If you have a previous project loaded, save it as a MAL file and then clear the dtm,cad,irrigation entities by using File New.

 

 

2) Once you have a clean system, determine you project's geo position.

That can be done by loading a kml file or capturing a Google maps image.

 

B) Getting the project's geo position (address)

There are a few ways to get the project going.

 

1) Define the area in Google Earth and save the project as a KML file.

Use File Import Kml to load the entities.

 

The kml import will check if your Irri Maker system settings are correct.

 

The correct settings will be shown if needed and you can change the system to that

 

 

 

 

 

   

 

2) If you do not want to start with a KML file, use Google Maps to get the position of your project.

 

Use Cad, Images, Google Maps. Locate the design area and capture the image.

 

 

C) Starting the design

 

a) Start by using the CAD to draw elements like lines and circles on the image.

 

b) If you loaded a kml file, the entities from there were already loaded in the cad.

 

c) You can also load a dwg file as drawing entities.

 

After using the cad to define needed detail, define DTM points over the design area so contours can be generated.

 

 

D) Define dtm points for a topo surface with contours and elevations.

 

a) The easiest way to create a topo is to use this sequence:

 

In the New Points dialog, set the grid spacing. The elevations for these points will come from Google Maps and if you have more than 3000 points you will have to create an account with Google to obtain their Maps API in order to elevate more points.

 

The Google Maps function will also ask if you want to create the triangles.

Answer yes so the triangle model is available to provide elevations for the irrigation design.

 

 

 

 

     

 

E) Start the irrigation design.

 

Access the irrigation menu. On the Project tab, set the correct units.

 

1) For a solid set or drip design, add new blocks and fill them with lateral pipes and emitters at the user defined spacing.

 

After the lateral layout is placed, define a valve and connect the manifold pipes. Mainline pipes can be connected to the valves.

 

 

 

       

 

 

 

 

2) To define pivots, use the pivot tab and pivot menus.

 

Add a pivot entry, define the spans and define the center position on plan.

 

When a pivot is defined, a valve is placed at the center and mainline pipes can be connected to it.

 

 

 

3) After the blocks are added, run the hydraulics of each block and resize the correct pipes.

 

Now activate the mainline (block 0) and define the pump and any other bend nodes.

 

Connect the pumps to the bend nodes and valves.

 

The pivot valves are already in the mainline.

 

 

 

After the mainline network is complete, run the mainline hydraulics and size the pipes if needed.

 

Print report if needed.

 

 

 

When complete (or during the design) save you data with File Save All
