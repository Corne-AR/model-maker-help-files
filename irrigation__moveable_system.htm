IRRIGATION : Moveable System

 Top  Previous  Next

The movable systems function takes a predefined area, then quickly divides it into pre-set laterals where each lateral is contained in it’s own irrigation block. The program then adds a valve for each block. A mainline is then connected to each valve automatically allowing the user to add the system to an existing mainline or simply add a pump to design the mainline.

What you will need:

1)A DTM surface of points and triangles to cover the whole area to be irrigated
2)A CAD segmented continuous CAD line indicating the area to be irrigated with a polygon shape.
3)A single segmented CAD line drawn in the position of where the proposed new mainline will be placed. It is important to make sure that the CAD line cross the intended polygon area

This is the movable systems setup window:

Placing:


To place the laterals, you may choose to place the laterals on only the left or only the right or both sides of the mainline

 

 

Neat grid will place laterals on either side of the mainline in perfect grid spacing.

Staggered will alternate laterals around mainline creating a sort of triangle spacing of laterals.

 

Emitters:

Must an emitter be placed at every outlet or only every second outlet? In this option you can select to have either an emitter placed for every outlet, or every second outlet. If Every second sprinkler is selected, the program will place a fitting for every alternate node.

Click on each node or pipe type button to select the appropriate nodes or pipes to be used in the design.

Finally select the area definition option that suit you best which will be used to define the sub area to be irrigated:

The area definition dialog will show, select the Define Area option and use Cursor Snap modes to track the area.

 

Accept the area

Then finally indicate the diagonal CAD line indicating it on the side where you want the block definition to start.

The final result is that IrriMaker automatically draw all the blocks. The user can now simply calculate pipe sizes for each block and to get a design flow for each block after which you can connect the pump node to the mainline and design the mainline as usual.
