The user title block option has been developed so that the user can produce plotted plans with complete title information along one edge of the drawing and CAD, DTM and other elements, displayed within a defined rectangular area.

 

This is a format that works extremely well except that it does not allow for the definition of an irregular active plotting area. This drawback is overcome by the inverted user defined title block option.

 

Background

The regular user title block is drawn in the following sequence.

1)                The complete paper area is activated.

2)                The user defined title block information is plotted.

3)                The active plotting area (the user defined rectangle) within the title block is activated.

4)                The drawing elements are plotted that falls within the active area. Elements outside the area or crossing the boundary are clipped and not displayed.

 

Figure 1

Take this layout as an example.

If the actual title block was defined as area a1 to a2 then the active plotting area could be:

 

A to B - rendering the area E to D ineffective plotting area.

or

C to D - leaving the area A to E ineffective.

 

 

The inverted title block option allows Model Maker to use the complete paper area for plotting purposes.

The plotting sequence is as follows.

1)                The complete paper area is activated.

2)                All the drawing elements are plotted first.

3)                Keeping the total area activated, the defined title block information is plotted second, on top of the drawing elements.

 

The secret to this method is that the user must define hatch areas in the areas where title block lines and text are defined.

Figure 2

Notice how 4 hatch areas have been placed at the corners to allow the title block information to be plotted on top of CAD elements.

Defining an inverted title block

Define the inverted title block in the same way as the regular title block except for the following.

 

                       Place a hatch in the area where the title information must be displayed.
                       When defining the plotting area, indicate the complete paper area. In figure 1 that will be area A to D.

 

                       Finally, when saving the title block, check the invert option.

 

A typical final result obtained by using the defined title block as shown in figure 2. Notice how the hatched areas fill the corners, erasing the original drawing in those areas.

 

Notes:

This format of title block can only be used on the newer ink jet plotters.

Plotting to ink jet plotters is done through an electronic canvas component which allows the adding and erasing of pixels before the final image is displayed on the printer.

So what really happens, is that the full paper area is initially covered by the drawing elements. When the title block is plotted, the hatched areas below the title areas actually erase what ever was visible before, again, creating a clear area in which the title lines and text can be displayed.

On pen plotters this would not work as lines already drawn cannot be erased again.

When plotting on regular white paper, also make the colour of the hatch - used as the title area back drop - white.

Ensure that the CAD plotting sequence is set that the hatch elements area plotted before the other elements.
