MENU : Cad - Text - Copy

 

Copy any CAD text to a new position.

 

Use the COPY+ option to increment the value of the text to be copied by one if the text is a valid integer number. Use this function to add incremental numbers on a plan, eg. area numbers.

 

NOTE : When the COPY+ option is used and you need to copy the original text onto a straight line for a number of times, keep the SHIFT key down when the position of the first new text is indicated.

In this example the first text is added as 'A1'. The COPY+ function is then used to number the rest of the blocks. When the first position is indicated the SHIFT (on the keyboard) is pressed and a dialog box prompting for the number of copies is displayed.

Enter the number of additional text copies you need. The new text will be based on the distance and direction between the base text and the 1st indicated position.

 

Example of copied text. (positive number of copies = extrapolated)

 

Sometimes it is more convenient to indicate the last position of the set of new text elements and have the program interpolate the new text in between the base and last position. To do that, enter a negative copy number.

 

Example of interpolated text by specifying a negative copy amount.

 

 

 

Text can be incremented alphabetically or numerically.

 

 

Use the Page up/ Page down or Ctrl Page up/Page down keys to rotate the text while copying.

Press CTRL Home to change the relative text size, angle or the rotation increment.
