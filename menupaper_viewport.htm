Viewport

 

 

Viewports are typically used to streamline the plotting of multiple views of the layout in a manageable format. It also allows you to compile different parts of the drawing into a single paper window.

How to set up viewports.

In this example we have 2 viewports (red and blue) and one paper window (pink).

 

1)Set the position of the paper window by using the NEWP option. Select an area outside of you drawing where there are now lines or drawing entities.
2)Use the F6 measure function and measure the length of a typical viewport that will fit inside the paper window. Also measure the height. Divide the height of the paper window in 2 (or more) to allow for the placing of 2 viewports.

If for example the paper window is 800 by 600 then the 2 viewports should be 750 by 200 (approximately).

Use the VIEWPORT function and enter the default values.

 

 

3)Use the ADD function on the viewport dialog box and indicate the origin of the first viewport. In this example, viewport 1 is not rotated. After indicating the bottom left point of the origin proceed to the top half of the paper window and place the viewport destination in that area.
4)Use the ADD function again and place viewport number 2. This time use the Page Up and Page down keys to rotate the viewport when setting the origin.

The Scale Factor

 

Use viewport 4 (above) as an example.

On the origin the window is going to be 40x40 meters in size (40x40mm at 1/1000 scale). On the destination window the size is going to be 400x400m and the elements inside the 40x40m origin will be plotted inside the 400x400 destination at a 10 times scale increase.

If you intend using a scale factor, enter the value in the last column before you actually use the ADD button. In this case the destination window will already reflect the scaled window size and you can position it correctly.

If you only change the scale value after the destination has already been set, the updated destination view will only be updated (in size) after the first replot.
