MENU : Cad - Id - Cutout                                

 

How to make a cut out of the CAD elements based on a defined area.

 

 

Cutout method

 

Move

 

Cut and move the elements from the original drawing to a new location. A blank space will be left within the defined area. Lines that cross into the area will be trimmed. The centre point of an arc/circle, the base point of text, the end points of dimensions, and the nodes of an arrow will determine if the lines are within the area.

 

Copy

 

Copy the cut out elements to the new position. This is almost the same as the MOVE option, but the origin drawing will not change.

 

Delete

 

Delete the elements from the drawing.

 

Save

 

Save the selected elements to disk.

 

Paper windows

 

Save the elements inside each paper window to disk.

The paper window areas will be used to define the cutout areas. When the first window is done you have to specify the KCD filename. Thereafter, the other windows will be saved automatically to KCD files with an incremental postfix.

 

EXAMPLE:        If you enter the first name as CUTOUT then the other files are called CUTOUT1, CUTOUT2 etc.

 

IMPORTANT :        Model Maker only asks you the first filename, thereafter the others are automatically generated and ANY other KCD with                        the SAME name will be REPLACED.

 

Hint:

Select a unique initial filename so that the others do not have to replace existing files.

To add a title block, have a pattern called TITLE loaded and check  the option to the left. (will be visible if  the TITLE pattern is loaded).

 

 

 

 

Line -> Zero

 

Move an existing line within the area, horizontally to the new position. For example, use this option if you have a road layout and you want to cut the straight sections of the road between the curves and align them all horizontally next to each other.

 

Scale change

 

When move or copy, apply this angle to elements.
