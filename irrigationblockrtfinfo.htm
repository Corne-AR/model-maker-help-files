IRRIGATION : Block RTF Info

These 2 function are used to copy the block information to cad.

Although there are other basic function to copy the information, this function allows you to define the order and format of the data you want to copy.

 

 

 

Use the Copy info function to generate the cad information.

 

The Move cad blocks function is used to change the position of the block info as one entity.

 

Other supporting functions that are used:

 

1) The RTF file is a file that is created in the block report function.

 

2) The resulting cad elements (lines, text and hatch) are assigned to cad blocks and can therefore be moved as one entity.

 

 

 

RTF file.

 

This is the file created in the block report. The OK button will only be enabled if the RTF file is available.

 

Settings:

 

Select if the text must be placed on top of a hatch back ground.

Select if the a border must be drawn around the text.

 

Normally, if a hatch background is selected, a frame is not needed.

 

Define the pen colors for the text, line and hatch.

 

Specify if the information of only the current irrigation block or all the blocks, must be copied.

 

Define the cad layer name for all the elements.

 

 

Press OK to generate the cad info.
