MENU : Cad - Id - Man_id - cadline Point                                

 

All the identified CAD lines are inspected and new co-ordinates are calculated at their endpoints.

The new points will be saved to an ASCII yxz file.

Use the load Asciidata option to load these points into a project.

 

 

 

The text in this box will be the code used for the new points.

 

If the Increment the numeric codes box is checked, it adds one (1) to the numeric value of the code each time.
