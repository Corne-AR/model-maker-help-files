MENU : Cad

 

Model Maker has a fully integrated CAD system that combines exceptionally well with the DTM functions.

The CAD can be used on its own or to extend the capabilities already provided by the other survey and related modules of the program.

 

Purpose

 

Although Model Maker was initially designed as a dedicated DTM program, CAD has become an integral part of the package. In these days of presenting drawings that are both functional and appealing, a CAD system within the survey, engineering and DTM environments is paramount.

CAD draws features on the surveyed plans, assist in drawing layouts for design work and add design detail and most other information associated with the engineering profession.

 

CAD functions

 

Model Maker's CAD provides all the basic drawing functions needed to create drawings as overlays on a survey or from the imported DXF files.

               At the top right of the screen is the cursor mode indicator.

               Use the space bar or point to the curser mode indicator to  change the cursor mode.

               At the bottom right of the screen, select the pen and line (P) for all drawing functions. A layer can be specified for all entities.

               CAD drawings can be exported to other CAD systems via a DXF file.

               Save the CAD drawings to file using the File function.

 

Lines

 

Lines are normally the element used most abundantly in a CAD drawing. Many different details like fences, roads, property borders, rivers, power lines, telephone lines and houses are usually represented by CAD lines. Model Maker allows you to draw lines with different colours and line thickness. Lines types can vary from solid lines to various types of dotted lines. The user may also define line types

 

Text

Adding text to a drawing is a very important part of the final detail. Model Maker recognizes that and uses a unique dialog menu that gives you total control over the font, size, colour, position and orientation of the text. Text elements can be copied, aligned, resized and manipulated with the greatest of ease. Pages of text, for example the report of the irrigation design results, can be imported and neatly displayed along with all the other CAD elements. Different fonts, including any installed windows font can be used.

 

Circles

 

All the functions necessary to add, edit, move, copy and to delete circles, are available. Similar to lines, circles can also be drawn with all the different line types and colours. Working with full circles or arcs you will find that Model Maker has all the functions available.

 

Hatch

The hatch option allows you to define any area and to fill it with either a grid of lines or with a solid colour. Use this to easily identify and draw attention to certain areas.

 

Patterns

Patterns are used to group a set of CAD elements together in a single entity. This entity can then be saved to a file and at any future stage, recalled from disk and inserted in other drawings. The pattern concept is normally used to draw certain terrain features like trees, houses, cars and other small shapes. The patterns can then be used repeatedly on different scales, sizes and orientations.

There are also functions that will automatically place a predefined list of patterns on survey points having a certain code

 

Arrows

Arrows are used to point to a defined position. The arrows can have different colours and the arrow head is user definable.

 

Dimensions

Dimensions are used to indicate the distance between 2 points or various other elements. Model Maker will calculate the distance but also allow the user to edit the final value.

 

Bitmaps

With the advent of the new ink jet plotters the manipulation and plotting of bitmaps images has become an integrated part of many CAD drawings. Up to 9 images in either BMP or JPG format can be loaded into Model Maker and plotted with all the other vector elements. The bitmaps can easily be positioned and resized allowing you to align any aerial photograph with the actual surveyed information.

 

Hints

CAD operates on its own or with any of the DTM projects. Usually, you would first load the survey points and then start adding CAD features to it.

Most of the time you will manually add elements to the drawing. Certain features can also automatically be added to your survey work. These functions are found on the lines, textand pattern menus. Powerful functions to add additional information to drawings are in the identify menu.

 

Examples

Use CAD Line Add functions to add a line between 2 points. Set the cursor mode before indicating the points. Also check the default line attributes that will be applied to the new line. Once the first point is indicated, a rubber band will connect that point with your cursor position as you move around. Use any of the screen manipulation functions while you are in the process of indicating the terminals of the line. The line will be fixed when you set the second point. Use the undo key to delete a line.

 

Use the detail function to automatically add lines between points based on certain parameters.

Use the speed menu to for quick access to some of the more frequently used functions
