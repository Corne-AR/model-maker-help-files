How to compare 2 projects

QUESTION:

Please point me at the correct help files: I have looked through everything and don't really know where to start

1. Import the survey file (File >Import ASCII and Other>DWG/DFX/DGN

2. Scale the drawing to 1:1 or 1: 100 say

3. Import the second drawing

4. Define the boundaries of both drawings to compare

5. Indicate the layers on both drawings that have the spot level text

6. Recognize the spot levels on each drawing

7. Draw contours/triangles

8. Compare the two drawings

 

ANSWER

In this walk through words in "CAPS" like NORMAL will be used. Open the help file and go to index, type NORMAL into the search box, and see the three topics found. Display the second topic: Plot, and read about NORMAL. In this way all unknown commands may be found as to easily follow.

Let's start by getting your first file loaded.

 

•STEP1: FILE IMPORT ASCII AND OTHER: select DWG/DXF/DGN. now OPEN your dwg, dxf, dgn extension file, and wait for the file to display. press OPEN. the file displays in the mode maker transfer window. now press OK and wait for the contents to load into model maker.

You will note that some drawings may be in millimetres, and a conversion needs to be applied. before you apply the conversion constants, you need to explode all the patterns: to do this:

 

•STEP 2: on the left main menu click CAD. at the sub main menu (bottom) : PATTERN, EXPLODE.  a dialogue box appears, select ALL, so all the patterns may be exploded. confirm your decision with YES. after exploding the patterns, press the NORMAL key. you will see the full picture now.

now  check the scale: go CAD, LINE, INFO. click any line, if you find the line lengths are very long, constants needs to be added.

 

•STEP 3: go to MAIN MENU: DATA, GENERAL, CONSTANTS AND CODES CHANGES. see the add coordinates box. we need to put *0.001 into the E and N row ( E and N for Northern Hemisphere). make sure the CAD DATA tick box is selected .

press APPLY CONSTANT.

now verify the distance of the lines is correct. go CAD, LINE, INFO. click any line and you will see the distance appears within limits.

some text and circles were not included in the conversion. It is necessary to reduce it's scale as well.

•STEP 4: go to CAD, GLOBAL, and select the following tick boxes: EVERYTHING and ALL. now click CHANGE, everything will light up in red (means it is identified). select TEXT, SIZE, and select the multiply by this value = 0.001.press OK, and MM  returns to the global screen, select CHANGE again, and now select CIRCLES, RADUIS. also multiply by 0.001. you will see the complete file is now in a workable format.

 

please save the file now as a .kcd extension .do the following.

•STEP 5: FILE SAVE, select the .KCD button, give a name in the filename box, like survey1, and save the file.

now to identify which layer the elevation text is.

•STEP 6: go CAD, TEXT, EDIT. click on one of the elevation text entities. at the top you will see the LAYER name. this layer now need to be identified.( for special purposes).close the box.

•STEP 7: go CAD, ID, NEW, and then DO_ID. select the button LAYER, and see all the layers displayed. now click the layer as found in step 6. press OK, and OK again. model maker highlights everything in the layer. press CLOSE to exit the id box.

•STEP 8: now to make DTM points from the selected text. go CAD, ID, MAN_ID. select the TEXT_POINT function, give code names, and press OK. DTM points are now extracted into project1. you may view them by switching off the display off the cad. to do: go SETTINGS, PLOT PARAMETERS: the button CAD appears under the word Global. You now need to click on it to switch it off. press ok when done.  all the dtm points will now be shown. to see the elevations of all the points, go back to PLOT PARAMETERS, and find ELEVATION, you will see by toggling these displays, the points' info change.

 

•STEP 9: to view points independently, go POINT (left main menu) and POINT,(Bottom, sub main menu) . click on any point and it's info will be displayed.

•Step 10 : now add triangles: (Left main menu) TRIANGLES, CSEARCH. then press NORMAL. make sure the display of the contours is on.(plot parameters) you have just created the survey dtm. triangles may be edited, and save the file as a .TOT file. follow the same procedure as saving a .KCD file (step 5).

•STEP 11: now clear the memory: (FILE NEW)

•STEP 12: Repeat steps 1 to 11 with the second file, and name it differently.

you will now have two dtm models which can be used for cut and fill. in the help file under working examples, you'll find example 6 which will help you to do cut and fill.
