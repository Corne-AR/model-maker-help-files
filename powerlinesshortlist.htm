POWER LINES : Short List

 Top  Previous  Next

 

MENU : gXt - Power - Edit pole - Short list

 

A short list of poles are defined for the purpose of selecting a pole type when adding or editing POWER line pole data.

 

The poles are added to the database on the SELECT page of the dialog box.

 

 

 

 

 

 

 

 

Multiple short lists can be created.

 

To start, click on the 'Create a new short list' button.

 

Enter the name of the new short list.

 

 

After the new name has been accepted, click it to select it.

 

Now that Model Maker knows which short list you are working with, click on the 'Short List' tab.

 

 

 

Initially the short list will be empty.

 

To add a pole to the list, select it from the available poles.

 

Click the 'Add Pole to list' button.

 

The pole name is added to the list.

 

 

 

 

 

 

After all the poles have been added to the list, click the  'Save Changes to List' and Model Maker will record the short list pole names.

 

Single poles can be removed from the list or select the ALL option to clear the short list.

 

 

 

The short list is mostly used in the Power Line design functions.
