MENU : gXt - Utility - Update_dtm

 

An indicated general string will be investigated to see if any DTM points on the active project lay exactly on any of the nodes of the string. The elevation of these points will be updated with the elevation of the nodes of the string.

 

This function is usually applied to strings that have all ready generated new DTM points on their nodes via the string>DTM function and where the elevations of the string were changed.
