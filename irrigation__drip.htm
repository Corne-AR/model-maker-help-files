IRRIGATION : Drip

 Top  Previous  Next

 

Drip lines consist of a selected emitter and selected pipe.

 

1) Add a drip line.

·In the emitter list, select a emitter and nozzle.
·In the pipe list, select a pipe diameter.
·Click the Add Dripline button on the bottom of the dialog box.
·You may then change the drip line's name, default spacing, cost per unit and part number.

 

2) Edit an existing drip line.

·Select the drip line to be changed.
·If the emitter has to change, select the new emitter from the list.
·If the pipe must change, select the pipe from the list.
·After the new pipe or emitter has been selected, click the Edit Dripline button and the new selection will be placed in the drip line.
·You may at any time change a drip line name, spacing, cost or part number by typing directly in the editor.

 

3) Delete a drip line.

·Select the drip line and click the Delete Dripline button.

 

You may use the Import CSV option to populate the drip line database if the csv file is in the correct format. Contact Model Maker Systems to enquire what the exact format is.
