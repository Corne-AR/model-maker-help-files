rAdius                : Change the size of the identified circles by a specified factor.

 4 

Size                : Change the size (height and width) of all the identified text elements based on a user defined factor.

Ratio                : Change the height/width ratio all the identified text

                                 elements to a new defined factor.

 

Constant        : Add a constant numeric value to the text.

                                 The constant can only be added to CAD text elements

                   that can be  translated to a numeric value.

 

                   Example : Add 55 to '500.12' = '555.12'

                                 Add 55 to 'A500.12' = A500.12' (Because the 'A' character is not numeric)        

                   This function is useful to change numeric CAD text values normally

                                 imported from dxf files.
