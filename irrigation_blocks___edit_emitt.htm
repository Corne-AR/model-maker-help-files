MENU : Irr - Blocks - Info and Pipes

 

To edit the emitter and pipe combination, follow the steps below. To change only the emitter or only the pipe is also explained.

The Info section displays the Block type. In this example, it is a Micro/Drip block.

 

 

Edit Emitter and Pipe:

 

Select the Update button.

 

 

 

Click OK

 

 

 

A message box will display the following message:

 

It is now required that the drip/micro lines be defined before filling a block with laterals.

In this dialog box, select a drip line, nozzle and operating pressure.

Drip/micro lines can be defined in either the node or pipe databases.

Or press the MANUAL SELECT button to pick your own emitter.

 

 

Select the emitter and pipe combination from the predefined list, the nozzle data and the operating pressure.

 

 

 

 

Manual Emitter Change:

If you only want to change the emitter, click MANUAL SELECT. Manually change the settings of the emitter in the menu below.

 

 

 

 

Manual Pipe Change:

To only change the pipe, return to the main menu. Select Pipes from the top tabs. And select Edit Type.

 

 

 

Select the pipe to be changed from the menu below.
