MENU : Utility

 

 

This standard communications dialog box is used to open and control the serial port.

The serial port is often used to communicate with a digitizer or to download data from an electronic survey instrument.

 

Before opening a comms port, set the correct parameters in the following boxes.

 

 

Enter the device name, baud rate, data bits, parity and stop bits.

The device name indicates the port number in use. The other parameters normally depend on the device that MM will communicate with and should be obtained from the supplier.

 

 

After the applicable serial communications cable is connected and the correct parameters entered, click on the OPEN button to set the parameters and open the port between the pc and the external device.

Click on the close button if the communication port must be closed.

The Clear button does not do anything about the communications but only clears the 2 display windows from text.

 

 

   

These options control the format of the communications. Normally, the default as displayed does not have to be changed. The experienced user may however change some of these settings when problems are encountered under normal conditions.

 

 

Two windows are displayed for information purposes.

 

Window 1 displays general information about instructions passed between the pc and device.

 

Window 2 displays the return bytes received by the PC from the device when a the device is prompted.

 

Use the Open and Save buttons to save the current settings or load previous saved settings.

Click the OK button when the comms port is successfully opened and proceed with the function that is going to use the comms.
