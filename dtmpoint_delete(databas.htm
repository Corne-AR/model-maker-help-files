MENU : Point - Delete - Database

 

Delete DTM points graphically or directly by editing the data base.

 

This command displays a screen where you can delete points permanently. All the points with a *.* code deleted and removed from the database.

Select OK to confirm.

You will be asked whether triangles must also be updated. Confirm for Model Maker to check and delete all the triangles that are connected to the removed points.

 

Triangles are related to spots in terms of the point numbers they are connected to. If a point in the database is deleted, then all the following points move up one number. This change in the structure of the database will be updated. The updating only applies to triangles currently in memory.
