MENU : Min - mine_Planning - Sections

 

Extract cross and long sections through all the layers loaded in the mining section. The result will be plotted according to set scales to CAD.  

 

Cross sections will be taken through all the models loaded into the mine planning function. The current horizontal alignment as defined in the Horizontal options is used. Sections will also be taken through project 2 and 3 if they have data loaded and their display buttons (right side menu) is set to ON. The sections will be extracted and you will be asked to indicate the position of the sections to be drawn into CAD.
