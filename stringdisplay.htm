MENU : gXt - infO - Display

 

Control the display parameters of the strings.

 

Node detail

Select any of the 2 check boxes to display the information with each node of the string.

 

Orientation

This controls how the node detail is to be displayed.

 

         Horizontal                :The text is always displayed left to right.

         Parallel                        :The text will rotate with the string and stay in line with each segment.

         Perpendicular                :The text will be at a right angle to the segments of the string.

 

Fill

Globally control if strings defined by the info dialog should use their fill selection when plotted.

 

String display

Globally select to allow or suppress the display of the strings with each replot.
