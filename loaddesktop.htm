Load Desktop

 

If it is noted that when Model Maker starts up it brings up unknown data each time, or your last set of data; check that the Load Desktop button is not activated/checked.

 

If checked, MM will automatically load the last saved desktop. The desktop can be saved in two ways:

       by checking 'DESKTOP save on exit?' box on the EXIT menu
       alternatively under File/Save Desktop (top left hand of Main Screen)

 

Use the desktop when you want to quickly save all the data on the disk without having to give file names for every type of data.

These files are generally saved in C:; depending on which drive Model Maker was installed.

         - The files saved in this folder can be opened manually if the Load Desktop function is not checked at the time of opening Model Maker.                

         -   e.g. of file names :        proj1.tot                -        points and triangles (project one)

                                 cad.kcd                -        current CAD

                                 centre.pid        -        horizontal alignment

                                 irri.mir                -        irrigation

                                 string.kcs        -        strings
