MENU : File - Open/Save - Find

 

 

 

This function searches for files and is available on the Files dialog box. Press the Find button to access the find engine.

 

 

Use this input box to specify the filename that you are looking for.

Enter the file name in the big box (eg. ZX) and the file extension in the small box(eg. TOT).

Check the ANY box to search all files regardless of the file extension.

If you want to locate all the files that, for example, start with the letters ABC, enter the file name as ABC*. T

 

 

The search will start in the specified directory and continue in all the sub-directories below it.

 

To scan the entire hard drive, select the root directory folder .

The results will be placed in the list box on the left.

There are 2 ways to exit the file find dialog box.

1) Click the CANCEL button, which will close the menu and return you to the regular file dialog box

2) Double click a sub-directory name in the result list box. The program will automatically change the sub-directory system to the double clicked name and return to the regular file dialog box. For example, double clicking on the sub-directory C:line in the above example will return you to the file dialog box with C:as the active sub-directory.
