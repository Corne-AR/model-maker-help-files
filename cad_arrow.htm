Arrows

Arrows are used to point to a defined position. The arrows can have different colours and the arrow head is user definable.

 

Model Maker CAD allows you to define arrows using any of the cursor snap modes.

An arrow consists of at least one line and an arrow head or up to 10 nodes and a head.

All the cursor snap modes may be used when adding or editing arrows.
