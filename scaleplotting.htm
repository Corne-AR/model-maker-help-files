MENU : Scale - Plotting

 

Model Maker makes uses user defined plotting scale when producing drawings and plans. The scale value may be changed in the Plot parameter menu and in the General paper window.

 

The purpose of the scale value is to allow you to work with real co-ordinates. Work with the data in terms of the correct physical Y&X co-ordinates and lengths. The scale is applied to the system when plans are produced to a plotter.

 

Therefore, while working with the data on the graphic screen, you are not bounded by any size. Zoom into any area or do a NORMAL plot which would display the whole of the project on the screen irrelevant of the scale.

 

The plot scale is considered on the following occasions:

 

1) Using the Magnify to scale function.

2) CAD text elements are sized in millimeters and displayed along with the rest of the drawing by relating the size in millimeters to that of the plotting scale. i.e. 1mm occupies 1m in distance at a 1/1000 scale but 2m in distance at a 1/2000 scale.

3) The window frames in the Gen - Paper menu are displayed according to the size of the sheet in relation to the scale.
