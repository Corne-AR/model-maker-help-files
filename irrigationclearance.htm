IRRIGATION : Pivot Clearance

 Top  Previous  Next

 

Use the Clearance function on the Irrigation menu to check pivot span heights over the terrain.

 

After the number of pivot spans have been defined, the user has the option of placing the pivot in position on the survey map.

 

 

 

 

 

 

 

Image showing cad circles indicating the tower tracks as defined by the pivot span lengths.

 

 

To access the clearance function, click the Clearance after the Pivot has been placed (by using the TEST function).

 

The clearance check dialog box requires the following values.

 

1) Tower heights. This is the height of the span where it is attached to the towers.

 

2) Clearance values. A color map will be produced depicting the areas where the height above ground, along any position on the spans are within the set values.

 

3) Check the Copy to Cad option if the color map must be retained as a bitmap image on the drawing.

 

 

This picture shows a typical bitmap image of the clearance values.

 

Light blue     : spans are more than 3m above the ground.

Light green   :  between 2 and 3 meters.

Red              : less than 0.5 meter.
