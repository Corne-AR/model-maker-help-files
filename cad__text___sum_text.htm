MENU : Cad - Text - sUm text

 

 1 

Indicate the text

After selecting the sUm text option you should select the text that you want to be summed. The text should contain numerical values that will be added.

 2 

Text indicated

After you selected the text to be summed, press escape (right click) and a window will appear showing the numerical values of the text accumulated.
