MENU : Cad - Line - moVe                                

 

Move any existing CAD line.

 

After indicating the CAD line the cursor will jump to the closest endpoint of the line. The line can then be moved around and placed in any position.

 

Example:

While moving to the line the following functions can be used.

 

1)  The (CTRL) keys changes the cursor mode.

2)   The Page up and Page down keys rotate the lines.

3)   The Home key makes the line horizontal.

4)   The Ctrl Home key changes the line length and angle.

5)   The Shift Tab key changes angle to that of the closest CAD line.  

 

Also see the list of shortcut keys.

 

The original pen parameters of the line is not affected by this action.
