MENU : Tri - DeT

 

Generate CAD lines or strings using DTM points with selected codes or tags.

 

 

Action Type

 

1) By characters

Draw detail lines between DTM points that use the < > and @ characters.

Use this function to automatically help in preparing a detailed contour plan based on intelligent information associated with the codes of the surveyed points.

 

Note:

The survey points with these codes must be sequential in the file. If the code of a survey point starts with >, e.g. >A or >01, Model Maker will join all the following A or 01 codes with a detail line. To end the detail line, start the code with a <. If the detail line needs to stop and be joined with the first survey point on the detail line, the code of the point to be joined with the first is started with a @.  The following is an example of detail codes:  01 >02  02  02 <02  03 >02  02 @02 >03 <03

 

2) By sequential codes

Draw string or CAD lines between DTM points that have the same codes and were surveyed in sequential order.

The DTM database is scanned from the top. The line will start at the first occurrence of the code and will join to the points following it. Points without the selected code will be skipped.

 

3) All points in sequence

All the points will be joined no matter what the codes are. Model Maker will start with the first point and end with the last.

 

4) Selected points

This option provides the most controlled way of generating detail lines.

Use the SCAN button to generate all the codes/tags that were used in the current project. From this list, select either ALL the codes or, with the mouse and CTRL & SHIFT keys define, the desired ones.

On the left, use the DEFINE CODE/TAG input box to select a range of points. The full code of the points may be entered or a range can be selected by using the * wild character key.  For example, T***** will select all the codes Tree, Top, Tennis etc. Double click the input box to activate the selection.

 

5) Incremental codes/tags

This function allows you to add lines between points with codes like A1, A2, A3 etc.

 

6) DTM - along indicated line

Add lines between DTM points with the same code that lies in an approximate straight line. Lines will be added to points within a set distance from a given temporary center line. Press Esc to skip a point.

7) Auto - within set angle

Add lines between DTM points with the same code that lies in an approximate straight line and where the change in direction between lines connecting the points is less than a given value.  

 

 

The light green lines are not actually drawn but are only indicated for this example.

In the example, point 1 to 8 have the same code. Join the first line from point 1 to 2 and enter the search angle (A).

Model Maker will now look for the first point with the correct code falling within the view angle (A).

This is point # 3. The line through 2 and 3 is now taken as the base and, with angle (A), the search continues for point 4.

Note that although point 3A has the same code as points 1 - 8, the search angle, based on the line through 2 to 3, prevents it from being used in the process because it is to wide.

Confirm each new point.

 

The yellow toolbar at the bottom of the page will give you information regarding the points that are being connected. You may also click the yellow toolbar if you want the program to generate the new line without confirmation each time.

This is handy if you have a long line and don't need to interrupt the sequence.

 

Most of the work involved in preparing a detailed contour map goes into joining road edges, fences, telephone lines etc. with CAD lines. The functions on this dialog assist in automating the initial connection of lines that represent above mentioned detail or break lines.

More than 50% of the CAD work can be done with this function if the field work is done with care in terms of the position and sequence of where the DTM spots were taken.

 

Output Results

 

This box regulates the output destination. The CAD lines and strings option will only be available if the specific program modules have been purchased. Use the display only option if you first want to inspect the results before converting it to elements.

Use the maximum leg length to skip the connection between points far apart and not relevant to the project.
