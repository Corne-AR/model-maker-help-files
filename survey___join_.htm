MENU : sUr - Join

 

 

Calculate a join (the distance, direction, slope and height difference) between two indicated points on the screen.  

Use any cursor mode  to specify the points.

 

General

The attributes of the 2 selected points are displayed in the info box.

The distance, slope and height difference between the 2 points are displayed below the info box.

 

Editor

 

Apart from having the results displayed in the dialog box, use the editor functions to copy the data to the editor for further processing.

 

Editor

: Transfer control the editor dialog box.

 

Header

: Produces a standard header line in the editor. Normally, you would output a header line to the editor before   a sequence of joins are calculated.

 

Info

: The actual join information is outputted to the editor.

 

                         EXAMPLE:

 

                            -------- JOIN CALCULATION ---------

         No Tag    Code     Y                X                    Z           Direction        Dist.

         From         39 tc     0        -5470.394   -4464.107   38.473        251.51.15

         To              40 tc     0        -5630.314   -4516.518   39.314        168.290

                                 

 

Manipulating the join

 

 

The update button  opens up a powerful addition to the join function. These functions allow for the manipulation of the elevation and position of the second point based on it's relation with the first join point.

 

 

The dialog box has 3 entries that all work interactively. The logic of this dialog box takes into account that 2 points are relative to each other based on the elevation difference, horizontal distance and slope.

 

 

If one of the 3 parameters is changed and a second is kept constant, then the change to the third parameter can be calculated.

 

Navigating the dialog box

 

 

The three items (Elevation, Slope%, Horizontal Distance) in the dialog box, each has a radio button next to it. The item with the checked radio button is the fixed parameter. The other two parameters can be changed and Model Maker will display the updated value in the third text box.

 

EXAMPLE : Assuming that point1 is at elevation 40, then point2 (at elevation 44) will be 40 meters away for it to have a 10% slope. If the elevation of point2 is raised to 48 meters and the slope maintained at 10%, the new distance would be 80 meters.

 

 

 

Use the Update points  button to apply the changes to the elevation and position of point2.

Model Maker will calculate a polar from point1 using the join direction, the new horizontal distance and place point2 with the new elevation. The triangles and contours will also be updated.

 

Apply update to multiple points

This option will, after applying the changes to point2 of the join, also allow you to indicate other points to which the same relative changes must be applied.

 

Automatically update GXT

When checked, any strings have nodes on the same coordinates as point2, will also be updated.

 

Show this page next time

When checked, the join dialog box will each time be displayed in the wide format allowing quick access to the manipulating functions.

Note:

Take care when moving points around in this way. If triangles are connected to point2, they can be forced into an undesired position.

Note:

The expanded join functions are only available if the original join was calculated between 2 DTM points.

 

Use the manipulating functions when designing surfaces based on certain slopes between the DTM points.
