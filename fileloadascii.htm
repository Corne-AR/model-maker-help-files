MENU : File - Open - YXZ

 

 

Load Normal ASCII Data. Model Maker takes data in an ASCII file as normal if the fields of data are in the sequence tag code y, x, z and the file extension is .YXZ (the line is space delimited). The first line in the file must have a project name.
