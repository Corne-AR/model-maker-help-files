MENU : Hor - Editg

 

Model Maker uses a standard dialog box when editing horizontal alignment nodes. In this dialog all the parameters of the PI are displayed and may be changed.

 

The first 2 input boxes relate to the alignment as a whole and defines the name and starting chainage.

 

Use this part of the dialog box to edit the parameters of the indicated PI. The Y&X co-ordinates are defined by the cursor when the node is initially defined. Edit any of the input boxes. The radius and the transition lengths in and out define the length of the curve at the PI.

 

The UPDATE button will be activated when changes are made to the data. Click the button to update the calculated results section described below.

 

This panel displays the relevant calculated results based on the PI variables defined above. A more detailed display of this results are available in the text editor.
