Menu : Min - mine_Planning - Load old format

 

The mine planning functions in Model Maker operate on points and triangles in the defined dtm projects.

 

 

 

 

A) Older versions of Model Maker used a triangle stack system to load surfaces to be used by the mine planning function.

This is what the grid editor looked like to manage the triangle stack.

 

 

 

B) To migrate from the old stack method to the current dtm project system, an old stack Master file can be loaded. The active project must be set to No. 1.

 

 

 

C) Once the Master file has been loaded, use the Copy to Projects function to unpack the surfaces from the stack into regular dtm projects.

 

 

 

 

 

Use the triangle stack editor to view and edit the loaded data headers.

: Use this field to describe the layer. Model Maker expects the original (NGL) surface to be loaded into layer 1. The surfaces below that will follow in layers 2 to 15.

 

               

 

 

The data in the triangle stack is used for all the options available in the mine planning.
