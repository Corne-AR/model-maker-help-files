MENU : Cad - Patt - Auto_pat

 

Place specified patterns on selected DTM points. This allows you to produce a detail drawing of the survey work without snapping to each point that identifies a specific feature.

 

 1 

The entry table

Enter the relevant information in this table.

 

 2 

Code                

The code column holds the description of the DTM point on which the pattern must be placed. Enter the codes manually select from the code list below.

 

 3 

Pattern        

Enter the pattern name in the second column. This pattern must be pre defined.

 

 4 

Scale                

The scale indicates the relative size of the pattern.

For example:

 

A typical survey may consist of points denoting several trees of different sizes. You do not need to create patterns for each tree size since patterns may have different scale values. Therefore, create one pattern called tree2 of a set tree diameter. The other tree codes will then be marked by the same pattern but with different scale sizes.

 

 5 

Use                

Enter Yes or No in the USE column. Yes means that if a code is specified and used in the current project and the pattern exists, the pattern will be placed on  the DTM point. No will ignore the item.

 

 6 

The code list box

This list box will initially be empty. Use the button to fill the box with all the codes used in the current project.

To transfer a code from the list box to the entry table, select the code in the list box and keep the mouse button down. Drag the item into the entry table. Once over the correct cell, release the mouse button to drop the item in place.

 

 7 

Available patterns

The list of all the patterns on disk is displayed. Using the drag and drop method, you can take elements from this list box and place them in the patterns column of the entry table.

 

 8 

 

This button clears the entry table of all the input.

 

 9 

 

The button below the scale column resets all input values in the column to 1.

 

 10 

 

The button below the USE column toggles the column to Y or N. This allows you to globally activate or deactivate all the lines.

 

 

 

An option is also available so patterns can intelligently display dtm attributes when placed ontop of dtm points.

In the pattern, place text on layers ?DTMZ? and the pattern text will display the elevation of the closest (<0.5m) dtm point in project 1.

Place text on layer ?DTMC? to display the dtm code.
