MENU : Tri - Par

 

 

To create the triangle model for the survey data, the searching for points must be done according to specified parameters.

 

If no triangles are found, or if there are specific areas without triangles after a search, then increase the search radius and decrease the search angle.

 

CAD ID Line Search

Used to create the break lines on a model, as defined by the CAD drawing lines. Use the CAD Lines option to draw the break lines between survey points, and the CAD Id option to identify these lines. If this option is set to ON then MODEL MAKER will not create a triangle leg which crosses a line identified by CAD. Only implemented in the normal search option.

 

Normal search (Allow intelligent Search)

This method of triangle search does take all the parameters as set in the search parameters section in to account while searching. The searching can be interrupted. The normal search is automatically used for any continue search operation.

 

 

TRIANGLE:

The option to restrict the search by area are separated from the Code and ID restrictions. You can specify that only points with a certain code WITHIN an area must generate triangles.

  Search All

  All - includes all active survey points in the triangle search.

  Search Not

  Exempts points with specified codes from the search.

  Search Only

  Only searches on points with codes specified.

  Only ID

  Only searches on points identified using the Point identification options.

  User Defined Area

  Only searches on points within a user defined area.

  Screen Area

  Only searches on points visible on screen
