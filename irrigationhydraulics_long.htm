New option in Design and Calculate, Info, LongSection.

This function will produce a ground longsection + the hydraulic gradient for the calculated pipe sgements.

 

This is the width, left and right of the pipe line within which Model Maker will pick up survey points to produce the ground long section.

 

This dialog box allows you to set the scale and plotting limits for the long section.
