MENU : File - Open (F3) / Save (F2)

 

 

How can this topic help me ?

 

Model Maker keeps the system data in memory while in operation. This data can be entered in various ways, but must be stored before leaving the program or starting work with another set of data. Transfer this data using the FILES menu. Previously saved data can be loaded back into the program. Current data can be saved to disk.

 

 

The Open file and Save file dialogue are the same except for the action buttons which will depend on whether the file is being saved or opened.

 

The Open file will open existing data.

 

To open a new file select the following.

1.                Select File in the main menu.

2.                From the drop down menu, select Open.

3.                Once the Open has been selected, the dialogue below will appear on the screen.

 

Open All

 

 

 

Please note that the YXZ (Ascii) option has moved to the Utility - Ascii Files - Load standard YXZ data

 

The Total file format has been replaced by the binary BOT format.

 

File Name

Displays the file name to be loaded.  The file name can be entered in the following ways.

1.)                Type the full file name in.

         Eg A: platngl.kcd.

2.)        Select a name from the file list

3.)                Use the history button.

 

Path Button

If the path box is checked it will display the full file name.

Eg C.kcd

 

 

File Name - History Button

This option will display the last twenty files saved.

 

Project

 

There are twenty one projects. This enables you to load data into more than one project. To load into another project click on the project number eg. 2.

Depending on the version of Model Maker you have, you will be able to load 9000, 32 500, or an unlimited number of points into each project.

 

File Open Selection

 

 

An open folder will be displayed next to each extension as long as the file name displayed in the file name box exists. The open folder picture will be updated automatically whenever the file name is changed.

 

The gray buttons represent the various file extensions. Only one extension at a time may be visible. In the above example only files with a KCD extensions will be displayed in the file list.

 

Select a combination of file types by checking the relevant boxes.

 

The older TOT file has been moved to the File - Import section

 

 

 

File Folder/Directory

Select the folder that you are looking for in the folder directory. Double click on the folder to activate it. This will display the files with the active extensions.

 

File Folder - History Button

This option will display the last eight folders used.

 

Path Names Button

This button allows you to view the long path names.  If the path name is too long for the box, you can press 'End' to view the last part of the name and then 'Home' to return to the beginning of the name.

 

 

Drives

Data cannot be entered into the drive box, Click on the drop down arrow to view the available drives and select the drive.

 

 

 

File List

The file list box displays the sub-directories with the selected file extensions.  To list the sub-directories select the folder in the folder directory For example in the file selection box.  the KCD button is selected.

Therefore all the KCD files will be listed for the that particular folder.  Selecting one of these will automatically update the file name box.

 

To list the sub-directories select the folder in the folder directory and double click. The sub-directories will appear with selected file extensions.

In this example the KCD files are listed as in the file selection box and the KCD extension has been selected.  Therefore all the KCD files for that particular folder will be listed.

To select the file you want click once and this will automatically update the file name box.

 

 

 

Always Ask To Replace

We recommend that this box is always checked. If this box is unchecked. Model Maker will automatically overwrite any existing DTM, CAD or Strings data in memory. this is also a useful way of combining data. If the box is checked a warning screen will appear.

 

Add File To Data

Check the box to automatically combine data. If the box is unchecked the existing data will be lost.

 

Open File

The file of the selected format will be loaded.

 

Once the file has been selected and opened, the drawing will appear in the main screen.
