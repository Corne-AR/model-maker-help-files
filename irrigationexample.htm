IRRIGATION : Example

Background

 

Model Maker's irrigation module  can operate independently from any other module. However, the DTM and CAD modules can greatly assist in the overall design and maintenance of the irrigation project.

 

The irrigation project consists of blocks, nodes and pipes.

 

More about pumps.

 

Standard nodes are user definable and entered in the node data base.

Standard pipes are also user definable and entered in the pipe data base.

 

 

Blocks

Blocks are defined to specify certain areas and conveniently group the nodes and pipes together.

 

Nodes

Nodes can be placed at any position and in any order when building the network. Before a node is added to the project,you will be given the opportunity to select the specific node type from the available list as defined in the database.

Nodes can be edited, deleted and moved.

Nodes can also be raised to the correct elevation by automatically interpolating the elevation from the triangle surface of the dtm project. When a node is moved, all the pipes connected to it will also be updated.

Nodes can be displayed by single points, the defined pattern, or the circle with defined spray radius in the case of a sprinkler. The node numbers can also be plotted along each node.

 

Pipes

Pipes are placed between any 2 nodes. The type of new pipe is first defined by selecting a standard pipe from the available list as defined in the data base.

Pipes can be edited, deleted and removed from one node and connected to another.

The calculated flow and velocity in each pipe can be labeled along the pipe.

 

Note:

If the block is defined for Micro/drip all the lateral pipes will additionally have the option to specify the CES. (Constant emitter spacing)

 

Certain global functions can be performed. They include editing of one node type to another, changing all the pipe types in a defined area, or copying an entire project onto a new position.

 

 

STARTING A NEW PROJECT

 

Generate the DTM

If you have survey data of the terrain and you want to use these in the irrigation design, use the DTM functions to enter the survey points and to create the triangle surface.

Follow the steps as described on the DTM definition page to get the survey in place.

 

If the irrigation design is started without the DTM in place all the nodes will be on the zero elevation.

 

Generate the CAD

Although the irrigation design can be done without any CAD data, your ability to design and plot the results will greatly be enhanced by utilizing CAD.

 

You may either load a previous Model Maker drawing, import a DXF file or use CAD functions to draw the terrain features.

 

Irrigation design

Activate the irrigation menu by clicking on the Irr button on the left menu.

 

1) Select the appropriate units on the project page.

2) Ensure that the nodes and pipes types are defined in the node and pipe data bases.

3) On the block page, use the functions to add a block and to fill the block with laterals. Select between either a sprinkler or micro/drip layout.

4) Copy an existing block to a new position or define all the necessary blocks.

5) Use the nodes and pipes functions to link up all the laterals with manifold pipes to the source valve of each block.

6) Use the Hydraulics option on the Global page to apply water to the system. Also use the pipe sizing functions if necessary.

7) Optionally inspect the nodes and pipes on the Info page to see that the pressures and flows are within the design standards. Make      adjustments if needed and repeat the hydraulics function.

8) Activate block 0, the main line and add the pump node. Use the pipe options to add main line pipes from the pump to all the block valves.

9) Use the Shifts page and assign the blocks to shifts.

10) Use the Hydraulics option to calculate and inspect the main line.

12) Get a report of the bill of materials.

13) Save the irrigation data to disk.

14) Plot the irrigation layout along with the other CAD data to a printer.
