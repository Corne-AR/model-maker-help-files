MENU : Cad - Id - do_Id - Global                                

 

Area

Identify all the elements in an area.

Layer

Display all the available layers in the project. Select the layer you want to identify by name. All the elements (lines, text, circles/arcs, arrows and hatch ) on that layer will be identified.

Pen

Display the pen grid colour dialog box. Select the colour you want to identify. The default selected colour in the grid colour dialog box will be the current active pen colour set with the pen parameters.

Line type

A dialog box with all the available line types will be displayed. Select the line type that must identified. All the lines, circles/arcs, arrows and hatches with the specified line type will be identified.

 

Font

All the available fonts will be displayed. Select the font to be identified. All the text elements of that font will be identified.

Duplicate

The CAD database will be scanned and all the duplicate elements will be identified.

 

The following factors will decide if an element is duplicated.

 

Lines                        : The terminal co-ordinates of the 2 lines are equal.

Text                                : The text base co-ordinate, the text itself and the origins are the same.

Circles/Arcs                : The centre co-ordinate and the radius are the same.

Arrows                        : The co-ordinates of the first and last nodes and the number of nodes in both arrows are                                                    the same.

Dimensions                : Both terminal co-ordinates and the dimension string are the same.

Hatch                        : The co-ordinates of all the nodes are the same.

By line

 

Identify all CAD lines that intersect the temporary user defined line. This method only works on CAD lines and has no influence on other elements.

Polyline

 

Identify the lines along a polyline.
