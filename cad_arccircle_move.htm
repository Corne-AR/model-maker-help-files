MENU : Cad - Circ - Move / Delete / cOpy                                

 

Move

 

Move an indicated element to a new position. When initially selecting the arc or circle, the cursor will snap to the centre point of the circle. This is then the base point that is used to position the circle again.

 

How to select the functions to move an arc/circle.

 

1.                Select the Cad button in the left hand side menu.

2.                Choose the Circ button in the bottom menu.

3.                From the sub menu select the Move option.

 

Delete

 

Use this function to delete an arc/circle by pointing to it. When indicating the arc/circle, either be close to the centre point of the element or close to the circle itself.

If you accidentally delete the wrong element, use the '+' key to undo the delete.

 

How to select the Delete function.

 

1.                Select the Cad button in the left hand side menu.

2.                Choose the Circ button in the bottom menu.

3.                From the sub menu that appears select the Delete button.

 

cOpy

 

Copy an indicated element to a new position. The circle or arc is always referenced by the centre point of the circle. The original element will not change and a new copy will be generated at the new co-ordinate point.

 

 

How to select the function to copy a circle.

 

1.                Select the Cad button in the left hand side menu.

2.                Choose the Circ button in the bottom menu.

3.                From the sub menu select the cOpy button.
