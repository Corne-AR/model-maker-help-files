MENU : Min - mine_Planning - strip Width

 

This function uses the mining database  to calculate a strip width that will yield a user-defined volume when excavated.

Specify the top of the two layers between which the volume will be calculated. The calculation will start from an indicated CAD line and proceed on the user-defined grid interval until the desired volume is reached. The volumes of the other layers within the same area will be reported. You may select to confine the calculations to a user defined area.

 

EXPLANATION OF MENU.

The description of the loaded mine layers are displayed. Click on the one that will form the top of the 2 layers to be used for the volume calculations. In the example, '2a2' is clicked, meaning that the volume will be calculated between layer 2a2 and 3a3. The volume for layers 1a1 and 2a2 will be calculated and included in the final report. It would be irrational to click 3a3 in this example, since the controlling layer as 3a3 is the lowest layer and has nothing below it to compare against.

 

This grid interval regulates the accuracy of the calculations. Depending on the speed of your machine, a value of 2 would be acceptable. A to small value could take too long to calculate, while to large value would produce inaccurate results.

 

The calculation process will be repeated until the required volume or the end of the pit is reached. The volume required will be compared with the volume calculated between the selected top layer and the one below it.

 

Check this box to confine the calculations to a user-defined area.

 

EXAMPLE:

An initial base CAD line must be defined.

 

An area is also indicated. Point to the base line on the side that the strip must be calculated. On the regular grid interval, Model Maker will calculate elevations on all the layers and constantly check the volume. The process stops when the desired volume is achieved within the new strip width. Note that the red crosses indicate the grid positions. Ensure that you are satisfied with the area that is covered.

A new line at width can be added to the graphics. Use this line as the new base line for the next strip width.

A summary of the volumes will be presented in the editor.

The width and volumes between the layers are displayed.

 

EXAMPLE OF SUMMARY

Width       a1       a2    a3

   2.0     340    408

   4.0     680    816

   6.0  1000  1200

   8.0  1320  1584

 10.0  1640  1968

 12.0  1940  2328

 14.0  2240  2688

 16.0  2520  3024

 

The above shows that at a width of 16m a volume of 3024m3 was calculated for the layers a2 to a3.

The first column is the width at regular intervals. The next are the volumes for layers a1 to a2 and a2 to a3.

Note that the column a3 is empty because only 2 sets of volumes can be calculated between 3 layers.

 

The purpose of the strip width definition is to determine the layout of the mine in order to achieve desired volumes within predetermined widths.
