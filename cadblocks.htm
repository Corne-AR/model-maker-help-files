MENU : Cad - Blocks

 

Blocks provide a means to group elements together so they can easily be manipulated, edited, identified and moved.

 

Blocks are like cad layers but with some differences.

 

·Each cad element is associated with a specific layer while the use of blocks are optional. This means you can assign elements to blocks when you want to, it is not required that any or all elements belong to a block.
·Elements from different layers can be assigned to a block.
·Blocks have specific text editing capabilities. Text assigned to a block can easily be managed in an Xcel grid style.
·All the elements in a block can be moved without having to identify them.
·The elements assigned to a block can be identified with a single function.

 

Assigning elements to a block.

 

Unlike the layer attribute which is assigned to any element when the element is created, elements are optionally assigned to blocks at any stage after the element has been defined.

·Initially any Cad element is not assigned to any block.
·After a block name has been defined, elements can be assigned to the block by:

 Defining an area.

 Assigning all the identified elements to the block.

  Block Editor  

 

 

This example shows the Cad elements and the associated display of the text in the grid editor.

 

Text in the (D) grid editor are grouped according to their origin coordinates on the graphics screen.

 

Text with the same column coordinate will be listed in the same column. Text on the same row coordinate are shown on the same row.

 

The values in the first column (E) shows the spacing between the rows in millimeters.

 

The text (connected with red lines) is an example of elements that do not line up correctly.  Functions, to line up any of the elements, will be discussed in the next section.

 

Note : A tolerance of 1 millimeter is used when text are assigned to a row and column. This means that text in the same column may have a different coordinate of up to 1 millimeter.

The Align Columns and Align Rows options are used to force text in the same column and same row to have exactly the same relative coordinates.

 

A) Changes to the actual text can be made by typing in the grid editor. To edit cells (text) the Edit option must be selected.

 To select a range of text elements (cells) the select range option must be checked.

Note : You cannot add text by typing in a blank cell. Only existing text can be edited. To add new text elements, use the regular cad text Add options.

 

B) This input box is used to specify a new (millimeter) spacing between selected rows. Enter a value and press Enter. The row spacing will be updated and all the rows (below) the selected one will be adjusted accordingly.

 

C) The 4 directional buttons are used to change the position (on the graphics) of the selected text.

 The keyboard arrow keys can also be used to move text around. To activate the arrows keys, click in the yellow box. The box will turn red, you may then use the arrows keys on the keyboard to move the text the selected text.

 

1) Allows you to edit any text on the graphics screen.

2) Does a refresh of the graphics screen.

3) Undo the last changes you made.

4) Align the selected text exactly to the column coordinate. (Initially, text in the same column may have up to a 1mm difference)

5) Align the selected text exactly to the row coordinate. (Initially, text in the same column may have up to a 1mm difference)

6) After text have been moved around, the elements may line up in different columns and rows. This function re-assigns the text and grid.

 

An additional set of functions are available when the right mouse button is pressed.

1) Select all the text in the grid.

2) Select all the text in the selected column.

3) The graphics screen will be zoomed so all the selected text will be shown on the right half of the graphics screen.

Hint: Move the Block dialog box to the left side of the screen so the zoomed text (to the right) is visible.

 

4) The regular text attribute dialog box will be shown. The attributes (pen, origin, font size) of the selected text will be updated by any changes made.

5) The selected text will be disassociated from the block.

6) All the Not selected text, although still part of the block, will not be shown in the grid. This allows you to make spacing changes to the grid text without effecting the text that is temporarily not shown.

7) The selected text can be moved left and right, up and down by using the keyboard arrow keys.

8) Use this function to apply a defined row spacing (millimeters) between all the selected rows.

9) All the selected text will assume the column coordinate of the column immediately to the right (of the selected column). Text can only assume the position to the left, if the cell to the left is empty.

10) The same as (9) but the text move to the right.

Notes :

When text is loaded into the grid editor, the first text entry defines the text direction Model Maker assumes the block to be. If the text is at an angle, then Model Maker will recognize the text direction and all the functions (moving, spacing text) will be done at the text direction angle.

 

To move all the elements in the block (not just text) click the top BLOCK tab. There you will find functions to ID the elements and also to move all of them without identifying them.
