As shape file is a file database system used mostly to define and display GIS information.

 

Model Maker can read a shape file, display the records for the database elements in the dialog (below) and plot the figures along with all the other Model Maker elements on the plotting screen.

 

 

A shape file normally consist of a 3 files with extensions : SHP, DBF and SHX.

 

The SHP and SHX files hold the actual coordinates of the shape elements. The DBF file is directly associated with the SHP file and holds any other user defined information fields that pertains to the shape element.

 

When Model Maker reads a shape file, the contents of the DBF data is displayed in the Lines (when it is normal 2 point lines) or Polygons edit boxes.
