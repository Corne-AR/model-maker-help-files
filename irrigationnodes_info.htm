MENU : Irr - Nodes

 

 

This dialog box is displayed when a single node is edited.

 

 1 

Info

All the relevant attributes of the selected node is displayed and cannot directly be changed. If any of the values in this panel must be updated, use the button to access the node data base.

 

 2 

Design / Actual

Two columns of performance data is displayed. The first column holds the design data and will initially relate to the values as defined in the node data base.

The second column displays the actual calculated values. The pressure, flow and radius will always refer to the values as calculated when the last calculation on the Results page has been performed.

 

If the node type is a valve or sprinkler you will be allowed to edit the design data manually. It is good practice to recalculate the design after changes to the design have been made.

If the node is the source valve of a block, the design flow and pressure values will be the values as calculated for the block. Each time the block is recalculated, the values will be updated.

 

The source valves of the blocks (when used in the main line) cannot be edited while block 0 (the main line) is active. Updating the source valves of the blocks, when necessary, should be done by activating each block and calculating the new values. Whenever a block is recalculated, the new values will be passed on to the main line nodes.

 

Sometimes it is very difficult to determine the position of the node in question. The Flash option will remove the Info dialog and continuously flash the node. Press any key to return to the Info box.
