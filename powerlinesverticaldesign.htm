MENU : gXt - Vert

 

This menu is displayed when viewing the string in the vertical mode

 

 

Plant poles

The elevation of an indicated pole will change to an elevation calculated as follows. The ground elevation is calculated. The bottom attachment height for the type of pole and type of line is read from the database and added to the ground elevation. The string node is placed at this elevation.  

 

Pole info

Design information of the indicated poles are displayed. Use the functions to change the pole type or pole parameters use the tower loading option for a report of all the poles .

 

Low points

The whole line will be checked. The lowest point per span will be indicated on the screen. The actual difference between the ground and the parabolic curve is indicated. No constants are added or subtracted from the calculated value. Use the dump graphics option on the Vert menu to produce a graphical picture on a printer.

In the DEFAULT menu you may set the low points to be displayed automatically.

 

Load report

Produce standard report of all the tower loadings to either a text file or a printer. If the screen is selected as the ----output device, Model Maker will check every pole against the maximum permissible values and report any pole that is overloaded.  

 

Add new

Add multiple poles to the line. This function operates on an existing segment (span). You have the option to divide the span into smaller segments of equal length or a fixed number of segments. Select the DO button of the selected type for Model Maker to perform the specific addition.  

 

Service 1

If the string is displayed with other strings defining crossing detail a report of all the intersections between the main line and the crossing features will be outputted to the editor.

The chainage at the crossing point on the main line is displayed

The hot & cold elevation on the main line is also displayed

The service string name and the elevation is displayed.

The last column is the height difference between the crossing service and the main line.

 

Service 2

The crossing services will be inspected and the service name and depth either above or below the main line will be -displayed.
