MENU : Tri - Add

 

Triangles can only be formed if 3 or more points exist in the active project.

The user can create triangles manually or automatically.

 

To create triangles manually.

 

1

Select the Tri button in the left hand side menu.

2

Select the Add button in the bottom menu.

3

If there are no existing triangles, a dialogue will appear asking for the three points to be indicated. Indicate all 3 points that will form the first triangle.

4

If triangles do exist a new triangle may only be added to an existing triangle with an open side. Read the dialogue which will appear as a guideline. An open leg is the side of a triangle where there is no neighboring triangle, or where it is not common to any other triangle.

5

Point inside an existing triangle that has an open side, as shown in the first example. Place the cursor within the area of the triangle, ensuring that the cursor is in a position closer to the open leg of the triangle than to any of the other 2 legs.

6

The open side of the existing triangle is connected to point 1 and 2 shown in the example. These two points automatically form the first 2 points of the new triangle.

7

Indicate the third point.

8

A new triangle cannot be created that crosses the lines of other triangles.

 

                                         

 

Note

It is not required anymore that you must click INSIDE the base triangle when adding a new triangle. Clicking outside the base triangle but still close to the base leg will also work.

In the above pictures, clicking outside the base triangle (near the K or ! of the "CLICK!" will suffice)
