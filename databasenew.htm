MENU : dB - data base-Gen

 

 

This menu clears the computer prior to accepting a new project. A window will be displayed where you can enter information on the new survey project. Enter the name and the number of stations in the project data. The range of stations is between 1 to 100. After you enter the number of stations, you will be transferred to the station data editor. If data is known for only one station, then specify only one. You may add other stations later using the DATA option .

 

Station Data

 

Enter the following data for each station:       *    Station name.      *    Y co-ordinate.      *    X co-ordinate.      *    Elevation.      *    Instrument height.      *    Correction angle (DD.MSS) - entered as Deg Min Sec                    e.g.  181 Deg 45 Min 05 Sec - 181.4505      *    Spots.          

 

Station Name

 

The station name may consist of any six alphanumeric characters. The name is also used as the code for the point when plotted. If the station occurs as an observation point, then put a period (.) in front of the name, e.g., Station A must be entered as (.A). This is because Model Maker skips a point with a full stop in front of the name in the triangulation.

 

Define a dummy station by entering the name as "KOORD". Only spots consisting of final y, x and z co-ordinates can be entered into a "KOORD" station.

 

Y & X Co-ordinates

 

The y and x values may be on a local, or a national LO system. When co-ordinates for two or more stations are entered, make sure they are calculated in relation to each other.

 

Elevation

 

Enter the ground elevation of the station.

 

Instrument Height

 

Enter the measured height of the instrument at the station.

 

Correction Angle

 

Enter a correction angle which is added to the horizontal angles of all the spots to swing the station into orientation.

 

After entering the data for your new station(s), you will be taken to the Station Configuration Screen.
