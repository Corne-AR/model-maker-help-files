EXAMPLE : Working Example 4 (3D model)

 Top  Previous  Next

 

MENU : 3D

 

Object

This example creates a three dimensional wire frame view of your model. Your data and triangles must be loaded for the model.

 

 

Procedure

 

STEP 1 : Create grid DTM

 

Select the 3D option from the left menu.

From the bottom menu, select the gridDtm option to extract points on a regular grid necessary for the 3D viewing.

Enter the grid interval and select the binary format.

Enter the file name and press "save file". Model Maker then extracts sections at the specified intervals and generates the DTM file to be used in STEP 2.

 

STEP 2 : View DTM

To plot the 3D on the screen, select VIEW  from the bottom menu. Load the file, set the attributes and view or copy the image to CAD.
