Main Screen:

 

This is Model Maker's main working screen. The example below reflects the main screen in a particular setup phase. The options displayed here will differ depending on what function is active at the time. This screen displays a great deal of information.

 

 

 

Version & Dongle Number:

 

This tells you what VERSION (4.00) of Model Maker you have and what your DONGLE NUMBER (98:286) is.  If it reflects DEMO where the Dongle Number should be, this means that Model Maker is running in Demo mode.  You can go to Settings - Configuration (main menu) to re-set the program to full access.

 

Angle & Co-ordinate System:

 

360 degrees - Southern Hemisphere

 

Cursor Co-ords Info:

 

               

This box, in the bottom right of the screen, shows you the co-ordinates information wherever your cursor is positioned on the screen.
