Reading files that have more than 32000 points

 

 

 

Read blocks of data from large files so that it can be thinned out to smaller workable numbers.

Specify the start and end line number of the block of data to read from the source file. In the example, the first 30000 points will be recorded. Reading the second lot of points will normally start at 30001 and end at 60001.

The End=Start plus option allows MM to add the desired number to the start value to calculate the end number automatically. Enter a zero value if you want to specify the end number manually.

 

Import certain codes.

 

Check ALL THE CODES if you want all the points between the start and end number to be imported.

Check ONLY THE FOLLOWING and specify the code (eg. A40) if only a certain code must be considered.
