MENU : Pnt - upcOde

 

This function assists you in changing the codes of selected DTM points to codes of certain CAD text values.

 

Often when a DXF CAD file is imported and DTM points generated from it, or loaded from files, the DTM points do not have a code. However, the desired codeis available in CAD text elements in close proximity.

 

Update the codes of the DTM points in 2 different ways.

 

Indicate

 

Indicate the DTM point of the code the to be changed. Then, indicate the CAD text element to become the code of the DTM point. Repeat the sequence until all the points are done. Press escape at any time to exit the function.

 

Identified

 

This method will scan through all the identified DTM points and automatically search for the CAD text closest to each DTM point . The code of the DTM point will be updated.
