MENU : Tri - Edit

 

Edit the leg between 2 triangles by placing the cursor close to the leg and click the enter button on the mouse. Deciding which leg to edit can be difficult.

 

Two triangles are formed between four points. The common leg is formed between points 1 & 3 or points 2 & 4. Because only the position of the leg can be selected between 2 options, it is easy for Model Maker to edit the leg when instructed by the user. The leg will flip to the other position each time it is edited. It is your responsibility to choose the correct position. Model Maker updates the contours for the 2 triangles each time the leg is edited. With experience you should be able to determine which position represents the actual surface on the ground correctly.

 

The following guidelines should be followed :

 

a)                The correct model would normally be the one that forms the best looking triangles.

         In some examples, the model cannot be created without long narrow triangles such as in cross-sections taken along a road.

 

         Creating triangles of a model like this suggests that you use the  NORMAL SEARCH with search angle set to +/- 5 degrees, use the  QUICK SEARCH  method or build the triangles manually.

 

b)                Always use the  EDIT  function. Do not try to delete triangles and rebuild them manually.

         Although nothing stops you from taking the long way to edit a model, you will find that by using the edit function you arrive at the same result in a fraction of the time.

 

c)                A leg of a triangle cannot be deleted. You can only edit a leg. Only a triangle may be deleted.

 

d)                Only the common leg between 2 triangles can be edited. Open sided legs of triangles cannot be edited, you may add triangles to open sides.
