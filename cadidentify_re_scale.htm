MENU : Cad - Id - Man-id - Scale                                

 

Scale all the identified CAD entities around a base point. You will be prompted for a reference point and a scale factor. The entities will be re-scaled around this point.

 

Note:

this function physically changes the sizes of the elements and it is not merely a plot scale change.
