MENU : Irr - Pivot - Pivot Parameters

 

 6 

 

a. The Pivots' Pen parameters can easily be set here.

b. These buttons leads to the Slope analysis and Clearance report windows respectively. Use these options to resolve any terrain issues you might experience and also to bring the current pivot up to standard.

 

 7 

After clicking this button you will have the chance to place the current pivot in the workspace. The mouse cursor will represent the pivots' Center Coordinates.

 

 1 

Here you will find a list of all the pivots in the current project.

 

 2 

 

a. Click this button to create a new pivot. After clicking it, go ahead and name the new pivot and give it a description.

b. Click this button to delete the currently selected pivot.

 

 3 

 

a. Name the current pivot.

b. Give the current pivot a description.

 

 4 

Here the current pivots' spans are entered and edited. Each span should have a length with and optional number of it. As one enter spans, the area of the pivot will be shown dynamically underneath the grid.

 

Please see the new format of managing the pivot spans.

 5 

 

a. Set the lengths of the Overhang as well as the Endgun.

These values will be added to the total length of all the spans and will be drawn with circles with a different line type.

b. Easily set the size of the current pivot here. The default will be a complete circle.

c. Here you can quickly clear all the spans of the current pivot.

d. The current pivot can quickly be copied to Cad. The spans of the current pivot will the be added as circles or arcs in the Cad.

e. If the current pivot has not been placed in the workspace yet, you will see a notice here.
