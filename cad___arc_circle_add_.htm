MENU : Cad - Circ - addCirc                                

 

An Arc/Circle can be added by several methods as described in the following section.

 

Before the arc/circle is added, select the pen and line type by using the (P)en mode on the right side of the screen.

Select the method of fixing the co-ordinates with the cursor mode functions.

 

How to use the add Arc/Circle functions.

 

1.        Select the CAD button in the left hand side menu.

2.        Choose the Circ button in the bottom menu.

3.        From the sub menu select the Add button.

4.

 

 

Add Circles

 

Centre + radius

 

Define the circle by indicating the centre co-ordinates, then enter the circle's radius.

 

3 Points

 

Define the circle by indicating three points on the circle. The cursor mode can be switched between the operations.

 

2 Point + radius

 

Define the circle by indicating two points on the circle itself. Then enter the circle's radius. If valid, two circles will be drawn through the two points. Delete the unwanted circle by placing the cursor on the circle and selecting it.

 

Centre + Point

 

Define the circle by first indicating the centre co-ordinates. Then, enter a point through which the circle must go. The cursor mode can be switched between the operations.

 

3 Lines                

 

Circle is drawn between three lines.

 

Add Arc

 

Centre + radius

 

Define the arc by indicating the centre co-ordinates. Then, enter the arc's radius. Enter the start and end angles of the arc as degrees, minutes, and seconds.

 

3 Point

 

Define the arc by indicating it's start and end point. Then, define the third point on the arc. The cursor mode can be switched between the operations.

 

2 Point + radius

 

Indicate 2 points going through the arc and then enter the radius. Delete the side of the arc that is not applicable.

 

Arc Parallel

 

Circles or arcs will be generated at fixed intervals from an existing element. Up to 8 elements can be added simultaneously. All the distances will be treated as relative distances away from the existing arc. Positive values mean that the new element is on the indicated side of the existing element. Negative values indicate that the new element is on the opposite side.

 

Exit after method

 

If checked, the box will close after a method is used.  If unchecked, the box will re-appear after a method is used.
