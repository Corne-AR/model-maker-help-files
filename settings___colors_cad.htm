A) Pen Colors

Change the 255 CAD pen colors by clicking on the color grid. Select or form a new color from the color definition dialog box.

 

Note: The assigned pen colors (0 to 255) are also used by the Projects. Changing the color of a pen (column A) will also change the color of the DTM element drawn with the same pen number.

 

B) Plotter Colors

The plotter color of each pen is defined in column (B).

·To quickly make the plotter color = the screen color, RIGHT click on the plotter color box.
·To set a plotter color, left click the plotter color box. The color definition dialog box will be displayed.
·To set all the plotter colors = the screen colors, click the reset (F) button.

 

C) Pen Thickness

To set the line thickness of a pen, click on column (C).

 

The predefined line thickness panel is displayed.

 

SELECT: Predefined thickness values (mm) are shown. Click on a value to assigned it to the select pen.

 

EDIT : The edit box for 11 values is shown. Here you enter the thickness values to be used then the Select list is displayed.

D) Screen Pen Widths

CAD lines are specified to draw at a certain width when outputted to a printer. When these lines are drawn on the screen the same definition cannot be selected as the monitor can only produce images in pixel sizes. The thinnest line on the screen is one pixel width.

To get some indication of the relative line widths on the screen, Model Maker allows you to specify the millimeter to pixel ratio. These values can be set according to your own choice and it is suggested that you experiment with different settings to obtain a desired result.

 

Use the check box to activate or deactivate this feature when plotting to the screen.
