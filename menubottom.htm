MENU : Bottom of main screen

 

The general screen layout allows menu buttons to be placed right around the main plotting area.

 

The menu at the bottom of the Model Maker screen holds the buttons (functions) that are related to the specific menu item that has been selected on the left menu.

General

 

       Use the F8 function key to select between the top and bottom menu as the active menu.
       The buttons on the bottom menu can only be accessed while the cursor is a white solid arrow shape.

               While any of the bottom menu functions is selected and the associated action is taking place, the cursor shape will change to a cross and access to the left and bottom menus are not allowed. To return to the neutral mode (white arrow) press escape.
