STRING : Add - Parallel

 Top  Previous  Next

 

MENU : gXt - Add - New

 

Create a new string parallel to an existing one. Point to the base string on the side of the new string. After entering the distance from the old string a new one is created depending on the type of the base string.

 

General base string

 

The new string is of type general and will have the same number of nodes as the base string. The Z values will be similar.

 

Alignment base string

 

The new string is of the general type and new nodes will be formed at specific chainage distances. The Z value of each node will be calculated from the corresponding position on the vertical alignment of the base string.  

 

 

 

 

To add a single new parallel string, use the Single String options.

 

Enter the distance between the base string and the new string. Use the ? option to measure a distance on the screen.

 

The elevation of the new string can defined in 2 ways.

 

1) Elevation difference - The new string will follow the slope of the base string but also have an elevation difference as defined.

 

2) Constant elevation - The new string will be level at the set elevation, no matter what the slope of the base string is.

 

 

 

Use the Multiple Strings option to add many parallels in one go.

 

A) Enter the offset and elevation differences in this editor. You may define up to 4 new string offsets.

 

B) This setting controls how many times the offset sequence (A) will be repeated.

 

    How many times. - Enter the fixed number of times in input (C)

                                In this example, 2 offsets will be repeated 5 times resulting in 10 news strings.

                                The last string will be 25 meters higher than the base string.

 

  Until maximum elevation. - The sequence of new parallel strings will be repeated until the first node on the new string reaches the maximum elevation.

Please note, the green arrow indicates where the start and end of each string is. It is best to start and end the string on a straight section and not on a corner.

 

If the start and end is at a corner, the intersection angle cannot be established and the required step back is not formed.
