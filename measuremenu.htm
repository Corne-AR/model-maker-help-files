Menu : Measure                (F6)

 

 

Lengths - Arc

 

Calculate the distance along an arc. The point on the arc must be indicated. Then indicate the arc. The distance along the arc from its start or end point is calculated. This depends on the position of the cursor when the arc in question is indicated. Model Maker will calculate the distance from the start point of the arc if the cursor is placed between the start and specified point. Model Maker will calculate the distance from the start end of the arc if the cursor is placed between the end and specified point.  

 

Option to set cursor snap mode format for 2 Point measure.

 

 

This setting allows you to control the snap mode function when using the single, 2 Point measure feature.

The input box must have 2 characters which correspond with the hot keys found in the cursor snap mode dialog box.

The above example will automatically set the snap mode to tachy Point for the start point of the measure and then to cIrcle centre for the endpoint.

To deactivate the feature, either uncheck the check box or clear the input box of any characters.

 

This feature is normally used when you have to do multiple measurements between the same type of endpoints.

 

Quick Swipe/Last action option.

After completing any of the measure functions, the swipe/last button is available to redo the same action without having to find and click the original button on the dialog again.

This button can be activated by either clicking it or just by moving the mouse over it.

 

If the Enable swipe action is checked the last action will execute by just moving your mouse over the button. If the Enable swipe is not checked, then you need to click the button to run the last action.

Also, if the Enable swipe is checked, the last action button will have focus and the action can easily be executed by just pressing Enter on the keyboard.

 

Example: Running many consecutive measurements.

Without the swipe/last feature, the user needed to find and click the correct button on the dialog each time after a measure is performed.

With the swipe/last option the last function button will always be available in the bottom left of the dialog and can be clicked (if Enable swipe is unchecked) or swiped with the mouse or by pressing Enter on the keyboard.
