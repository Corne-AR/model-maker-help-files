PROJECTS : Active project

 Top  Previous  Next

 

Up to 21 different DTM projects may be loaded in combination and displayed simultaneously.

 

Only the active project may be edited.

 

The active project is indicated by the project buttons on the side.

 

The active project plays an important role throughout the program and not only on the graphics screen.

1.                All the functions available under the DATA  option will manipulate data in the active project.

2.                When using any SAVE  functions involving the DTM points and triangles, only the data in the active project will be saved.

3.                When using any  LOAD  function which involve the DTM points or triangles, the data will be loaded into the active project.

 

When the  TRIANGLE  and  CONTOUR  (or  QUICK ) functions are used on the top MANIPULATE menu, the result will only be generated for the active project.

 

EXAMPLE:

Activate Project 1 and draw the contours. Only the contours as produced by the triangles of project 1 are drawn.

 

Activate Project 2 and draw the contours. The contours of project 2 are now drawn resulting in a contour plan of both models.

 

In many cases you would select a different active project between 2 parts of a command.

 

EXAMPLE:

Draw a CAD line originating from a tachy point in project 1 and ending on a tachy point in of project 2.

 

Go to  CAD - LINE - ADD - SINGLE. Set the cursor mode to tachy point. Change the cursor mode by pressing by the SPACE-bar and then select the correct option.

 

Jump to the tachy point in project 1.

 

Before indicating the end point of the CAD line, go to the project button on the top right of the screen and place the cursor in project 2. Press Enter. The active project has changed to # 2. This is done while in the process of adding a CAD line. Now indicate a tachy point in project 2.
