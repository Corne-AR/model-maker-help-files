MENU : Cad - Line - Special - Blockmove                                

 

There are various ways to move lines to new positions. Block Move works similarly to the move point function.

 

Define an area

All the endpoints of lines in the area will be moved from a base point to a new defined base point. You may specify a rotation angle.

 

Note that if any line has only one endpoint in the area, the other endpoint will stay fixed in its original position, resulting in a possible stretch and/or rotation of the line.
