MENU : gXt - Power - Edit conductor

 

In the design of power lines, the conductor type and the associated characteristics are important since they define the shape and behavior under different temperatures and span lengths.

In this dialog box, enter up to 40 different conductor types and edit their associated SPT files.

 

 

This scroll box displays the existing conductor names.

 

Use the name input box to add new names or edit existing ones.

 

The SPTeditor allows you to enter the SPT values for each line type.

See the design section for an explanation of the format.

 

This button will become available after any changes are made to the SPT data. Save the data to the data base if you want the changes to be applied to the conductor type.
