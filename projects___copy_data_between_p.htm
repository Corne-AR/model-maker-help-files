PROJECTS : Copy data between Projects

 Top  Previous  Next

 

MENU : prO - Copy

 

Apart from being able to load DTM data directly into the active project  Model Maker provides you with a powerful copy  function to move data between projects.

 

 

 1 

Source project #

Select the project number from where the data must be copied. In this example project 1 is the source project.

The 19 displayed next to project 1 indicates the number of identified points in that project. It is useful to note this number if you only want to copy the identified points from this project.

 

 2 

Destination project #

This is the project number to which the data will be copied. In this example it shows that the points will be copied to project 2. The numbers next to the project numbers indicate the total number of points in the project. This example shows there are 18137 points in the source project and zero points in the destination project.

 

 3 

Copy?

The selected points (All, identified or Triangle points) of the source project is copied to the destination project.

All the points                : All the points of the source project will be copied to the destination.

Identified points                : Only the identified points will be copied. In this example the 19 identified points will be transferred. Use the POINT  and  ID  functions while in the active projects to identify the correct points.

Use the triangle points option if you only want to copy the points connected to triangles without first identifying them.

 

 4 

Source Stations

This option is only available when all the points are copied.

The Retain Station Relation copies the data from the source to the destination project in station format. The destination project will end up with the original number of stations plus the number of stations added from the source project.

 

The Combine data option will take all the source data and add all the stations and points to the last station of the destination project. If the destination project is empty a new'KOORD'station will be opened and the data copied to it, otherwise the data will be added to the last station of the project. The copied data will only be kept as final co-ordinates in the new project.

 

 5 

Clear destination

If the clear destination first option is checked, the points in the destination project will be cleared before the new data is added to it.

 

Delete source

The copied points are not deleted from the original  project unless the Delete Source Data After Copy is checked.

 

Match codes

Match the codes between the source and destination projects as entered on the Codes tab (6)

 

 6 

Codes

The codes entered here will be used to replace the each corresponding one in the source project with the one in the destination project.

 

 

 

Move the cursor over the graphic and when the appears press the enter button on the mouse and a more detailed explanation of the function in question will be given.

 

 

Copy triangles from source

 

If required, the triangles connected to the copied points of the source project may be copied with the points to the new project.

Triangles are connected to 3 corner points and if not all three of the corner points of a triangle is copied, that triangle in turn will not be copied to the new project.

 

Delete common triangles

 

Any triangles in the new project that models the same area as those of the copied points are automatically deleted before the new triangles are copied. This check box should normally be selected as duplicated triangles are not desired.

 

Continue search

 

If this box is selected, Model Maker will automatically perform the continue search option after the models have been merged.

 

Delete redundant

 

The points in the destination project that will be covered by the new points from the source project will be deleted after the new points have been added.

 

·To perform the continue search and delete redundant points manually, (If the automatic boxes are not selected) do the following.

 

Activate the new project and draw the triangles and the sides.

Notice that there is a gap between the triangles of the old data and the new data.

To close this gap, use the CONTINUE SEARCH  function. The program will automatically build triangles between the two areas.

MAKE SURE NO TRIANGLES OVERLAP BEFORE USING THE CONTINUE SEARCH OPTION.

 

All the old points falling below the new triangles are now redundant and should be removed.
