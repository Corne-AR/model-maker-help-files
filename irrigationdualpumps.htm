MENU : Irr - Dual Pumps

 

The hydraulics of the system is calculated by entering the water into the system through the source node.

               For regular blocks the valve is the source node.

               For the main line, the source node is the pump.

               Irri Maker can accommodate up to 20 pumps in the main line.

               The main line can be an open or looped system.

               Multiple pumps can only be used in a looped system and has to be connected to the system by means of single branched pipes.

 

EXAMPLE: multiple pumps. Note that the valid pumps are connected to the loop by means of single pipes. The pump which is part of the loop is not allowed.

 

Single pump

When a single pump is specified, you are required to enter the operating pump pressure when the hydraulic calculations are performed. Irri-Maker will then calculate the flow into the system based on the available pressure.

 

Multiple Pumps

When 2 or more pumps are specified the hydraulics gets much more complicated. In this case a balance must be found between the pressure and flow of each pump.

Increasing the pressure in one pump will normally result in the other pumps contributing less water to the system.

 

Irri Maker allows you to use 2 different options when specifying the flow balance.

 

1) Specify the main pressure and both flows.

         

 

 

In this case the main pump will set the pressure in the system.

 

The pressure set in this box will be that of the main pump. The pressure in the other pumps will be determined based on their contributed flow.

 

 

The combined percentages of the flows must add up to 100.

 

 

Note:

Each node may be given a user defined node name. Use the node edit function to assign different names to the pumps so that they can easily be identified.

You will also be allowed to specify the percentage of flow (contribution) that the main pump puts into the system. In the example all the pumps supply an equal percentage of the flow.

In the hydraulic menu you will then enter only the pressure of the main pump. Based on the main pressure and the percentage of contribution, the system will be balanced and the required pressure at the other pumps calculated.

 

2) Specify the pressure at each pump.

 

In this case you will enter the pressure at each pump. Irri Maker will then iterate, balance the system and calculate the flows at each pump.

Depending on the pressures and elevation differences this formation can often result in a situation where one pump actually starts to reverse another. In such a case, Irri Maker automatically assumes there is a check valve at the pump. The pump will then not contribute to the system but will also not take any water out of the system.

 

3) Pump Curves

The pressure against flow curve as defined for each pump in the node database will be used to balance the flow in the system.

 

It is therefore important that correct relevant data has been entered in the node database for each pump.

 

If for some reason the system cannot balance, use one of the alternative methods and look at the flow and pressure at each pump to see if the pump's performance curve data can actually supply the needed pressure and flow.
