MENU : Min - mine_Planning

 

Mine planning allows you to plan and manage open cast mining activities. Bore Hole information is used to model and contour surfaces of different reefs or seams. Information like volumes, average elevations and sections can be calculated and saved to spreadsheet files.

 

 

 

 

Blast Grid

 

The blast grid function is used to create dtm points on planned blast grid positions.

 

Layer Depth

 1 

Volume

Calculate the excavation volumes during the planning stages for an open cast mining operation. Use the bore hole data from the exploration to create triangle models of the different horizons. Excavation strip areas are then defined and volumes between the different horizons under each area will be calculated and saved in a spreadsheet file.  

 

 2 

DVolume

Calculate the volume between a design surface and then layers in the mining stack.

 

 3 

Sections

Sections will be extracted through all the mining layers and plotted to CAD.

 

 4 

Elevations

Determine the average elevation within each geo area on a regular grid and actual points.

 

 

 5 

Setup Layer

Setup Layer : Define the dtm projects to be used as mining surfaces.

 

 6 

Manage Old Format

Manage old Format : Method to import older stack data into new dtm project format.

 

 7 

Stip Width

Calculates strip width to accommodate a volume.

 

 8 

Update Surface

Improves bore hole surface based on OGL model.

Borehole Manager

Borehole manager : Go to the borehole menu.
