EXAMPLE : Working Example 1 (Create DTM)

 Top  Previous  Next

 

Object

 

This example creates a DTM and generates a contour plan using existing data by showing you how to:

               Load the survey data from a file.

               Create a network of triangles.

               Draw the contours.

 

Before attempting this example, please familiarize yourself with the following:

       The general menu structure to gain a good understanding of the screen layout.
       The cursor control keys to eventually know all the shortcuts.

 

It is also a good idea to check out the total picture example.

 

Procedure

 

               Load DTM data from either an ASCII file or from an existing file.

 

         If you do not have any survey files to use, load the ZX.MMS sample data used in the Total Picture example.

         File - Open (F3) -

 

         

If you have to load survey co-ordinates from an ASCII file, different to the normal Model Maker format, use the import ASCII function:

 

         File - Import ASCII & Other - Define format

         If you have to input co-ordinates in another way, see the input co-ordinates example.

 

         Data - DTM DATABASE - Stations

 

       After the file is loaded the points will be plotted on the main screen.  To display the codes or elevations of the points, select the options in the screen dialog box on the right hand of the main screen or the GOP menu on the top of the mainscreen.

 

 

       On the top left menu of the main screen go to the Triangle menu.

 

 

 

The triangle options will appear in the menu at the bottom of the main screen.

       Select the Csearch option.

 

       From the second dialog box select the option Quick Search.

 

The triangle network will be created between the points. After the triangle search is completed, draw the triangles and contours.

 

 

       Choose the Tri option on the main menu at the top of the main screen to draw the triangles.

 

       Select the Quick option on the main menu at the top of the main screen to draw the contours.
