MENU : Cad - Line - Special - Bend_arc                                

 

Generate an arc with an indicated line using the 2 endpoints of the line and the position of the cursor.

 

The result is similar to the 3 point arc. The original line is deleted once the new arc is formed.

 

This function is effective when modeling a curved alignment of the edge of a road.

 

This is an example of a road center island when lines 'a' and 'b' have been bent into a smooth arc.
