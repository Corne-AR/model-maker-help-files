MENU : Irr - Blocks

 

Model Maker uses the block concept to define certain areas and to group the nodes and pipes in logical working units.

When a new project is started, block 0 is already added to the system. Block 0 holds all the information regarding the main line.

The main line will normally consist of the pump, fittings, valve nodes, the pipes connecting these nodes and may even incorporate individual sprinkler nodes.

Add additional blocks to the project when certain areas must be grouped together or if a shift system, turning valves on and off, must be used.

 

Blocks may be copied, deleted and automatically filled with nodes and lateral pipes.

 

A maximum of 600 blocks per irrigation project is allowed.

Info

 

The info page will be active once the lateral layout is in place. If the block was originally defined as a sprinkler system, the info page will indicate it as such. For a defined micro/drip block, the info page will indicate the specified emitter type to be used globally on all the lateral pipes defined with the CES value. You may use the UPDATE button to select a new emitter (sprinkler) type. Remember to recalculate the hydraulics after changes have been made to the design.

Block Info

All the applicable information about the current block is displayed in the block info panel.

 

# Perimeter points        

The number of points on the perimeter of the block. A maximum of 40 points may be used.

 

Block area                

The area defined by the block.

 

Perimeter Length                

The circumference of the area.

 

Node & Pipes.

The number of nodes and pipes defined in the block. Initially, when a block is defined it would not have any nodes or pipes.

 6 

Open in Shifts

Blocks can be defined to operate in different shifts. The four input boxes indicates the shift numbers in which the block is active. You may edit the numbers in these boxes but the more convenient method of managing the shifts is done in the shift page.

 

 7 

Block list

The highlighted block number is the active block. Functions applied to the nodes and pipes will be performed on the active block. In a project with multiple blocks, the active block can be selected by clicking on the block number in the block list or by using the button found below the block list. After the Pick button is pressed, point to the selected block on the graphics screen to make that the active block.

To see the actual layout of any block, double click on the block number in the block list box. The graphics menu will be displayed and the selected block will flash. While the boundary of the active block is flashing, press any key to return to the irrigation menu.
