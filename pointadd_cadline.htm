MENU : Pnt - Add - cad Line

 

Add DTM points along an indicated CAD line.

 

Line information

 

 

Length

Indicates the length of the indicated line.

Direction

The angle of the line using the start point as the line node closest to the cursor position when the line was indicated.

 

Position

Parameters controlling the placement of the new points.

 

 

At start

Check to add a DTM point to the first node of the CAD line. (The node closest to the cursor position when the line was selected)

At regular intervals

The in-between points will be placed at this regular distance along the line starting from the first point. The remainder will be left at the end of the line.

Number of points                

Use this as an alternative to the regular distance option. Divide the length of the line in equal parts and new points will be placed accordingly.

At end

Check to add a DTM point to the last node of the CAD line.

 

Point Attributes

Select to Manually enter the elevation and code of each new point or allow the program to automatically add the points with the pre-defined parameters.

 

 

 

 

 

Automatic info

Based on the information supplied, Model Maker will add the points without user intervention.

Pause at points

Model Maker will stop at each new point allowing you to make changes.

 

Automatic Parameters

These parameters are used when the automatic check box is selected.

 

 

 

 

 

 

 

Start elevation

The elevation of the point at the beginning of the line.

End elevation

The elevation of the last point. If other points are added between the start and end point Model Maker will interpolate the elevation of each point based on it's position on the line.

Code

The code used for each new point.

Increment code

If this is checked and the proposed new code ends on a numeric value (eg Tree5), the subsequent codes will have one added to it (eg Tree6, Tree7 etc.).
