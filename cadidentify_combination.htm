CAD : Identify - Combination

 Top  Previous  Next

 

Identify cad elements based on a combination of selections.

 

 

On the left side, check the attributes that will be identified.

 

On the right are the values of the attributes.

To change the values, use the Line Menu and Text Menu buttons to access the attributes dialog boxes.

 

Elements (lines, text, circles, arrows, dimensions and hatch) will be identified based on the selected criteria.

 

If the AND button is pressed, only the elements that satisfy ALL the selected options will be identified.

 

If the OR button is pressed, elements that has any one of the selected attributes will be identified.
