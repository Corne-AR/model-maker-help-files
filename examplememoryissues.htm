MENU : Example - Memory issues

 

Model Maker works in WINDOWS and relies on the windows environment to manage the memory. Using the fastest computer with the most memory is always the preferred system but the program would also work on a system that has :

 

               128 Mb of RAM

       40 Mb hard drive space for installation

               20 Mb free hard disk space for operating

       a high resolution monitor (minimum settings  of 800 * 600 and 256 colours)
       a CD drive
       a Mouse

 

If you run Model Maker and on the first screen it says 'OUT OF MEMORY', please check that your hard drive is not full. By that we mean it should have more than 100Mb of free space because windows creates a swap file on the hard drive that dynamically grows and shrinks and makes use of that space.

 

When  Model Maker is active and working with data, it saves the data in RAM memory.

               If the computer is switched off before you save to disk, the data in memory will be lost.

               Use the FILE - SAVE option to save the data before existing the program.

 

Model Maker has an automatic back-up facility which makes regular backups of the data. This is intended only to be used in an emergency like a power failure.

 

Note:                

Use the F4 key to force a back-up at any time.

 

In general, it is your responsibility to go to the  FILES  menu and SAVE  data on a regular basis to disk.

 

The time interval for the automatic back-up function is set using the BACKUP  option under the FILE  menu.
