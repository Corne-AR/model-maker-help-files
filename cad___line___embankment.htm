MENU : Cad - Line - Special - Embank                                

 

High

 

The embankment lines will be drawn from the indicated position.

 

Low

 

The embankment lines will be drawn toward the indicated line.

 

Indicate

 

Indicate the base line of the embankment.

 

A special hatch function indicates embankments around excavations or other earthworks. This is not a normal hatch function because:

 

a)  The area is not specifically indicated and defined.

b)  The lines used to indicate the hatch are normal CAD lines and may be manipulated individually using the line functions.

c)   The lines identifying the toe points of the embankment must be defined and identified. (Using the Id functions).

d)   The platform line is indicated by the user pointing to the corner points.

e)   The embankment lines will always be placed to the left of the indicated platform line. Always follow the platform in a clockwise                      direction.

f)   The Move Point or Move Line functions are used to rotate the embankment lines on the corner points.

g)   The new lines are defined by using the current Pen parameters.  Use the correct line attributes before the action is performed.

      The lines should be placed on their own layer name.

 

Parameters

 

Enter the percentage length of the long embankment lines. If an embankment pattern is specified, Model Maker will apply the percentage to shorten the pattern.

Embankments will be drawn using normal lines if this name is left blank. If a pattern name is entered, Model Maker will use the specified pattern instead of a line.
