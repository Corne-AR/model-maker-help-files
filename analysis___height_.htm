MENU : Ana - Height        

                 

 

 Example:

Fill Type

 

Model Maker analyzes height by filling the area of the model in three different ways. The full fill and contour band fill will create a bitmap image. The crosses option generates crossed lines in CAD.

 

Pixel Interval

 

The resolution of the bitmap is defined by the pixel interval value. A value of one will create the smoothest bitmap but may take some time to prepare. Model Maker creates the bitmap by checking the relative elevation of every pixel. The pixel is then coloured according to the full fill colour box.

 

Full Fill Colours

 

The pixels with the lowest elevation will be in colour1. The highest elevations will be colour2. The colours in-between will be a smooth mix of the 2 extreme colours. Click on the colour button to change the defined colour of 1 & 2.

 

Copy

 

To record the exercise as a bitmap image, check the check box.

 

Contour Band Fill

 

This is similar to full fill, except for the colour definition of the pixels. Use the band fill colours to define the 10 available colours. Use the merge colour button to smoothly combine colours between the first and last colour. This exercise creates a bitmap image.

 

Example:

Crosses

 

Model Maker generates crosses using CAD lines to indicate the difference in elevations. Small crosses depict the low areas and large crosses show higher areas.

 

Define relative elevations of the different intervals manually or automatically.

 

 

 

 

 

Example:

 

Manually (User defined elevations)

 

Check the USE USER ELEVATIONS box (as in the example) and enter relevant elevations in the user defined table. Define up to 10 elevations. Model Maker will check the elevation at each grid point in order to determine the pen colour. The AUTO FILL button fills the table with data relevant to the current model. No.1 will hold the lowest elevation on the model, while No.10 will have the highest value. The in-between intervals will be filled with increasing values. To use less than the available 10 intervals, fill the remaining table values with zeros.

 

Automatically

 

Model Maker will determine the total height difference across the whole model. Ten colours are automatically used to indicate the different levels.

Hints:

 

Height analysis is often used only to see the visual effect. To plot the resulting image to a plotter, consider the practical implications. Older pen plotters cannot reproduce a filled area. In those cases, use the Crosses method to do the analysis.

Create a bitmap file using either the full fill or band fill options. This bitmap is not part of the CAD file, but is only displayed before the rest of the elements are plotted. Use the CAD Bitmap function to manipulate the bitmap properties.
