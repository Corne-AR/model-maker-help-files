MENU : Settings - Language

 

 

This option allows you to print reports in a user defined language.

 

By default the Original option will be selected. In this mode all the reports will be printed without any translations.

 

If the alternative option is checked, 2 things will happen.

 

1) Whenever a report from the Irrigation functions is produced, all the headers will be recorded in the original editor of this dialog box.

2) If the header text was already entered in the original editor, the text will be substituted with the alternative text before placing it on the regular text editor.

 

Changing the editor text - Step by Step

 

       Visit the Settings, Language option and check the ALTERNATIVE checkbox.
       Go to the irrigation menu and produce the desired report. (Example > Global - Report)
       After viewing the report in the regular text editor, come back to this language dialog box.
       Notice that all the headers of the report will be listed in both the original and alternative editor boxes.
       In the alternative editor, type the correct words for each header in your own language.
       Go to the irrigation menu and produce the report again. Notice that all the headers will be displayed in your own language.

 

Each time a report is produced and any of the headers is not already listed in the language editor, the original line will be recorded and you then have the option to

specify the word in the alternative language.
