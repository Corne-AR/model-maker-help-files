The list of the actual dtm projects is displayed in this dialog box.

 

 

1) Dtm project number

 

2) Number of triangles in the project.

 

3) The project description.

 

4) The description of the volume between the 2 surfaces. It will be used as headers on the reports.

 

5) Minimum depth. When the planning volumes are calculated between the surfaces, the volume will be assumed as zero where the layer depth is less than the specified minimum depth.

 

This will preclude volumes to be included where, on a practical bases the layer is to thin to be excavated separately.

 

6) The SG value to be printed on the reports.

 

Note : The surface of a dtm project will only be used as a mining layer if the SG value is larger than zero.
