MENU : gXt - Vert - >>3D - Copy_cad

 

Copy the image of the current string in vertical mode to the CAD as normal lines. Use the menu  to specify the correct vertical and horizontal scales. Manipulate the long section  afterwards in the CAD with all the normal CAD functions. Take note that once the vertical image is in the CAD it only represents a normal CAD picture. The original string is not influenced in any other way.
