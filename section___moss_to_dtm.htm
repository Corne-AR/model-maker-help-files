SECTION : Sections to DTM

 Top  Previous  Next

 

MENU : Sec - xx1>Dtm

 

The 3 functions described here will load data in section format. By using a defined horizontal alignment, the functions will translate the offsets on the sections to 3 dimensional DTM co-ordinates.

 

If an applicable alignment is loaded, the positions of the offsets on the sections will, according to their chainage distance, be calculated and added to the active DTM project as regular points. If no alignment is available, the sections will still be converted, but a temporary regular straight line alignment will be used.

 

XX1 > DTM

Regular Model Maker and Road Maker cross-sections will be loaded from disk and translated to DTM points.

CRD > DTM

Sections in the CRD format will be loaded and converted. The end result will be the same as when loading XX1 files.

 

GRO > DTM

The GRO format is the regular file format that Model Maker and Road Maker use to store long section (profile) data.

The offsets on the long section (chainage and elevation) will be translated to dtm points. Because the GRO is only a section along the center line the resulting DTM points will be placed on the horizontal alignment.
