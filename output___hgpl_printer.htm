MENU : Output - Plotter/Printer

 

Create a plot file or plotted output to the installed printer. Model Maker uses the standard WINDOWS printer drivers to produce the plot. It is important to specify the printer.

 

This is the dialog box for the output process.

 

 

Plotting to

The device driver that generates the output. Use the drop down button to select other installed options. For more comprehensive settings of the output device, use the button to jump to the printer setup menu.

 

Current scale

The selected plotting scale set in the plot parameters menu.

 

Maximum scale

The scale at which the current drawing will fit within the available paper size.

 

Area needed

The width and height in millimeters needed to produce the drawing at the current scale.

 

Area available

The size of the paper in millimeters.

 

Plot To Plotter

 

3 options are available to specify the paper output result.

 

Expand area

This option is only available when the size needed to produce the drawing is smaller than the selected physical paper size. Model Maker will centre the drawing on the paper while the frame will be drawn on the boundary of the physical size. The grid lines, if selected, will be drawn up to the drawn frame.

 

Only area needed

This option is also only available when the size needed to produce the drawing is smaller than that of the selected physical paper size. Model Maker will place the drawing at the bottom left corner of the paper while the frame will be drawn around the area needed for the drawing. The grid lines, if selected, will stop at the edge of the small frame.

 

Paper window

This option is only available if windows has been set in the paper menu. Model Maker will use the paper size and produce the drawing inside of it. Use paper windows if the size of the drawing and the selected scale makes it impossible to fit the whole drawing onto a single sheet.

 

Paper Window

 

These options are available if paper windows have been set.

 

Plot to number

Up to 10 windows may be defined. The Plot to Number allows you to specify which of the set paper windows will be plotted.

 

Stack #

Allows you to produce strip plans (windows) on a single sheet of paper. Use the paper size option and set, for example, the A2 size to 150mm height and the length of an A0. Four sheets will fit on a 840mm height A0 sheet. If the plan that you want to produce consists of a road survey, use the A2 sheet and place it along the road with a slight overlap between the sheets. Change the paper size to an A0 and use the Output - HPGL option. Use the stack option as above and stack from window 1 to 4. The space input box that currently has a 10 in it, is the distance in millimeters that will be kept between the strips.

Put North On Plot

 

A north arrow will be drawn at the position that you specify by using the numeric keypad keys  Position 1 will be the left-hand lower corner. Position 9 will be the top right-hand corner etc.  You may make your own north arrow shape by saving it as a pattern under the name NORTH. Model Maker will automatically use the shape saved as pattern named NORTH for the north arrow. If no pattern with the name north exists, Model Makers own standard north arrow will be drawn

 

 HINT

 If text within the NORTH.PAT pattern is on a layer called SCALE, that text will be replaced with the actual scale value.
