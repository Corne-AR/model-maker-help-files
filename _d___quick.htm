MENU : 3D - quick3d                                

 

Use this function to create a temporary 3D image of the current data. The initial parameters control your view of the image. Once transformed into 3D mode the other regular functions, apart from the plot manipulation options, are disabled and the data cannot be manipulated. (There is one exception that will be discussed later.)

Use the options on the bottom menu to change the image's appearance, dump it to a printer, copy it to CAD or export it to a plotter.

                               

Vertical view angle (0=horizontal)

 

This parameter controls the steepness of the picture relative to the horizontal. A zero value will generate a side image of the data.  A value of 90 will produce a plan view of the terrain. You may experiment with this value but, in general, a value of 30 is suitable.

 

Vertical distortion (1..50)

 

This value controls the amount that the vertical scale is enlarged relative to the horizontal scale. A value of 2 is normally adequate but for a flat terrain you can enlarge it up to 20 (or more) times.

 

Horizontal view angle

 

This angle is similar to the normal survey orientation with zero being south and 180 north. Enter any value between 0 and 360. This angle defines the initial view of the terrain.

 

Minimum rotate delay (sec)

 

This allows you to control the rotation of the model by seconds.

 

Auto fill of triangles

 

Check "on" or "off". "On" will hatch the triangles of the terrain after incrementing the view angle in the slide mode.

 

Apply depth colour

 

If checked, surface in the back will be darker than surface in the front.

 

Colour1 and Colour2

 

Model Maker uses two colors to hatch the terrain according to steepness, shade, and direction to the light source.

Choose Colour1 to hatch the shaded areas,ie., the areas away from the light. Choose Colour2 to indicate the bright areas.

Since the image is a relative 3D object, the normal                                                    scaling and grid options do not apply. You do not have the option of setting a window over the picture. Therefore, the scale must produce a picture small enough to fit the paper size.

 

Note:

Exception: Although you cannot use the other menus while in the 3D mode, the speed menu options are still accessible. This allows you to edit the triangle legs while viewing the model in 3D.

 

Apart from using the CAD copy option, you may also use the Utility> "Bitmap to clipboard" option to capture the screen image to be viewed as a bitmap.
