TUTORIAL - CREATING THE DAM WALL MODEL AFTER USING QUICKDAM FUNCTION

 

To follow this tutorial you must have a model (project 1) of the terrain on which the dam wall must be build and the geometric alignment string representing the centre line of the dam crest.

 

The centre line string must be at the elevation of the crest and the endpoints must meet the ground at each end.  To check this:

 

GXT - VERT - VIEW - indicate string and show string with ground profile.
Use SPECIAL - MODE MOVE to move end points to meet ground.
Use EDITZ afterwards to make sure endpoints is still at correct level.
When done go back to HORIZ mode.

 

We will now use strings to model the rest of the dam wall.

 

Add strings to represent the outside edges of the crest

 

GXT - ADD - NEW - PARALLEL - Distance between strings must be half crest width, Elevation difference zero (0)

 Indicate each side of the centreline string for the two new edge strings.

 Because the crest is usually very long we need to add additional nodes along it.

 

GXT - ADD - NODES+ - FIXED at intervals of say 10m.  - Press NODES button - Indicate the two crest edge strings.

 

To model the crest now activate DTM project 2 (right top corner) and then create DTM points and triangles from the two outside strings.

 

GXT - UTILITY - STR_DTM - indicate the outside strings (DTM points to project 2).

 TRI - CSEARCH - QUICK SEARCH.

 

The toe line strings must now be added

         

GXT - ADD - TOELINE (Do upstream; downstream separate) - Enter the cut/fill slope ratio you want, Interval of toe points 10m, Normal toe points, To project 1.

 

Now we must create DTM point and triangles for the toe lines.  Activate project 2 again and then

         

GXT - UTILITY - STR_DTM - Indicate the toe line strings (DTM points to project 2.)
TRI - CSEARCH - PARAMETERS button - Set radius to 100m and angle to 2 deg. and CLOSE - CONTINUE SEARCH.

 

You now have a complete model of the dam wall.  Save this model (project 2) with a new file name. Now calculate the earth works volumes using TER - CUT_FILL - GRID.

 

To get the final water capacity above the dam wall you need to merge the dam wall model into the original model.

 

PRO - COPY - Source project = 2,  Destination project = 1,  Copy all points and  combine data,  Also copy the triangles, delete common triangles,  Continue search and delete redundant points . - Activate project 1 which now has the complete model and save it with a new name.

 

Calculate the water volume now

 

TER - CUT_FILL - PROJECTS - Make project 1 the design and project 2 the ground, OK - Select the option "Or flat surface @ elevation" and enter your final water level into the input box. - GRIDAREA - Indicate the area you expect the water to be above the dam wall.

 A normal volume calculation will be done and the cut result will be your water volume.
