CAD : Identify - On the FLY

 Top  Previous  Next

Actions that can be performed when the mouse is in neutral mode (looks like this) and the on the FLY option is selected on the Identify dialog box.

 

Note : If the ON THE FLY option on the identify dialog box is not selected, the default mouse cursor has no function on the graphics screen except when clicking on actual buttons.

 

The normal way to identify elements is to select the CAD, Identify, Do-Id option.

 

The "on the fly" option opens the following shortcut options:

1.Just by pointing to a cad element, the element will be identified. If the element is already identified it will be un-identified. (Always passive ID mode)
2.By clicking and dragging the mouse over an area, all the elements in the area will be identified.
3.By clicking the RIGHT mouse key while keeping the SHIFT key down, brings up an abbreviated Identify dialog box.

 

1) Click on an element.

In the neutral mouse mode, clicking on a cad element is very similar to using the following on the identify dialog box.

 

The on the fly option however has the following differences.

 

1) You do not specify what type of element you want to identify. The element closest to the neutral cursor will be identified.

 

2) On the fly automatically works on a default include/exclude basis.

 

Eg: if you identify an incorrect element, just click it again and it will be un-identified.

 

Passive ID mode.

 

2) Left Click and drag an area.

To identify an area on the fly, click the left mouse button, keep it in and drag to the opposite corner of the area.

The elements inside the area will be identified.

 

3) Right Click with Shift key down.

 1 

The top window shows the number of identified elements.

 

 2 

Use MOVE Id, COPY Id, DELETE Id and ROTATE Id to perform these functions on the identified elements.

 

 3 

id All: Identify all the cad elements.

id New: Clears the id list.

id Layer: Identifies all the elements on a selected layer.

 

 4 

The Manipulate ID option opens the regular Man-ID dialog box.

The Do ID menu opens the regular Do-ID dialog box.

 

 5 

Alternating between Passive and Active ID Modes.

Active: New while ID.

Passive: Add to what is already ID'd. Clicking on elements to id and un-id always use this mode.

 

 6 

Hide

This will hide the current active Layer.

 

Summary:

Use the regular CAD, ID options to use the complete list of ID function.

Use the On The Fly option to quickly perform certain ID functions when the full set of CAD ID functions are needed. (and mutiple clicks away)
