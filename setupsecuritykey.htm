MENU : Settings - Configuration

 

 

All MODEL MAKER SYSTEMS software is protected by a HASP security key. This key must be plugged into a parallel port in your computer. If you have two Model Maker security keys and they are placed together in line the second key will not be seen by Model Maker, but a Model Maker key can be placed in line with any other software vendor security key.

 

Each program consist of a number of modules which are activated / de-activated through a setup file. Both the security key and a correct setup file must be available to enable the program in full access mode. A setup file will only work with a specific version of the software and each security key has a unique number.

 

 

Dongles

If, for example you have a security key number 96:15 you must have a setup file named MMS96_15.MRP

If your security key is faulty or unplugged or your setup file is incorrect or not available, this dialog box will be displayed.

 

 

Security keys found

Here the number of the attached security key will be shown if it is detected. If none is found:

 

               Check the HASP driver. >From the WINDOWS task bar select START - RUN to test the driver and run the program HINSTALL.

 

       Type C:- INFO (Accepting that Model Maker is loaded in to the default directory). If the driver is installed it will show the driver version number or else you can

install the driver by running HINSTALL again. Type C:-I ·

 

       If you have another computer available test the key on it as well. Your parallel port must allow for bidirectional communication. In the computer's BIOS settings

various port modes may be set. Get the help of a qualified technician to change the port modes. Test the key with each mode available.

 

 

If you have a network key select the 'RUN IN DEMO MODE' option and on the next dialog the 'SETUP' option. This will take you to the configuration dialog where you must make sure the 'SECURITY DEVICE PORT' is set to 'NETWORK' and not to 'SEARCH'.

Contact your program vendor.

 

 

Setup files found

A setup file corresponding to the connected security key's number must be shown here. After an upgrade of the software it might happen that the file is shown but not recognized for the program version. Contact your program vendor for a new setup file.
