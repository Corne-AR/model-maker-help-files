MENU : Utility - Electronic data

 

 

This utility may be used to download survey data from an electronic instrument to a user defined ASCII file. This file can then be imported and reduced to final co-ordinates and plotted on screen.

 

The file create button is displayed when the dialog box is activated. A file must be opened into which the data received form the instrument will be stored.

Click the Create File button and specify a file name and extension. Note the folder and filename that is used as this file may be used by MM at a later stage.

 

 

 

The interface setup button is displayed after the file has been created. If the button is displayed in this format , you must first activate the comms port.

Click on the button and create a link between the pc and survey instrument.

If the comms port is successfully opened, the open button will look like this and the transfer box is displayed.

 

 

Click on the receive data button. MM will send any input received from the instrument, through the opened comms port, to the new ASCII file.

The dialog boxes will be replaced by the receive data option.

 

 

When this dialog box is displayed, the communication lines are open and MM is ready to receive data from the electronic instrument.

MM does not prompt the instrument for data by sending out any instructions. The instrument must be manipulated and triggered to start sending data.

If at this stage you don't see any input coming through in the receive window, you will have to enter the correct commands on the survey instrument to initiate the transfer process. MM does not have the capability to trigger the instrument from the pc's end.

 

If all the data has been downloaded or you need to exit for any other reason, click on the Terminate transfer button.

 

Click the button to exit the download menu.
