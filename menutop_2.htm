MENU : Top of main screen

 

 

The functions on the top menu allow you to:

               Load and save data.

               Access the DTM data base.

               Check the status of the project.

               Output the plotted image to a printer and DXF files.

               Access the utilities options - to import electronic data and digitize data.

               Set default parameters.
