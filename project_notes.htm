When working on a project, you have the ability to add information into the project note editor.

 

 

To activate the Pro Notes option (if not visible), open the and check that Information - Filename is selected.

 

To make use of the Pro Notes, button and enter any information you want.

 

NOTE : Pro Notes are only stored when working with MAL files.

 

 

1) Check this box if the time when the file is opened until it is saved, must be listed in the editor.

2) Check this box if the user name and dongle number must also be recorded.

3) Option to save the info to excel so you could use a formula to calculated the total times.

 

If the boxes (1 and 2) is not checked, the pro notes will remain blank unless you manually enter any information.
