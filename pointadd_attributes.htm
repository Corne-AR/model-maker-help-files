MENU : Pnt - Add

 

This dialog box is displayed whenever a DTM point is graphically added to the database.

The point attributes, including the code, tag and elevation are defined in this box.

 

 

 

 

The default code of the previous point added is displayed. You may accept the code, change it or use the drop down button to choose a previous code from the list. If the default code ends with a numeric character (as in the example) and the INC? box is checked, the code number will automatically be incremented.

 

       Manipulate tag as you did the code.

 

 

 

 The co-ordinates of the point are defined. These co-ordinates were derived from the cursor position indicated on the screen. You can change the co-ordinates no matter how they were defined initially.

 

The elevation of the new point is defined. The value may be entered manually or derived from the following functions.

 

 

 

The interpolated elevation based on the triangle directly below the new co-ordinate will be calculated and used as the Z value. This function cannot be used if no triangles exist or the point falls off the triangle model.

 

 

The elevation will be calculated using the slope of an indicated triangle and extending it to the intersection with the new DTM point.

 

The elevation of an indicated existing DTM point will be used for the new elevation.

 

 

You will indicate a piece of CAD text. This text, if it can be translated to a valid numeric format, will be used as the elevation. This function is useful when you have a design model defined as CAD lines and text. Add the DTM point to the end points of the CAD lines and grab the elevation from the CAD text.

 

Enter the elevation. Use the drop down button to select a value from the history list. Use the button to zero the value.
