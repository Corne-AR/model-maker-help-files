LOGIC WHEN SELECTING LINES

 Top  Previous  Next

 

Here are two examples of what will happen when selecting a line:

 

 

EXAMPLE 1:

If you select the line on the right hand side of the "invisible centre line", the horizontal alignment will track back to the start of that line, then go back up and continue along the line leading off to the right.   To bypass this line, you need to break / move it away from the main line so that the track will carry on past it.

If you select the line on the other side of the "invisible centre line" the horizontal alignment will go in the opposite direction.  This is because the point of selection picks the closest end point, therefore, use the "invisible centre line" method.

 

 

 

EXAMPLE 2:

If you select the line on the left hand side of the "invisible centre line", the horizontal alignment will track back to the start of that line, then go back up and continue along the line main line.

 

If you select the line on the other side of the "invisible centre line" the horizontal alignment will go in the opposite direction.  This is because the point of selection picks the closest end point, therefore, use the "invisible centre line" method.
