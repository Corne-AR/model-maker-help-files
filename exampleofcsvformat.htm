·Typical entries in a csv file. Note that each column must be separated by a comma (,)

 

The origin of this option is the format in which the Stardust DTM program saved data from survey data originally imported from various data logger programs.

·The first character of a line indicates the line information type.
·All directions are in the format (degree.minutesecond)

 

 

·Description of entry types as defined by the first character of each line.

 

{        Setup

 

The following info is defined in the setup line.

 Station name

 Y,X,Z coordinates of the station

 The instrument height

 

 

%        Orientation

 

The following info is defined in the orientation line.

 Trig name

 Horizontal angle

 Vertical angle

 Distance

 Prism height

 

 

e        Electronic (prism) Spot Shot

 

The following info is defined in the spot shot line.

 Point Code

 Horizontal angle

 Vertical angle

 Distance

 Prism height
