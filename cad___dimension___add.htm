MENU : Cad - Dim - Add                                

 

TYPE Page

 

Seven different types of dimensions are available.

 

 

Linear

 

Indicate any 2 points. The linear distance between the 2 points will be calculated and the dimension line will be placed between the 2 points.

 

Horizontal

 

Indicate two points. The horizontal distance between the 2 points will be calculated and the dimension line will be placed horizontally between the 2 points.

 

Vertical

 

Indicate two points. The vertical distance between the 2 points will be calculated and the dimension line will be placed vertically between the 2 points.

 

Survey

 

This option calculates to the linear dimension between to indicated points. However, no dimension line is drawn. Only the distance and direction between the 2 points are displayed.

 

Circular

 

Indicate a point on the circle. A dimension line will be drawn across the diameter of the circle from the indicated point through the centre point.

 

Angular

 

The intersection angle of the indicated arc or 2 lines will be used to calculate the dimension.

To calculate an arc's dimension, click onto the arc.

To dimension the angle between 2 lines, indicate the first line. Model Maker will then ask you to indicate the second line. The position of the dimension will be placed at this

cursor position.

 

Radial

 

The radius of an indicated circle/arc will be used for the dimension. Indicate a point on the circle/arc. The dimension line will be drawn from the circle centre to the indicated point.  

 

Arrow type

 

 

There are 3 standard arrow types. You may define another 6 types.

These types must be defined in the MODEL.SHM file and must have the names #ARROW4, #ARROW5, etc.

 

Note that the name is always ARROW and the number at the end of the name denotes the user arrow type that may be selected

The arrow shape in the MODEL.SHM file is displayed in true-to-scale millimeters. The defined arrows must be pointing to the right.

 

 

This is the type of dimension to be defined and is selected when the dimension is initially added.

Hint:

You can use the editing  option immediately after you have defined the dimension.

 

Survey Notation

 

This menu sets various dimension options. The number of decimal places for lengths and angles can be defined. The character defining the decimal point can be changed when survey dimensions are used. You may specify only length, only angle or both properties to be recorded.
