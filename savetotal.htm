MENU : File - Save - TOT

 

This is an ASCII file which saves the points and triangles in one file. It will save the identified points or the whole project. Use the  POINT ID  menu to identify the correct points. If only part of the project is identified, Model Maker will inspect the triangle model and include in the total file only the triangles that are connected to identified points.

 

This format of saving data was initially not intended to be used to save data on a long term basis. It was designed as a method to transfer a model between projects. You may however use this method in any way applicable to your situation.
