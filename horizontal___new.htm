MENU : Hor - New                                

 

There are three ways of defining a horizontal alignment (see below):

1.                Pointing to each PI on the screen.                

2.                Selecting a polyline drawn in CAD.

3.        Direct input of co-ordinates in the text editor.

 

 

Data

When defining an alignment, you will be required to enter a name for the alignment eg 'Main Road To Nowhere' (this is not the file name). This is for your reference.  It will be the first line in the horizontal alignment definition file (PID).  It will also be  used as a default layer name when copying the alignment to CAD.

You must also give the alignment a starting chainage or value. Typically, it is zero, but negative starting values are allowed. The default is zero.

 

 

HOW TO - Define an alignment by:

 

1.  Pointing to each PI (Point of Intersection)

Define the alignment in free hand mode. Select the position of the PI using any of the snap modes. After each point is selected, the Radius, PI Name, and Transition lengths IN and OUT can be added or changed. Once the end of the route is reached, press ESC to terminate the data collection operation. Select Plot to show the alignment and curve data.

2.  Selecting a poly line

This is an intelligent option, using existing CAD lines and arcs that already define a route. If lines have been filleted with set radii, Model Maker will track the line by calculating the theoretical PI as it goes. For this function to work correctly, your CAD work must be correct. If you find that the alignment does not go as far as expected using this method, then check the CAD work at the point where the alignment stopped. Use the ' Plot ' option to see how far the alignment has gone. You will probably find that there is a break in the line or that lines overlap.

 

Once all the PI's are found, you can choose to ' use the current CAD arcs as initial radius values '. Model Maker will assign the radius of the curve closest to each PI. The curves need not be tangential to the lines defining the PI's. Check the alignment on completion to make sure that the correct radii has been assigned. Model Maker cannot tell which curves belong to which lines. It is possible to have two or more curves near the start of a line. This could be intentional or due to sloppy drawing. Therefore, check the result at the end.

 

When you select the line, the end of the line closest to the cursor will become the start point of the alignment. You do not have to change your cursor snap mode because Model Maker does it for you. Once Model Maker defines the route, it will ask for a name for the start point. By default PI1 comes up. If you accept this option the following PI's will be named PI1, PI2, etc. Any name ending with a numeric character will automatically be incremented for you. Remember that there is a limit of 10 characters for the names. Longer names will automatically be truncated to 10. There is no limit to the number of PI's.

3.  Text editor

This option allows you to type in each PI name, Y & X (Southern hemisphere) or N & E (Northern hemisphere), radius, and transition curve lengths in and out. The transition deflection angle is calculated using the following formula:-

                         l^2/6RL

 

               where                l   = distance along transition

                         R = radius of circular curve

                         L  = length of transition

 

 

Co-ordinate

This options allows you to Insert or Delete a co-ordinate.  An arrow will appear. Click on the line you wish to delete or insert.

 

Print

The Print button will automatically print  your information.  The Setup button will take you to the PRINTER setup.

 

Editor

The Edit button will take you to the File - Editor.

 

Files

Use the Open option to open an existing PID file. Use the Save option to store the data. The alignment is stored in an ASCII file with a PID extension.

 

Display

Do a normal plot before tracking a poly line

You often need to zoom into an area before you are able to select the start of a POLYLINE for tracking. When the tracking starts, it often moves off the screen and cannot follow the process. Check this box to perform a normal plot .
