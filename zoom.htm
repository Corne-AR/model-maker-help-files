MENU : Zoom

 

 

 

ZOOM

Indicate the 2 opposite corners of the new zoom area. Identify the indicated corners in any sequence. Model Maker will perform a replot with the new screen limits as indicated.

 

Note:

The screen ratio will not be distorted irrelevant of the shape of the indicated zoom rectangle. Use the odd zoom function to perform the zoom as indicated.  

 

ZOOM  :  Oddzoom

The elements within the indicated zoom window will be stretched out of proportion to the screen ratio.

Used this function when the survey points that are very close together in a specific direction.

 

ZOOM  :  Add

Up to 10 zoom windows can be indicated and saved without having to do a normal plot every time.

 

ZOOM  :  Delete

Delete any one of the ten previously defined zoom windows.  

 

ZOOM : Recall zoom

You will be prompted for the zoom window number to display. A dialog box will be displayed indicating the positions of all the stored windows. Point to the zoom window you want to use.

 

ZOOM  :  Clear

Clear all the zoom windows that are set.

 

ZOOM : Save Window

The limits of the current screen will be stored.

 

ZOOM : Recall Window

Model Maker will recall and zoom to the limits of the saved window.

 

ZOOM : Coordinate

Zoom to a specified co ordinate
