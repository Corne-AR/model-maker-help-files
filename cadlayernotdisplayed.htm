CAD : Layer not displayed

 Top 

 

MENU: Cad - laYer - not displayed                                

 

This menu is displayed as a precaution when the active drawing layer is switched off. It warns you that new elements added to CAD using this layer will not be visible on a replot because the layer is de-activated.

 

Ignore

If you ignore the warning you may proceed to add elements to this layer but keep in mind that they will appear to be gone when doing a replot.

 

Display

Select display to reactivate the layer and make the elements on that layer visible again.

 

Display menu

You will be transferred to the CAD display menu. Select or deselect the layer to be displayed.
