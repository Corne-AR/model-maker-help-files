MENU : Pnt - Add

 

DTM points (co-ordinates) may be added to the active project in a variety of ways. Detailed below are five different options used to add points.

 

Add to project

 

Select the DTM project number to which the points will be added.

 

Incorporate new points

 

If this box is checked and the new point is added onto an existing triangle, the point will be incorporated into the model and the triangle updated accordingly.

If this check box is not checked and you want to incorporate the points manually you will first have to delete the triangle that covers the new point and then use the continue search or add triangle options to cover the area again.
