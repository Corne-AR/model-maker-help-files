MENU : File - Backup

 

 

Model Maker has an automatic backup system. All the system data is saved to the backup subdirectory on a regular interval. You define the interval and set the Files-backup menu. Press F4 to manually backup all the data. Data from the backup files may be loaded by :

 

1 : Selecting the backup button on the startup menu.

2 : Selecting the Files - Backup - Load menu.

 

EXAMPLE:

 

Access to the backup menu allows you to set the backup time interval, open the previous backup file, or save a backup file.

Use the F4 key to manually make a backup of the data whenever necessary.

Up to 10 backup versions are saved. Each time a new backup is saved in position one, all the previous versions are moved down one position and version 10 gets lost.

 

Open

Model Maker will load the data from the specified backup version. This function recovers work that was in progress in the case of a system crash. If the computer was turned off or re-booted since the last session, Model Maker might not recover all the data. Remember to save the recovered data again in the normal formats.  

 

Idle time

The backup system monitors the following.

1) Changes made to the data.

2) Changes to check for on the next time interval.

3) Mouse actions. When the mouse is idle for the specified time period (or more), the program will              automatically do a backup.

 

Hints:

The backup system is not intended to replace regularly saving of data. Always save your data when exiting the program or loading other data. The backup system saves working data in case of an accidental loss of data. For example, if the backup timer is set to 10 minutes, you would lose no more than the last 10 minutes work if the system crashes. Model Maker will try to load the data used in the last working session. This function recovers work that was in progress in the case of a system crash.

If the computer was turned off or a re-boot was done since the last session, Model Maker might not always be successful in recovering all the data.

Remember to save the recovered data again in the normal formats.
