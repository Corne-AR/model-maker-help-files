MENU : Point - Delete - Graphically

 

Delete DTM points graphically or directly by editing the data base.

 

1) Delete single points by indicating them individually.

2) Delete points within an area by indicating the area.

3) Delete all the identified points.

 

Mark point as deleted

Points may be marked as deleted, meaning that the code is changed to '*.*'.

A point with this code in Model Maker is inactive but remains in the database.

Remove de-activated points from the database using the DATA menu.

 

The WIPE TRIANGLES? check box allows Model Maker to remove any triangles connected to the de-activated points. This is the best option since triangles connected to de-activated points can lead to problems if not controlled correctly.

 

Remove point

Remove the points. The triangles are updated automatically.
