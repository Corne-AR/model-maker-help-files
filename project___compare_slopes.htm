PROJECTS : Compare slopes

MENU : prO - cOmpare2

 

This is a specialized function mostly used on a terrain after some rehabilitation has been done

 

After an area has been rehabilitated the following two criteria are used to determine the successfulness of the operation.

1) The slope of the terrain must be gentle within certain parameters.

 

2) Adequate topsoil must be placed over the area and is measured in topsoil depth.

 

Model Maker produces a report to determine if the terrain meets the industry standard specifications.

 

Two projects are used to do the comparison.

 

1) The first project will hold the -elevations- that are to be checked. Normally, a project will consist of points that model a terrain. The elevation above sea level is the Z value. In this function the Z value should not be a true elevation but the height (thickness) of the topsoil on the terrain.

Usually, this information can be obtained by drilling holes in the rehab area to measure the topsoil depth. The position of the drill holes would then be surveyed and when entered into Model Maker, the Z value will be substituted with the measured topsoil depth.

 

2) The second project consists of a normal topographic survey of the final rehabilitated terrain. From this model, the program can calculate the slope of all the triangles on a regular grid. By comparing it to the elevation of the first project (topsoil depth) at the same co-ordinate position, the program can determine the relation between the 2 items.

 

Loading the two projects

 

1) Activate project 1. On a regular grid over the rehabilitated area, measure the depth of the topsoil layer at each position and also record (by means of survey methods) the Y and X co-ordinates of each position. This will result in a list of co-ordinates. Activate project 1 and enter these co-ordinates into Model Maker and enter, for the elevation of each point, the depth of the topsoil at that co-ordinate. Create the triangles. This model represents the thickness of top soil placed over the area.

 

2) With general survey methods, obtain survey coordinates of the rehabilitated area after the topsoil has been placed.

Use project 2 and load the survey points into this project and create the triangles. This model represents the surface of the completed rehabilitated area.

 

 

Enter up to 4 different interval values. Model Maker will on a regular grid inspect the elevation (topsoil depth) of project 1 and categorize it according to the given depth intervals. If, for example, the topsoil depth at a certain grid point is 0.65m, Model Maker will compare it to the list and put it in category 2 because it falls between the 0.5m and 1m interval.

Always start with the smallest value in the first entry box.

 

Enter up to 4 different slope values (%). If the slope on the model in project 2 is within the specified intervals, the value will be added to the report in the appropriate category position.  If, for example, the slope at a certain grid point is 1.1%, Model Maker will compare it to the list and put it in category 1 because it falls between the 0% and 2% interval.

Always start with the smallest value in the first entry box.

 

Copy index text to CAD

 

If a grid point falls within any of the above specified intervals, a text value is outputted on the screen. If the check box is selected, the text will be added to CAD as a text element and placed on the specified layer name.

The text will consist of any combination of 11, 12, 13, 21, 22, 23, 31, 32, 33 and will always have 2 letters. The first indicating the elevation interval the point fall into and the second the slope interval.

EXAMPLE

:  Elevation intervals        0                0.5                1.0                2

:  Slope intervals                0                2.0                5.0                10.0

:  Index                                              1                  2                  3

 

An index of 12 means the elevation (topsoil depth) is between 0 and 0.5m and the slope is between 2% and 5%.

An index of 33 means the elevation (topsoil depth) is between 1 and 2m and the slope is between 5% and 10%.

 

The areas on the plan can easily be identified with this numbering system.

 

Copy index points to DTM

 

The same system applies as for CAD text. The grid points will be copied to the indicated project number and the code of the points will indicate the index position. By activating this feature you can then go and activate the third project (that now holds the result points) and identify points of certain codes.

 

The density of the grid points used to inspect and compare the 2 models.

 

After clicking the OK button, indicate the area within which the calculations must be done. Normally this would comprise the whole survey model.

Model Maker will, based on the selected grid interval, inspect both models, project  1 (for topsoil depth) and project 2 for the slope value and then produce a report of the actual combinations.

The result = area m2 within each category.

Note that the report consists of a 3X3 array and can have 9 possible entries. Within each of the 9 positions, the area of the terrain which falls within that category, is reported.

 

In the above table, it is easy to determine that a total of 32300m2 (index 33) had a slope of steeper than 5% but less than 10% and that the area was covered with more than 300mm of topsoil.

 

The best rehabilitated area would be the 31 index.(26700m2), meaning the terrain has more than 300mm topsoil and is not steeper than 3%.

 

To determine if the rehabilitation was completed satisfactory, one has to compare the results to that of the desired standard. If, for example the standard specification requires that 50% of the terrain must have a topsoil depth of 0.1m or more and a maximum slope of 5% or less, use the following calculation.

 

The topsoil categories that comply with the specification would thus be: 2 and 3. (topsoil between 0.1 > 0.3 and 0.3 > 1.0)

The slope categories that comply with the specification would thus be: 1 and 2. (slope between 0.0 > 3.0 and 3.0 > 5.0)

 

The combinations are then : 21, 22, 31,32

 

The total area within specification is then:  26700+12200+29600 + 11300 =                79800.

The total area is :                                                                                                                155900.

The percentage within specification is:        79800/155900                                                51%

 

EXAMPLE: index output on screen:

 

 

Model Maker uses the category combination to graphically indicate the state of the terrain at any point. In the example the 32 codes indicate that the rehab terrain in that area has more than 0.3m of topsoil and the slope is less than 5 percent.

To pinpoint all the areas that are not satisfactory, locate all the codes of  11, 12, 13, 23 and 33.
