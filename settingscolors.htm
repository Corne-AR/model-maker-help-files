MENU : Settings - Colors

 

Use this dialog box to set preferred colors for the following defaults.

 

Projects

 

Set the default screen colors for the DTM points, triangles and contours of the 50 projects.

 

A) Click on the color panel that you want to change.

B) A colour grid showing the available colours will be displayed from which a new color can be selected.

 

C) Just as a reference, the plotter color and line thickness is shown for the selected screen color.

 

This color grid (B) with the available colors is defined in the Pen Palette tab.

 

 

Example: Project 1 (dtm points) is drawn with pen number 2. (green). If the color of pen no. 2 is changed on the Pen Palette tab the dtm points of project 1 will automatically be drawn in the new color of pen 2.

 

 

 

 

 

Use the project selector to scroll through all 50 projects.

 

 

 

 

 

Menu

Four panels representing the side borders are displayed. Click on the panel, a colour definition dialog will be displayed from which a new color may be selected.

Three other panels are also displayed. The second one on the left represents the color of the text in the main menu panel. The other two represent the top and bottom menu colors. The active color defines the color of the menu which accepts direct key board inputs. Use the F8 key to swap the top and bottom active menu.

 

 

DXF

In this dialog box you may change the pen numbers that must be used when a DXF file is created. A .dxf file is defined by specifying pen numbers and not pen colours. For that reason, Model Maker allows you to map a pen number used on the screen to a different pen number when defining an element in the .dxf file.

For the DTM, select the pen numbers to be used in the .dxf file when defining the frame, grid, DTM points and contours.

 

For the 255 CAD pens, select the appropriate pen number to use in the .dxf file.

With this facility you are able to limit the pen numbers defined in the .dxf. If you know that the program receiving the .dxf cannot accommodate a line drawn with pen number 64, then map the 64 pen to say pen number 16. By doing this, when the .dxf is defined, elements drawn in Model Maker with pen 64 will be outputted to the .dxf as using pen number 16.

 

Model Maker now shows the pen numbers and associated colors. This makes it a lot easier for users to see what their end result will look like.
