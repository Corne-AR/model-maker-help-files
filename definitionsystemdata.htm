DEFINITION : System data

 Top  Previous  Next

 

Model Maker operates on basic data called system data. System data are used by the following applications.

 

1) DTM                        This consists of the co-ordinates and triangles of the 3 different projects.

                         The points may be saved as tachy (mms), ASCI (yxz), co-ordinate (mmb) or total (tot).data.

                         The triangles are saved in triangle files (tri) or as part of the total (tot) file.

 

2) CAD                        The CAD data is made up by the lines, text, arcs/circles, dimensions, arrows and hatch elements.

                         This data is saved in the CAD (kcd) files.

 

3) STRING                The string data consists of the strings as defined by the options in the string menu (GXT).
