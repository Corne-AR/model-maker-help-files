Indicate

 

The head of an indicated arrow is changed to a new head type.

 

Specify

 

All the arrows having the selected head type will be changed to a new type.

 

All

 

All the arrow heads will be changed.

 

iDentified

 

The heads of all the identified arrows will be changed to the new head type.

 

 

 

 

Drawing the symbol in CAD

 

Run Model Maker and set the scale to 1:1000.

Use the normal line functions in CAD and draw the lines of the shape you want to create.

Use the ID function and identify all the elements of the symbol.

In the CAD General menu, locate the L-shape option and save the identified elements to a file.

 

Using the symbol editor in the Irrigation menu

 

Run Model Maker, activate the Irrigation menu and proceed to the defaults page. Locate the shape editor and use that to manipulate the SHM file. Remember that changes made in this editor will only become effective after Model Maker is exited and started again.

 

Arrow head type

 

Arrow heads are usually a standard shape in Model Maker. You may define another 6 shapes numbered as 4, 5, 6, 7, 8, 9.

Define the new types in the MODEL. SHM file. Name them #ARROW4, #ARROW5, etc. Note the name is always ARROW. The number identifies arrow type.

The arrow shape in the MODEL. SHM file is displayed in true-to-scale millimeters. The defined arrow must be pointing to the right.

New arrows will always have the head type of the last defined arrow.

 

Definition of a segment

 

MMW.SHM file holds the definition of the shapes that make up the arrows and typically looks like this:

# ARROW 3

0.00 2.00 0.00 -2.00

0.00 -2.00 4.00 0.00

4.00 0.00 0.00 2.00

 

Each line defining a segment has the following format and consists of 4 entities.

StartX StartY EndX EndY

The 4 entities define the X and Y co-ordinates of the start point and the X and Y co-ordinates of the end point of each line in the shape.

The co-ordinate values on these lines should be entered as real millimeter values and separated by one or more spaces.

 

The SHM file can hold up to 50 symbols and each symbol  may be defined by a maximum of 80 segments.

The blue lines are only displayed to assist you in understanding the local grid concept.

The horizontal blue line is the X axes reference and the vertical blue line is the Y axes reference.

When prompted to indicate a base point, indicate the origin (0,0) position of your shape.

 

After the symbol is saved, use an ASCII editor to open both the new file name eg. ARROW.SHM and the original MMW.SHM file.

Cut and paste the content of the new file into the MMW.SHM file. The new symbol can now be included in any line definition in the STM file.
