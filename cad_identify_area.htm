MENU : Cad - Id - do_Id - Area                                

 

Identify a polygon area. Specify if you want to identify all the elements inside or outside the area.

 

Id CAD Area Inside

 

Identify all the CAD elements with endpoints within the polygon area. Use the Include/Exclude option to change the Id format

 

Id CAD Outside

 

Identify all the CAD elements with endpoints outside the polygon area. Use the Include/Exclude option to change the Id format.
