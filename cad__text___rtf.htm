MENU : Cad - Text - rtF

 

This function allows you to add text in an editor and then display it along with all the other cad elements on the plot.

 

The editor is used to enter the actual text to be displayed.

Use the regular editor options, ie. font size, font type and other properties to change the text appearance.

 

Use the first button to define the top left and bottom right of the text area on the screen.

Use the second button to maintain the area size but move the area in total by defining the bottom left.

Use the color bar to define the color of the selected text.

 

 

The normal display format would be text only. A pattern can be associated with the rtf text. This pattern can be placed and moved along with the text box.

Use this option to specify which of the entities must be displayed.

 

 

If a frame is drawn around the rtf block, specify the shape of the corners.

Select if a line frame must be drawn around the block.

Specify if the block must be filled with a back ground color.

 

 

Specify the frame color and line width.

The width of the line is specified in actual millimeters. On screen it will appear wider depending on the plotting scale and zoomed state. On paper it will be drawn in exactly the specified mm width.

Set the back ground

 

 

A left margin may be set. Any text that is left aligned will only start the specified number of mm away from the left edge. This allows you to enter text without having to put spaces at the beginning to move the text away from the border.

 

Set as default will save the settings and use that each time a new rtf entity is added.

 

 

If a pattern is associated with the rtf text, use this button and input box the specify the pattern name.

 

 

Use the scroll box to specify the relative pattern scale.

Initially a pattern is placed at the top left corner of the rtf box. Use the Move option to give the pattern it's own base coordinate.

 

 

 

If you have many rtf in the cad, use the rtf List to switch between them for editing.

 

The first line of each rtf is shown in the list.  Click on the selected line and then click on the Editor tab. The selected rtf page is shown.
