MENU : Input Co-ords - Electronic field book

 

Station number

This is the number of the current station.

 

Check boxes

Normally these boxes will always be checked.

When would you uncheck one? When you do not want to make use of the measurement in calculating the correction angle.

               When you have surveyed more than 1 beacon for orientation purposes and you find that either you have the wrong co-ordinates or the wrong name of one or more                 of the beacons.

               When you find that a particular direction looks totally out of line with the others and want to ignore it.

 

Notice how the 3rd measurement is totally out of line with the previous two.

 

This can either be that the wrong beacon name is specified, the wrong co-ordinates used or that the surveyed angle is off.

 

In this example the effect of the 3rd measurement has been eliminated by unchecking the box on the left.

 

Notice that in the Correction column a zero value is entered and that the average correction is only calculated based on the values of the first 2 lines.

 

Click the RESET button to set the Accepted angle equal to the Average value.

 

Use distance weighted average

If this box is checked and more than one orientation beacon is surveyed, the distances to the beacons will be taken into account when calculating the correction angle. The measurements to the further beacons will have more weight than those to the closer beacons.

If the box is unchecked, the correction angle will be a regular average of all the calculated correction value.

 

Hold off calc.

If this box is checked and in the electronic field book you have surveyed multiple orientation beacons from the station, the Correction Angle dialog box will only be displayed once - after all the beacons measurements are read from the file and before the first spot shot is reduced.

If it is not checked, the Correction Angle dialog box will be displayed each time an orientation beacon reading is read from the file.

 

Average correction angle

The calculated average correction angle is displayed based on the weighted or unweighted mean value.

 

Accepted correction angle

This the final value that will be applied to the station and would initially be the same as the average correction angle.

               You may override the calculation process by entering your own correction value in here.

               Whenever the average and accepted correction angle values are not the same - you have changed it manually or some other check boxes were switched on and off - click the RESET button to use the calculated average as the accepted value.
