POWER LINES : Pole Info

 Top  Previous  Next

 

MENU : gXt - power

 

This dialog box displays the relevant information for the selected pole.

POLE DATA : The pole name (node name) is shown. This name may be changed by selecting another from the short list.

 

Use the button or double click on a selection to place the selected pole type in the name edit box.

Use the edit list to setup a range of poles.

 

The chainage value is the distance that the pole is placed along the design line.

 

The elevation is the value that you entered. Or, if the pole was planted automatically by Model Maker, the bottom attachment height is the elevation. Check the STRAIN NODE box if the pole must be a fixed pole.

 

SPAN DATA : The actual and maximum values for the different spans are displayed. The bitmap indicates that the maximum value is exceeded while the picture indicates a good value.

 

Node Types

Nodes are normally pinned.

 

Explaination of images on the vertical view.

 

 

A) Indication of a crossing service.

B) White line : Position and hight of lowest point (to the ground surface) on the span.

C) Gray line : A node on the profile defining a bend in the plan position of the line.

  Plan view of a bend node indicated with a gray line in vertical view.

 

D) A node (pole) that has got up-lift. This will be indicated in green for Strained node and red for pinned and suspension nodes.

E) A red arrow pointing towards a span indicates the span is more than 1.5 times the length of the span on the opposite side of the arrow.

F) The straight line between nodes.

G) The hot and cold powerline profiles.
