IRRIGATION : Hydraulics - Total Head Calc

Use this function to calculate the various values of the complete pumping system.

 

NOTE : This function is normally used to finalize your design. To determine an approximate pumping pressure at the start of the design in order to calculate pipe sizes etc. use the proposed pump pressure functions on the pipe sizes page.

 

 

 

SUCTION

 

Enter all the relevant suction information in this panel.

 

The flow will be entered by the program based on the required flow needed in the critical shift.

 

 

Critical Valve

 

Based on the design information the program will determine which valve is the most demanding on the system and the critical information to determine the pump pressure will be taken from here.

 

 

DISCHARGE

 

Enter the values that describe the discharge.

 

If the discharge is not a factor, enter zero values for the length, height and secondary losses.

 

 

 

Total Dynamic Head

 

This part of the function calculates the actual pressure required at the pump to run the system.

It accumulates the required pressures from the other input boxes and allows you to cater for filters and an over design factor.

 

Select the pump type. If it is a turbine no other results are shown. For a centrifugal pump the following inputs and results are shown.

 

Enter the required NPSH based on the pump you have selected. This value can either be entered as a pressure value (first input box) or as an equivalent length value (second input box).

Directly below the input boxes for the required values you will find the available NPSH in both pressure and length units.

 

Also check the pump elevation as the height above sea level has an effect. By default the elevation of the pump (as it is in the irrigation design) is input here. If by any chance your survey data's elevations are not in relation to actual sea level, enter the correct pump elevation here.

 

The minimum submergence is calculated based on the suction values and the velocity of the water.
