CAD : Lines - Spline

Menu : Cad - Line - Special - Spline

 

Spline a set of CAD lines based on user defined parameters.

This same function is also used when splining strings. (Gxt Add Spline)

 

 

For CAD lines:

Model Maker allows you to indicate the lines to be splined. Start from a side and indicate every line in sequence. The lines must follow each other and their endpoints must meet.

Press Escape after the last line is indicated. The spline parameter dialog box is displayed.

 

For STRINGS:

Indicate the string.

 

Spline parameters

 

 

 

 

 

 

 

General

Number of segments/nodes :

 

This is the number of new nodes per line segment to be added to the original line to perform the splining. Two factors need to be considered.

         1) More nodes will produce a smoother result, but more lines will be added to the                      database.

         2) Too few nodes often make the spline too rough.

                    A practical number of nodes is between 2 and 10.

 

 

 

 

Spline amount :

This controls the amount that the spline will deviate from the original position. Enter any percentage up to 100. Model Maker will check the spline amount and only spline up to the maximum practical size.

 

 

 

Start deflection

Choose from 2 variables.

 

1)The actual angle in degrees that the first spline segment must deviate from the original line. The maximum angle should not exceed 90 degrees. The ? button lets you indicate a  

   line to calculate the start direction. This usually is the line before the 1st spline line.

 

2)The relative deflection percentage. This value controls the deviation angle for the first

   segment from the original.

 

 

 

End deflection

Select an actual angle or relative deflection percentage as for the start point.

 

 

Test

Use the test button to view the splined result after making changes.

 

Press OK if the spline is correct. The original lines will be replaced by the new lines that makes up the spline.

Press CANCEL to discard the spline and keep the original lines.
