MENU : Cad - Text - Search_ replace.

 

The dialog box has 3 main components:

 

 1 

Here you enter the text string you want to search for. To activate the search, either press Enter in text panel A or click on the >> button (a)

The actual text found in the cad will be listed in the (4) list box. In this example, the cad contains 2 text strings with the word 'DATE'.

 

 2 

To show a list of all the cad text elements, click button B. The text is shown in list box (3).

 

 3 

The numeric value in this list box indicates how many instances of this text is present.

 

 4 

Click this to open the text editor.

 

 5 

The list box showing the found text.

 

 6 

Click to switch to the Search and replace dialog box.

Examples showing the functions in action.

 

 

The text 'aw' was entered in the search box and enter was pressed.

 

The drawing was scanned and all the text containing 'aw' or 'AW' is shown in list  5  .

 

Notice that list  3  is empty. To simply search for any part of a string, the complete list does not have to be shown.

 

When list  5  was populated with the found text, the graphics screen automatically zoomed to the first entry.

To zoom to the second entry, double click on line 2, list  5  . To zoom to any of the text lines in  5 , just double click the entry.

 

To edit the text, click on  4  to open the text edit dialog box.

 

 

 

 

If you do not want to enter text in the search box but want to zoom to a specific text string in the cad :

 

 1  Click the populate button. The list box is filled with all the text strings.

 

 2  Double click on the text you want to zoom to.

 

The text is automatically copied to the search box and the screen is zoomed to the text position.

 

A copy of the text is also shown in the found list box. If there are multiple instances of the text is in the cad, all would have been shown. See next example.

 

 

 

In this example, there are 10 cad text with the wording  D/P in the cad drawing  1 

 

By double clicking the text, the string is copied to the input box and since there are 10 different text in the cad, they are shown in list box  2 .

 

Double click each one in turn and the graphics screen will be positioned with that cad text in the center of the screen.

NOTE:

 

The Quick search & zoom dialog box and the Search & Replace dialog box are both accessed by using the CAD, Text, Search option or the speed button.

The dialog box that opens, will be the one you used the last time. To use the functions in the other dialog box, click the button in the bottom left of the dialog box that are shown.
