DEFINITION : Speed button

 Top  Previous  Next

 

 

 

Speed buttons are displayed along the left border of the graphics screen

 

 These buttons provide a short cut to certain functions which are normally only accessible through the regular menu structure. The buttons on the example provide short cuts to triangle functions, add triangle, delete triangle and edit triangle. The advantage of the speed buttons is that you do not need to change the main activity menu to get to the desired command. The speed button may be used irrespective of the current selected main menu item.

 

Use the SETTINGS  Speed Menu function to select the options on the speed panel. 13 buttons are available. The dialog box controls the speed panel's visibility.

 

Note: After a speed button is used, press Escape to end the function before another speed button function is selected. The action of a speed button activates an internal loop in Model Maker. Failure to exit each loop before another loop is entered, may result in Model Maker running out of resources.
