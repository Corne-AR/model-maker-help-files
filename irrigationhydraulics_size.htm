MENU : Irr

 

Irri Maker assists in the correct selection of pipes for a specific application. If the active block is block 0, the main line sizing dialog boxes will be present. For the other blocks, the functions to size the laterals and manifold pipes are displayed.

This page will not be available when calculating the hydraulics of a looped main line.

 

Regular blocks

The pipe sizing is done according to specified pressure loss tolerances.

 

In this dialog box, enter the percentage factor that the pressures at the lateral nodes may vary from the design value. If for example the design node pressure is 20 meters and the percentage factor above and below is 10% then the laterals will be sized to maintain the pressure at each node between 18 meters and 22 meters.

 

Max # of pipes

Irri Maker will select a different pipe size when necessary. This could result in multiple size changes along a lateral. You may force the pipe sizing function to use no more than the specified different pipe sizes by setting a limit.

Press the LATERALS button to size the pipes.

 

In this dialog box, enter the percentage factor that the pressures along the manifold nodes may vary from the design value.

Press the MANIFOLD button to size the pipes. If the Lateral Also box is checked, the laterals will be sized automatically after the manifold is done.

For the manifold, also set the maximum number of different pipe sizes that may be used.

 

Depending on the selected tab, Irri Maker will limit the pressure differences to either a % of the design pressures or if the second tab, limit by pressure, is checked the pressure range will be controlled by the actual user defined pressure value.

EXAMPLE : If the design pressure of the sprinklers are for instance 25 meters, then a pressure limit of 10% or 2.5 meters will result in the same pipe sizes.

 

Velocity

Use the input boxes to specify the maximum velocity in the pipes. Model Maker will first try and use the smallest pipe diameter that would satisfy the pressure requirements. If the velocity in a pipe exceeds the maximum limit, the pipe size will be increased even if it means that the pressure range cannot be maintained.

         

After the pipes have been sized, return to the RESULTS page and rerun the calculations. All the information will be updated according to the new values.

Fixed Sizes

 

If the pipe has been set as a fixed diameter the line number in the first column will have a '#' and the pipe sizing will use the existing pipe diameter and not change it.

 

 

 

The update panel

 

 

 

The update panel is displayed before the actual proposed pipe sizing takes place.

On this panel, set the maximum allowable velocity in the pipes.

In the list box, select the maximum pipe size that may be used when calculating the proposed pipe set.

 

Model Maker will endeavor to find a combination of smallest pipe diameters that will convey the water from the pump to the valves so that the resulting available pressure at the valves is equal or more than the design value.

After the maximum velocity and pipe diameter are set, click on the update button at the bottom.

 

 

New proposed sizes

To calculate new pipe sizes, two methods can be used.

 

Pressure only

This method calculates the pipe sizes based on the user defined pressure at the pump and the desired pressure at the valves.

The method does not look at pumping costs but only at the smallest total pipe costs that is still hydraulically suitable.

 

MM determines the flows in each pipe within each shift and gets the maximum flow for each section.

The velocity limit is then applied to all the pipes and sized so it does not exceed the limit.

 

For each shift, the flow and friction losses are then calculated and the pressure at the nodes are compared to the design pressures. If any node pressure is below the required value, the route to the pump is traced and the pipe with the smallest diameter is increased. The flow and friction is again calculated, node pressures compared and pipes adjusted if needed. Once the node pressures are satisfied, the new pipe diameters are proposed.

 

If the proposed new diameters column displays ERROR then either the biggest pipe diameter was selected but still exceeded the maximum velocity limit or the biggest pipe was selected but the pump pressure was too low to provide sufficient pressure to the nodes.

 

Pressure, pipe costs and pumping costs

This method will compare different pump pressures, applicable pipe diameters and operating costs and optimize it to propose the cheapest system that still satisfies the hydraulic parameters.

The cost component of the system comprises of the capital cost and operating costs.

 

Capital costs

1)                The cost of the pump. This value is set in the node data base.when a part number is associated with the pump.

2)                Cost of the pipes. These values are set in the pipe data base and defines the unit cost of each diameter.

         

Operating cost

This component takes into account the cost of the energy to run the pump at the different pressure levels.

Finding the optimum pressure value

This page is used to activate the interpolation process.

 

 

 

Click on the best cost button to start the calculations. Model Maker will inspect the full pressure range of the pump and at each interval, define a set of pipes that hydraulically satisfy the system based on the pump pressure.

The pump price ( a constant value ), the cost of the pipes (for the specific input pressure) and the operating cost (to maintain the input pressure) are added up and the input pressure with the lowest total cost is then determined.

 

The best pressure to operate the system at is then indicated and the specific pressure is then displayed at the bottom of the page.

 

Using this optimizing process does not automatically change anything on your actual design.

If you want to apply the new pressure and associated pipe diameters to your project, click the Operate system button at the bottom. Model Maker will set your pump pressure to the calculated best pressure and also update the pipe diameters.

To view the hydraulic calculations of the changed system, click the Calculate button on the front page.
