MENU : Min - Mine Planning

 

Volume

 

 

 

 

 

The volume between all the loaded mining surfaces will be calculated.

 

This calculation is done on the gridded method and may be done within the areas as defined with the CAD Geo area functions (This is a separate module found on the CAD menu) or the total volume between the surfaces may be calculated.

 

It is useful to save the calculation results onto a spreadsheet file when the volume within Geo areas is used. These results can then be manipulated with any spreadsheet program.

 

 

DVolume

 

Calculate the volume between any design surface and the mining layers as defined in the mining module.

 

 

 

The volume will be calculated either as a single value (per mining layer) in relation to the design project

 

or

 

If geo areas have been defined, the volumes between each mining layer and the design project can be done per geo area.

 

The calculation method is as follows:

 

Define a grid. Model Maker will calculate the elevation at every grid point on the design project as well as on every layer of triangles defined in the mining module.

The difference in elevation between the point on the design project and each mining project is determined.

The volume of cut or fill is then calculated and the result added to a spreadsheet file containing the cumulative volumes between the design and mining surfaces.

The total cut and fill volume is then reported along with the split of  these volumes into their respective layer contributions.
