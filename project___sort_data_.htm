MENU : prO - Sort

 

Sort all the points of the active project by either the code, tag, Y, X or Z co-ordinates.

The database of the active project will be updated accordingly.

You can sort co-ordinates by increasing or decreasing values e.g. E - W or W - E.

The triangles in memory will be updated as well. Other triangle models that fit on the data will not be updated.  Therefore, this function must be used with caution if more than 1 model has been created from the same database.
