MENU : gXt - Utility - str_Dtm

 

A string in Model Maker exists independent of any DTM points in either of the 3 DTM projects, meaning that you cannot directly build triangles to nodes of a string. The string to DTM option will generate new DTM points on all the nodes of a string to the current active DTM project. This is usually used on general strings since alignment strings have nodes on the PI intersections only.
