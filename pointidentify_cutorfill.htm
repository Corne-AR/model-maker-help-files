POINT : Identify - Cut or fill

 Top  Previous  Next

 

MENU : Pnt - Id - do_Id - Cut&Fill

 

 

Identify all the points in one project that fall either above or below another selected project.

 

The current active project is displayed.

 

Choose to identify points above or below the other project.

 

Then, select the project against which the comparison must be done . It is possible to compare points in the active project, with the active project. In this case, the points in the active project must not be joined to triangles.

Model Maker will then go to each point in the active project, and calculate whether it is in cut or fill (with respect to the other selected project) at that position.
