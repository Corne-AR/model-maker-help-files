MENU  : Tri - Csearch

 

Generate Triangles Dialogue

 

This dialogue allows the user to setup and control the way the triangle network is created. There are basically 2 ways to create the network.

 

Quick Search

 

This option is very fast. It will create a complete network of triangles and there will be no holes. At this stage it cannot be given any real intelligence and it is left up to Model Maker to determine how the network will be created. It is approximately 10 times faster the intelligent method.

 

Intelligent Search

 

With this method the user can set certain constraints and thereby get Model Maker to compute the triangulation network the way it should be done. One can restrict the area to be used, the DTM points to be used and various other factors. Normal CAD lines can be used a breaklines to improve the network generation

Radius and Angle

 

This dialogue displays the current search radius (30m) and minimum angle (10 degrees) that will be used while searching for triangles. The values shown here can be altered in the Parameters box above. These values are only applicable if the Intelligent search method is used
