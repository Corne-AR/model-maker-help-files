MENU : Cad - Line - Polyline

 

 

Edit Polyline Nodes

 

1  Node Add

Add nodes to existing polylines. These nodes may be added to the start, end or between other nodes. When indicating the polyline, the cursor placement on the polyline is important. To add a new start or end node, place the cursor close to the first or last node on the first or last segment. To insert a node, place the cursor on the segment joining two existing nodes. To insert the node on the first or last segment, do not place the cursor close to the first or last node.

 

2 Node Delete

Delete any node of a polyline. If one of the last 2 nodes are deleted, the polyline will be deleted.

3 Node Edit

Edit the actual values of a node.

4 Node Move

Move a polyline node to a new position. The lines connected to the node will adapt to the new position.

General

 

5 Node extend

Extend a polyline's end node longer or shorter.

6 Polyline Coordinates

Use the Coord. Editor from the dialog window to change the coordinates of the polyline's nodes.

 

7 Convert Reg>POLY

Convert a regular created polyline (not real) into a real polyline. This can be done by selecting a single line or after you identified a couple of lines. All lines have to be connected (touch each other) in order to be converted.

Edit Full Polyline

 

8 Resize area

A polyline can be resized larger or smaller. In bother cases, a equivalent polyline will be drawn along the perimeter of the selected one. The new polyline will be either smaller or larger, based on the size that was entered compared to the selected polylines' size.

9 Poly Move

Move a polyline to a new location.

10 Poly Combine

Combine a polyline with another one by using Segment or Meet.

Segment will add a new line between the nodes of the 2 polylines.

Meet will add lines from the polylines to the point where the 2 selected nodes will meet.

11 Poly Copy

Copy a polyline to any new location.

 

12 Poly Attributes

13 Poly Split 1

Remove a line (leg) from a polyline.

14 Poly eXplode

Break the polyline into individual lines. All polylines or only selected polylines can be exploded.

 

15 Poly Fill

Fill all polylines or only identified polylines with an entered percentage of fill.

 

16 Poly to GXT String

Convert all polylines or only identified polylines to strings.

17 Check this box to return to this menu after the function is executed.
