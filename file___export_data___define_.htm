MENU : File - Export ASCII - General YXZ

 

 

Save the co-ordinates of the active project to an ASCII file in a defined format. Specify the order of the columns and the column delimiter character.

 

A regular DTM point in Model Maker consists of 5 attributes.

 

C - Code :                The description of the point.

T - Tag :                        The second description of the point.

Y Co-ordinate                The Y or Eastings co-ordinate

X Co-ordinate                The X or Northings co-ordinate

Z - Elevation                The height of the point.

 

Any combination of the specified capitals may be used in the definition. Enter a combination of eg.YXZ that would only place the co-ordinates in the file.

 

 

Select the type of character that will be placed between the columns. Certain programs will only operate correctly if the appropriate delimiter is specified. The space character is the default value for Model Maker.

 

 

Select the number of decimals. This controls the accuracy of the exported file. Values higher than 6 are not meaningful.
