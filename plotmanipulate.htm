MENU : Top action bar on main screen

 

The top action bar of the screen holds the functions to manipulate the plot.

 

 

The top menu is always active and manipulates the graphics.

               The cursor shape will be an arrow pointing to the top when any one of the top menu functions are active.

         The function needs to be completed or Esc pressed before the cursor will resume normal action again.

               The cursor will be a magnifying glass when the zoom menu is active.

               The functions can be selected by pointing to it or by pressing the capital letter of the button.

 

Note:

the top and bottom menus are coloured red or blue. The red menu is the active one, meaning that Model Maker will activate the function with the selected capital letter from there. Use the F8 button to select the active menu.

 

               The current CAD line parameters and CAD text parameters are displayed on the second line of the menu when the attribute panel is activated

 

 

The plot is spaced so that all the points are displayed on the screen. This command is used as an initial plot, or to see all points after a magnification.  

 

The plot is repeated using the same plotting area. This is used when a fresh plot of the zoomed area is needed. The difference between normal and replot is that a replot is done on the same area, whereas normal fits the whole area on the screen.  

 

 Move the center of the screen to an indicated spot, keeping the same zoomed state.

 

Plot the screen to the previously plotted window. Up to 10 previous windows will be saved and redisplayed in a revolving manner.

 

To use the magnify option, move the cursor to the place you wish to enlarge. A rectangle image will indicate the magnified area. When you press Enter, the area around the cursor will be enlarged by a factor the set factor

Alternatively, use the Page up and down keys to increase or decrease the magnify area. Press and hold down the CTRL key when pressing the Page up and down keys to produce an accelerated change to the magnify area.

 

The screen will be magnified to approximately the correct size as indicated by the current scale. Use this option to preview your graphics at the correct scaled size.

 

Reduce the plot size by a set factor

 

 Select the zoom menu and the top action bar will be replaced with an action bar for zoom commands. You can define a rectangular area, enlarged to the screen limits, by indicating the diagonally opposite corners of the area.

 

Draw the triangles of the active project on the screen. Nothing will happen if no triangles exist in the project. The triangles must then be loaded from disk or created from scratch.

 

Use this option for a fast redraw of contours. The contours generated by this method will not label the contours values and even if spline is selected, the contours will be drawn as straight lines.

 

Draw contours at the specified contour interval if triangles exist.

 

Use the PAN function to move the screen view by indicating a screen position.  This is similar to SHIFT, but the second point is user defined.
