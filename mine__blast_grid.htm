MINE : Blast Grid

 Top  Previous  Next

 

Use this function to create dtm points between existing boundary points and an indicated cad base line.

 

 

 

When opening the dialog box, the Ok button is disabled until the baseline has been indicated.

 

Use the Indicate Baseline and point to the cad line that will be used as a base for the calculation.

 

 

 

This is a plan view of a typical scenario where the Blast Grid can be applied.

 

1) The existing dtm points that form the edge of the blast grid.

2) The cad line that will be used as the base.

 

3) A cad line illustrating the position of proposed blast points from the dtm point, perpendicular to the base line.

Note: the cad line @3 is just for demonstration purposes and is not used in the calculations.

 

 

 

When the Indicate baseline is used, the program will then inspect all the selected dtm points and calculate the distances perpendicular to the base line and summarize the results.

 

Use the stagger option if every second point must be placed at the offset distance. If stagger is not used, all the new points will be in straight line between the original point and baseline.

 

Select how many lines you want. New points will be placed at each line. If an additional point must be placed on the baseline, select that option.

 

Define the code of the new points and press OK.

 

Before new points are actually created, a preview of dots are shown on the screen.

 

If you are satisfied with the result, click Yes and the new points will be created.

 

 

 

This is a snapshot of the resulting new points.

 

a) The original perimeter points.

 

b) The new blast points will be tagged as follows:

 

  C : indicating Column.  (each line from base point to baseline)

  C1, C2, C3...  indicating column and the column number.

 

c) The new points will have the following codes:

 

   B : user defined indicator.

   B1, B2, B3.... indication each line number.

 

 

Summary :

 

All the points tagged C1 were generated from the first base point. C2 from the second base point etc.

 

All the points coded B1 form the first line closest to the base points. B2 the second line etc.
