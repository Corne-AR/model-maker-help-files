MENU : Input Co-ords - Manual entry

 

Getting survey data points into Model Maker can be done by various methods.

This example shows the different ways that can be used to type data into the system.

Also see the getting started example which illustrates the other data entering methods.

 

How to clear the system

From the top menu select FILE - NEW and use the functions to clear the memory of any DTM points if a complete new project must be started.

 

Background

Model Maker can hold up to 21 different survey projects simultaneously. Normally only the first project is used, the others are usually only used when certain cut and fill volume, and other calculations are performed.

 

Within a project, up to 50 survey stations can be opened - each with their own set of surveyed points measured from them. These surveyed points can be entered in unreduced format - as distances and directions - or as final Y&X or easting&northing co-ordinates.

Survey points can also be added to the project by using the graphics commands and entering the co-ordinates by pointing to positions on the graphics screen.

 

Unreduced data

Model Maker accepts unreduced data (meaning raw survey data in the format of distance, angles, and directions) and reduces it to co-ordinates.

The surveyed data will always be related to a survey station, defined by the co-ordinates of the position where the survey instrument was set up at the time of the survey.

One or more survey stations must be opened before raw field book data can be entered into the data base.

Graphics

Data can also be entered on the GRAPHICS screen. On the left menu, select the PNT option and then the ADD option at the bottom.

This function allows you to place DTM points graphically on the screen.

 

Add single points using any of the cursor positioning modes. Points may be placed freehand, by entering the exact co-ordinates or by placing them on CAD Line endpoints. You can also place points at regular intervals on a selected cad line, at the endpoints of a CAD polyline or along a circle.

 

               Use theIdentify function to manipulate groups of points.

               Use the project options to copy points between projects.

 

Model Maker allows a maximum of 50 stations to be defined per project.
