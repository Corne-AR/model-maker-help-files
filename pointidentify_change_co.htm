POINT : Identify - Change - Code

 Top  Previous  Next

 

MENU : Pnt - Id - Man-id - cOde

 

Globally change the codes of all the identified points. There are 2 options.

 

 

New Code

 

Select this option to change all the codes of the identified points to a new value. Enter the replacement code in the new code input box. You do not control the order that the renaming will take place  It will be done according to the points position in the database. To selectively renumber points, use the "Pnt -Ren+" option.

 

Data Base Number

 

The second option is to automatically rename all the identified points according to their number in the database. This ensures a unique name for each point and is often used when tables of co-ordinates are required. You can add a constant to the database number. This allows you to start numbering the points from any value. This would be used if you already had a list of co-ordinates that had been automatically numbered. If the last number happened to be 400, you could then renumber your set of data and start at 401.

Enter the constant value in the data base constant input box.

 

Increment Code

 

If the new code ends in a numeric (eg MH1) and the increment suffix box is checked the codes will be replaced with MH1, MH2, etc.

If the new code starts with a numeric (eg 1A) and the increment prefix box is checked the codes will be replaced with 1A, 2A, 3A, etc.
