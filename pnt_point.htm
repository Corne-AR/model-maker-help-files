MENU : Pnt - Point

 

Model Maker will display all the attributes of the indicated DTM point

 

Point number

 

The top left block informs you about the current active project number, the station number to which the point belong, the name of that station,and the position of the point in the database, eg., 16.

 

Manipulate

 

The manipulate block allows you to delete the point, change the identification status and presents you with a number scroll and input box to jump to a point with a specific database number.

Press the delete button to change the current code of the point to that of *.*, indicating that the point is marked for deletion. Use the data menu to remove it from the database.

 

Entering a valid point number in the input box, less or equal than the total number of points in the project, will redisplay the attributes of the point in question.

Co-ordinates

 

This block displays the attributes of the point and may be changed. Any changes made in this dialog box are automatically updated in the database.

Use the + and - buttons to change the number of decimals used in displaying the values.

 

Constant

The value entered in the constant box can automatically be added or subtracted from the point elevation by clicking the buttons.

 

Elevation

 

Here are two ways of defining the elevation of the point other than physically typing the value .

Calculate the extrapolate elevation value by pressing the  button. This value will be placed in the position that holds the 0.00 value.

The interpolated value is calculated automatically and would normally be the same value of the elevation of the point,except when the point is not connected to triangles. In this case, the elevation base on the interpolated elevation of the underlying triangle is used. The value would be zero if triangles are not present below the point.

Use the appropriate arrow to instruct Model Maker to use the new calculated value as the final elevation of the point.

 

Update

 

Use the update function when the elevations of a series of points need to be changed to a constant value this is how to do it:-

 

Jump to the first point and change the elevation to the desired value. Without pressing OK or Cancel, press the Update button. The new value will be remembered. Now select the points to reset the elevations. As you select the point, the elevation will be updated and plotted in red to indicate that it has been assigned the new elevation or Z value. Keep selecting points until you are satisfied.  Press ESC or right mouse button to stop. You can press Replot at any stage to check.

Original elevation

 

This value is only displayed as reference and can be used to check the what the original elevation was before the elevation value has been edited.
