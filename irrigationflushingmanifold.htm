IRRIGATION : Flushing Manifolds

 Top  Previous  Next

This function allows you to simulate the flushing of dripline laterals.

 

Under normal circumstances the laterals are all calculated as being closed at the outer ends. For the purpose of flushing, these laterals are not really closed but connected to a flushing manifold which in turns have a valve that controls the flushing action.

For the program to exactly calculate the hydraulics, including the flow in the flushing manifold, one has to make use of loops (or closed circuits) which the program cannot do in a block situation. (Loops can only be simulated in the main line.)

 

A simplified approach is presented here that avoids the use of loops for the actual hydraulic calculations but is still able to give the designer all the necessary information to make intelligent decisions.

 

The basis of this analysis is that the designer wants to calculate the flow in the block under normal operations and also the flow when all or some of the flushing manifolds are open. The additional flow in each lateral is calculated by taking the pipe diameter and minimum flow velocity (in the lateral) into account and thereby determine the flow in each lateral that passes through it, into the flushing manifold.

 

This additional flow is created by placing a unique fitting node, called FLUSH VALVE at the end of each lateral. These FLUSH VALVES are then assigned to groups and can either be opened, demanding the additional flow through the lateral or closed. In the closed state the lateral is calculated normally without any water emitted at the end.

 

The group association options are used to allocate and assign these unique FLUSH VALVE fitting nodes to the end of the laterals.

 

 

First set the current group number. The assign options will assign this number to the FLUSH VALVE nodes so they all open or close simultaneously.

 

Use this option to select individual laterals that must flush in the current group. Do not point to the actual lateral but rather point to the flush manifold node that the lateral is connected to.

 

Use this option to assign multiple laterals to a group number. Use the cursor and draw a temporary line over the selected laterals. All the laterals being crossed by the temporary line will have their end nodes changed to the unique fitting node type called FLUSH VALVE and they will also be assigned the current group number.

 

If you have added flushing manifold pipes to the end of the laterals (see the pipes marked by 1 and 3) you may point to the flushing manifold and all the laterals that connect to the indicated flushing manifold pipes will be assigned to the current group.

One should note at this time that the flushing manifold, the pipe connecting all the laterals to a flushing valve, does not really exist in the system as a regular pipe element. The velocity is just used to report a theoretical pipe size. This is the pipe that should be able to handle all the flow of the laterals of the same group.

 

Please note: If the flushing manifold was really part of the system as regular pipe segments the system resulted in a looped analysis. For that reason it is just assumed that all the laterals assigned the same group number will eventually drain into this theoretical, reported flushing manifold.

What does this feature allow the user to do?

 Calculate the hydraulics under normal flow and also when the laterals are flushed.

 Determine how many laterals can be grouped together and serviced by a flushing manifold.

 What the size of the flushing manifold should be.

 The overall pressure drop in the system and also the additional flow when flushing is being done.

 Calculate flushing without the need to actually incorporate flushing manifolds in the layout. Optionally, you may add flushing manifolds to the design so they may be shown on plan and listed in the bill of materials.

 

Note: Flushing the laterals can be calculated whether flushing manifold pipes have been added to the system or not.

 Since the flushing manifold pipes are not really part of the system they do not effect the hydraulics.
