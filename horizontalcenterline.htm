HORIZONTAL : Centre line file

Top  Previous  Next

 

MENU : Hor - cenF

 

Save the centre line co-ordinates of the current horizontal alignment.

 

Chainages

Enter the start and end chainages of the report. By default the start and end chainage of the alignment is displayed. Change the values if only a section of the whole alignment must be considered.

 

Chainage Interval

 

 Regular interval        :                Values will be calculated at the indicated interval.

 Include BC & EC        :                The begin circle and end circle co-ordinates will also be added.

 User Define                :                Access the dialog box and enter individual chainages.

         

Alignment Offset

Normally the chainages are calculated exactly on the centre of the defined alignment. If the co-ordinates must be calculated at an offset, enter a negative distance to go to the left and positive to the right. Model Maker will traverse down to centre line of the alignment and determine the co-ordinate at the centre point and bearing perpendicular to the centre line. The offset point will then be calculated by polaring out in the calculated direction.

 

Report destination

If the centre line file is generated based on the request of the user, the data can either be stored to file or printed in the editor.

 

File

:        The list of co-ordinates will be stored in the selected file using a standard format of 5 columns.

                  1                               2                                   3                          4                       5

                 0.0000      457.83200      555.44800   221.8863   221.8863

 

Column 1                The chainage.

Column 2                The Y or E co-ordinate

Column 3                The X or N co-ordinate

Column 4                The bearing perpendicular to the centre line. (decimal (360) degrees)

Column 5                Equal to column 4 unless it is a user defined chainage with a user                                                              defined angle other than 90 degrees to the centre line.

 

Editor

 

:        The list of co-ordinates with the defined number of decimal places and bearing format will be outputted to the editor.

 

Dtm project

:       Make your chainages a code and specify the number of decimals you want on your centerline

 

Alignment type

If the centre line is generated to extract a long section the following options are available.

 

For a Road

 

:                The centre line will be followed by tracking the curves (if a radius is define) at each PI.

 

For a Pipe

 

:                The centre line will run directly from PI to PI ignoring any defined curve.
