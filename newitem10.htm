MENU : Ana                (Shift A)

 

Certain functions analyze the triangular model of the active project for slopes, height, valleys and areas. Most of the analysis functions derive their information from the triangles in the model. Be sure the data is loaded and the model is created before using these functions.

 

1)                Select Ana in the left hand side menu.
