CAD : Text - Align

 Top  Previous  Next

 

MENU : Cad - Text - aliGn

 

Align 2 or more text elements relative to the base (first selected) text.

 

Select all the text elements to be aligned and then set the following:

 

Position

 

No change

The position of the text will not change. Use when only the orientation of the selected text must be manipulated.

Space equally

A line will be drawn from the first to the last selected text. The origin of the other text will be dragged onto this line.

If the proportional box is checked, equal spacing will be placed between them. Otherwise, the distance between the text will not be changed.

 

Align to base V

The direction of the base text will be used and the other text will be aligned directly below it.

The proportional check box will determine if the text spacing will be distributed evenly between the base and last text.

 

Align to base H

The direction of the base text will be used and the other text will be aligned directly next to it.

The proportional check box will determine if the text spacing will be distributed evenly between the base and last text.

 

Check this box to space all the selected text between the base and last text evenly.

 

Orientation

No change

 

The orientation (direction) of the text will not change. Use when the position of the selected text must be manipulated.

Space equally

 

If the direction of the base and last text differs, Model Maker will align all the in between text proportionately.

Align to base

 

All the text will have the same orientation as the base text.

 

Use the Test options button to visually inspect the result of the selected options. (Press ENTER or ESCAPE on the keyboard to return to the dialog box. Optionally, click ABORT button when the inspection is done.)

 

Click OK to record the changes. Use the '+' key to undo.
