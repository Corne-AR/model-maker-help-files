MENU : gXt - Power - Edit pole

 

The three main elements in the design of a power line are the ground data, the line conductor and the poles (pylons)that are defined in this dialog box.

 

Power Line  Pole Data Edit Menu

                               

ADD

 :  Open a new standard edit to enter the data of the new pole type.

EDIT

 : Edit information about the pole. Or, use the EDIT tab on the top of the page.

DELETE

 : Delete the selected pole from the list.

SORT

 : Sort the poles by name.

EDIT

After selecting a pole from the list, use this menu to change the data of the pole.

 

The pole name identifies the pole. This name is used when the short list is displayed and the info of a pole is requested. The name may have a maximum of 10 characters.

The type is not used, but may be entered as a longer description of the pole.

 

Enter the name of a specific pattern that will be used to display the pole in the plan and vertical modes. The pattern must be defined in the CAD.

Use the default menu to select that the patterns is not to be used.

 

The physical dimensions of the pole are entered. None of these values are used in the design process and are only entered as references.

 

When adding new poles, use this button to copy the information of an existing pole to the current pole. This saves you the time of retyping the data. It is then easier to edit the different values between the 2 poles.

 

Conductor

The names of all the defined conductor type are displayed in the scroll box. By clicking on a selected name, the dialog box is extended to allow you to enter the basic pole specifications relative to that type of conductor.

 

It is important to enter the correct maximum permissible values that the pole can accommodate for each conductor type. These values are used in the loads report. Spans exceeding the limits will be identified.

 

The 2 heights entered here are important. When poles are added to the string and planted on the ground the bottom attachment will be used as reference.

When checking for crossing details 2, scenarios will be evaluated.

         1) Cold temperature with top attachment for going below a higher service.

         2) Hot temperature and bottom attachment for going over lower services and to check for low points (clearance) relative to the ground.

 

Password

 

 

 

You may assign a password to a single pole or to all the poles.

 

To set a password

·enter the password in the password text box.
·Set the action type to : Set Password.
·Click the Select Pole or All the poles button.

Important - Remember the password you have set. Only you will know what the password is.

 

To remove a password, set the action to Remove password and click the appropriate button.

 

Notes regarding password protection.

 

·A password protected pole can not be edited unless the correct password is entered in the password edit box.

 This means you do not have to remove a password before a pole can be edit. The correct password just has to entered.

 

·The following message will show when a protected pole is edited and the correct password is entered.

 

·If a protected pole is edited and the password is not entered or incorrect, the following message shows and all the data boxes will be disabled.

 

 

 

The user will be able to view the data but cannot make any changed.

 

·A pole that is not password protected may be edited at any time.
