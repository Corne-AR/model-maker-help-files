Function keys

 

F1               -  get help menu.

F2               -  activate the save data menu

F3             -  activate the open data menu

F4               -  force a backup of the data

F5               -  invoke the TRACKING menu

F6              -  invoke the MEASURE menu

F7               -  No action.

F8             -  toggle between the top and bottom menus

F9             -  invoke the built-in calculator

 

 

Keyboard keys

 

Arrow keys                                The arrow keys on the keyboard can be used to move the cursor on the screen in a horizontal or vertical position. When the SHIFT key is pressed and held

                                         down while the arrow key is used, the cursor moves in larger increments in the selected direction.

Insert                                                Invoke the default pen & line type menu.

Delete                                                Record all the pen parameters from the CAD line or circle closest to the cursor or  the text parameters from the closest text depending on the current CAD menu option.

Page up                                        When copying or moving elements, rotates the element 1 degree counter-clockwise.

CTRL Page up                                Does a 45 degree counter-clockwise rotation.

Page down                                When copying or moving elements, rotates the element 1 degree clockwise.

CTRL Page down                        Rotate 45 degree clockwise.

Home                                        Put the element back to a 0 rotation.

 

CTRL home                                Opens the menu to change certain attributes (angle and scale) of the element that is being added, copied or moved.

SHIFT TAB                                Rotates the copied or moved element to the direction of the closest CAD line (eg. when text needs to be aligned.)

SHIFT (T,P,G etc)                                Activate the left main menu.

ALT (1,2,3)                                Activate a specific project.  

CTRL (P,L,Y etc)                                Change the cursor mode.  

 

Manipulate Plot Menu

 

N – Normal. To zoom everything into view or to zoom extents

R – Replot. To refresh the graphics on screen

S – Shift. To shift the cursor position to the centre of the screen. This is the same as pressing the wheel button.

M – Magnify. To magnify everything in the zoom block into view.

D – De-magnify. To de-magnify with a predetermined zoom factor.

Z – Zoom. To zoom in to a defined rectangular area.

Q – To view quick contours temporarily.

O – To view normal or splined contours with labeling temporarily.

T – To view the triangles temporarily.

G – Plot Parameters

 

 

Project Activation

 

1 – To activate project 1

2 – to activate project 2

3 – to activate project 3

 

 

Cursor Snap Modes

 

Ctrl+P – DTM Point.

Ctrl+U – DTM Point Number.

Ctrl+Y – Any line or circle endpoint, intersection, Circle Centre, Arc endpoint, Text Origin etc.

Ctrl+L – Line Endpoint.

Ctrl+A – Arc endpoint.

Ctrl+X – Text Origin.

Ctrl+H – Arrow head.

Ctrl+M – Line Midpoint.

Ctrl+R – Perpendicular to an existing line.

Ctrl+K – Track any irregular cad shape. (Tip: The CAD shape may contain arcs and lines but should be drawn exactly from endpoint to endpoint for this function to work properly).

Ctrl+S – Distance on a line based on the point on the line closest to the cursor.

Ctrl+N – Near Line.

Ctrl+T – Tangent to circle or arc.

Ctrl+E – Near Circle or Arc.

Ctrl+I – Circle Centre.

Ctrl+F – Freehand meaning snap is disabled.

Ctrl+B – Intersection of two line segments

Ctrl+Q – Intersection of line segment with circle or arc

Ctrl+C – Intersection of two arcs

Ctrl+V – Intersection of the offset position from two intersecting line segments

Ctrl+W – Pre defined Y & X or N & E co-ordinates

Ctrl+D – The dY & dX position from a predetermined position

Ctrl+O – A horizontal distance and survey direction from a predetermined position

Ctrl+G – A position on a evenly spaced predefined grid

Ctrl+J – A position halfway between two indicated positions

 

Using the "+" key

Most of the activities in Model Maker may be undone using the '+' key.  For instance, in the CAD, when adding lines, press + to delete the last added line or, when deleting lines, press the + key to undo the deletion of the last line.

The + key has the function of an UNDO utility, it does the reverse of the last user action.
