MENU : Limits - Dialog box

 

This dialog box is often used when Model Maker requires you to enter a lower and upper limit value on certain actions.

 

In the CAD, when identifying elements, many of the processes are performed based on the specified limits.

 

EXAMPLE: Text sizes, line lengths and circle radii.

 

Enter the upper limit and then the lower limit. All the elements with attributes within the specified limits will be manipulated.

 

Check ALL THE ELEMENTS if the lower and upper boundaries must be ignored in the selection process.
