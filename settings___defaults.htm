MENU : Settings - Defaults

 

 

How can this topic help me ?

Change general default parameters in this dialog box:

 

Set the general and CAD plotting order

 

 

The numbers do not represent the order of plotting.  The top item in the column will plot first.  When you experience a problem with text being obliterated by hatching, you need to check your plotting order.

 

 

 

 

Magnify Factors

These factors affect the mag and demag buttons

Filename On Plot

When ticked, this will add the file name on the drawing when plotted.

 

 

Set the rotation interval when using the page up and down keys when copying or moving elements.

This can be changed to suit your requirements.

The rotation interval can be user defined.  By default, page up / down will move in increments of 1 degree and Ctrl / page up / down in 45 degree increments.

F5 display - +snow screen - picture select the various ratio buttons and their options. This information would be displayed on the bottom right screen about the co-ordinates.

General editing defaults

 

 

Adding hatch and areas

 

·Indicate each point

You will indicate each perimeter point.

When defining a hatch, the nearArc snapmode will pick up the complete circle/arc. The tracKline option is not available.

 

·Use the Area Definition Mode

The regular area definition functions are available. The nearArc mode will only produce the a single coordinate on the arc and the tracKline snapmode will track a polyline including curves, if the track poly around curves is selected.

 

Number Of Decimals

Set the number of decimal places for distance and direction when copying to CAD and General co-ordinates on screen and in Pnt Info for plan and Z

 

 

Use this dialog box to assign installed windows fonts as regular fonts to be used by the CAD text option

 

 

 

 

 

Two lists are displayed.

       The first list displays all 13 Model Maker fonts and all the windows fonts. Notice that the Model Maker fonts start with the letters MM.

These are vector drawn fonts that are part of the program and operates outside of the standard window fonts.

 

       The second list contains the font names that will be displayed on the shortlist when cad text are added.

 

Use the arrow keys to set up our selected of font names in the shortlist.

 

Troubleshooting

When plotting to printer / plotter, if the text is not plotting:-

       check pen colours
       check that all 13mm font files are present in the Program folder.

 

 

Note:

Model Maker saves the font attribute of the CAD text elements in the KCD file as a font name. If a cad file is displayed on a computer that does not have the particular font name installed, windows will automatically substitute it with a different font.
