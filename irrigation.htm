Model Maker Irrigation

 

The irrigation module is a comprehensive designing system to develop irrigation projects.

Operating within Model Maker, Irri Maker integrates with the powerful CAD and survey functions to provide the most complete software package available.

 

Irri Maker makes the design of irrigation systems a pleasure by dividing the whole process into logical compartments. The two main components are the nodes and pipes. The system (blocks or main line) are build by placing nodes on the ground which are then connected by means of pipes. Blocks are defined in which the sprinkler or micro/drip layout is placed and managed.  Each block consists of lateral lines (connecting the emitters), the manifold line (connecting all the laterals) and the valve which connects the block to the main line.

 

The main line transports the water from the source to the valves of all the blocks. The blocks are easily grouped to operate in different shifts. Comprehensive hydraulic functions are used to size the pipes and to calculate all the pressures and flows. A complete list of parts and clusters can be defined and automatically placed on the nodes, providing a detailed list of materials with prices. Impressive drawings of the design are produced along with the CAD detail and contours by means of the plotting functions provided by Model Maker.

 

Shifts

Use these functions to group blocks into logical operating sequences.

 

Pivot

Functions are available to assist in pivot design.

 

CAD

Copy information from the irrigation design to CAD elements.

 

Defaults

Set certain plotting, editing, calculations and design defaults.

 

Parts

This department allows easy manipulation of part numbers to be allocated to the different nodes of the irrigation project.

 

At the bottom of the irrigation menu you will find the plot button.

 

You may at any time click on this button which will hide the irrigation dialog box allowing you access to the graphics screen where you may use any of the plot manipulation functions.

Access the main irrigation dialog box by clicking on the Irr button on the left or by pressing Shift I on the keyboard.

 

All the irrigation functions are grouped together on the main dialog box. Pages with descriptive tab names will lead you to the individual functions.

 

Project

Set the project units. Use the LOAD and SAVE options to save the irrigation project to disk. The irrigation data is saved separately from the Survey and CAD, which are saved using the Files option.

 

Main Line

Work and manipulate the main line by activating block 0. The main line must have one pump node in addition to any other nodes needed to connect the pump with the blocks. Only the main line layout may be in a loop configuration.

 

Blocks

Irrigation design is done by grouping different areas into separate blocks. The main line will then distribute the water from the source node to the valves at each block.

Block zero is not an actual block but relates to the main line.

 

Blocks

Blocks are added to the system by indicating the block perimeter points. Up to 40 perimeter points may be used to define the block area. Once the area has been defined, use the automatic function (Fill laterals) to define the lateral pipes and sprinkler nodes.

 

Nodes and pipes may also be added to the block using any of the other manual methods. Before using the Fill Laterals method select the correct sprinkler option by checking the sprinkler or micro/drip options on the page.

 

Each block must have one valve. This valve node is used as the source node to the block and will be used to link the block with the main line.

 

Nodes

The irrigation project consists mostly of nodes and pipes. Nodes can be sprinklers, pumps, valves or fittings.

Use the Database option to enter the standard available node types.

Use the Plot parameters to specify the display format of the nodes.

Each node can have no more than 4 pipes connected to it. One feeding water into it and 3 transporting the water to the down line.

 

Pipes

Pipes can only be added between defined nodes. Use the Database option to enter pre-defined types. Pipes are defined in 3 categories. Laterals, manifold and main line pipes. The laterals may either be a regular pipe or a CES (Constant emitter spacing) pipe which is used to simulate a micro/drip system.

 

Global

Nodes and pipes may be entered in any sequence. However, before calculations can be done, Model Maker will renumber the whole system by using the valve as the source node for the blocks and the pump as the source node of the main line. The source node would be the node where the water is entered into the system, eg., the pump in the main line and the valve for each block.

You will not be allowed to use the hydraulic option unless the main line (block 0) has one pump and each block has it's own valve.

 

Use the Design & Calculate option to apply water to the system. For the main line, the water will be entered into the system (all the nodes and pipes of the main line) through the pump node.

The hydraulics of each block will be calculated using the valve as the source node and distributing the water to all the down stream nodes.

 

Automatic sizing of the distribution pipes is also available.

Use the Group, Copy, Move, or Delete options to globally manipulate a section of data.

Use the Flow Path function to ensure that all the pipes are connected to the source node.

The Report button gives a Bill of Materials for each or all the blocks.
