MENU : sUr - Area                                

 

Calculate the size of a user defined area. Change the cursor positioning mode to indicate the polygon.

In the area function, when using the Track option you can hold down the Shift key and then individually select arcs, just like using the neararc function in Hatch.

The lines forming the area and the area size can be written into the CAD environment.
