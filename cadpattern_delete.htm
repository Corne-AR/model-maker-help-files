CAD : Pattern - Delete

 Top  Previous  Next

 

MENU : Cad - Patt - Delpat

 

Delete any of the patterns that have been loaded and added to the drawing.

 

Single

 

Point to a pattern to delete it. Use the '+' key to undo.  

 

Name

 

Delete all patterns with a specified name.

 

All

 

Delete all patterns on the drawing.

 

Duplicated

 

Delete only the duplicated patterns. A pattern is considered a duplication if  2 or more same named patterns share the same base coordinate.
