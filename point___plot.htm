MENU : Pnt - plOt

 

Model Maker normally plots the points in a standard format. The output to a plotter are more user controlled and may have different settings than the ones used when displaying the data on the screen.

 

This function will plot to the centre, but will apply the parameters normally used when outputting to a plotter  Use this function to preview the plotter output.
