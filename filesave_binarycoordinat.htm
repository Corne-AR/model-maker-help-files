MENU : File - Save (F2) - MMB

 

Save data consisting of codes, y, x and z co-ordinates in this format. It is in a binary format with a .MMB extension. It is recommended that co-ordinate data, which does not consists of TACHY stations and points, be saved in this format because smaller files are produced. This format saves exactly the same information as the ASCII (yxz) file, but in binary format.
