MENU : Cad - aRrow - Edit        

 

These functions allow you to move, delete or add nodes on an arrow.  

 

 

 

 

Add node

 

Add new nodes to an existing arrow if less than 10 nodes are already defined.

Model Maker determines where to add the node in the following way:

         

·        To add a node to before the current tail, point close to the tail point, but not on the side of the line connecting the tail to the first node.
·        To add a new node between two nodes, point to the line joining the two nodes.
·        To add a new head, indicate close to the head node.

 

Use the '+' key to undo the add command.

 

Move node

 

Move any arrow node to a new position. Point close to the node, then indicate the new position.  

 

Delete node

 

Point to a node to be deleted. if the wrong node is deleted, use "+"-key to undo the action.
