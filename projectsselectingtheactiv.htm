PROJECTS : Selecting the active project

 Top  Previous  Next

 

Model Maker provides the following options to change / select the active project :

 

       On the graphics screen - right menu:

 

Point the cursor arrow inside the

 

 

(1,2,3 project buttons) to activate the relevant project.

         Or

Press the  ALT - key plus the required project number eg. ALT-2 for project 2.

Use any of the methods to switch between projects.

 

               On the FILES menu a different active project can also be selected.

 

               Set the active project on the DATA BASE dialog box.
