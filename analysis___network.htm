MENU: Ana - pipeNetwork                                

 

This function assists you in creating the necessary support data file for dedicated pipe network programs.

The 1st line of the file contains the project description.

The attributes of the nodes (numbers, volumes and co-ordinates) are listed either in single rows or in multiple rows

 

 

SINGLE ROWS:

Network 2

PIPE    1    1    1    1     798.747     440.610      59.235   100.000    100.000    2     798.747     1440.610      59.235     60.000     0.000       1000.000

You will be asked to indicate the node positions. Model Maker will record the co-ordinates plus the ground elevation if triangles are available to interpolate the value.

 

Once your whole network has been traversed in major and minor loops, exit the function. The file will be saved to disk.

 

The file will have a PRN extension and is an ASCII file.
