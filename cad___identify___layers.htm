MENU : Cad - Id - laYers                                

 

Delete and rename layers without first identifying the layer.

 

 4 

CAD elements appear in the order that they were created in. Below, the RED layer have lines that were drawn after the lines on the YELLOW layer.

 

If you wanted to draw all YELLOW layer (yellow) lines on top of the red ones you can draw the RED layer sooner, drawing them underneath the yellow lines.

 

 

 1 

To Delete the contents of the layer, first select a layer name from the existing layer list box and then press the DELETE button.

 

You will be asked to confirm your request before all the elements (lines, circles, text, hatch, arrows and dimensions) on the layer are deleted. Once the delete action has started, the process cannot be reversed.

 

 2 

To Rename a layer, first choose the selected layer from the existing layer list box.

Enter the new name in the input box.

 

Click   to start the renaming process. All CAD elements with the previous layer name will be renamed. If the new name matches any of the existing layer names, the elements from the selected layer will be placed in the new layer along with the elements already in that layer.

 

 3 

All the layers in use for the drawing are displayed in the list box. Select the layer to be deleted or renamed. The selected layer name will be displayed at the bottom of the box.

 1 

Assigning a pen style to multiple layers can be done here. One default template can be stored (Update template) and recalled (Fill from template) as needed. The template holds all the assigned Pen Styles and their associated layer names.

 2 

The available layers are present here with a Pen Style field next to each. Drag the Pen Style to bind it to that layer.

 3 

A list of all the Pen Styles are visible here. Drag the Pen Style from here over to the layer to bind it.

 

 4 

After the Pen Styles have been connected to their layers, press this button to apply the Pen Attributes to the elements on the layers.

All the elements on the assigned layers will have their pen numbers, line types and layer names changed to the values in the pen style.
