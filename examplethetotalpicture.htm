MENU : Example - The total picture

 

This example shows you how to do the following without going into too much detail.

               Load a survey data file and create a triangle model.

               Use Cad to draw a couple of lines

               Create a simple irrigation pipe line.

 

If you are a novice to the program, use this example to get a general feeling of how the software operates.
