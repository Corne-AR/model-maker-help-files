MENU : Tri

 

This menu allows you to create and manipulate the triangles.

 

               A triangle network must first be created between the survey points before information such as contours and x-sections can extracted.

               Triangles are created according to the set search parameters.

               Use the edit functions improve the triangle model when needed.
