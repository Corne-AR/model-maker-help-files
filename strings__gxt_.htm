STRING (GXT)

MENU : gXt                (Shift+X)

 

       

 

Strings are powerful design tools.

 

They can be used to define road alignments, areas, shapes, positions and elevations. A string is defined as either a general string or an alignment string.  A string can be manipulated in various ways. The points on a string are called nodes.

 

Delete

 

Delete any string by pointing to it. Undo the last deleted string by pressing the '+' key.

 

Vert

 

Put the indicated strings into the vertical view mode. One or more strings may be viewed simultaneously. The co-ordinate system will then work, not as Y&X co-ordinates, but as chainage distance and elevations (Z value of nodes). The chainage system will be dictated by the first indicated string. The vertical distortion, that is the ratio between the horizontal and vertical axes, may be set. Another option is to extract and draw the long section as taken from the active DTM project, along with the strings. This section will always be drawn in white and may not be edited. Any of the vertical grading options may be used while in the vertical view mode.  

 

     STRING : Hor

 Set the graphics back to the plan view mode. The strings displayed in vertical mode are placed back into the plan format.  

 

Clear

 

Delete all the strings currently in memory. Make sure that the string data has been saved.

You can also clear the string memory with the File New function.

 

Files

Strings are saved to disk and loaded from disk in this menu. Strings operate separately from CAD lines and are stored in separate files. The file extension is .KCS.

The main files menu may also be used to save the strings.

 

     Save identified

 Only the strings that have been identified are saved to disk. Strings are identified with the ID option.

 

Id

Identify strings. This means that a string is flagged for it to perform certain tasks. These tasks are usually found in the mining module of Model Maker, and include options like rehabilitation and reshaping land by strings.

 

Identified strings may also be manipulated on a global scale
