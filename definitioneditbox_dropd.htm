DEFINITION : Edit box - Drop down

 Top  Previous  Next

 

An edit box is the place where most of the input from the user is transferred to Model Maker. A normal edit box does not have a drop down option on the side:

 

 

A drop down edit box has an arrow on the side to display other default values or previously entered items:

 

 

Three edit boxes are displayed. The third box shows its drop down list.

You may select any of the items in the list, or; enter a value in the edit box. The drop down is an additional tool but does not prohibit you from entering your own data when necessary.
