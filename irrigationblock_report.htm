Use this function to define your own irrigation reports.

 

The content of this report consists of static text entered by the user and coded text generated by the program. Each one of the coded text refers to a specific information type in the system.

By combining the user's caption text with the dynamic coded text of the system, a variable format report is produced.

 

The report editor is divided in 2 main sections.

 

The Header Section.

 

The header section may have any number of static text entries and any of the 6 defined program values.

How to use the # options

 

 

The report is a combination of static text and program values. Program values are defined by the # character and appropriate data number.

 

To print the block flow as an example, define your own header and then use the #16 option.

 

Here are some examples.

  Block Flow  #16_______ gpm

or

 Valve flow (gpm) #16_____

 

The # identifies a program value, the (16) indicates the value type and the ______ defines the space the value will occupy.

 

NOTE: The ______ part of the definition MUST start directly after the # part.
