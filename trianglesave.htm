MENU : Files - Save                (F2)

 

Transferring triangle data is done in the  FILES  menu. Save triangles in following manner :

 

1)                The triangles will be saved in a dedicated triangle file with a  .TR2  extension.

         This is a binary file and can only be read by MM.

 

         

 

2)                Save Total        .TOT extension

 

         

 

         EXAMPLE:

         

 

The above 2 methods are the only ones designed to save triangles so that MM can load them back from disk.

Exporting data should not be confused with saving data.

 

Note

Saving data in the MAL format also saves the triangles if the survey data is included in the MAL setup.
