MINE : Borehole manager

 Top  Previous  Next

 

The borehole manager allows you to:

 

1.Import borehole logs from an xcel CSV file.
2.To enter borehole logs manually.
3.To assign colors and DTM layer numbers to the log categories.
4.To visually compare logs on a same scale.
5.To view the logs in Model Maker's virtual 3D option.
6.To export the logs to a CSV file.
7.To create dtm projects based on the log coordinates and layer depths.

 

 

Main Page

 

On this page the list of boreholes are shown. You can also change the plotting parameters and export the boreholes to dtm projects.

 

 

To set the size and font type with which the numbers or names will be drawn, do the following.

 

1) Open the Cad Add Text box and set the desired size and font name.

2) Return to the Borehole manager and click the Size & Font button.

 

The default size and font is recorded and all future plotting of the name or number will be done with these parameters.

 

Export borehole data:

 

The dtm projects as defined on the Settings page, will be populated by the borehole data. Please see the example (below) describing the general process to manage and export the boreholes.

Borehole Legend manager

 

On this page the allocations of geological colors and dtm layers are done.

 

The borehole log is shown in the middle, the dtm project allocations on the left and the geological colors on the right.

 

The purpose the of the DTM projects. (eg. Ngl, Top Hard, TC and BC)

When the borehole exports itself to the dtm projects, (see the manager page) the program needs to know which layers in the borehole log will generate coordinates and elevations in the dtm projects.

 

 

On the settings page the relation between the layer description, layer color and dtm project is defined.

 

In this example, when the top hard surface is associated with a layer in the borehole log and the borehole is exported to the dtm, the elevations of the top hards will be placed in dtm project # 2.

 

 

The geological layer descriptions and colors are also defined on the Settings tab.

 

On the Legend tab, use the drag and drop feature to assign geological colors to the borehole log.

 

 

 

 

The assignment of geological colors to the log is only needed for visual purposes.

 

In this example, no geological colors have been assigned and that would not be a problem.

 

Model Maker does not really use these colors except to display the log in a more detailed view.

 

What is important are the DTM layer definitions.

Without the dtm definitions, the boreholes cannot populate the relative dtm projects correctly.

 

 

Compare plots

 

Use this page to compare three boreholes.

By default, the current borhole is shown in the middle and the 2 closest boreholes (y and x coordinates) are shown on the sides.

The boreholes are all drawn to the same scale making it easy to compare the depths.

 

To select different boreholes on the view, do the following:

Change the active center borehole by using the left and right arrow buttons. As soon as the active borehole is changed, the default closest 2 boreholes are again shown on the sides.

To change any of the side boreholes, click their names. You can then point on the graphics page to a desired borehole.

 

 

Import Single CSV files

 

If you have the borehole information available in an Xcel file, save it as CSV files and then import the information to the borehole manager.

 

Before using the import function, define the column and row numbers (in the csv file) where the information must be extracted from.

 

Note: The template definitions must all be numeric. If the borehole name is in cell B3, specify it as column #2, row #3.

 

Save and load the import template settings by using the File option.

Examples:

 

To define new boreholes, either import it from CSV files or type the borehole name, y, x and z on the manager page.

 

To change the colors on the borehole logs, specify the relative descriptions and colors on the settings page. On the legend page, drag and drop the geological descriptions and the dtm layers onto the log.

 

To inspect and edit the detail of the borehole, enter the information on the boreholes page.

 

To enter the analysis data for each borehole, select the analysis page on the boreholes page.

 

To see how the log of a borehole compares to that of the 2 closest boreholes around it, use the compare plots page.

 

To create dtm points that represent the boreholes in Model Maker's survey module, use the export function on the manager page.
