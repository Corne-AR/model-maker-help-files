MENU : Irr - Global

 

After the project is defined by adding nodes and pipes the global functions can be used to process the data and do editing of groups of nodes and pipes.

 

Note:

To return to the irrigation menu after having used the option in the menu, click on the HORZ button. This action returns the segment to the horizontal layout format and then back to the irrigation menu.

Apply water (Hydraulics)

 

This function will calculate the hydraulics in the system when water is applied to the source node.

 

Before the calculations are done, Irri Maker first checks the integrity of the system. Hydraulic calculations will only be allowed on the blocks with an open layout. An open layout is when there is one node more than the number of pipes and no pipes are in any loop format. If the block does not comply to the requirements the following warning is displayed.

 

Use the flow path function to assist in the inspection of the layout in order to make it an open system.

 

A system of loops are only allowed in the main line. The main line can either have one or 2 pumps.

 

 

Calculation the next block or calculating all the blocks.

 

The Next option on the hydraulics panel allows you to calculate the next block or all the blocks.

 

Normally, after all the blocks have been designed, it is good practice to go through and run the hydraulics for each block again so the final results can be calculated.

 

The N function gives you 2 new options.

 

 

1) Select the valve pressure option.

     The first option will not change the existing valve pressures when each block is calculated.

 

     If the seconds option is selected, the pressure of each block will be set to the user defined value before the block is recalculated.

 

2) Auto Settings

     If the Automatic option is selected, all the blocks, from the current to the end, will be calculated without further user assistance.

     In the automatic mode, a hydraulics report of the results will be printed. This report can either be in the standard Hydraulics Info format or you may click on the report definition button to define your own format.

 

  If the Automatic option is not selected, the user must press the Next button each time to proceed to the next block.

Info of Nodes and Pipes (Automatic reporting)

With this function a theoretical report is automatically produced. All the nodes are analyzed and reported according to the number of pipes connected to each node. Nodes are then categorized as:

 

End piece

: Only one pipe connected.

Reducer

: 2 Pipes of the same or different sizes are connected.

T Piece

: 3 Pipes are connected to the node.

Cross

: 4 Pipes are connected to the node.

 

The nodes are also grouped together according to the node type. The final printout will thus list all the nodes of the same type, having the same number and diameter of pipes connected to it.

Edit group of nodes & pipes

 

 

Note:

All these group editing function only operate on the active block.

You cannot copy elements between two different blocks.

 

 

These functions allow you to work with all the nodes and pipes that follow an indicated node in the sorted down line order.

 

Copy Nodes & Pipes

 

All the nodes and pipes in the down line of the indicated node will be selected and attached to the cursor as a ghost image. Like copying CAD lines, you can move and rotate the system before placing it at the new position. The original elements will be not be changed.

 

Move Nodes & Pipes

 

This function is identical to the copy function except that no new nodes are created but the original nodes are manipulated and moved to the new position.

 

Delete Nodes & Pipes

 

Delete and remove from the project all the elements in the down line of the indicated node.

 

Delete Nodes & Pipes A-Z

 

This function will cut out all the pipes and fitting nodes between 2 indicated points. Use this function to quickly remove the manifold from a block design without having to redo the all the laterals.

 

Flow Path

 

Use this function to visually inspect the layout in order to determine if all the nodes and pipes are connected to the source node. Irri Maker will also indicate how many and what the position is of any nodes not connected to pipes.

 

 

In this example the area indicated by the yellow circle is not connected to the valve because of a missing pipe in the white circled area. The green circle identifies the area that is linked to the valve or source node.

Notice that two colours are used to indicate the connected and disconnected areas. This is the same colour used to in the CAD identify functions and can be changed in the colour dialog box.

 

 

Divide Segments

 

Use this function to divide a single pipe into multiple segments.

 

 

Long Section

The pipe line and ground profile will be drawn to CAD as a long section.

 

Note 1: The pipe profile and sections through all the DTM projects (ground profiles) will be shown.

 

Note 2: The produced Long section will only have the basic information regarding node positions and ground line.

To get a long section plus all the hydraulic information, calculate the hydraulics and use the Long Section option the Info page.

 

 

 

Edit Section

 

This function allows you to pick a pipe line and look at a profile of the pipe and ground surface. Functions are also available to add nodes to the pipe while in section mode. This option uses the same functions as the String Vertical mode menu.

 

Node Distance.

Option to pick 2 nodes and calculate the distance (pipe length) between them

 

Note: Long Section and Node Distance is not available in a looped system (not open) since there could be multiple paths between the 2 indicated nodes.
