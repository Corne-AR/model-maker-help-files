MENU : gXt - infO - Names

 

This dialog box allows you great control over the strings by working with the string names.

 

String name list box

 

All the string names will be displayed and sorted in this list box. Even though strings may have duplicated names, all the names will be displayed and repeated if necessary.

 

Double click on a selected string. Model Maker will point out the string by repeatedly flashing it on screen. Press any key or a mouse button to terminate the flashing.

 

Identified strings will be flagged by an ID added to the string name. Strings not identified will have a only the name displayed.

 

Click on a string name to select it.

 

Edit info of selected string

 

This button will transfer to the Info dialog box.

 

Identify selected string

 

Strings may be identified or not identified.

 

The ID status of the selected string will alternate between the 2 options above.

 

Identify all with the same name

 

All the strings with the same name as the selected one will have their ID status changed.

 

Delete selected string

 

Delete selected string. This action will have no effect on other strings with the same name.

 

Use the UNDO button to reactivate the last deleted string.
