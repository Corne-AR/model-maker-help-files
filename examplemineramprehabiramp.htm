EXAMPLE : Mine ramp rehabilitation

 Top  Previous  Next

This example uses the rehabilitation function, -Identified strings- in the Mining module.

 

 

Before using the function, prepare the model by doing the following:

 

A) Define the center string. (Blue line)

 

 

Use the string functions and place a general string along the center line position of the ramp.

 

 

B) String in vertical view mode.

 

After the horizontal position of the string is in place, use the Vert View option and grade the string according to your set drainage parameters.

 

 

C) Define the cross section profile of the trench. (Red and yellow strings)

 

 

Use the Add Parallel strings function and define new strings at offsets from the centerline string.

                 

 

 

Image of typical cross section as defined by the 5 strings.

 

D) Create a toeline string to have the profile intersect at a defined slope with the surrounding surface.

 

 

 

Typical scenario where the theoretical trench profile is higher than the ramp surface and a toeline surface must be created.

 

 

 

 

 

 

 

Use the String Add Toeline option and add the left and right intersecting string.

 

Use the Force toepoints UP option. This will ensure the string will be created so water drains down into the trench.

When forcing an UP toeline Model Maker attempts to find a toeline intersection at the defined slope.

(1 in 5 in this example)

If no intersection is found for that slope, the slope is incrementally lowered until an intersection with the ground is found.

 

 

 

 

 

 

 

 Image showing a typical toeline string. (Green)

 

 

 

 

Sketch of the complete theoretical cross section.

 

Please note, these drawn cross sections are not really visible in Model Maker while you are going through this process. Mathematically they have been defined by means of the strings but not until the surface has actually been graded to the parameters of the strings, will you be able to actually see the profile as shown here.

 

 

 

After the toeline strings have been defined, visually check their positions and if needed, add, move or delete some of the nodes where Model Maker maybe did not find exactly the correct position of the toepoint on a very undulating surface.

 

In this image, the slope (right) is nice and even and the toeline looks smooth. On the left there are many spoils and the toeline may need some editing.

 

E) The Mining Rehab function requires that an indicated string must defined the area in which the rehab must be done. Since the adding of the toeline strings resulted in 2 strings to the left and right of the trench, you need to use the String Combine function to make them one.

 

 

 

Select one end of the ramp and use the Combine function to make them one string

 

Strings combined.

 

F) Identify all the strings and then use the MINING, REHAB, REHAB String option.

 

       

 

G) When asked to show the boundary string, point to the green toeline string.

 

 

 

 

 

 

 

When doing the cut and fill needed to change the existing surface model to that defined by the identified strings, Model Maker can do a balanced cut and fill or an 'as is' process.

 

1) Balanced.

For all the dtm points inside the string area, the new surface as defined by the strings is determined and if the point is below the new surface, it is raised to the new elevation. This imply that material needs to be imported to achieve the result and in the balanced mode, material is cut from points outside the string area.

 

Preferred cut slope. When material is taken from outside, Model Maker will only remove enough material from a point so that the resulting slope from that outside point to the trench meets the preferred slope.

If material from all the points within the maximum hauling distance has been removed and still more material is needed to build the trench, Model Maker will lower the cutting slope to get more material but will not reduce the slope to a value lower than the minimum slope.

 

If the material from all the outside points (within the maximum hauling distance and minimum slope) have been cut, and material is still needed to fill the ramp, a deficit occurs and the cut and fill balance will not be equal anymore.

 

2) If the As Is option is selected, Model Maker will do the cut and fill without importing or exporting material over the boundary string. This option does not guarantee a balanced volume distribution and is normally used when doing rehabilitation of mine dumps and not mine ramps.

             

 

 

This is a sketch illustrating the process when material is cut from outside the string area when using the Balanced option.

 

A) This is the point that is re-elevated to the theoretical elevation as defined by the strings.

 

B) The closest outside point that material can be cut from.

 

C) The point on the edge of the string design surface, closest to the (B) cut point.

The slope between C and B is checked and adhered to.

 

Initially, point B will be cut to a 1 in 5 slope (example) when A needs material. If all the material within the hauling radius has been cut to 1 in 5, a new round of cuts will be made at a slope less than 1 in 5 but not less than the minimum defined slope.

The gridded dtm points will now be elevated to the theoretical design as defined by the string

 

Typical section showing the rehabilitated surface in relation to the original ground surface.

 

 

 

After the REHAB by Identified strings is complete, use the REHAB ALL function to rehab the outside area and tie it in with the string rehabilitated area.

 

To run the Rehab All function, proceed as follows.

 

H) Un-identify all the string and only ID the boundary string. (GREEN)

 

I) Run the Mining Rehab ALL function.  Choose YES.  

 The surroundings will be rehabilitated but the area within the green (identified) string will not be altered.

 

J) On the next dialog box select the SIDEWAYS or Picked Up option.

 

K) Select yes and define the overall area that must be rehabilitated.

 

Once the process is complete, verify the results by doing a Terrace Cut and Fill between the rehabilitated model and the original model. The cut and fill volumes should be in balance.

 

 

Summary:

 

This example describes a very effective method to rehabilitate an open pit ramp in the most economical way and it also ensures that the water will drain out of the ramp if the slope of the initial centerline string was defined as a drain.

This approach also eliminates the need to work with cross sections and gets to an acceptable design and requires much less user design time.
