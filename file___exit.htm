MENU : File - Exit                (Ctrl+X)

 

 

How can this topic help me ?

Select this option to leave Model Maker and return to WINDOWS. Be sure to save your data before you select this option.

INDIVIDUAL FUNCTIONS: Model Maker EXIT box

The status of the system data is displayed as follows:

 

DTM Project Data & Status

 

 

 

This box indicates the status of your project data.

The green check indicates it is saved as final.

 

CAD Data & Status

 

 

 

The checked  box indicates that there is a KCD file present in  your data.

The red dot indicates that it is in its final state and has not been saved as final.

String Data & Status

 

 

 

The unchecked box indicates that there are no strings present in your data.

The green check indicates it is saved as final.

Irrigation Data & Status

 

 

 

The unchecked box indicates that there is no irrigation data present.

The green check indicates it is saved as final.

Save Button

 

 

 

Use this button to save any data not in its final state.

Status Button

 

 

 

This button takes you to the status menu at the top of the main screen.

Desktop

 

 

 

If this box is checked, Model Maker will automatically save the last desktop on EXIT.

OK & Cancel Buttons

 

 

Select OK once you are happy with the information in the box.  Model Maker will EXIT if all data is saved and nothing has been changed. Select Cancel if you do not wish to EXIT Model Maker.
