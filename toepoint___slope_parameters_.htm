Cut slope

 

Specify the cut and fill batter slopes required for use in

toe-line calculations.

Other parameters are set as well Specify the slope of the

toepoints when the design is below the ground surface.

 

Fill Slope

 

Specify the slope of the toepoints when the design is above the

ground surface.  The slopes are set as a ratio of vertical difference / horizontal distance. In the example, the value of 1:2 means that for every horizontal 2 meters the elevation changes by 1 meter.

 

No. of points

 

Check the toepoints checked against the open sides of the

perimeter design triangles. Each side is checked individually and

the points for that side are determined. This parameter

specifies the number of toepoints that will be calculated per side.

 

Toepoints @

 

This parameter works with the previous one and regulates the

distance between the toepoints.

 

 If the (# of toepoints)  value is greater than zero, Model Maker ignores the (toepoints @)  and calculates the set number of toepoints per side.

 If the (# of toepoints) is zero, Model Maker will calculate toepoints at the specified distances.

 

Add every toepoint

 

If set to YES, Model Maker will add every toepoint to the database whether or not the points fall on a straight line. A NO setting will allow the program to only define points at a change of slope.

 

A NO setting is advisable. It generates a more economical model. Use the YES setting only if you need toepoints at regular intervals for setting out or for mining applications.

 

Sharp cut corners

 

This setting is related to toepoints in a cut situation and controls

the corner options. A YES value will produce a sharp end (see

bottom picture) and a No value will produce a rounded corner.
