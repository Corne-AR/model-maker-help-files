EXAMPLE : Working Example 7 (Plotting)

 Top  Previous  Next

 

Object

This working example shows you how to set up your system to get drawing output to an online plotter or printer or plot file.

Before starting this working example, load some data and triangles from any existing project of your own or use our ZX.MMS model.

 

Procedure - pAper - siZe / sCale / new P

 

 

These are the basic steps for plotting - General : siZe - sCale - new P. Find below a more detailed explanation.

 

STEP 1 : General information about plotting

Model Maker will plot directly to a plotter or printer that has a windows driver installed to generate the plot.

The plotter or printer driver must be installed using the normal WINDOWS install function (disk / CD supplied by manufacturer). Model Maker creates a standard plot instruction file that is sent to the current selected driver. The driver translates the instruction to the correct format for the specific device.

 

STEP 2 : Plotter connection

Using the normal WINDOWS functions to install your printer.

 

STEP 3 : Plot parameters

Set the plot parameters to obtain the correct output.  Click Parameters (main menu - top of screen)

 

 

Set all the settings to obtain the correct output :

To make sure the actual output will be to your requirement. Check it on the graphic screen before outputting to the plotter or file. Select NORM  on the top main menu and a screen plot will be done according to the plot parameters. If anything is missing or unnecessary, go back to the plot parameters and correct the settings. Note that the screen plotting is not done according to the set scale. The magnify to scale function produces a screen plot to scale.

 

 

STEP 4 : Testing the plotter connection

Set the communication parameters on the plotter and in the windows driver to the correct baud rate, data bits, etc., if the serial comms is used. No parameters need to be set for parallel communication.

 

Load some data to plot and put paper in plotter.

On the left menu, activate the GEN  button and the pAper  option at the bottom.

 

Activate the siZe  button at the bottom and select Printer setup for the paper, printer and orientation information box.

 

Activate the new P  button on the bottom menu and the paper plotting area you selected will be displayed on the screen.

If the paper window is much larger than the model, change the paper size or scale. Place window over model and click to register the paper window. The area covered by the paper window will be included in the plan when the plot is generated.

EXAMPLE: The white line around the model is the paper plotting area/window.

Model Maker does not communicate with the plotter directly. The physical paper size is obtained by getting the size from the printer driver which is defined by the user at the time when the driver is installed. The orientation of the paper windows is controlled by the landscape or portrait settings in the driver.

 

 

STEP 5 : Assign pens

You may assign specific pen numbers to the various  DTM  entities, eg.  CONTOURS, GRID  or  POINTS  or CAD.

From the top main menu, select the Settings - Colors option.

Click on the  button to go to the PLOTTER PEN SETTINGS screen.

Select a DTM > PLOTTER or CAD > PLOTTER entity and then select the pen number to be used from the palette titled CAD PEN : 0.

The required width for each pen can also be specified.

 

 

STEP 6 : Plotting

Plotting to the plotter is done selecting the Output Plotter/Printer option on the top main menu.

 

 

STEP 6c : Plot to .dxf

Select the Output DXF option from the top main menu and choose the options.

While the data is plotted to the selected file type, it will also be plotted on the screen.

 

 

Note:        

Only the entities drawn on the screen at this stage will be in the file. If anything seems incorrect, check your plot parameters for the necessary options.
