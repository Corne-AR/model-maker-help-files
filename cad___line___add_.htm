MENU : Cad - Line - Add                                

 

Add lines - CAD lines can be entered in various ways.

Use the Pen menu function to specify the type of line, pen and layer.

Use the spacebar to specify the method of point entry. (Cursor mode)

 

 

REAL POLYLINE ENTITY :

If this checkbox is checked the new lines will be created as real polylines. A real polyline is a single entity - although it may have many segments - and can be copied, moved, deleted etc. as one element.

 

It should be noted that a polyline (as individual segments) and a real polyline (as one entity) does not appear much different on the screen or plotter. The main difference is the way they are manipulated.

 

To break a real polyline up into individual segments, use the Identify function or the Explode option on the Cad, Line menu.

 

 

2 Point line

 

Enter single lines between two points. Use the cursor modes to fix the endpoints.

Press the "+" key  to undo newly added lines.

 

Polyline

 

 

Enter a polyline between indicated points. Indicate the start point, then indicate all the other nodes along the line.

Press Escape to end the sequence. Although the function is called POLYLINE, the resulting lines are each individual elements (unless the REAL POLYLINE ENTITY is checked) and the identify or global functions must be used to manipulate the segments as a whole.

The (P)en and cursor mode functions can be changed while you are entering lines. Press the "+" key if you want to delete the last line segment.

 

 

Use the Shift Screen together with Polylines to help minimize panning. This will centre the screen position on the point you snap to. Use this when drawing lines on a Road or pipeline survey manually. Normally if the survey Codes are not adequate for the  “Add Using DTM” function.

 

 

Rectangle

 

Enter a rectangle by indicating the two corner points. The four lines around the rectangle will be added to the database as regular line segments. The Rectangle can be drawn as 4 individual lines or with the Poly tick on as one Real Polyline.

Use the Keyboard shortcuts to draw a Square (R) or Rectangle (i) If you press (i) again the function reverts back to default Horizontal vertical.

 

3 Point Rectangle

 

 

Draw two connected lines and automatically have Model Maker construct the remainder of the rectangle. If the two lines are not perpendicular, the resulting shape will be a parallelogram. See 2 Line Square for more detail.

 

2 Circle Line

 

 

Add a new line tangent to two circles. Normally, there are 4 possible line positions: one on each side of the circles and the 2 lines crossing in the center that join the 2 opposing tangent points. If the circles touch each other, the last 2 options are not possible. If one circle falls completely within the other, no new lines are possible.

 

 

Sketch

 

Use this function to sketch a line when the POLYLINE function does not have the desired effect.

The input box next to the button specifies the length of each segment in millimeters based on the current drawing scale. In this example, 5 mm is specified. If plotting at a 1/2000 scale and in meters, then Model Maker will automatically create a new line segment when the cursor has moved 5 mm or 5*2000/1000=10 meters from the previous point.

 

 

Parallel

 

Use the parallel add dialog box to specify the positions of the new lines.

 

 

Par-Trim

 

 

 

Create a parallel trim around your image by clicking on the series of sides you want to be trimmed.

 

 

 

Copy line

 

Copy any existing CAD line to another position.

Indicate the CAD line. The cursor will jump to the closest endpoint. An image of the line can now be moved around and placed on any position. While moving to the new position, the following keys can be used.

1)   The (CTRL) keys changes the cursor mode.

2)   The Page up and Page down keys rotate the lines.

3)  The Home key makes the line horizontal.

4)   The Ctrl Home key changes the line length and angle.

5)  The Shift Tab key changes the angle to match the closest cad line.

 

 

Line

 

 

 

Draw the number of lines chosen from the radio box on the midpoint between two lines. (Only 1 line shown below)

 

 

 

Angle

 

 

 

Draw the number of lines chosen from the radio box on the midpoint of the angle between two lines. (Only 1 line shown below)

 

Add using DTM

 

 

 

Use the special dialog box that allows you to add CAD lines automatically onto survey points.

Exit after method

 

If this check box is selected, the program would not return to the dialog box each time after lines have been added using the desired method.
