MENU : gXt - Add - Grade

 

Generate a new string based in a set string slope and the active model.

 

Indicate a starting position. Model Maker will determine the elevation on the triangular surface at that point and start defining a string that follows the ground at the defined slope. If the slope value is positive, the string will continue uphill until it cannot extend further (eg. the top of a hill, the edge of the model).

 

A negative slope will force the string to go downhill.

 

Specify if the string should go to the right or the left around the hill.  If a specified slope value is greater than the maximum slope of the terrain, the string will take the most direct route up the hill at the maximum ground slope.

Sharp Corners

 

 

 

 

 

 

 

 

1) The minimum slope at which the string will be generated.

 

2) Check this box if steeper slopes should be used to try and avoid sharp corners.

 

3) If the change of direction is more than this value, the program will increase the slope to try and decrease the sharp corner.

 

4) The maximum slope that will be used when adjusting the direction.
