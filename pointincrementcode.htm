MENU : Pnt

 

Specify the value of the text (for CAD text COPY+) or the DTM points (REN+) to be incremented each.

It is important to enter an original text value that ends on a numeric character. This function will have no effect if the original text is eg. TREE.

The text must end on a numeric eg. TREE5 or TREE10.

The original text will be used during the first round where after the increment value will be added.
