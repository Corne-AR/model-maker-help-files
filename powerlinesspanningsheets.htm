After using the Span sheet function, a copy of the string and all the node (pole) information is copied to cad.

 

A user defined title block is used, the string is broken into the required lengths and stacked on multiple sheets.

What is needed to create the desired results?

 

1) A string defining the powerline.

 Follow the normal procedures to design the powerline.

 

2) A user defined title block

 Use these steps to define the title block.

 Before defining the elements of the title block, read this topic in total to find out how the layers should be named so the span sheets function can automatically populate the title block headers with the correct information.

 

3) The correct data entered on the Span Sheets dialog box.

 

A user defined title block is really an extended definition of a regular pattern.

 

When the pattern is used as part of the title block, the program will check for cad text elements on certain layers. The original text on the specific layers will then be replaced by the powerline design data at the time when the title block is used.

 

Below is a list of the assigned layer names.

 

Note that the layer name must have a '?' as the 1st character.

 

The base text in the pattern must have a series of '*' characters to mark the actual position of the replacement text in the original string.

 

 

 

 

       

Pole Name/Number is the text that will be written next to each pole. The last character is normally a numeric value and will automatically be incremented for each pole.        

 

Default Text Font refers to the font defining all the new text elements that Model Maker adds to the drawing.

 

Title Block Name. The title block (pattern) used to create the spanning sheets.

 

Also copy cad? If this is checked, any regular cad elements around the original position of the powerline will also be copied to the span sheets.

In addition to the items on the standard page, the user may add up to 9 text replacement items of his own.

 

User Description. This is not used anywhere and only assists the user to better reqocnize each entry.

 

Cad Layer Name. This refers to the layer name in the cad on which the cad text to be replaced, has been defined. Remember, these layer names must start with ?

 

New Text. This is the text that will replace the ******* in the original cad text.
