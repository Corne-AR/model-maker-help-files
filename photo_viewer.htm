DTM PHOTO VIEWER

The DTM Photo Viewer in Model Maker is activated by selecting the photo viewer icon on the “Speed menu – CAD and other” toolbar:

1.For this function to work, the user must load DTM points into an active DTM project where:
·the CODE field is called : #PHOTO
·the TAG field is the file name of the image it will represent. Do not include the file extension as part of the TAG field

For example:

It is not a requirement to group photos of similar meaning together in folders as long as the CODE is #PHOTO and the TAG is the image name; Model Maker will recognize it as such and show the image preview specific to the point TAG description and only those points of which the CODE is also called #PHOTO.

 

2.To associate a DTM project with a specific folder on your PC the viewer will start by displaying the Source Directory.
 

Step 1:
Please select the folder where your images are located.
 

Step 2:
Then click the Viewer button to start viewing your images by simply hovering your mouse over the DTM points with images associated with them.

 

3.There can be any number of folders with images in it. These folders can be associated with different DTM projects in Model Maker as follows. Let’s assume you already loaded correctly formatted DTM points (see step 1 above) into project 2 and you want to associate a new folder with that DTM project.

Step 1:
Activate the project number containing your new points
 

Step 2:
Select the appropriate folder on your PC, it can be any folder on any drive with any description.
 

Step 3:
Click the Viewer button. This step is needed to record your correct folder and to link the folder path to the correct project number.
 

Step 4:
For the active project number, hover your mouse over the DTM points and view your images.

 

4.The linked address of the folder is stored in the Project Notes as follows:


 

The number in the square brackets [ ] represent the DTM project number and immediately after the ] the folder path that the DTM project is linked to. Should your folder location change for whatever reason or should you share your photos with another Model Maker user with his/her own folder structure,  the values in this list can be edited manually. Or simply follow the steps in section 3 above to relink your DTM project with the correct folder.

 

Please be reminded that:

·Project notes are ONLY stored if you save MAL files.
·Images are stored on your PC in their own folder and are not saved as part of the MAL.
