STRING : Vert - Relative grade

 Top  Previous  Next

 

MENU : gXt - Vert - Rel_grade

 

 

Functions that would regrade or change the elevation of a string based on the position and elevation of an indicated base (controlling) string.

1)                

Recalculate the elevations of the indicated string based on its position relative to the base string. Each node on the indicated string will find the elevation on the base string directly opposite its own position. The elevation of the node will changed based on the base string and the method used.

 

 

1A)   The node elevation will be equal to the base string node +- the difference specified.

 

 

1B)   Calculate the distance between the node and the base string. The new elevation will equal the base elevation +- the distance between the strings * the % / 100.

 

 

 

2)  The same formula above is applied to an indicated section of the string.

 

 

3)  Model Maker will calculate an imaginary surface between the 2 base strings. The string will then be related to the imaginary surface and re-elevated accordingly.

 

 

Updating elevations based on position or elevation.

 

The 3 formats described above will update the elevation of the string based on the selection made in this dialog box and the selections of either 1A or 1B.

 

         1A

Section showing base string (A) and indicated string (B) to be regraded with 1A (Height difference).

The new elevation will be determined from A + H:

Keep elevation constant : B will move along arrow P until it meets the new elevation.

Keep position constant : B will remain on the same position but the elevation will move along arrow Z until it meets the new elevation.

 

         1B

Section showing base string (A) and indicated string (B) to be regraded with 1B (Slope percentage).

The slope will be determined from A:

Keep elevation constant : B will move along arrow P until it meets the slope.

Keep position constant : B will remain on the same position but the elevation will move along arrow Z until it meets the slope.

 

 

4)                

This option will not really regrade a string, but rather add a constant to all the Z values of a string. The constant by which the original values are changed is calculated as follows.

         1. The base string is indicated.

         2. The regrade string is indicated.

         3. The base node of the regrade string is indicated.

         4. An elevation on the base string is calculated for the base node of the regrade string.

         5. The difference between the elevation of the base node and the new elevation on the base string is calculated.

         6. This difference is then applied to all the nodes of the regrade string.
