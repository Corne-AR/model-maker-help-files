MENU : Hor - Vehicle_turn

 

Use this dialog box to specify the vehicle that should be investigated.

Model Maker will use the current horizontal alignment and graphically track the path of the vehicle as it travels along the road.

Up to 3 units may be defined including the horse (pulling vehicle) and 2 trailers.

 

 1 

The image at the top of the dialog box is used to indicate the position of the measurements that are needed to simulate the vehicle configuration.

There are 14 values to be specified if all 3 vehicles are identified.

 2 

Input window is used to specify the values and is divided in 3 columns and 6 rows.

The data of each unit is entered in the designated column. The 6 rows are used to indicate the distance value at each point.

All the distances are entered relative to the front (point 1) of the first vehicle. In the example only 2 vehicles are used as no values have been entered for the second trailer.

Even if all the vehicles are specified, the connect at input box for the horse column would always have a zero value.

 3 

This window is used as a quick reference to remember what each code number is related to.

 

 

Plot Path

 

Draw the images of the vehicles in plan as it travels along the horizontal alignment. Each vehicle will be drawn to scale according to the defined sizes.

Use the input box to specify the increment interval. Experiment with the distance value before plotting the image to CAD.

 

Sight Distance

 

Apart from plotting the vehicles, Model Maker can also plot the envelope that is needed to maintain a selected sight distance. This is not related to the size of the vehicle and the only input is the distance of the sight.

 

This example shows the alignment and the sight line after it has been copied to cad.

The sight line is generated by going around the curve at regular chainage intervals. At each interval where the vehicle is, a second point is determined on the alignment where the distance between the vehicle position and the second point is equal to the defined sight distance. A theoretical straight line is then drawn between the 2 points and the edge of this line is used to determine the sight envelope.

The blue radial lines in the image represent the envelope area between the base alignment and the edge of the sight line.

 

 

An example of the sight distance measured along the alignment and the straight line connecting the 2 points. By determining the edges of lines created by multiple sight positions around the curve, the sight line is generated.

 

 

Options

 

Center Offset

Use this input box to specify a distance the vehicle will travel to the left (negative value) or to the right (positive value) from the defined center line of the horizontal alignment.

 

 

Plot in reverse

This setting will simulate the vehicle travelling from the end to the beginning of the defined horizontal alignment.

 

Copy to CAD

 

When selected, the lines used to define the sight or vehicle path will be added to the cad. If not selected, the lines will be drawn on the screen but the image will disappear when a replot is done.

 

Use patterns

 

Each vehicle can be represented by a defined cad pattern.

·Use the cad, draw the vehicle to scale and save it as a pattern.
·The vehicle must be drawn, pointing right to left.
· In this example, the horse is drawn as if travelling to the left.
·The pattern base point must be defined in the center of the front of the vehicle. This will be position '1' for the horse, '6' for trailer1 and 'b' for trailer2.
·If patterns are not selected or if the pattern does not exist, a generic box will be drawn to represent the vehicle.

 

Envelopes

 

The left and/or right envelope line can be calculated and drawn. When this option is selected, the plotting takes a few seconds longer to complete.

The horizontal alignment length must be less than 5000 meters if the envelope option is to be used.
