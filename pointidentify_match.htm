POINT : Identify - Match

 Top  Previous  Next

 

MENU : Pnt - Id - Man_id - Match

 

All the identified DTM points of the active project can be stretched between two co-ordinates. You will be prompted for a base point, a new position for the base point, a reference point, and a new position for the reference point.

This function uses scaling, rotating and moving on the identified points.

 

EXAMPLE:

BEFORE

AFTER

                                                                                 

In the example, DTM point at position A is matched to the line end point at B and the DTM point at position D is matched to the line end point at position C.

The original distance and direction between A and D has been transformed to coincide with that between B and C. This swing and scale factor is then applied to all the other identified points.
