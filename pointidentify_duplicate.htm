MENU : Pnt - Id - do_Id - Duplicated        

 

 

Before the the duplicate points are identified, the code dialog is displayed so the user can choose if all the duplicate points must be identified or only those with certain codes.

 

The code select option will be displayed and the points within the duplicate parameters will be identified.

         

 

General

 

The process of looking for duplicated points will scan the whole database and present the above dialog box each time a duplication is found. Click the OK button if you want the selected point to be identified, click the CANCEL button if the point must be skipped. To break out of the loop in total click the ABORT button.

 

The DTM points of the active project will be scanned and duplicate points identified.

 

 

 

First point

 

If 2 points are found to be closer than the specified radius, Model Maker will identify the first of the 2 points. The points are scanned in the order they were entered in the database. The first point will have the smaller database number.

 

Second point

 

The same as above, but now the second point will be identified.

 

User selected point

 

Model Maker will stop when duplicated points are found. Using the dialog box, indicate which point is be identified.

 

Radius or exact duplicated

 

Enter a radius value which Model Maker will use to find points. Check the EXACT duplicated box or set the radius to zero if you only want points to be identified that have the same plan co-ordinates irrelevant of the elevation differences

 

Elevation differences

 

If consider elevation differences is selected, the user need to enter the maximum value and only duplicate points with elevations within the maximum difference will be identified.

 

 

 

This dialog box is displayed when Model Maker is scanning the DTM points for duplications.  A duplication is found and the user chose to identified a user selected point.

 

 

 

Select point

 

Check either #1 or #2 to indicate which of the points must be identified.

 

Y & X values

The position of the points.

 

Number

The database points numbers of point 1 & 2

 

Code, Tag

The information of both points.

 

Elevation

The Z value of both points.

 

Triangle

This feature allows you to instruct Model Maker to change the triangles from one point to the other. In the example, the icon indicates that there are triangles connected to point number 681 and that point 1669 is going to be identified. If you would rather connect the triangles to point 1669, click on .

 Model Maker will update the database so that all the triangles previously connected to point 681, will now use point 1669 as the dtm point. You would normally use this option to move the triangles away from the identified point if the you intended to delete the identified points later on.

The SWAP button will be inactive if the distance between the 2 points is greater than 0.01. You are not allowed to swap triangles between points further apart from each other as the chances of creating overlapping triangles are then a real problem.
