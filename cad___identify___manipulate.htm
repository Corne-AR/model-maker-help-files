Lines

Write distance

 

cadline Point

 

DTM points on end of lines. This option will take all the identified CAD

lines and generate DTM co-ordinated points on their endpoints.

 

The new data may either be saved to an ASCII (.yxz) file or added to the selected active project.

 

Intersection point

 

DTM points on intersections of lines.

The same as CAD LINE POINT, but DTM points will be placed at all the intersections of the identified lines.

 

intersection cUt

 

All the identified lines will be inspected and all the common intersection points determined. The lines that do intersect with other will be divided into segment.

         

In this example, 4 lines are identified. After it has been cut at the intersections, the image remains the same but now there are 12 lines as each original line now consists of 3 individual segments.

 

Trim area

 

trim Line

 

All the identified lines that intersect with a selected base line will extend to the intersection point.

This has the same effect as trim line except that all the identified lines will be manipulated in one go.        

 

House keep - Option1

 

This unique function determines if the end points of all the identified cad lines are closer to each other than a pre defined user distance without being exactly on the same coordinate. Some dxf files that contain lines that are suppose to be poly lines (continuous segments meeting each other) have a small gap between them. When imported, Model Maker extends both the terminals of lines falling within that category to the average co-ordinate point ( (x1+x2)/2 and (y1+y2)/2 ) to form a continuous line.

Endpoint A of line 1 and endpoint B of line 2 will move to position C if RAD distance is less than specified.

 

Option2

Here the end points will move an identified dtm point

Endpoint B of line 2 will move to dtm point C if

dB is less than specified tolerance value.

 

The same for line 1. If dA is smaller than the defined tolerance value it will also move to dtm point C

 

Option3

Move end points to meet or trim to lines.

Endpoint A will extend and meet with line D

Endpoint B will trim and meet with line D

Line D and line C will extend to meet.

(if the move distance is less than the tolerance)

 

All the identified "real" polylines will be exploded to individual cad lines. The individual lines will still make up the image of a polyline but they can be manipulated as separate elements.

 

Text

Text point

 

Place new DTM points at the base co-ordinates of all the identified text elements. These points will be added to the active project and the elevation will be the same as the original CAD text string if the string had a numeric format.

Example : Text string '100.123' ; DTM elevation=100.123; text string 'ab12x'; DTM elevation=0.00

 

Text file

 

Write the identified text strings to an ASCII text file with a PRN extension. Each text string will occupy a single line.

 

All actual text of all the identified text elements will be copied to the editor.

 

 

All the identified text elements will be aligned in relation to a fixed text entity. The regular Text Align function is utilized

 

Circles/Arcs

Arc point

 

Place new DTM points at the base co-ordinates of all the identified circle/arc elements. These points will be added to the active project with a zero elevation. If a DTM model is loaded in project 1 and project 2, then you may specify that the elevation should be the height difference between the surfaces of the 2 models at the circle centre point.

 

Circ linez

 

The identified circles will scan all the identified CAD lines and trim the lines whenever a line crosses the circle perimeter.

 

Example:

 

BEFORE

 

AFTER

 

Circ line Chord

Place CAD lines along boundary of identified circles.

The regular Circle Divide function is used and applied to all the identified circles.

 

Circ info

Copy the centre co-ordinates and radii of all the identified circles as CAD text.

 

 

This option allows you to transform an id'd circle. After the start and end angle of the circle are entered, the id'd circle assumes the new values.

 

 

 

 

 

The Stope Sheet function is used in a mining environment and is used to stretch the identified cad elements in one direction.

 

1) A base point must be indicated.

2) A Slope direction must be indicated.

3) The dip angle must be entered.

 

All the identified cad elements will be moved relative to the base point in the direction of the slope by applying the cosine formula, using the dip angle value.

 

The stretch are calculated from plan to projection or visa versa.
