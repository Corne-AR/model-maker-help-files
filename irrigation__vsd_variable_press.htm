IRRIGATION : VSD (Variable pressure)

 Top  Previous  Next

 

The functions on this page allows you to define different pump pressures when running mainline shifts.

Normally all the shifts will be calculated based on the pump pressure specified in the pump pressure box. By defining variable pressure values for each shift and also check the box, the hydraulics for each shift will be calculated by using the defined shift pump pressure (B).

 

The basic way to manage the page would be to . The grid editor will be filled with the calculated data and the suggested shift pressures displayed in (F)

 

If the suggested shift pressures are what you want to run the system at, click and the pump pressure (B) will be changed to the suggested values.

 

Check and the hydraulics for each shift will be done with the set pressure for each shift.

 

 

   

 

1)

Initially the grid editor will be empty. Use this button to do the calculations and fill the grid with the new values.

2)

When report (1) is run, the new suggested shift pressure is calculated so that the pressure available at the critical valve will be that of the valve required design pressure plus the margin entered here.

3)

Use this button to do one of 2 things. To either set the pump pressure for all the shifts to a (4) constant value or set the shift pressure to the new calculated (5) suggested value.

4)

If this is checked, the shift pressures will all be set to the entered constant value when (3) is pressured.

5)

If this is checked, the shift pressures will be set to the suggested pressures as displayed in column (f).

6)

Check this box if the regular hydraulic calculations must be run with the pump set at the designed shift pressure. If this is not checked all the shifts will run a the global pressure set for the pump.

Note : This check box does not remain checked. Each time you exit this dialog box and return to the main irrigation menu it is unchecked. You need to check it again when returning to hydraulic calculations.

7)

The contents of the grid editor will be copied to clipboard allowing you to paste it in Excel for instance.

Grid editor columns

a)

The flow of each shift

b)

The pump pressure for each shift.

c)

The actual pressure at the critical valve if the shift is run at the pressure in (b)

d)

The design pressure of the critical valve in the shift

e)

The difference between the actual and design pressures. A negative value indicate the pressure at the critical value will be insufficient.

f)

The suggested pump pressure for the shift so the pressure at the critical valve will be that of the design pressure plus the values specified in (2)


 

When the hydraulics is run with (6) checked, the reports on the Info tab will indicate the shift pressure accordingly.

 

8) Run Total Simulation. Open the Simulation dialog to run and analyze various combinations of open valves.

 

 

 

 

User set Pressure:

 

Apart from a constant pressure or suggested pressure (F) the user may enter a desired shift pressure in (G) and set the starting values (B) to this user defined value.
