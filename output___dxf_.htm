MENU : Output - DXF

 

This option creates a DXF ASCII file that can be read by most CAD programs.

 

                       Enter the file name without the DXF extension when saving the data.

         All the data selected by the plot parameter menu will be exported.

                       To add the contours of the DTM to the file, activate the CONTOURS check box in the plot parameter menu.
                       If the output is done in 3D and the TRIANGLE check box in the plot parameter menu is selected, the triangles will be outputted as 3D faces in the DXF file.
