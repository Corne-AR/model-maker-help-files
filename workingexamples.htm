Working Examples

 

Example.0 : All the steps in pictures.

 

Example 1 : to help you create your first DTM to obtain a contour plan.

 

Example 2 : to show you how to enter survey data manually using the editor.

 

Example 3 : to show you how to create the horizontal alignment on which to extract sections and then to extract the long- and cross-sections.

 

Example 4 : to create a three dimensional wire frame perspective view of your model. You must have your data and triangles loaded for the model.

 

Example 5 : to show you how to determine toe points for a terrace model and then to extend the model to the toe line.

 

Example 6 : to show you how to calculate a cut and fill volume of the earthworks necessary to build a typical platform on existing ground levels using two projects. The example uses data that is supplied with your Model Maker package.

 

Example 7 : to show you how to set up your system to get drawing output to an online plotter/printer or plot file.

 

Example 8 : to show you how to create your own line styles or edit existing line styles to be used in Model Maker's CAD module.
