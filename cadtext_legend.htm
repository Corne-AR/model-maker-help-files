MENU : Cad - Text - Load - Dtmlegend

 

 

 

Place a legend describing the codes of the DTM points on your drawing .

 

Use this to maintain your own sample legend and add to it from the codes used for each new project.

 

When the legend is copied to CAD, all the elements will be saved as CAD lines and text.

 

Explanation of dialog box.

 

Select Type : The function can either work on the codes or tags of dtm points.

 

 

The codes used in the active project are displayed. Choosing the different codes used in the project could take some time. For that reason, the box is initially blank. Use the  button to display all the codes. The example shows the codes already loaded.

Use the Scan ID option if only the identified points must be used.

 

 

Use the button or double click on a selected code to copy it to the legend.

 

 

This data will be displayed in the legend. It consists of the code and the user defined description of the code.

The data base of codes and descriptions are retained by Model Maker, even if you exit the program. You may work with different databases. They are saved and loaded from disk with the following buttons:

 

 

Use the button to display the entries in the legend in alphabetical order.

 

The button will transfer the database to the legend in the cad. Select WHOLE LEGEND to copy all the entries in the legend to CAD. Select PROJECT LEGEND to copy only the codes in the current project.

 

The resulting string in the cad can either be a single string where the codes/tags are combined with the description as one entity. If the 2 TEXT Entities is selected the codes/tags will be a column of text elements and the descriptions will be in their own column of text elements. In this instance, after the base point of the codes/tags is indicated and the data copied to cad, you will be asked to indicate a second base point where the descriptions will be copied to.
