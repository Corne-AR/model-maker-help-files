Line type

This displays the line type. Click the Change button on the top right hand corner to change the line type and pen colours.

 

Pattern

When the hatch has been selected this screen will appear.

 

Apply

To copy a colour from a hatch click on it and select the next hatch. The hatch will then change colour.

 

 

 

 

Layer

Use the drop down arrow to view all the current layers and change if necessary.

 

Change

Once the hatch has been defined, click on the Change button to change the line type and colour.

 

Line type

This displays the line type. Click the Change button on the top right hand corner to change  the line type and pen colours.

 

Pattern

This shows the current hatch type.

 

Spacing

The spaces can be defined between the hatch lines. Enter up to 5 spacing distances in millimeters. The number of hatch lines in an area will depend on the scale of the plot.

 

Fill Pattern

Border                : Click the box to draw a border around the hatch area.

                   Use this area if the hatch area is not already defined by CAD lines.

 

The hatch pattern may be made up by the following:-

 

Single lines

: The single line pattern uses the angle as set in Angle 1

  Below shows a hatch using single lines.

Cross lines

: The cross line pattern uses the angle as set in Angle 2

 The example below shows a hatch using cross lines.

Fill                

: The hatch will be completely filled as shown below.

 Define the percentage using the spacing input box.

 The fill percentage may vary from 0% producing a white colour to   100% which is equivalent to the selected line type.

 

The hatch line type is also defined by the Pen parameters. Set the correct line type before the hatch is indicated. Use the CHANGE button to change the line type and colour after the hatch has been defined.

 

Default

When this box is checked the current set of parameters automatically become the default.
