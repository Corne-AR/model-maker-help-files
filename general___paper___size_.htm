MENU : Gen - pAper - siZe

 

Model Maker uses the Windows plotter/printer drivers to output a scaled drawing to paper. The drawing is generated according to the current scale and to the paper size specified in this dialog box.

Select five theoretical dimensions or the physical paper size.

 

 

Physical

 

This paper size relates to the paper size as specified in the windows drivers installed for the printers. Use the button to access the dialog boxes used to control the printer attributes.

With this setting, Model Maker uses the entire space of the specified paper size.

 

Theoretical

 

Enter the dimensions for each paper size in millimeters. The sizes are automatically saved for later use. The buttons on the left indicate the current selected paper.

These buttons allow you to plot to, for example, an A0 plotter. But only use an A3 size of the paper.

The same result could be achieved by changing the physical paper size in the option.

 

General

 

Model Maker installed with the standard values for the ISO papers. In some cases, this may result in a window frame that is too large for your plotter, because the pinch wheels use space on the border of the paper. If you find that the top and/or right hand side of the paper border is not displayed on the plot, decrease the theoretical paper size until the frame fits on the paper. You can also specify a border area around the paper window.
