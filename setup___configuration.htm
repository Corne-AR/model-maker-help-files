MENU : Settings - Configuration

 

Here is a example of the setup screen.  All the fields except the Modules may be edited or set.

 

 1 

These two boxes allow you to enter your details which will be printed as headers in reports and data lists.

 

 2 

a.

Set the survey system axes and increment to cater for survey co-ordinates in the

Southern or Northern Hemisphere.

 

b.

The system can work in the regular 360 degree angle format or in grads.

The grads system works in decimal angles and divides a full circle in 400 degrees.

 

 3 

b.

This grants the user 2 levels of access, either full or demonstration.  If you have a registered copy of the software you may run it in full access mode.  All functions may be evaluated in DEMO mode, but you may not export drawings and quantities as they may not be correct.  Take note that if the security key is removed while the program is running it will revert to DEMO mode and the data may be lost.

 

d.

Allows the user to select which unit they want to work with.

 

 4 

Use alternative language.

Toggle this box to switch between using the alternative language definition and the default language (English). The alternative language option will display some of the dialogs in the text as defined by the MM_Language.def file

 

 5 

 

Shows which program modules are active.

This is determined by the modules purchased. Only functions for the active modules may be used.

This example shows that five the modules are active.

 

 

Security Device

The number of the security key connected will be displayed. In this case it is Dongle Number 98:286

 

 

Search Dongle

Click here to tell the program where to search for your dongle or security key. The key works off the parallel port. There could be more than one port on your computer. You need to instruct Model Maker where to look. If you are not sure select the "Search" option as  shown here. Click on the down arrow on the right of the input box to see what other options are available.

The example shows that dongle 97:108 is connected. 97 shows the year the dongle was issued and 108 indicates the 108th key issued in 1997.

Security keys are valuable items and should be insured for the replacement costs of your package. If they are lost or get stolen they will not be replaced.

 

 If they are faulty a replacement dongle will be issued provided the faulty dongle is returned. The current price of the actual dongle will only be charged.

Remember that the dongle is an electronic device and is therefore subject to the normal hazards that computers are subject to. Power surges and faulty ports can cause the dongles to malfunction. It is up to the user to insure that the computer and any attached devices are working correctly.
