MENU : File - New

 

 

Delete DTM , CAD, String and Irrigation data.

 

 

 

Status box

These button will either have a green yes mark or a red no mark. Green means the data is saved in the final format ,while red means the data is not in a final format.

Save the data first.

Go to the status menu.

 

Deselect all the check boxes.

 

Select all the check boxes. The whole memory will be cleared.

 

Check this box for Model Maker to ask you to confirm the action before the data is finally cleared.

 

Click the OK button to start the clearing action. All the points of the selected data  will be deleted from memory. Use with caution, the data cannot be undeleted except by loading an original file from disk if the data was previously stored.  

 

Click the CANCEL button to exit without clearing anything.
