MENU : Tri - Wipe

 

 

TRIANGLE : Spot

Delete all the triangles connected to an indicated spot. If a certain point influences the contours in such a way that it is unacceptable, do not delete the point, but just remove all the triangles connected to it. This leaves a gap in the model which can be filled with triangles using any of the  ADD TRIANGLE  functions.

 

Remember, when closing the gap in the model, build the triangle so that no triangles are connected to the incorrect point. The continue search option cannot accomplish this task without taking some precautions.

Change the code of the incorrect point by adding a "." to the code. For example, Rock becomes ".Rock" would automatically instruct  MODEL MAKER  not to connect any triangles to this point.

 

You may also add one triangle manually in such a way that the incorrect point is covered by the new triangles. The  CONTINUE SEARCH  function would then ignore the point because any point falling under the triangle model is automatically excluded from the triangulation process.

 

Delete all the triangles connected to an indicated dtm point.

 

 

        PIC 1                  PIC 2                   PIC 3

 

You may find that a dtm point should not be considered to influence the contours. This often only becomes evident when the contours are inspected. In (pic1) is an undesired point connected to triangles. The wipeout function deletes the triangles connected to the point and creates the gap (A) in (pic2). The triangles numbered 1,2 & 3 are added manually, resulting in the point not influencing the model.

 

 

TRIANGLE : Last

Delete the last found triangles. Model Maker will automatically supply the value of the number of triangles that were generated during the last continue search option. By accepting this number, Model Maker will delete the last generated triangles, leaving the model in its original state.

 

Use this option when, after two models have been combined and continue search is used to fill the gap between the models, you find that the search process goes in a loop, generating duplicated triangles over each other. The wipeout last function will delete all the incorrect triangles.

 

TRIANGLE : Area

Inside

Delete triangles within a block. Define a rectangle or polygon. Only the triangles having all 3 points inside the defined area will be deleted.  

Outside

Delete triangles outside a block. Define a rectangle or polygon. All the triangles having all 3 points outside of the defined area will be deleted.  

 

TRIANGLE : Line

Delete all the triangles cut by an indicated line. This is often used to delete all the triangles along a road, between 2 separate projects, or at any instance where a strip of triangles needs to be deleted. Delete all the triangles that are being cut by a user indicated line.

      BEFORE                                 AFTER

 

Indicate the line in any position.

 

TRIANGLE : Id

Delete all triangles connected to identified points.

 

This method holds many advantages that may not be apparent. Model Maker automatically applies the  WIPEOUT SPOT  function on all the points that have been identified. Identifying the points is done in the  POINT ID  menu.

 

Although we will get to that section of the program, at this stage remember that identified points can delete certain triangles.

 

TRIANGLE  :  Duplicate

Model Maker will check for triangles duplicated unintentionally and delete them. Duplication can occur when triangle files are linked together.  

 

TRIANGLE  :  Legs

 

 

The QUICK search method of generating triangles often produces triangles on the side of the model with long legs.

 

 BEFORE

 

 

The wipeout legs function automatically inspects the model and deletes any triangle on the side of the model with legs longer than the specified value.

 

 AFTER

 

Legs longer than 60 meters have been deleted.
