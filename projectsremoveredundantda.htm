PROJECTS : Removing redundant points

 Top  Previous  Next

 

Use the POINT ID NEW  function. This will ensure that no points are identified. Remember to activate the correct project. Now identify the points connected to triangles.

 

The identified data may now be copied to another project (leaving the redundant data behind) or saved as a total file to disk.

 

The fact that the Project Copy  function operates on the  identified  points of the active project makes this process very powerful and can be used to merge data, separate data and indirectly, delete data.

 

The data of the active project is in no way disturbed during the copy process.
