MENU : Irr

 

On the nodes page of the irrigation menu you can select that the nodes must be represented by the images as defined in the shape (mmw.shm) file. This shape file can either be edited with a regular text editor program or you may use the functions in this dialog box to manipulate the shapes.

 

This dialog box is divided in 3 parts.

 

On the left is the Shape Names list box. All the current defined shape names are displayed. To view the co-ordinates of a particular shape, click on the shape name.

The co-ordinates of the segments within the shape will be displayed in the centre grid editor.

An image of the shape will be displayed in the graphics box on the right.

 

You may at random, change, delete or add new segments in the co-ordinate grid editor. Any change will be reflected in the graphics box.

 

When you are done editing the shape, click the button. The new shape will be recorded in the MMW.SHM file. Click the cancel button if the changes must be discarded.

 

Click the button to add a new shape name to the list. Activate the new shape by clicking on the new name in the shape name list box.

 

Click the button to completely remove a shape from the shape files.

 

Use the value to temporary enlarge the image of the shape while doing the editing.
