EXAMPLE : Working Example 2 (Enter survey data)

 Top  Previous  Next

 

Object

 

This example shows you how to enter survey data manually using the editor.

 

Procedure

 

To begin, you need the final co-ordinates, elevations, instrument height and correction angle of each survey station.

 

STEP 1 : Start new project

 

At the main menu on the graphics screen, select the DATA menu option. This activates the DTM DATABASE EDITOR.

The editor consists of three individual screens which deal with : STATIONS, POINTS and GENERAL.

Use the NEW button on the stations menu to clear the system.

 

STEP 2 : Enter station data

 

The "STATIONS"-screen will now be displayed. On this screen a maximum of 50 stations per project can be defined.

 

 

This example shows three actual survey stations and one coordinate station.

               For the first three stations the name, tag, co-ordinates, instrument height and correction angle are entered.

         These stations are used to hold unreduced survey data.

               The fourth station is called KOORD and does not need any coordinates.

         This type of station is only used to hold survey data consisting of final coordinates.

 

Once you have entered all the data for the stations, select the "POINTS"-screen to enter the data for each individual station.

 

 

STEP 3 : Enter station configuration

 

Highlight the station and select the "POINTS"-screen to automatically activate the "STATION CONFIGURATION"-dialog box.

 

         

 

The configuration for each station may be different. Select from the following options depending on your survey method:

 

ANGLE FORMAT        

 

 

DISTANCE AS

 

ELEVATION AS

-  Choose whether the measured angles will be entered as a decimal value or in degrees, minutes, seconds. If you choose DMS, enter in the format  DD.MMSS.

 

-  Select how the position of each observation will entered according to your survey method.

 

-  Select how elevation of each observation will be entered and calculated according to your survey method. Once the station configuration is specified. enter points.

 

STEP 4 : Enter survey data

 

In the editor, first enter a code for the observation point. Then use the TAB-button on the keyboard to tab from entry to entry. Model Maker calculates the final co-ordinates and elevation for each observation once all the data is entered.

 

Enter the data for each station separately. Once you are finished with a station, press the OK button to exit the editor.

 

Note:

If you enter actual survey data for a station and the data must be changed, update the station data. All the observations from that station will then be recalculated.

 

STEP 5 : Plot data on the screen

 

Confirm that the data entered correctly by viewing it on the graphics screen.

 

Once OK is selected on the "POINTS EDITOR" you will be taken back to the graphics screen.

 

All the points will be plotted on the screen. If the orientation of certain points look wrong, check the station data first. Usually the correction angle is incorrect.

 

Edit points individually using the DATA option on the main menu and then the POINTS-page after that. Then, select the station that holds that data you want to edit and continue from STEP 3.

 

Note:

If your project has more than 50 stations, enter the data for the first 50 stations, plot it on the screen to check it, save it to a disk as tachy data, and then also save it as ASCII data. You may then recall the data as ASCII and all the data of the 50 stations will be under one station as Code, Y, X, Z data. The rest of the stations for the project can then be added.

 

If you have observations taken to stations, they will be duplicated in the database and Model Maker cannot create a triangle model if duplicates exist. A good practice is to put a period (.) in front of your station names. Model Maker will then ignore the station in the triangulation process and only use the observation point. If you have any other points in the database that should not be part of the triangulation, put a full stop in front of their codes.
