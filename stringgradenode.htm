STRING : Vert - Grade Node

 Top  Previous  Next

 

MENU : gXt - Vert - gradeNode

 

 

Grade all the nodes between 2 indicated nodes of a string to an even slope.

The grade menu is displayed after the 2 nodes have been selected. The grade menu works interactively. Changing any of the components automatically updates the information of the other parts.

 

 

Change the elevations of node1 or node2. Node2 is related to node1 via the indicated slope. You may change any of the 3 input values. Model Maker will automatically update the other part of the formula. Changing the EL2 value from 10 to 5 will update the EL difference automatically to 4 (5-1). You could have also changed the EL difference yourself to 4, resulting in the elevation of EL2 being updated to 5.

 

 

This is the slope parameters as measured from #1 to #2. Any changes to these input values will immediately recalculate and update the elevation of EL2. The elevation of EL1 is fixed unless you change it directly in the input box. The slopes will have negative values if EL2 is less than EL1.

 

 

 

Selecting CURVE 1 will change the elevations of all the nodes between indicated node 1 & node 2 to that of a parabolic vertical curve. eg.

CURVE 1

CURVE 2

Selecting CURVE 2 will also produce a parabolic curve but the shape of the curve will not be based on the slope of the segments of node 1 & 2. A third node must be indicated. Model Maker will calculate a curve from node 1 to node 2 that runs through node 3. Use the special node move function if you want to place node 3 in the correct place before the grade is done.

 

 

 

If this is checked and the nodes of the string fall exactly on existing DTM points, Model Maker will change the elevations of these points to elevations of the regraded string. The same result could be achieved manually by using the Utility update option.

 

Grade segment

This option is similar the grade nodes except that only a single segment is indicated. The start and end nodes of the segment are then manipulated in the grade menu.
