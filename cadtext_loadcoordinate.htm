MENU : Cad - Text - Load - loadcoord

 

A specific ASCII file will be read from disk and placed on CAD at base points as defined by the ASCII file.

 

Hint:

The following lines should be viewed with the help screen enlarged to the full screen size.

 

Example of a line in a typical file:

!     !     ~         ~      *                *      %            %    ^             ^      &             &    #             #  (this line only for explanation)

BH1           10.123             24500.234              5460.201            1202.234           1201.900     1200.123

 

The co-ordinates of the base point must be on every line of the text file . You will be asked to specify the start and end columns for the Y and X co-ordinates. In the example, the column numbers are * * and % %.

 

The ASCII characters between these columns determine the position of the text for that line. The text will be placed at Y24500.234 and X5460.201.

Specify the columns of the text's start and end points to be used on the line.

If the bore hole name must be placed at the defined co-ordinates, enter the column for ! !.

Or if the whole set of elevations should be written next to the point, in which case the column should be specified from ^ to the last #.

 

It is important to have the default text parameters set correctly before the operation. It is also good practice to set the drawing scale correctly beforehand.
