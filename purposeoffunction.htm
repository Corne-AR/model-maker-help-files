Model Maker extensively uses the identification process to globally manipulate groups of points.

       Select the points to be identified by using the various options.

 Remember that the identification process will only work on the currently active project.

 

Many operations that can be performed on the identified data set.

 

Here are a few brief examples:

       Identify a set of points by a certain code (eg TB) and change the code to BB.

 Remember that the search for codes is case sensitive. TB is not the same as Tb.

 

       Identify the points on the side of the triangulated model and then copy them  to project #2 to be triangulated again.

 If the original model (say in project #1) constituted a mine dump or quarry, the new model created in project #2 will then        represent the surface of the original ground.

 Quantities may then be calculated between the two.

 

       Identify points within an area and then change all the elevations by a constant

 

This clearly shows that manipulating identified points is a handy versatile tool which could be applied in various ways.
