Irrigation : Pipes - Length

 Top  Previous  Next

 

Use this function to change the length of a single pipe, user indicated pipes or the laterals being cut by a user defined base line.

 

There are 2 ways to get access to this dialog box.

 

1) On the pipe edit dialog box, click the Length option.

The length dialog box will be shown with these available functions.

The Remove nodes and User defined base line options are not available.

 1 

Get the length from and indicated pipe.

 2 

The new length.

 3 

Only apply to the edited pipe.

 4 

Apply to the edited pipe and other indicated pipe.

 

 

2) On the main irrigation dialog box, click the Length option.

 

The length dialog box will be shown with these available functions.

 

Since you did not access this dialog box from a specific pipe, only the indicate pipe length and User defined base line options are available.

 

The Remove nodes function is also available.

 

 1 

Nodes between 2 same type pipes, in a straight line, will be deleted to form a single pipe segment.

 2 

User indicate a base line, laterals intersected by the line will change lengths.
