MENU : Irr - Shifts

 

Apart from manipulating the shifts in the regular shift grid editor or by typing the shift numbers directly in the block page, Irri Maker allows you to manage the blocks and shifts in the graphics screen.

 

For explanation of the shifts in Turf design, click here.

 

 

Shift Numbers

Select any number, the blocks active in the shift will be displayed. Double click on a shift number. The active blocks will flash.

 

The list box display a list of all the possible shifts that can be used. Click on any number, the graphics will display the updated shift numbers within each block.

 

By double clicking on the shift number, the active blocks will flash, indicating the areas that operate in the selected shift.

 

The number next to the shift list indicates the number of blocks that are active in the selected shift.

 

 

 

 

Three colours are used.

 

ACTive

 

 

Dead

 

OTHer

: The blocks active in the selected shift will be labeled with a square block having the selected Active colour.

 

: Blocks that are not active in any shift are labeled with this colour.

 

: All the blocks that are active in another shift will be marked with the OTHer colour.

 

 

 

 

Inc/Exc

When a shift number is selected, the active blocks will be identified with the ACT colour. Any of the blocks may then be added or removed from the selected shift.

 

To add a block to operate in the shift, just click inside the block. If the selected block was active in another shift, it will be taken off that shift and included in the current shift.

 

If the EXC option is selected, the block will be taken out of all the shifts and labeled with the DEAD colour.
