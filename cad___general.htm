MENU: Cad - Gen                                        

 

Use this dialog box to manipulate certain utilities and copy DTM elements to CAD.

 

 

Copy

Converts DTM and other information and into CAD lines. You may enter the layer name onto which it will be copied. Once the information is in CAD, all the normal CAD functions can be applied. Use with caution because there is no check to prevent multiple copies of the same information.

 

Grid

Copy grid lines to CAD. Set the grid line spacing in millimeters. The number of grid lines on a model will depend on the specific scale at the time the copy is made.

 

 

Contours

Draw the contours as defined by the triangles of the active DTM project, into CAD. Set the contour interval in the plot parameter menu.  

 

 

Triangles

Copy the triangles of the current project to the CAD as regular lines.

 

 

Side

Copy the lines that identify the side of the current loaded triangle model into CAD.  

 

Proceed with copy

Copies the elements as selected in the COPY box.
