DATA BASE : Delete points

 Top  Previous  Next

 

MENU : dB - data base-Gen

 

 

 

Use this function to remove all the surveyed points that have been marked for deletion from the data base.

 

 

Click the Remove button. All the dtm points of the active project with the *.* code, will be deleted.  The triangles will also be updated to fit on the newly arranged points.

 

Note:

There are several places where points may be marked as deleted.

               See - deleting points graphically and the point function.

               The audit and change codes functions will also mark points as deleted.

 

 

 

Background

 

Points are not removed from the database when deleted on other menus. Only the code of the points are changed to  *.*.

This delete and remove menu is the only place where the points marked as deleted, are removed from the database. The only other function that will also remove points from the data base is found in the point,delete menu.

 

               The reason why points are not just randomly removed is that the triangles must be updated to fit on the project after the points have been taken out.

 

Note:

Other more complicated methods can also be used to remove points from the project.

 See identifying points,

 Copying points between projects and

 Saving a total file.
