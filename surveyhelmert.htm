SURVEY : Helmert

 Top  Previous  Next

 

The Helmert transformation procedure transfers Dtm,Cad and Strings data to a new system based on a set of Old and New coordinates.

 

The Input Data editor is used to enter sets of coordinates on the existing (old) and new system.

 

The coordinates (old and new) for 3 or more base points must be entered.

 

 1 

For each base point, define the Y and X coordinate values on the old (or from) system.

 2 

Define the Y and X coordinate for each point on the new (or to) system.

 3 

The Helmert function uses the difference between the Old and New systems and calculates the residuals for each base (common) point.

 

Before converting the data, inspect the residual values to determine the projected accuracy of the conversion.

 4 

The conversion function can be applied to the Dtm, Cad and Strings data. Check the relative options.

 

 5 

After the base coordinates have been entered and the residuals are acceptable, click the Convert Data button to transform the coordinates.

 6 

Base points with undesired residual values can be deactivated. To activate/de-activate a base point, select the point by clicking in the editor and then click the Activate option in the top menu.

In the above example, points 5 & 6  are NOT active (X) and will not be used to calculate residual values.

 

 

 

FILE : function to save and load the base (old & new) coordinates.

GRAPHICS : function to pick coordinates from the screen. Most of the time the user will manually enter the old (A) and new (B) coordinates in the Input Data editor. If any of the base coordinates are already part of your drawing and you want to select it. use the Graphics function.

First click on the coordinate line which you want to populate and then use the correct option.
