Load User Title block and manage import of other user titles.

 

 

 

 

1.Title allows you to choose the User Title block that you want to use to print.
2.Select the Titleblock Name and
3.Ok, to use a specific Title Block.
4.Delete will delete the highlighted Title block

1.Import allows you to add additional Title blocks.
2.Import Tab window
3.Load the Titleblock Definition file. On a computer with a Model Maker Installation you will find the “Ubatch16.def” file on this file path. “C:\Users\Computer User Name\.MMSystems\MM\Settings”

This is also where your existing Titleblocks are saved.

Load a Ubatch.def file from any other user to use their Titleblocks.

4.Once you open the file you have an option to Combine the Titleblocks with your existing Titleblocks.
5.Add a prefix to allow you to see if there are duplicate names witch ones are new.
6.The second option is to Replace your existing Titleblocks with the new File.
7.The * sign next to the Pattern name shows that the Pattern linked to that Titleblock is not in the Patdata Folder on your computer.

To correct this you need to Copy the patterns with that name into the “C:\Users\Computer User Name\.MMSystems\MM\PatData” folder onto this computer.

8.On the Import Popup select Yes or no to complete the Import.

 

 

13. The Manage Tab Shows the Imported Information including if the patterns are available or not.

14. Put your Cursor in any Title Name to change the name.  You can also delete a title here by changing the Title Name to a blank.  (No characters)

15. Save the Changes when you are satisfied with the Changes.
