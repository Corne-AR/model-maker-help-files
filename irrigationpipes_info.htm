End Nodes

These are the numbers of the 2 nodes at the pipe terminals. You may edit these nodes and the pipe will be connected between the new nodes numbers. You are, however, not encouraged to edit the nodes manually in this way as there are more elegant graphical functions to perform the same operation.  The node elevations are displayed next to the node numbers.

Application

Pipes are divided into 3 application categories.

 

Laterals

These are the pipes connecting all the sprinklers in a line

 

If the block type is defined for micro/drip purposes, all the lateral pipes will display the additional CES (constant emitter spacing) value. This is the distance at which the block defined emitter type will be spaced along the lateral pipe. A lateral pipe with a zero CES value will convey the water through it without allowing for any discharge. All the lateral pipes in a sprinkler block will have a zero CES value.

 

Manifold

These are the pipes connecting all the laterals to the block valve.

 

Main Line

These are the pipes operating in the main line and connects the pump to the block valves.The main reason why pipes are classified in these categories are for the function to work effectively.

 

Flushing Manifold

The flushing manifold type is a unique pipe application and is only used to simulate the flushing manifolds at the end of drip line lateral pipes.

 

Pipes belonging to the flushing manifold application group have the following restrictions and applications:

·They are not considered when the block hydraulics are calculated. In terms of the hydraulic calculations they are considered to be dummy pipes and do not really convey water when the flows are calculated. If they were included in the actual calculations it would have resulted in closed, looped pipe systems which are not allowed in the individual blocks.
·The reason they are available as elements in the layout of the block is two fold.
·They are shown as regular pipes in the layout drawing and therefore indicate the position and dimensions of the flushing manifold when it is required.
·As regular pipes they are part of the bill of materials.
·They are added to the layout by using any of the pipe add functions. One would normally use the add lines function to place them on the end nodes of the laterals.
·They are also useful when determining the number of laterals to be flushed by a single valve.
·The flushing manifold functions allow you to globally change the pipe type and pipe diameter of the flushing manifolds based on theoretical assumptions.
·The flow and pressure values of flushing manifold pipes will always be zero since they are not included in the hydraulic calculations.

 

 

Fixing the pipe size for main lines.

 

You may set and fix the pipe diameters in the main line when designing existing pipe lines. When doing pipe sizing, these pipes will not be resized.

To set the FIX option to a range of pipes, do the following:

 

1) Use and edit the first segment of the existing pipe.

2) Set the option to true.

 

Now use and point to the first and last node of the range of pipe segments you want to change and fix.

3) When it asks you to select the new pipe type, use the GET TYPE option, point to the first segment (the one you have just fixed the size of in (1) and all the segments between the 2 nodes (A to Z) will take on the type (including the fix size) of the GET TYPE pipe.
