MENU : Tri - display

 

Control the background appearance of the triangles when drawn to the screen and printers.

 

Triangles are normally drawn by showing the lines representing the 3 legs. To automatically draw the triangles on each replot, set the appropriate check box in the plot parameters menu.

To redraw the triangles use the manipulate plot functions.

 

If the triangles must shade themselves with each replot to the screen and output to the plotter, use the settings in this dialog box.

 

Shading of the triangles are controlled globally as per each project or within areas as defined by the selected strings.

 

Options

The global options are defined for each of the 3 projects. For this exercise, we will use project 1.

 

Check this box to shade all the triangles of the project. If this box is unchecked, the triangles can still be filled when the box is checked.

If both the ALL and STRINGS boxes are unchecked, the filling (hatching) of the triangles for the specific project is switched off.

 

Filling all the triangles

 

3D fill

Check this radio button to shade the triangle model using the relative slopes and orientation of the triangles. Model Maker will determine the relative position of each triangle and calculate the amount of light that is reflected from the surface. Slopes facing the light source will have more of the colour as specified in SHADE1. Triangles facing away from the light will have more of the SHADE2 colour.

Click on the SHADE panels to access the colour control. Select or mix a preferred colour.

 

EXAMPLE: a shaded mine dump.

         

By triangles colour

The triangles will be filled with the standard colour as set in the colour dialog box.

 

Defined colour

Click on the colour panel to change the colour. The triangles will be filled with this colour to both the screen and printer.

 

Fill %

Select the density by which the triangles should be filled. The filling feature has been developed as a background picture. A colour that is too dense or dark can hamper your identification of the other elements on the screen. Changing the fill % to lower values, such as 20, produces a lighter background. Model Maker intelligently applies the percentage value relatively between the screen back ground colour and the fill colours. A low fill percentage will produce a picture that does not differ much from the screen background colour.

When outputting to a printer. a white paper background is automatically used. A low fill percentage will produce a very light (closer to white) image.

 

Strings

 

If the box is checked, the settings from the strings page will also be considered.

 

General explanation

Strings are very flexible design tools. Because strings consist of a continuous polyline, it is often used to define certain areas. Here, this feature is applied to the triangle fill concept.

 

How does a string know it must fill the triangles?

A simple and effective method links the desired strings and the filling of triangles together.                

This is done using of the LAYER name (Pen) on which the string was created.

         

The grid box.

 

In this grid box, enter up to 20 layer names. When a replot is done, all the strings will be scanned to see if any strings exist on the specified layer name. If so, all the triangles falling within the area of the string will be filled depending on the method. The selected method can have any of the following numbers.

 

0 : The string will be ignored.

1 : The triangles will be filled with the 3D method as described in the Options page. (Explained above in (1))

2 : By triangle colour. See explanation above in (2)

3 : By string colour. The colour of the defining string will be used when filling the triangles.

 

 

If the printer option for the specific layer is selected, the string lines will also be outputted to the printer. If not selected, the string will define the area but will be invisible on the printed output.

Strings on the defined layers will always be visible on the screen and are not influenced by this setting.

 

Updating the settings in the grid box.

The only typing that you will do in the grid box is when the layer name is entered. The Selected and Printer fields are controlled from the panel.

General

 

The picture produced by filling the triangles on the screen is generally not the same as  outputting to a paper plot. Although it could be practical to work with filled triangles on the screen that has a certain fill %, the output to the printer will often look too dark with the same fill value. The factor provides an easy way to adjust the fill ratio when outputting. This factor will reduce the fill % as specified on OPTIONS page by the set value.

 

EXAMPLE:

If the general fill % on the options page is 50%, then the output to the printer will be performed at 50%*20% = 10% of the specified colours

 

Set Tri Colors

 

Use this menu to apply colors to triangles by first setting the default color and then use the functions on the left. Once a color is assigned to a triangle the triangle remembers the color and will hatch itself with that color when the BY SET COLOR is selected on the OPTIONS page. The SLOPE COLOR option assigns the default color to triangles with slopes within the selected range.

This is a once off apply function and future changes in the triangle slope will have no effect. Use the GET COLOR function to set the default color to that of an existing triangle. If the FIXED color is checked, the triangles will be drawn in exactly their set colors. If it is unchecked a shading effect will be applied and colors will be graded depending on the orientation of the triangles.

 

Hints:

Only in extreme situations would you use the different layer names. Remember that multiple strings may be defined on the same layer and that each string may have its own colour. By specifying a single layer name and selecting format 3 (by string colour), the area of the triangle model falling within the string area will have a different colour.
