SURVEY : Traverse

 Top  Previous  Next

 

MENU : sUr - traVerse

 

This function allows you to enter a known start co-ordinate, traverse directions and distances to other points and then to close on a known co-ordinate again. The misclosure will be computed and the intermediate provisional co-ordinates adjusted using the Bowditch method. The final result may copied to the editor to be printed or saved, copied to the DTM data base as regular survey stations or copied to the active project as survey points, or to the CAD as regular lines.

 

INPUT Page

 

Use the input page to enter the start and end co-ordinates and the distance and directions of the traverse legs.

 

Start traverse from

Enter the station name and co-ordinates of the first known point.

 

Traverse legs

This input box allows you to enter the actual survey. Notice that on line 1, the station name of the start co-ordinate is automatically duplicated. Enter the distance and direction measured from the starting point o the second station in line 1.

 

Note:

The distances must be horizontal distances. This function does not accommodate slope distances.

Enter the distance and direction from station 2 to that of station 3, in the second line. Do the same for all the other legs. The last leg that you input in this box should be the readings taken to the co-ordinates of the end station.

 

This input box can either have 3 columns or 5 columns. If you are working in orientation only 3 columns are displayed. This allows you to enter each station name and the distance and direction to each next station.

 

Note:

Working in orientation means that each time you setup at a new station you would turn the horizontal disk angle of the instrument to a position so that the foresight from the previous station and the back sight from the active station, differs by 180 degrees.

 

 

This diagram shows, for the southern hemisphere, 2 stations in orientation.

From station 1, the forward direction to 2 is 250 degrees. Station 2 has been orientated that the back sight reading to station 1 is 250-180=70 degrees.

 

 

If you are NOT working in orientation, meaning the back sight direction is not equal to the foresight minus 180 degrees, select the  calculate corrections option.

 

 

Two additional columns will be added to the input box. In the  (Reading to previous station) column, enter the back sight direction taken to the previous station. Model Maker will then calculate the correction angle that should be applied to all the horizontal directions measured from the station.

Calculate page

 

The calculate page displays the results of the traverse. The traverse is calculated in the following manor.

 

1-        Provisional co-ordinates are calculated for each station.

 

2-        The provisional co-ordinates of the last station is related against the actual co-ordinates and the difference calculated as Delta y and Delta x.

 

3-        The provisional co-ordinates are adjusted by the bowditch rule.

 

4-        Joins are again taken between the adjusted co-ordinates and the directions and distances displayed in the second column.

 

5-   The adjusted co-ordinates are displayed.

 

6-        The SUMS value is the total length of the traverse.

 

7-        The accuracy is displayed.Bowditch

 

Copy as Station

Use this option to copy the names and co-ordinates of the traverse points to the DTM data base. New stations will be created. These stations can then be used to enter fieldbook survey data.

 

Note:

The traverse does not accommodate vertical angles and as such no elevations for the stations will be calculated.

 

Copy to Graphics

Points may be copied to the active project or/and cad lines can be drawn between the traverse legs.

         Points                : Survey points will be added to the DTM. Elevations for the new points will

                                                 be calculated from the DTM surface if  triangles are loaded.

         CAD                : Select the pen, line type and layer name. The lines will be added to CAD.

 

Copy to Editor

The traverse results will be copied to the editor. From there it can be saved to disc as a regular ASCII file or printed to a printer.
