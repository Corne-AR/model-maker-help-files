Layer Depths for Boreholes.

 

Load all your Pojects into Model Maker.

Add DTM points in the Project you want to record the depths from, on all the Borehole or relevant points. (in the example below it will be Project 2)

Identify all the Relevant points in Project 2.

 

 

Select the Mining Icon

 

On the Mining Function Select the Setup Layers first.

 

Change all the Layers you want to be recognized as Mining Layers to a S.G. bigger than Zero.

 

On the Main Page of the Mining Function Select the Layer Depths Button.

1.Select the Project you want to use as the Controlling Project.
2.Select if you want To plot to Cad or have a Xcel Report.
3.Select to Use the ID points or all the points in the Controlling Project.

On the Output Format report to Excel you can save the results to Excel or ASCII.

 

On the Output Format Report to Cad Text the Depths are plotted to Cad at each DTM point. The text size is the current Text size setting.
