?Set3

Use this option to obtain survey data directly from a Sokkisha Set3 theodolite. Data is not added to Model Maker's database, but a file may be saved with all the survey data in.

The instrument must be connected to one of the serial communication ports.  When this option is used , you will be taken to the next sub-menu.  Set communication parameters - the communication parameters between the instrument and the computer must be set the same when requested for tachy comms info.  To initialize the instrument, specify a basic mode code.  Refer to the instrument manual for the code of your specific instrument.

 

The defaults for the Set3 are usually:          

* Baud         1200           * Parity       None           * Stopbits     1           * Databits     8           * Port         1 or 2           * Mode Code    % for Set3 G / Xf for Set3 C.

 

After all parameters are set, select OK to  automatically save the settings.  It will be the same when Model Maker is run again.  Set  Y,X,Z Hi & Corr angle of station - a temporary station from which observations will be reduced must be created before any observations are taken.

The resulting data for each observation point will be final Y, X, Z values calculated, as polars from the temporary station.  Use this option to specify the Y, X, & Z co-ordinates and the height of the instrument at the station.  This data is not saved and must be entered each time the instrument is moved.  The instrument height must be equal to the prism height.

After the data for the station is entered, you will be asked if you want to save the survey data in an ASCII-file.  Save the data of each station consisting of the station data and all the actual observation data from the station. The file will  have a .SE3 extension and it may be read into Model Maker using the options - Files - Load - ASCII - Other - Columns. Select the columns of the final co-ordinates.  Receive measurements from tachy - after the communication parameters and station data is set, this option may be used to do the survey.  To receive data from the instrument press the Enter key on the keyboard.  If data (co-ordinates and triangles) is loaded, the elevation of the surveyed survey point may be compared to the elevation at the surveyed survey point position on the model.  The survey data (Code, Y, X, Z) of the survey point and the difference in elevation above will be recorded in the ASCII file that is created.
