MENU : gXt - Add - Toeline

 

Create a new string next to an existing one at a position determined by the specified slope value and the elevation on the triangle model of the active project.

The indicated string will be followed. At regular intervals, cross-sections will be interpolated on the surface of the active project. Model Maker will then determine the position of the toe point by comparing the elevation on the string at that point to that of the ground.

 

 

The picture presents a cross-section through the indicated string. A new node is added to the new toe line string at the position where the toeline based on the specified slope meets the ground. The original string is above the ground and in a fill position. Model Maker will automatically go down in a fill situation to get the toe point and up in a cut situation to determine the toe point position. You may,however, force the toepoint to be calculated as a cut or fill situation. In the picture, the forced toeline is calculated as a cut toeline, although the string is above the ground.

 

Parameters

 

Slope

This indicates the embankment slope. Model Maker expects the slope to be specified as one over a user value,ie., a 10% slope is 1:10 (enter 10), a 25% slope is 1:4 (enter 4).

 

Interval

Nodes will be added to the new toeline string at the specified interval.

 

Special instructions:

Normal toepoint is where the toeline will go down in a fill situation and up in a cut situation.

Force up                        : The toepoint will always be above the main string if an intersection with the ground is found.

Force down                     : The toepoint will always be below the main string if an    intersection with the ground is found.
