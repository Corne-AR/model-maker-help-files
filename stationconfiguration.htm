MENU : Station configuration

 

 

This screen has a number of windows where you can specify the method  to reduce the distance and elevation data of each spot. The configuration that you set will be saved and will be used as defaults when you enter this option again.

 

 

Angle format

Enter angles in decimal or a degree, minutes and seconds (DMS) format.

 

Distance method

Distance is entered and reduced as follows:

 

 

Mid-wire & Top-wire

Enter a top and mid-staff reading.

Model Maker will calculate a slope distance reading:

 

Slope distance = (Top-wire - mid-wire) x 200            

HD = Slope distance x 0,5 (sin [290-va]), where

va = vertical angle,

HD = horizontal distance.

 

 

Slope Distance (Cos va)

Enter the slope distance.

Model Maker will reduce this to a horizontal distance by:

HD = S.distance x cos (90-va).

 

Enter the reduced horizontal distance

Model Maker uses this distance in the calculations.

 

 

Y,X Co-ordinates

Enter the y, x co-ordinates of the spot.

Model Maker uses these co-ordinates in the calculations.

 

 

Slope Distance (0,5Sin [2 va])

This is the same as the first option, except that only the mid-wire reading is entered.

 

 

Bottom & Top - Wire

Enter the bottom and top-wire reading in meters.

Model Maker will calculate a slope distance:

Slope distance = (Top-wire - bottom-wire) x 100

HD = Slope distance x 0,5 (sin [290-va])

 

 

Elevation Method

Elevations can be entered and reduced as follows.

 

 

Vertical Angle

Enter the vertical reading in degrees, minutes and seconds.

A vertical height difference is calculated as follows:          

HD x (tan90-va)2 for options 1 and 5;

or        

HD x (tan90-va) for option 3, in distance format.

 

 

Height Difference

Enter the height difference in meters.

MODEL MAKER uses this in all calculations.

 

 

Final Elevation

Enter the final height.

Model Maker uses this in all calculations.

 

 

No Height

If there is no height, then Model Maker will not request an elevation and the points will be automatically excluded from the triangulation search.

 

 

Height Difference (tan)

Enter the height difference as a tangent of the vertical angle.

The horizontal distance will be multiplied with the tangent factor.  

Points top/mid wire

Top/mid-wire Whenever you have specified a station configuration where a staff reading is concerned, enter a value.

Mid-wire and top-wire readings need to be entered if a tachy has been used.

Model Maker will only request the mid-wire reading where a single staff reading has been taken.

Slope/ horizontal distance Slope or horizontal reading is entered, depending on the station configuration.

 

 

Note:

Model Maker always reduces the slope value to the horizontal equivalent and saves it as such. When a previously entered spot is edited, any entry that does not change need not be re-entered.

The only exception is the distance column. The slope distance must be entered each time a spot is edited because this entry is always reduced to horizontal distance.

 

Horizontal Angle

The horizontal angle is entered in the Model Maker angle entry format. To calculate the Y and X co-ordinates of a spot, Model Maker will add the correction angle to the entered horizontal angle.

 

 

Vertical Angle

The angle is entered in the Model Maker format.

 

 

Height difference

This is the height difference between the instrument and the target. Enter the value as a positive or a negative value as necessary.

 

 

Final Elevation

Enter the elevation.

 

 

Y and X co-ordinates

Enter the Y and X co-ordinate values (if applicable).
