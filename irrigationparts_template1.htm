MENU : Irr - Parts

 

Node Type

From the display list, select the node type which the combination applies to.

Use the ANY format if the specific node type is not important.

 

Pipe Type

Define the type of 4 pipes that connect to the node.

If the type is not important use the ANY format.

If less than 4 pipes are connected, enter the other fields as NONE.

 

Pipe Diameter

Enter the outside pipe diameter. Enter a value of zero if the diameter is irrelevant.

 

Part Number input box

Enter the part number here that fits on the defined combination. This is done by selecting the part number from the parts data base or cluster number from the cluster data base and then paste it into this input box by double clicking in this box.

 

Parts data base

This is the list of available parts. Use these part numbers in the part number input box.

If you need to access the parts data base itself, click the data base button at the bottom left of the dialog box.

 

Cluster data base

This is the list of available clusters. Use these cluster numbers in the part number input box.

If you need to access the parts data base itself, click the data base button at the bottom left of the dialog box.

 

Template data base

This is the list of the defined combinations. Each entry consists of the part/cluster number, the node type and the 1,2,3 or 4 connecting pipes and their diameters.

 

Control panel

Use these buttons to add, edit and delete template data base entries.
