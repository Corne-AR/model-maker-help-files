MENU : Data Types - DTM

 

The DTM manipulates survey data from various sources in the co-ordinate database. Twenty one DTM projects may be loaded simultaneously.

 

Normally only one project with all the data will be used. For instance, when creating a contour plan from survey data, all the data will be loaded into project  # 1.

 

Projects 2 & 3 are used when doing volumes between 2 surfaces, meaning the original surface (points and triangles) are loaded into  # 1. The new surface, eg., after excavation, is loaded in  # 2. The volume can then easily be done between the 2 projects.

 

Model Maker's DTM consists of a database that holds co-ordinates and triangles.

 

Many of the functions available in Model Maker are designed to manipulate these co-ordinate points.
