The chainages of all the ground (dtm projects) will be shown on the section.

The elevations of the ground will be shown.

 

 3 

The ground elevations as interpolated at the chainages of the string node will be shown.

Typical section showing ground chainages and elevations and also ground elevations at string node chainages.

 

 4 

The elevation differences between the projects are shown at the string node chainages.

In this example, the string and 3 dtm projects are shown. The first elevation is between the string and project1, project 2 to project1 and project 3 to project 2.

Regular intervals:

 

Options 1 to 4 relate to the defined ground and string node chainages.

 

The Regular Interval selection allow to show most of the information on a user defined chainage interval.

 

 5 

Define the chainage interval at which the following information will be listed.

 

 

 6 

The depth values between the dtm project will be shown. The sub selection (7) allows the user to choose the depth to be calculated as:

 

 7 

Depth : Related to Project 1.   The depth between the string and project 1 and all the other dtm projects in relation to project 1 will be shown.

 

 8 

Depth : Difference between projects. The depth between each project and the previous project will be shown. Project 1 will show the depth to the string.

 

 9 

Select this option if the actual interpolated groundline elevations must be shown and not the depth between values.

 

 10 

The text that will be posted as header for this section.

 

 11 

In all the selections, the string and it's profile is normally shown. If you do not want any string information to be displayed, check this option.

 

 

 12 

Note that the information for all the dtm project will be shown. If you do not want the section of a certain dtm project to show, switch the display OFF on the projects display options.
