MENU : Cad - Dim - deFault                                

 

All the relevant dimension parameters are found in this dialog box. This dialog box is used when a new dimension is defined and also later when editing an existing dimension.

 

 

PARAMETERS Page

 

The dimension text will be entered in the 2 input boxes when a new dimension is defined.

The 2 lines of text can have up to 80 characters and are normally used to enter the length and angle values.

The first line of text is normally written above the dimension line and the second line of text below the dimension line.

 

Use the button to have Model Maker flip-flop the 2 text items.

 

Text

Height                                The height of the dimension text in millimeters.

Width                        The width of the dimension text in millimeters.

Layer                        The layer on which the dimension is added to the database and displayed.

Decimals                The number of decimals used to display the length of the dimension.

                         Note that the survey notation will control the accuracy of a survey dimension.

 

Factor

The length value of the dimension will be multiplied by this factor. It could be used to draw in meters, but the dimensions must be in millimeters. Therefore, a factor of 1000

will display a measured distance of 2.12 meters as 2120.

 

Colour

Use the colour grid box to select any colour for drawing the dimension.

 

Move the cursor over the graphic and when the appears press the enter button on the mouse and a more detailed explanation of the function in question will be given.

 

 

POSITION Page

 

Text Position

 

Above

Both lines of the dimension text will be displayed above the dimension line.

 

Centre

The first line of the dimension text will be displayed above the line. If the second text is used, it will be displayed below the line.

         

Below

Both lines of text are displayed below the dimension line.

This box will be checked if the dimension text was moved using the Edit function and placed at a user defined position

 

Line Position

 

Two factors control the position and size of the dimension line.

       A : The size of the terminal pattern when arrow type 1,2 or 3 is used. User arrow types are displayed in the size                                      that the pattern is defined.

                         B : The distance in millimeters away from the indicated terminal position of the dimension

This box will be checked if the dimension line was moved using the Edit function and placed at a user defined position

 

Defaults

The specified dimension parameters as set in the dialog box may be saved to disk and also loaded from disk.

Three different settings may be recorded. Use this feature to save the basic layout of your general dimension formats.
