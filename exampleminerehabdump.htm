MENU : Example - Mine rehabilitation

 

Model Maker has a comprehensive set of functions that plan and design rehabilitated models. This includes functions found on the 3D, Mine and Strings(GXT) options. Most of the mining actions use strings that define areas, sections or slopes. The rehab has two major categories: REHAB ALL and REHAB STRING.

Remember that all the mining applications operate on a gridded file.

 

Rehab all

In its simplest form, this option operates without any defined strings. However, the whole sequence will be discussed and the different options highlighted.

Start with the original gridded data in the active project. After activating the REHAB ALL option, answer the following question:

 

Use identified strings to keep points undisturbed ?

YES : Identified strings placed around areas that will not be rehabilitated will exclude those points from the rehab process. This is normally done where a dump is next to a

         fence or an obstruction like a road.

 

If areas are excluded from the rehab, something must be done with the material when the normal rehab process wants to push the material over the boundary into the excluded area.

 

Your options are:

 

1) Ignore       : The material is pushed over the boundary, but the undisturbed points are not updated. The material is not accounted for. This is not an ideal situation and

would normally not be used.

2) Sideways : The material will be pushed up to the boundary and then sideways along the boundary until a position is found where it can be placed not exceeding the slope.

 

This is where the material will eventually end up.

 

3) Picked up : Material about to be pushed over the boundary will be picked up and placed in the closest area to the boundary that has a slope less than the maximum. This material dumped in the area will again be rehabilitated during the next sweep.

 

This indicates the material being placed (red arrow) and then rehabilitated again (yellow).

 

4) Picked up area : The material will be picked up and then placed on the closest flat surface within a user defined area. You control the area where the material is dumped. Use this when you know where the fill part of the model is going to be. This option simulates an operation where trucks are used to move the material from the boundary to

a specific dumping site. To use this option, define a string around the area used for dumping. Add the string in the GXT menu before the mining-rehab is started.

 

Enter maximum slope

Enter the slope that the final model must not exceed.

 

Specific areas with slopes other than general

The maximum slope applies to the whole model. You may, however, define areas within the model where the slope is different to the general slope. In that case, use the area definition option to indicate those areas. Also enter the new slope for each area.

 

Do rehab in defined area

Specify the perimeter area of the total dump to be rehabilitated. The rehab will not go beyond the perimeter. The first question - Use identified strings to keep points undisturbed - is applicable here, especially when the boundary is reached and the dump not yet rehabilitated to the full. Whatever you specified in the first question will be used to apply the correct procedure to the material that meets the boundary.

 

Confirm grid interval

It is important to apply the rehab process to gridded data. This information box only confirms the grid interval. A warning is issued if the program finds the data is not on a regular grid.

 

Update contours while rehabilitating

Model Maker can give you a running update of the contours of the new model while the rehab is in action. This update can either be immediate or after each sweep of the model. Updating the contours as the points are changed may take unnecessarily slow the machine down. Enter a 1 for Model Maker only update the contours after each sweep.

 

Model Maker will continually apply all the slopes and restrictions and rehabilitate the points. Interrupt the action by clicking the ABORT button. The program will stop automatically when all the points fall within the maximum specified slopes.

The final question offers these options:

 

DO NOT UPDATE ELEVATIONS : The process is aborted and the data left in the initial format.

UPDATE TO NEW MODEL           : The new rehabilitated elevations is applied to the model.

RECALCULATE PART VOLUME : This option is not applicable to the REHAB ALL function and will have the same result as the DO NOT UPDATE ELEVATIONS.

REHAB STRING                           : This function makes extensive use of strings as defined in the GXT menu.

                                                            Two types of string rehabilitations are available. The first one (One string slope) works on a single string and is mostly used to design a water course. The second method is more involved and relies on the user to design the strings correctly by using the GXT Vert option extensively.

 

The ID STRING uses of the designed strings and updates the DTM model based on the elevations calculated from the strings. The strings must be designed correctly with the following functions:

 

1) Use the String Add New Indicate General string function and place strings across the area to be rehabilitated. The strings should run parallel to the direction of the critical slope. Use the Point cursor snap mode, then the start and end of the string is defined. This automatically elevates the string nodes to the correct z value.

The vertical view of the original sections will look like this:

 

 

Use the String Vert View option to see a section of the string. Now use the Special N_Add option to add new nodes to the string so that the new shapes comply to the design criteria. To add a node to the string based on the slope to the existing two nodes on either side, use the CTRL B (Line intersection) snap mode. Because you are in the vertical mode, Model Maker allows you to add the slopes from both sides. The new node is placed at the intersection of the 2 slopes.

 

 

Note:

The slopes to the new node are also displayed as a percentage in the bottom left of the screen.

A combination of adding new nodes and moving existing ones with N_Move will eventually give you a satisfactory section. Use the AREA option on the same menu to calculate the cut and fill areas between the design string and the ground. Always try to design the string so that the cut and fill areas are equal, to give a balanced total volume across the whole model. Use the HOR option to revert to the horizontal plan view of the model.

The sequence of Vertical viewing strings and adding nodes must be done for all the design strings. When finished identify the design strings and add a new boundary string around the area of interest.

Go to the Mine options and press the Rehabilitation rehab_String button. Confirm the grid interval and select ID_string. Indicate the boundary string after which the calculation will start.

Model Maker will inspect all the dtm points within the defined area and re-elevate each one. the closest string will be found and the elevation at that point on the string interpolated. The string directly opposite the first one is picked and the elevation calculated. The dtm point has 2 elevations on both its sides. A new elevation for the point is calculated based on the opposing 2 Z values.

 

Each point re-elevated = final contours. The sweep through all the points is done once, since each point can only have a unique elevation. You will again asked:

 

DO NOT UPDATE ELEVATIONS : The process is aborted and the data left in the initial format.

UPDATE TO NEW MODEL           : The new rehabilitated elevations is applied to the model.

RECALCULATE PART VOLUME : Specify a certain percentage of the volume to be moved. Model Maker will redo the rehab. Once the specified amount of material is moved, you can accept or abort.

 

Use the F5 key to inspect the slopes on the model and calculate a final volume between the original and the design by using the terrace option. If you find that the slopes or volumes are not satisfactory, inspect the model and make the necessary changes to the design string. Re-run the rehab function after changes have been made to the strings. With some experience you will be able to create a design in a short time.

Use the move volume function to see how the material must be moved between the ogl and the design model.
