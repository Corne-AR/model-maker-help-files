MENU : Pnt - Del

 

Survey / DTM points may be marked as deleted or may be physically removed from the database.  

 

POINT :  Delete indicated point

The indicated dtm points will be flagged and then either deleted or marked as deleted.

 

POINT : Delete identified points

All the points identified with the Point ID function will be marked as deleted or removed from the database.

 

POINT : Delete in Area.

All the points in an indicated area will be marked as deleted or removed from the database
