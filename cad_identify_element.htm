MENU : Cad - Id - do-Id - Q All        

 

Identify specific CAD elements.

Toggle between including or excluding all identified points in other operations. Use this option to exclude previously included points.

 

 

This check box is important. Whenever an element is identified and the INCLUDE button is selected, it will be added to the identified list.

Oppositely, when you identify an element in the exclude mode, it will be removed from the identified list.

Example : If the CAD drawing consists of 100 lines and you want to identify all but 5 lines, it is easier to identify all the lines and then to un-identify the 5 lines, rather than picking all desired 95 lines individually.

Therefore, set status to INCLUDE and identify all the lines. Change the status to EXCLUDE and individually select the other 5 lines.

 

Single elements

Identify individual elements by first selecting the element type button and then picking the elements on the graphics screen.

 

All the elements

Identify all the elements of a particular type.

 

         LINES :

         Screen the lines may be screened based on their lengths. Enter a lower and upper length value.

         Example : Lower value=10  Upper value=30    All the lines >=10 and <=30 will be identified.

 

 

         

All

Identify all the text elements with the actual text similar to the specified string. Use the '*'  character as a substitute for any character.

 

Size

Identify all the text with height value in a lower  and upper limits.

 

Ratio

Identify all text with a ratio (width over height) falling between the specified limits.

 

Lorg

Identify all the text elements with a label origin similar to the current active label origin.

 

Angle

Identify all the text with an angle (direction) within a defined lower and upper limit.

 

 

CIRCLE/ARC

Identify all the elements with a radius within a defined lower and upper limit.

 

DIMENSION

Identify all the dimensions.

 

ARROW

Identify all the arrows.

 

HATCH

Identify all the hatch elements. The hatch elements are identified based on colour or percentage fill.

 

         All                Identify all the hatch elements.

         Colour                Select the desired colour of the hatch to identify them from the colour grid dialog box. Use the GET                                VALUE FROM ELEMENT and point to a default hatch if to use the colour from it.

 

         Percentage        Set the lower and upper % limits within the hatch. Use the GET VALUE FROM ELEMENT and point to a                                default hatch to use the percentage fill.
