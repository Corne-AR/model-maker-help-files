MENU : Cad - Id - Man-id - Match                                

 

All the identified CAD entities can be stretched between two points. You will be prompted for a base point, a new position for the base point, a reference point, and a new position for the reference point.

This function uses scaling, rotating and moving on the entire drawing.

 

EXAMPLE:

 

 

In the example, the drawing within the smaller rectangle is matched between points a,b,c,d.

The drawing was first identified by selecting the smaller area.

Then base point 'a' was matched to point 'b' and base point 'c' to 'd'.
