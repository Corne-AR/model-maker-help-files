MENU : Cad - Patt - Explode

 

Explode all the elements of a pattern into normal cad elements. The elements can then be manipulated with the normal CAD functions.

Explode a single pattern, patterns by name, or all the patterns.

 

The elements within a pattern cannot be manipulated individually unless the pattern is exploded to be part of the drawing elements. Once a pattern is exploded, all the elements will lose their relation to each other. However, they will retain their original layer names. Elements of an exploded pattern cannot automatically be put back into the pattern.  Remember, the original pattern, as defined on the disk, is not affected by this action, only its image is manipulated.

 

Single

 

The indicated pattern is exploded.

 

Name

 

All the patterns with a defined name is exploded.

 

All

 

All the patterns on the drawing are exploded.
