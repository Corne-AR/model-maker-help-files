SCALE : Text update

 Top  Previous  Next

 

This menu is displayed whenever the plotting scale of Model Maker is changed. This may happen when you change the scale in the plotting parameters in the Gen Paper menu or when a CAD drawing of a different scale to that of the system, is loaded.

 

 

The menu in the example was displayed when the plotting scale was changed from 1/1000 to that of 1/500.

Three options are available.

 

Use CAD scale (1/1000)

Model Maker will change the system scale back to 1/1000, which is the scale of the current CAD drawing, meaning that no changes are made.

 

Use plot scale and update text

The new scale of 1/1000 will be retained and the CAD text elements will change in size.

The scale change from 1/1000 to 1/500 implies that the text size will be multiplied by 2.

The text will appear to be the same size in relation to the positions of the other elements on the drawing.

 

Use plot scale : do not update text

The new scale of 1/500 will be retained and the cad text elements will remain the same physical size.

The text will now appear to be half the original size if compared to the rest of the CAD drawing.
