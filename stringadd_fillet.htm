MENU : gXt - Add - Fillet

 

Create a new string that fillets the intersection between 2 existing general strings.

 

 

 

 

 

EXAMPLE: Roads AA and BB are intersecting. The shoulder break point of both roads have been off-set as well.

 

The 4 red dots indicate the possible positions of a fillet between the roads.

 

 

 

 

 

 

 

 

 

To fillet the any one of the 4 quadrants. Select the 2 base strings by clicking on them, as close as possible to the predicted position the fillet circle is going to touch them.

 

In this EXAMPLE, to add the bottom left fillet of the intersection, positions A and B were indicated.

 

When 2 strings are indicated, Model Maker actually inspects all 4 possible positions and selects the fillet closest to your 2 indicated points.

 

 

To fillet the top left quadrant of the intersection, click in the 2 positions as indicated.

 

After the 2 base points have been indicated, Model Maker goes through the following steps.

 

       The 2 general strings closest to the cursor positions are selected.
       The fillet radius is requested. The distance between the nodes on the new general string (that will follow the fillet curve) is entered.
       A general strings consists of line segments. All the segments of both strings are inspected in turn to determine if a valid fillet can be constructed between

       them.

       The four possible fillet positions are calculated and the one closest to the 2 original base points, is selected.
       A new general string is constructed along the fillet curve.
       The elevations on the new string follows a straight line interpolation between that of the 2 base strings.

 

Notice in this EXAMPLE how the yellow fillet string ties in to the elevations of the 2 base strings.

 

Note:

       The original base strings are not altered in any way.
            If you want to combine the 2 base strings with the new fillet string, use the Split string option on both the base strings. (To create new end points for the fillet string to combine with)

         Use the Combine option to make one string of the 2 base strings and the fillet string.

 

 

In this EXAMPLE, the 2 base strings have been split and the segments removed where the fillet string must tie in.

In this EXAMPLE, all 3 strings have been combined in one.

 

       To check the elevation of the new string, use the Vert view option.

 

 

 

All the new nodes on the fillet section of the string called Fillet and can easily be identified by switching the name option on, in the vertical mode.

The elevations on the fillet section can also be improved by using the Grade node option and using the Curve 1 format.

 

 

When the fillet is not possible, this warning will be shown.

 

 

The reason could that the radius is too big or the following situation occurs.

 

 

 

 

 

One, two and three are nodes on the string.

 

A indicates a theoretical extension of segment 1>2

B indicates a theoretical extension of segment 3>2

 

X and Y indicate the possible tangent points of a proposed fillet with the extended segment.

 

Model Maker determines the position of the fillet by checking where a proposed fillet will tie in with an existing segment on the base string.

 

In this example, the base string is on a slight curve.

         

The base string is on a curve and the segments on the string is too long.

A fillet can therefore not be connected to any of the segments. It could work out to the extension of a segment but then the fillet is not tangent to any of the segments.
