MENU : File - Editor

 

 

 

Model Maker uses the text editor extensively to display information or calculation results. You can manipulate the data in the editor, print it, save it to disk, or just view it. The editor can also be used to type in new information or to recall any existing ASCII file from disk. The editor holds a maximum of 400 lines.

 

 

Files

 

Use the FILES buttons to load an ASCII file or to save the current data to disk. Use any extension to define the filename. Model Maker will keep the defined extension as the default until you change it again.

 

Printer

 

 

The Print button will output the data to a printer. Use the Setup button to change the printer type and font size for the printer.

Column Edit

 

The Insert button inserts a column and the Delete button deletes a column. Highlight one line in the area you want a column inserted or deleted.  You can also bring the columns closer together by highlighting one line between the two columns and pressing the Delete button.

Defaults

 

Clear before new: Select this check box to clear the editor of its current contents before any new data is added. If the box is checked, the contents will be cleared every time a file is loaded or the results of calculations are added to it. You would not want to clear the editor, for example, when volumes are calculated and you want all the results to be combined in the editor.

Save on close: If this box is checked, Model Maker will save the contents of the editor to a user defined filename before the closing editor.

Use print font: If this box is checked, the font will print the same as what you see on the screen.

Word wrap: This box, when checked, allows long lines of text to wrap to the next line.

Width of dialog box

 

These arrows increase / decrease the width of the dialog box

Clear

 

The Clear button erases the current data on the editor, leaving a blank screen.

Recall

 

Each time you clear the editor, load a file from disk, or when Model Maker uses the editor to display new information, a backup of the current data in the editor is made. The RECALL button will add the previous backup data to the current data in the editor. Use this function to add 2 files from disk. For example, load the first file. Now load the second file and then recall the backup (the first loaded file). The data of both files are on the screen.
