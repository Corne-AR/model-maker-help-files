STRING : Utility - Drape onto DTM

MENU : gXt - Utility - drApe

 

Take a previously defined string and do the following:

 

1) If no new nodes must be added:

All the existing nodes of the string will be re-elevated to the elevation as interpolated at that specific point on the triangle model of the active DTM project. Any nodes not falling within a triangle will not be altered.

 

2) If the string must add new nodes at all the changes of slope:

All the legs of the string will be inspected. A new node will be generated on the string at all the points where a change of slope is detected on the triangle model. The string will represent an exact profile of the DTM model along the horizontal position that the original nodes have defined.
