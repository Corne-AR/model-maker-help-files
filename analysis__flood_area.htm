ANALYSIS : Flood area

 Top  Previous  Next

 

This function determines and show the area that water will flow when released from certain positions.

 

 

 

 

 

 

 

 

 

The parameter input box is used to control the function and the output.

The water release points are defined by either a Cad polyline, GXT strings or on a regular grid interval

 

You may either select individual lines or strings or all the identified lines or strings can be analysed as a group.

 

Please note, if the Cad does not have any polylines then the polyline options will not be shown.

The GXT strings option will also not be available if no strings are loaded.

 

The Indicated Poly/Grid option requires a regular polyline to define the area in which the grid points must be analysed.

When the Poly/Grid option is selected, the actual grid interval is specified in the input box above the Flood Depth.

The grid points are not real points. It is just a virtual position within the area.

Within the area defined by the user indicated polyline, the program will assume that each grid point is a water release position and determine the flow from there.

 

Typical result:

 

 

A) The source = Cad Polyline or Gxt string.

 

B) The streams (flow paths) flowing away from the source.

 

C) Flood indicators.

 

 

This can be dots, circles or arrows showing the area that the water will cover when flowing along the streams at a user defined depth.

 

The streams will always be shown as a solid line but the flood indicator can be dots, crosses, circles or arrows.

 

Dots and crosses will be shown on the screen but not stored as long term elements.

 

Circles and arrows can either just be shown on screen or copied to cad. While testing the parameters one will just show the elements on screen. If the result has to be shown on a paper plot or saved with the cad, select either the Circles>Cad or Arrows>Cad option. Running the function with the CAD option selected takes longer since checks have to be done to prevent duplicate circles or arrows.

The 2 options that copy the indicators to cad, by default also copy the streams as cad lines to cad.

 

                       

 

The distance interval determine the density of the calculated streams.

 

When the grid option is selected, this value will define the distance between the grid points.

 

 

 

 

The desired distance interval may vary from model to model. Experiment with the value before selecting a flood indicator that copies the result to cad.

 

 

The flood depth value controls the flood indicators along the streams.

If the flood depth value is zero, only the yellow flood streams will be drawn.

 

If the flood depth value is larger than zero, then indicators will be drawn along the streams where the terrain elevation (to the left and right of the stream) is between the Z value in the stream and the Z value plus the flood depth.

 

 

Section X-X

 

A) The yellow stream position.

B) The flood depth.

C) The area to the left and right of the stream point that falls within the flood depth.

 

Stream path.

 

Each stream starts at the source and continues until it reaches the end of the model or when a ponding area is reached. When the stream ends in a pond, the indicators will be created to show the full water level of the pond. That is the elevation at which the pond will overflow.

 

Theoretically, the stream could again continue from the overflow point. This function however does not attempt to simulate that and the stream terminates at the first pond it reaches.
