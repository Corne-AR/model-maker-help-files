STRING : Add - Alignment

 Top  Previous  Next

 

MENU : gXt - Add - New

 

The ALIGNMENT string consists of two separate sets of nodes. The horizontal alignment consists of nodes in the form Y,X co-ordinates, radius & transition curve values. The plan co-ordinates are defined similarly to the co-ordinates of the general string. Enter the curve information (radius & transitional lengths) separately by the Node/Node info menu or with the dedicated alignment editor in the Info/Co-ordinates menu. The curve is part of the string and cannot be edited via any CAD menu option. Enter and edit the vertical alignment of the string using the Vert menu.
