MENU : dB - data base-Gen

 

 

 

Add co-ordinates

Use the 3 input boxes to specify the value to be added. Put a * in front of the value if the co-ordinates must be multiplied by it.

Example:  A constant *0.1 will have the effect of multiplying all the values by 1/10.

 

What type of data

Select which of the DTM, CAD and/or strings must be updated by the constant.

If the DTM is selected:

               Define the range start and end point number. By default values are from one to the last survey point number.

               Select if all the points or only the identified points must be updated.

       Note: The DTM option will be grayed out if the project consists of tachy stations.

 

         If the CAD is selected the constants will be added to all the various elements. Lines,text, circles etc.

         If Strings are selected the constants will be added to all the defined strings.

 

Apply constant

Press this button to apply both the co-ordinate and rotation constant.

       The rotation action is carried out first and then the constant values are added to the co-ordinates if both actions must be performed        simultaneously.

         Note: You can manually also do it in 2 steps. First apply the rotation, reset the values and then apply the constant values.

 

Reset Inputs

Press this button to reset all the constant input boxes back to zero.

 

 

 

 

 

 

 

 

 

 

 

Add rotation

In addition to adding regular coordinate constants, a rotation amount can also be added to the data.

               Specify the coordinates of the base point. The data bases will be rotated around this point.

               Specify the rotation angle as degrees minutes and seconds

               When adding a rotation value, check and compare the positions of a couple of reference points before and after the rotation has been added to ensure the calculations were done with the correct value and in the right direction.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

RANGE

Set the range of the point number that must be updated.

The start value will initially be set to 1 and the end value will be set to the total number of survey points in the active project.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

CODE : TO CHANGE

Enter the code you want to change.

Use the * character to substitute for any other character, e.g., 12** will change 1234 or 12AB etc. to the new code.

If you want to change every second, third, etc., points, enter !!2 or !!3 or !!4, etc.

 

 

 

 

 

 

 

 

 

 

 

 

CODE : REPLACE WITH

Enter the new code to be applied to the selected points.

               Use the *.* code to deactivate points.

               To delete every second point, enter the code to change as !!2 and the code to replace with as *.*

 

 

 

 

 

 

 

 

DO CHANGE OF CODE

This button starts the process.

 

 

 

 

 

 

 

 

 

 

RESET INPUTS

Reset the input values to the original defaults.
