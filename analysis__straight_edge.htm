ANALYSIS : Straight Edge

 Top  Previous  Next

This function will slide a straight edge over a draped GXT string and report the positions where the surface is away from the edge line.

 

A typical application of this feature would be to check how even the surface of a concrete floor is.

 

Explanations:

 

In this image:

 

Green line : Side profile view of a string that was draped over the surface.

Purple : Sample positions where the straight edge will be analyzed for depth differences.

 

1) A typical scenario where the end of the straight edge is in the air when going over a high point.

or

2) The edged of the ruler are on the ground and the depth of the 'valley' is picked up.  

                                             

 

 

 

a) The length of the straight edge.

 

b) The maximum difference between the edge and the surface before it will be recorded as text or shown as a cross in plan.

 

c) When the test is run, all deep depths will be shown as a cross. Optionally the actual depth can be added to cad as text.

 

d) Option to set the correct text parameters, including the layer name, before the test is run.

                       

 

 

A typical output after the analysis has been done.

 

Red lines : Identified GXT strings.

 

Red crosses : All the positions along the string where the edge if off the surface.

 

Blue text : Actual depth values as cad text


 

How it works:

 

1) Draw general GXt strings across the model to form a grid. Depending on the size of the platform and how accurate you need to be, these strings can be 0.1 to 2m+ parallel to each other.

Hint: Draw one string across and then use the add Parallel string option to generate the other.

 

2) Once the strings are defined, Identify all the strings and use the GXT, Id, Man-id features to Drape all the strings onto the model.

 

When doing the drape, also add nodes at slope changes. These nodes are important to measure the depths.

 

3) After adding the strings and they are identified and draped, proceed to Analysis - Straight Edge function and let the ruler run along the strings.`

 

Things to remember:

 

GXT strings indicate the line on which the ruler is tested. These strings must be identified and draped onto the correct surface (dtm project) so the strings are an accurate representation of the surface.

 

Edge length is defined in meters and the depth in mm.
