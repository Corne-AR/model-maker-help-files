STRING : Info - Area 2

 Top  Previous  Next

 

MENU : gXt - infO - Area-2

 

Calculate the slope area between 2 general strings.

 

Indicate any 2 strings laying approximately parallel to each other. Model Maker will divide the length of each string in 1000 intervals and calculate the co-ordinates including the elevation of each interval point.

 

Imaginary triangles are then built between these points and the slope area of the triangles are calculated.

 

In this diagram, 1 and 2 indicate the 2 strings. 3 (the yellow lines) indicate connections between the intervals on both the strings. No.4 (the green lines) indicate the legs of the triangles formed between the intervals.

In the above example, 16 triangles are automatically formed between the strings and the slope of each is calculated.

 

Although the technical description of how the function operates is given, note that the structures of the strings are not changed. The points at regular intervals and the created triangles exist only for calculation purposes and are discarded after use.

 

Note:

It is important that both strings start from the same side, since the intervals will be generated and connected on regular chainages from the first node on the string.

If the strings in the above example started in opposite directions, the following will occur.

 

Use the INDICATE START function to change the string direction.

 

The total slope area will be reported but a detailed report of areas per elevation can also be printed.

 

 

 

 

 

Interval areas can be reported to the editor or the xcel format editor.

 

Also set the value of the interval at which the areas must be reported.

 

 

 

 

 

The report breaks down the intervals into 2 formats.

 

By contour height:

 

   The interval slices are based on regular horizontal contour elevations in relation to a zero basis.

 

 

By Relative height:

 

   The interval slices are in relation to the elevation of the lowest string at each point of intersection.

   In practice it is the areas that run parallel to the lowest string.
