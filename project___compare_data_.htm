PROJECTS : Compare data

Top  Previous  Next

 

MENU : prO - cOmpare1

 

Data of 2 projects can be compared to find points that are close to each other. This is usually used to compare design data to as-built data.

 

 

Check Z to triangle model

 

All the points of project1 will relate to the interpolated elevation taken from the model of project2. The height difference will be reported.

DY DX

Output in mm

: If checked, an arrow will be drawn between each pair of points. Arrows are drawn from the point in the second project to the point in the first project. Minimum length. The arrow will represent the actual distance between the 2 points. Depending how far the points are apart and the current scale setting, this could result in minute small arrows. If a minimum length is specified any arrow shorter than the minimum (in millimeters) will have at least the minimum length.

 

: The direct distance between the 2 points will be written to the cad as normal text.

: The direction between the 2 points will be written to CAD as normal text.

: The difference in the Y co-ordinate and the X co-ordinate will be added as CAD text.

: If checked, the distance and DY DX values will be multiplied by 1000 before being added to CAD.

 

If points are found within the limits, a report is compiled and sent to the editor.
