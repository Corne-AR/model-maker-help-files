MENU : Irr - Plot

 

Plot Button

 

Click this button to temporary hide the irrigation dialog in order to view the Model Maker plot screen.

You may use any of the plotting functions like Magnify, Normal etc. while in this mode.

Press escape to return to the irrigation menu after viewing the graphics screen.

 

Calculate Button

 

Click this button to go directly to the hydraulics calculation menu. This shortcut button could sometimes be grayed out and unavailable. That is when the irrigation layout has not been checked in which case you must activate the global page and access the hydraulic calculation menu by selecting the regular flow calc button.

 

Block Number

 

The number of the current active block is displayed. The main line is referred to as block zero.
