Secondary Loss - K factor

In this input box, set the k value to be used in the secondary friction loss calculation formula.

         Secondary Loss formula :  Friction = k v^2 / 2g

                         v = velocity in pipe

                         g = gravitational acceleration (9.81 m/s^2)

 

 

When the actual hydraulics are calculated in the hydraulics calculation page, specify if the secondary losses must be taken into account or ignored.
