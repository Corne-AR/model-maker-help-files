 Top  Previous  Next

 

MENU : File - Save (F2) - YXZ

 

Save the data in an ASCII file consisting of a tag code, y, x and z co-ordinates.

 

EXAMPLE:

For a spot height at top of embankment:

 

Tag is                     -    TAG    

Code is                  -    EMT    

Y co-ordinate is    -    25000.00    

X co-ordinate is    -    69000.00    

Height is                -    1425.00      

 

ASCII file format for the above data:    

 

TAG  EMT    25000.00   69000.00  1425.00
