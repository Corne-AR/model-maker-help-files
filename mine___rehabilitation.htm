MENU : Min - Rehabilitation

 

Rehabilitate original models to set slopes or to strings as defined in the string module. These options should only be applied to gridded data files. This data can be generated from any triangular model with the 3D functions. Use a relevant grid interval and save the data in ASCII format. Load this gridded file as the design project and apply the rehab functions.  

 

Rehab All

Rehabilitation of a gridded model is done according to a set maximum slope. Every grid point will be inspected and slopes calculated to all its adjacent points. If any slope is steeper than the set maximum the higher point, it will be lowered and the low point lifted with half of the height needed to bring the slope to within the maximum value. This process pushes the material down hill. The crest of the dump will be lowered and the toe line will move outwards. This process will be repeated until all the slopes comply to the maximum.  

 

Reshape by dual slopes

A gridded model will be reshaped to the limits set by the minimum and maximum slope values in two directions. This is normally used to gradually level a terrain so that the direction and velocity of the runoff water can be controlled.

Dump Material

This functions takes a defined area and a user specified amount of material to be dumped in the area. Model Maker will inspect all the points and any hollow area - a point that has surrounding points higher than itself - will be filled in. This function will not build a dump as such only smooth out an area by filling the low spots.

 

Rehab String

Use strings to manipulate the shape of a design surface.

Use the string functions and place general strings onto the model. The design model must be a gridded data project. (Use the 3D option to put the original surface into a gridded ASCII file).

The general strings must be designed so that if the gridded data comply to the elevations of the string nodes, the model will satisfy the design criteria.

 

If the result is not what you require, make changes to the strings via the string module and re-run this option.

 

The difference between this function and the general rehab option on this same menu is that you can force your own design onto the model via the strings where the general rehab's only criteria is the maximum slope. The strings used for the design must be identified.

 

A single string or the identified strings may be used as the design.

Update Model

Model Maker has calculated new elevations for the design points based on the layout and design of the strings. These new elevations are temporary.

 

Do not update.

If this option is used, Model Maker will exit this routine and will not apply the new elevations to the database. The whole exercise is for inspection purposes.

 

Update all

The new recalculated elevations will be applied to the dtm data and the database updated accordingly.

 

Update part

The new calculated elevations will not be applied to the database. However, you can redo the whole exercise with only a certain percentage of volume to be moved.
