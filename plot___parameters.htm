MENU : Parameters                (G)                                

 

Use this option to select the appropriate plot parameters. To plot any one of the file options, set the plot parameters and then plot it to the screen to make sure all the necessary information is plotted.

 

File

 

Save plotting setup

Displays the screen for saving plot parameters.  All the parameters  will be saved to a file with extension .PDF. The plot parameters saved in the file called MMW.PDF are used as the default each time the program is started.

 

Open plotting setup

Display the screen for loading plot parameters. The parameters are read from disk and set accordingly. The parameters must already have been saved.

 

 

General

 

 

Plot Scale

Specify the plotting scale for plotting to the plotter or to a file.  

 

Constant Deduction

Enter the value to be subtracted from the elevations for plotting purposes only. This does not effect the database Determine the lowest elevation in the database by clicking on the question mark.

 

Number Of Decimals  

Specify the number of decimals when the elevations are plotted.  

Plot Rotate

Specify the angle which the plot must be rotated through to obtain a better fit on the paper. Clockwise is positive. You may also define a plot window that is rotated on the graphic screen with the Paper option

 

Grid Interval  

Specify the distance between grid lines in millimeters.

 

Text Size (screen)

Specify a text size factor for the text when plotting to the screen. A suggested factor is 0.3.

 

Contour

Select to draw contours automatically with each replot. On the screen the contours will be drawn by means of the Quick method. To the plotter or DXF file, normal contours will be drawn. If the plot is exported to a plotter or other file, this option must be selected to include the contours.

 

Triangles

Select to draw triangles automatically with each replot.

 

Detail

Draw the >< and @ detail.

 

 

Contours

 

General Interval

Specify the interval at which contours must be drawn.

Major Interval

Draw major contours using a different pen to distinguish them from general contours.  

 

Label @

Specify the interval at which contours must be labeled.  

Top Level

Specify upper and lower contour limits. Contours will only be drawn between these limits. Specify the upper contour limit.  

 

2 & 4 are the multipliers. In other words if 2 and 4 have values in them other than 0, these values will be multiplied with the general interval (1) resulting in the major interval (3) and contour label interval (5).

 

This enables you to only have to change the general interval value (1) and that the major and label contour will be automatically calculated as a factor of whatever the general interval is.

 

Bottom Level

Specify the upper and lower contour limits. Contours will only be drawn between these limits. Specify the lower contour limit.  

Spline Contour

Specify which contours are to be drawn as smooth or straight lines. Splined contours may be attractive, but they increase plotting time and the size of the .dxf file.  

?

Takes you the to Default Text Sizes window.

 

 

Global

 

CAD

Plot the CAD drawing. Deactivate this option if you only need to plot the DTM data.  

DTM

Plot the DTM data. Deactivate this option if you only need to plot the CAD drawing.

Gxt (Elevation)

Plot the position of the survey point and the point elevation.  The Point option must also be active.  

Frame

Draw a frame around the border.

Grid

Plot the co-ordinate grid.

Title

Draw the standard MODEL MAKER or user defined title block.

Heading

Descriptions used in the title block can be changed. This is the drawn by (drawn for) project and note labels put into the Model Maker title block or the labels defined in the user title block

 

 

DTM

 

None

Use this option if the points or any description of the points should NOT be plotted.  

Point

Show the position of all the points.

Number

Plot the position of the survey point and the point number in the data base.  The Point option must also be active.  

Code

Plot the position of the survey point and the point code.  The Point option must also be active.  

Tag

Plot the position of the survey point and the point tag.  The Point option must also be active.  

Elevation

Add the point Z value to the plot.

 

NOTE: All the text related to the DTM point is drawn in the standard Model Maker font. To display the text in another font, copy the elements to cad and then change the cad font.

 

 

Code Selection

 

 

Plot Code Selection

Specify up to 10 codes for points to be plotted. Enter the codes exactly as they occur in the database.

Clear

Clears the code selection box

View Database

Use this to see which codes were used in the survey.

 

All                All the points will be plotted irrelevant of the codes

Only                Only the points with codes as in the above list will be plotted.

Id                Only the points that have been identified will be plotted.

Not                Points with codes as specified in the list will not be plotted.

 

Show other as points

Specify what is done with the other points.  

All edit of other

Jump and edit points that are not in the list but displayed as points.

 

Other

 

Text line & other sizes

 

 

Elevation of codes

Elevation of codes starting with . (dot) codes will be shown.

 

 

Replot on exit ?

 

 

If checked, Model Maker redraws the graphics when you exit the window.
