Points add single

 

Points can be added in various ways. Given a list of co-ordinates, these can only be entered individually by typing in the  Y,X and Z  value.

To do this change the snap mode to use defined YX co-ordinates. (Ctrl W).

 

Listed below are examples of the various ways of adding a single point using the snap mode.

 

At the endpoint of any CAD element which includes a crossing line intersection.
At the endpoint of a CAD line.
At the endpoint or centre of an arc.
At the text origin.
In the centre of a line.
Perpendicular to a line.
Near a line which has been selected

       By specifying a distance from the line endpoint. (+ or -)

Draw a line tangential to a circle.
On the arc or circle closest to the current cursor position.
At the centre of a circle.

 

The cursor positioning modes may be used to place the new point. The point will be added to the current project.

When the position has been indicated, the dialogue below will appear.

 

Listed below are the various methods that can be used to elevate the points.

 8 

Tag

 

The default tag of the previous point added is displayed.  The tag may be accepted, be changed or use the drop down button to choose a previous tag from the list.  If the default code ends with a numeric character (not in this example) and the INC box is checked, the tag number will be automatically be

incremented.

 

 

 9 

Co-ordinates

 

The co-ordinates of the point are defined. These co-ordinates were derived from the cursor position indicated on the screen. The co-ordinates can be changed irrespective of how they were defined originally no matter how they were defined initially.

 

 10 

Enter z                

 

Enter the elevation. Use the drop down button to select a value from the history list.

Use the button to zero the value.

 1 

Project to use

 

Scroll up and down to select the desired project to interpolate or extrapolate from triangles.

 

 

 2 

Interpolate from triangles

 

The interpolated elevation based on the triangle directly below the new co-ordinate will be calculated and used as the Z value.  This function cannot be used if no triangles exist or the point falls off the triangle model.

 

 

 3 

Extrapolate Triangle

 

The elevation will be calculated using the slope of an indicated triangle and extending it to the intersection with the new DTM point.

 

 

 4 

Elevation of DTM                

 

The elevation of an indicated existing DTM point will be used for the new elevation.

 

 

 5 

Elevation of CAD text

 

Indicate a piece of CAD text. This text, if it can be translated to a valid numeric format, will be used as the elevation This function is useful when there is a design model defined as CAD lines and text. Add the DTM point to the end points of the CAD lines and grab the elevation from the CAD text.

 

 6 

Allow duplicate point codes

Points will be allowed to have the same codes.

 

Do not allow duplicates (case sensitive)

Points with the exact same codes will not be allowed ie: PT3N = PT3N

 

Do not allow duplicates (ignore case)

Points that have the same codes will not be allowed ie: Pt3n = pT3N or PT3N = PT3N

 

 7 

Code

 

The default code of the previous point added is displayed. The code may be accepted, be changed or use the drop down button to choose a previous code from the list. If the default code ends with a numeric character (as in the example) and the INC box is checked, the code number will be automatically be incremented.

 

Cad Text

Function will use selected cad text as the point's code.

 

Increment code: Alpha

Optionally increment the code by ALPHA characters. A will become B C etc up to Z and the ZA ZB.
