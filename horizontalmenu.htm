MENU : Hor

 

The horizontal menu allows you to do the following:

               Create a horizontal alignment - to be used in extracting sections.

               Plot the alignment to CAD - to show where the road or pipe line is located.

               Interrogate the alignment to know what the chainage distance value is at any position.

 

 

A horizontal alignment is used to define a route for a road or pipeline. Once an alignment is defined, long sections and cross sections can be extracted.

This information can be stored into an ASCII file for use in:

-                Model Maker                : horizontal alignment will have a .PID file extension (Point of Intersection Data)

-                Pipe Maker                : long sections will have a .GRO file extension

-                Road Maker                : cross sections will have a .XX1 file extension and long sections will have a .GRO  file extension

 

 

There are three ways of defining a horizontal alignment:

1.                Pointing to each PI on the screen.                

2.                Selecting a polyline drawn in CAD.

3.                Direct input of co-ordinates in the text editor.

 

Hints:

If you hold your cursor on the Hor button on the left hand menu and a horizontal alignment is loaded, the file name will be displayed.
