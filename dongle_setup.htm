START USING MODEL MAKER SYSTEMS SOFTWARE

 

From the WINDOWS TASK BAR select START, PROGRAMS, MODEL MAKER SYSTEMS, MODEL MAKER to activate the MODEL MAKER software.

 

 

When you run MODEL MAKER for the first time, it is important that you setup the basic configuration by selecting the SETUP option. On the configuration dialog enter your company name and telephone number for use on all print outs. Make sure the security port is set to SEARCH for a single workstation key or NETWORK for a network key. The PROGRAM ACCESS MODE must be set to FULL ACCESS.

 

 

Once you are in the main program screen make sure that your security key number is displayed in the WINDOWS header at the top of the screen. If the program is active in DEMO mode exit, using FILE – EXIT and restart the program again. If you do not get full access activated, contact your supplier for assistance.

PROBLEMS WITH THE SECURITY KEY OR SETUP FILE

 

When you activate the software it will first try to find a valid security key on your computer or the network. Once a key is found it will then try to read the setup file for that specific key.

 

If no security key is found the following screen will be displayed.

 

 

Make sure the correct key is connected and use the TRY AGAIN option. If this fails it might be that your computer's USB / parallel port is faulty. If possible test the key on other computers and if it does not work contact your software supplier for assistance.

 

If a valid security key is connected but a valid setup file is not found, the following screen will be displayed.

 

 

 

A valid setup file must be present in the C:\Program Files\MMSystems folder. If the file name is shown on the right hand side it might be corrupted or in READ ONLY mode. Copy the setup file from your backup validation disk or contact your software supplier.
