MENU : Vol - Triangles                                

 

Two models must be used. The first model is the dump itself. The second model is the base model, created by triangulating the perimeter points. The two models will be called into the computer one at a time. The volume difference between each model and a theoretical horizontal surface will be calculated. The actual volume of the dump is the difference between the two calculated volumes. Only use this method if the two models have the same plan area and perimeter.  

 

Recall model

Select this option if a new triangle model must be used in the calculations. You will be prompted for a triangle file to load.
