POINT : Identify - Copy to cad

MENU : Pnt - Id - Man-id - cAd

 

Copy the identified points as text into the Model Maker CAD environment.  

You have control over 2 parts of the attributes to be copied. The first is the method used by Model Maker to mark the spot of the point and next the text that must be written along it.

 

 

1

Points Identified

Select the method that must be used to mark the origin of the CAD point in the CAD.

The following is an example of the 4 different versions.

Size (mm)

Specify the size of the cross or radius of the circle in millimeters. The dot and cross will be drawn as CAD lines while the circle will generate a circle in CAD that may be manipulated with any of the regular options.

 

 

 

3

Text

The four basic attributes of a DTM point may be copied. Check the combination of your choice. If you only want the spot to be marked as a point in CAD, uncheck all four boxes.

4

Elevation Format

Allows you to choose between a space or a decimal.

5

Apply Layers

 

As set: This option will copy the dtm elements to CAD using the current default layer name.

As defined by code: With this format, the points will be copied to CAD using the point's code as layer name. New layers will automatically be created in CAD for points with unique code values.

When the option is selected to place a circle on each point (1) then the circle layer will also look at this setting and either use the default cad layer or set the layer according to the point code.

6

Allows quick access to the Text, Line and Plot Parameters. Use the plot parameters to deduct a constant from the elevations if necessary and to change the scale.

7

Graphical Output

Option to write the point coordinate values as Geographical format values.

2

Format

The information can be copied to CAD using 4 different formats.

Detail format :

This format will show the resulting CAD text elements in the same way the original DTM points displayed the point attributes.

Please note that copying the co-ordinate values to CAD is not available in this option.

Table Format :

This format will display all the attributes to the side of the DTM point, all lined up one below the other.

The point co-ordinate values can be copied to CAD in this format.

Notice that the item prefix is added to the attribute values when this format is used.
