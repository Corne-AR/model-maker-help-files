MENU : Cad - Text - line Inc

 

 1 

Unnumbered Erfs

This function is used to incrementally number erfs without numbers.

 2 

Enter a starting erf number

Once the option is selected from the Text menu, you have to enter the starting erf number.

 

 

 3 

Draw a line through the erfs to be numbered

Pick the erfs you want to number by  drawing a line through them.

Note:

To rotate the text correctly, hold in Shift before you select the second line point.

 

 4 

Numbered Erfs

After you select the second line point the numbers will show up on the erfs.
