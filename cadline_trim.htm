MENU : Cad - Line - Trim

 

Extend the indicated end of an existing line to the intersecting point of that line and a previously indicated base line.

 

BEFORE

 

Do not use this function on two parallel lines.  

 

AFTER

 

Note:

The base line must be indicated first. Then individually indicate the lines to be extended to the intersection point. The position of the cursor is important.

The first line was identified with position (1), which was closer to the end point furthest away from the base line. The indicated end point moved to the intersection point.  Positions 2,3 & 4 were closer to the end points closest to the intersection points, resulting in the lines being extended. Whichever result you want, use the cursor position to achieve it.

Use the '+' key to undo the last changes.

 

·If the SHIFT key held down while the trim line is indicated, the endpoint closest to the base line will be trimmed irrelevant of the cursor position along the trim line.
