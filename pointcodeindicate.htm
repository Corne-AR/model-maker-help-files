MENU : Pnt - Code/Tag

 

 

This function allows you to locate points with selected codes or tags in the plan.

 

 

 

When the dialog opens, all the codes of the points in the active survey project is shown.

 

 

 

Click on a code to select it. After a code has been selected use any of the 3 buttons to help you locate the point.

 

Show selected code in plan : All the points with the selected code will be highlighted in plan with a small white image.

 

Jump to first occurrence : The dialog will close and the mouse cursor will be placed on top the first point with the selected code.

 

Zoom to selected code : The dialog box will close and the screen will be zoomed so that the point having the first occurrence of the selected code will be in the screen center.

 

Search : When you have a long list of codes, use the search input box to find the code you want in the list box. As you type a character, the list box will scroll to the 1st matching code. Once you locate the code, click on it to select.
