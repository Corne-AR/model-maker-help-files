Example of YOM data:

(Format 1)

 

Z070197 08 01 01 1200                      (The Z at the start of the line denotes a stations with name 08 and instrument height 1200mm)

N01

0250 03503640 985801 -100                (The -100 denotes an orientation to trig # 100)

N02

3827 0250 03503640 985755 100                (The positive 100 denotes a tachy shot to a point coded 100)

N03

0250 290502 885410 04                                (The positive 04 denotes a tachy shot to a point coded 04)

Summary :

         Start a line with 'Z' if it is a station.

         Give the point a negative code if it is an orientation

         Give the point any positive code if it is a spot shot.

 

(Format 2)

 

Z120597 03 00 00 1400                                                (The Z at the start of the line denotes a stations with name 03 and instrument height 1400mm)

N1 32 4 357.4157 87.1144 1.490                                (Tag   Code   Type   Slope distance   Horizontal angle   Vertical angle   Prism)

N2 37 5 12.190 356.2016 88.5758 1.490

N3 48 5 11.466 338.3645 88.3617 1.490

Summary:

         Start a line with 'Z' if it is a station.

         The first column is the Tag

         Second is the Code

         Third is the Type. (4=orientation   5=spot shot)

         The distance and prism have 3 decimal places

         The horizontal and vertical angles have 4 decimal places.
