How to manage the pivot spans.

 

 

 

 

The spans dialog.

 

 

 

 

 

Since version 16.01 the management of pivot spans has changed.

 

In the past, the user would have used this list box to manually enter the span length and number of spans.

 

The Add/Edit Spans option is now used to open the span management dialog box. In this new dialog box

you can still enter the span lengths manually but you now also have the option of selecting them from

a database.

 

 

Clicking the Add/Edit button will open the following dialog.

 

 

1) The list of spans for this pivot

 

2) The OEM name. If a database is linked to this name, it will be shown in the

   DBASE list (5)

 

3) If an OEM is linked, use this drop down to only show certain models in grid (5)

 

4) The units that the database was defined in. If the database is defined in English units (a typical Sennpaq span file) and you work in metric units, the values in the list will be converted to metric.

 

5) The spans as defined in the linked database. These spans can be used to define the pivot.

 

6) Click Manufacturers to open the OEM database manager.

There you can assign OEM databases to use.

There is also an option where the program can create a basic database for you. You can then use that database to manually enter your own span information as basis when selecting spans for your pivot.

 

7) Use these 2 buttons to insert or delete existing spans in the pivot.

 

8) To make use of a span in (5), click the span, then click (1) the span number that you want to use from the

   databasis. Now the system knows which database span must be copied to the selected pivot span.

  Click COPY TO SPAN #.

9) If the last span in the list has an O for type, MM will assume it is an overhang and automatically define the overhang length as such.

 

NOTE:

 

The option to use an OEM (or self generated database) is optional. The most basic way of adding spans would be to click on a span number in (1) and just enter the span length manually.

 

The ID#, diameter, number of Outlets and spacing are purely informational and not used by the program at this time. The span length is used by the program.

 

If the actual pipe Diameter is defined in the list, it will then be used to automatically populate the span diameters on this page.

 

If the Type of the last span is O  (Overhang), the length of that span will be assigned to the overhang length.
