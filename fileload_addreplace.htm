FILE : Load - Add/Replace

 Top  Previous  Next

 

MENU : Shown automatically when data is loaded.

 

This dialog box is shown when new data loaded from disk will overwrite existing data in memory and allows you to use various functions to manage the situation.

If the data that you load is not CAD drawing data, the following dialog is displayed.

 

 

Use the following options as needed :

 

Replace

The existing data in memory will be replaced by the incoming data from the file.

Add

The data being loaded will be added to the data already in memory.

 

                                                                                                                                    Note:

If you want to combine 2 survey projects, rather load each one in a separate project and use the Project Copyfunction to combine it into one design.

 

Project #

If you are loading survey points and you do not want to replace the existing data in memory, use the project option and select a different project number into which the points must be loaded.

 

 

If the data that you load is a CAD drawing, the following dialog is displayed.

 

 

Use the following options as needed :

 

Replace

The existing CAD data in memory will be replaced by the incoming data from the file.

Add

The data being loaded will be added to the data already in memory.

 

                                                                                                           Note:

                                                                      When CAD data is added to the existing drawing, two additional options are available

 

 

Indicate new position

If this option is checked it will allow you to indicate the position of the cad drawing relative to the rest of the plan.

Use this feature to place the in coming CAD file of to the side of the existing elements if you do not want an overlap.

If the option is not checked the incoming CAD file will be placed on it's original co-ordinates. (The position that it occupied when saved to disk).

 

Prefix to layer

Any characters entered in this edit box will be added to the layer names of all the incoming elements. This feature can be used to easily identify the new elements to

be manipulated separately at a later stage.
