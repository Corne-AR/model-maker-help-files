MENU : CAD - Dim - Line_text                                

 

Line_text

 

This is not a standard dimension function as the length and angle of an indicated line will be calculated and recorded onto the line as normal text elements.

Set the text size and pen using the normal text manipulation menu.

 

Choose between 3 variations.

1: Only the length

2: Only the direction

3: Both.

 

These options are set in the Dimension DefaultType Notation menu.

 

The positions of the text are set according to the following conditions.

         The length is always added on the cursor side of the indicated line.

         The angle is always added on the opposite side of the cursor.

         The direction of the text is always away from the cursor.  

 

 

Use the functions on this panel to control the orientation of the text elements.

 

2Pnt_text

 

This function works similarly to the LINE_TEXT function. However, the base of the dimension is not indicated by selecting a regular CAD line but by manually indicating the start and end co-ordinates of the line.
