MENU : Parameters (Alt+P) - Border sizes + Text & Line Sizes

 

Use this dialog box to change the relative sizes of the attributes when plotting to a plotter.

 

 

DTM text sizes

 

This relates to the DTM elements.

 

 

Elevations

The elevations height and ratio regulates the size of the text on the plotter when the DTM points are displayed with selected detail

This is an example of a dtm point plotted with the elevation and code around the point. The elevation will be plotted at a height of 1.2mm and a width of (1.2*0.9) = 1.08mm.

 

Code/Elevation ratio

0.8, meaning that the code (bz45k in the example) will be plotted above the elevation at 80% of the size of the elevation.

 

Grid text

The height and ratio of the co-ordinate values on the grid lines.

 

Contour text

The height and ratio of the co-ordinate values on the contour lines.

 

Crosses

The size of little mark (cross)indicating the position of the survey point is set by this parameter. The unit is in millimeters and is independent of the scale.

 

Symbol sizes

 

This option regulates the sizes and distance between the patterns on user    defined line types.

 

Example of a fence line type with crosses as patterns.

Two definition formats may be used.

 

 

FORMAT A (Standard format)

 

The following values are used:

 

0 :                 At a scale of 1/1000 and less (1/500 1/200 etc) the symbols                                   will be the size as defined in the shape file. The symbols                                                                     will become progressively smaller for scales above

                1/1000, eg., 1/2000, 1/5000 etc. with the minimum size at                  1/25000 where after it will remain the same size.

          The distance between symbols will vary according to the                                   same formula as described for the symbol sizes.

 

Any value other than zero :

 

The symbols will be the factor of the defined size and will not vary according to scale.

 

FORMAT B (Range format)

 

Use this format to define specific symbol sizes at selected scale values which allows you to manually decide how the symbols should behave. For example you could specify that the symbols should stay the same size up to scale 1/5000 and only there after reduce in size. The distance between symbols and the size of the symbols can also be set independently.

 

Click on the Default button to use the standard settings. Remember to save the plot parameters if the changes must be recorded for future use.

 

Paper Borders

 

The original paper size as  defined will be made smaller  by the border  amount and the  drawing will be placed on the  paper with the border  amounts  away from the physical limits.

NUMBER OF DECIMAL PLACES

 

This specifies the degree of accuracy when the contour is labeled. eg: a Contour at elevation 10.125 labeled with 2 decimals = 10.13

 

ALLOW ZERO DECIMALS ON FULL CONTOURS

If this box is checked contours with full elevation eg 10, 15, 100 etc. which would normally have been labeled as 10.000, 15.000, 100.000 will be labeled without any decimals.

 

CONTOUR DESCRIPTION                                                

This description will be added to the contour elevation. eg : 12.25m, 50.20vt etc.

 

Arrow and Dtm

 

 

 

 

 

The sizes of arrows and dimensions can be controlled by these 3 options.

 

Cad: Same size scaling as what is defined for cad line symbols in the Sizes tab.

 

Fixed : Actual millimeters, true to scale.

 

Scale : The elements will grow larger and smaller according to scale. See hint on the option in the program.

 

 

 

 

 

Dtm stations and Dtm points can be defined with different symbols.

 

Select the symbol for each entity and also the mm size of the symbol. The symbol will be drawn, true to scale at the specified size.

The millimeter symbol size is applied both on screen and when plotting to paper.

 

Since the symbols are true to scale, their image on screen will increase in size when zooming in and decrease when zooming out. On certain projects with many points close together, the symbols could become too big when zooming in, clutter the screen and make it difficult to decipher.

 

Checking the on screen option and setting a maximum pixel value will prevent the symbol from growing larger than the limit, making it easier to visualize dtm points on projects with smaller scale values. This setting does not effect the symbol size when printing to paper.
