The pipe lengths option is used to divide the manifold into user definable length segments.

To do this, you first have to click on the MANIFOLD option so only the manifold pipes are shown in the hydraulic gradient graph.

 

 

Click the PIPE LENGTHS menu option. The following dialog is shown.

 

 

The bottom table allows you to specify your own pipe lengths.

 In the above example the existing length of 75HDPE pipe is 56.69 meters.

 If you want to build this section with 6m pipe segments you will either use 60m or 54m of pipe.

 To be conservative you will rather use more of the bigger diameter pipe and round the 75HDPE off to 60meters. Because you are using more than the 56.69 meters available, the program will shorten the length of the 63HDPE pipe and the result will be as follows.

 

 

 

Notice:

 

In the above example we worked from the valve down to the end of the manifold pipe. (Valve to end mode)

If you wanted to start from the smaller diameter pipes and let the program allocate the remainder to the segment that connects to the valve, use the End to valve mode.
