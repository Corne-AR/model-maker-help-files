MENU : Pnt - Code

 

This dialog box is displayed whenever you need to select dtm points by means of their codes.

The initial box will appear as above. The top right input box will initially be blank and the scan codes button visible.

 

Scan codes

 

The DTM points in the database may have many different codes. The scan codes button instructs Model Maker to display all the codes used in the project. This allows you to select the codes by clicking, rather than to typing the exact code of each one in the enter code input box.

The available codes are not displayed automatically since scanning data can take a while, especially if most of the DTM points have different codes. Progress will be indicated in the top right corner of the screen. Click the abort button if the process takes too long.

 

After the scan process is completed, the menu look like this:

 

The available codes are listed and 4 new buttons are displayed. Use the mouse and click on the codes to be identified.

The clear selection button changes the selected codes to the unselected state.

The select all button automatically selects all the codes.

The info from element button graphically indicates any existing dtm point from which the code will be taken and selected in the list.

The identify every second button is not code related. It will identify every second point sequentially. This is a useful function if the data was generated by aerial photography and taken to close to each other. Identifying and deleting every second one trims the data to a more practical format.

 

Enter Code

 

This input box allows you to type the specific code of the points to be identified. The code description must match the point code exactly using capital or small letters. The * character stands in for any other character.

For example, Tr** will identify codes Tree, Trim, Trap, Tr01, Tr02 etc.

The maximum length of the DTM code is 6 characters. An input code of ****** identifies all the points with any code.

 

Use the button to let Model Maker execute the identify command after you have specified the desired enter code.
