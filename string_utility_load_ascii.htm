MENU : gXt - Utility -More - loadAsci

 

Strings can load blocks of data (DTM points) from any Ascii file of co-ordinates. All the strings will be checked to see if any of the co-ordinates loaded from the file fall within the definition area of the string. If so, the co-ordinate will be loaded as a need DTM point and added to the current active project.

 

This function is helpful in extracting data from large co-ordinate files when you need to work only the data falling within the defined boundaries of the strings.

 

Filters

The height value will be used to filter the new points to be loaded. The elevation of a point to be loaded will be compared with all the points within the set radius value and only be loaded if the height difference is more than the Z exclude value. A value of 0 will not filter according to elevation.
