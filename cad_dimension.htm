Dimensions

Dimensions are used to indicate the distance between 2 points or various other elements. Model Maker will calculate the distance but also allow the user to edit the final value.

 

Use the dimension functions to add additional information to the drawing. Dimensions can easily be added, edited, moved and deleted.
