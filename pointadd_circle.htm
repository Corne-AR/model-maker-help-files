MENU : Pnt - Add - Circle/arc

 

Add DTM points to an indicated arc or circle. The points will be added to the active project after the code and elevation of each point is entered.

 

Arc/Circle Information

 

After the circle is selected the following dialog is displayed. Some basic information about the circle is shown.

 

 

 

 

 

 

Start direction

The direction as viewed from the circle center to the start point of the circle.

Angle difference

The angle difference between the end point and start point of the circle.

Curve length

The length along the perimeter of the circle from the start to end point.

 

Adding Points @

This dialog box indicates how new points are added.

 

 

The above 3 formats allow points to be placed along the arc only. If you want additional points at the start and/or end of the arc check the appropriate box in the ALSO panel.

 

 

Regular angle

Starting from the first point, all the points will be placed at a distance along the arc as calculated by the angle interval. The distance is calculated along the curve using the formula.(distance=angle interval*radius)

Regular distance                

Starting from the first point, all the points will be placed at the indicated distance.

Number of points                

The indicated number of points will be placed at equally spaced intervals along the arc.

 

Direction of Travel

This option allows you to control the direction around the arc. Remember that the START POINT and END POINT will relate to the opposite end when toggling between clockwise and counter-clockwise rotation.
