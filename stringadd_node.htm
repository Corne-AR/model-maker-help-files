STRING : Add - Node+

 Top  Previous  Next

 

MENU : gXt - Add - nOdes+

 

A powerful feature of strings is that they can add multiple nodes or patterns to itself. These elements may be added by 4 different methods or spacings.

 

The distance between the new nodes is specified by the interval.

 

FORMAT

 

 

The nodes will be added by checking each segment (the leg of the string between 2 nodes) individually. The new elements will start at the interval distance away from the start node and continue at equal spacings.

 

For example, a segment is 110 long and the interval 25. The new elements are placed at 25, 50, 75 and 100. The last 10m is then ignored.

The sequence is repeated for every segment.

 

 

The segments will be checked individually.

 

For example, a segment is 110 long and the interval 25. Four new elements will fit into a distance of 100m. Half of the remainder, 110-100=10, will be placed at the start (5) and half (5) at the end. The new elements are placed at 5, 30, 55, 80 and 105.

The sequence is repeated for every segment.

 

 

 

The string is manipulated as a whole. The first segment is treated as described in the FIXED format. The new elements on the second segment will start at a position which is calculated based on the remainder of the previous segment.

 

For example, the first segment is 110 long and the interval 25. The new elements are placed at 25, 50, 75 and 100. The last 10m is carried over to segment 2.

Segment 2 is 55m long. The first new element is placed at (25-10)=15m and then at 25m there after, ie., at 40m.

The last 15m is carried over to segment 3.

Segment 3 is then started at (25-15)=10m. The process is repeated for all the segments of the string.

 

 

New patterns will be placed at the existing nodes of the string.

 

 

 

Add and increment the node name if the increment option is selected.

 

 

 

 

 

 

 

Click on either the Nodes or Pattern buttons to start the action based on the selected FORMAT. The pattern option has the following 2 settings.

 

         Pat Angle        : This angle will be added to the rotation of the pattern when placed on the string.

         Rotate to string        : The direction of each segment of the string will be determined and the patterns                                                   automatically rotated   parallel to this angle.

 

General

 

Patterns are added to the cad drawing. The string is not changed in any way.

Nodes are added to the string.

Check your intervals before action is taken as a multitude of patterns/nodes can be added inadvertently.
