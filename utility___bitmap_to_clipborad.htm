MENU : Utility - Bitmap > clipboard

 

 

Copy the current screen image within a selected rectangle area to the clipboard as a bitmap. The clipboard may be accessed by other programs like image editors using the normal Edit, Copy and Paste functions.

 

Note:

Indicate the rectangle area that must be copied to the clipboard.

 

Use the CAD bitmap function if you want paste the contents from the clipboard.
