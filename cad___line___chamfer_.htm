MENU : Cad - Line - Special - cHamfer                                

 

Chamfer two lines with a third line that is placed at a specified distance away from the indicated line endpoints.

 

Indicate both lines. Point to the endpoints of each line from which the chamfer line must be measured. Enter both chamfer distances. Remember that distance 1 relates to the line indicated first. Distance 2 relates to the second line. The chamfer line will be drawn in the current pen attributes set by the Pen button.

Press the '+' key to undo the last chamfer. You will then be asked if you want to repeat the last exercise to specify different chamfer distances.
