MENU : Insert                (on keyboard)

 

New elements added to the CAD are drawn using the default pen & line type attributes.

 

       Activate this menu by clicking the  button on the right side of the screen; or by pressing the INSERT key for fast access; or by clicking  on the top info panel.

 

How to set default pen widths

Line Type

 

Line 0 is the solid line. Lines 1 to 7 are fixed dotted line type defined by Model Maker. User defined line type begins with number 8.  

Note:

User defined linetypes will resize when dimension leaders are resized.

 

Line Styles

 

Click the button to edit the line styles

 

 

 

 

 

Layer

 

 

All the elements on the CAD drawing are associated with a layer. Entering a new layer name will automatically add it to the list of already defined layers. Use the drop down to view and select pre defined layers.

 

Line Width

 

The width of the lines on the screen can be normal or wide. A normal width means the line will be drawn on the screen as a narrow line (1 pixel wide). When outputted to the plotter, it will be as wide as specified in specific millimeters.

 

Pen Colour

 

Select the color of the pen from the 255 defined options. This is the color of the pen on the screen and may be different from the color when generating a plotter drawing.
