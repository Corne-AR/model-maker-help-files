MENU : Cursor (Spacebar) - Positioning modes

 

How can this topic help me ?

When prompted to indicate a position on the screen, you may specify how the cursor reacts. At the prompt to indicate a position, the cursor will become a cross.

 

Cursor mode indicator.

 

Click on the CURSOR MODE INDICATOR to activate the cursor menu or press SPACE BAR on the keyboard.

 

EXAMPLE:                                                                                                                        

 

                 

 

       

                                                                                                                                         

Appears when SHOW SPEED MENU is checked.

 

Each options on the menu has a letter associated with it. Change the cursor mode without activating the menu by pressing CTRL & the related letter on the keyboard. The cursor can jump (draw) to, or move to an indicated point. It can be positioned on to a free hand position, a survey point, a number of a survey point, various positions of a CAD line, or you can enter final co-ordinates.

 

 

 

This relates to the DTM part of Model Maker.

 

Point

Move the cursor around and point to a dtm point. The cursor will jump to the closest DTM point and the co-ordinates of that point will be used.

 

nUmber

Enter a dtm point number. The cursor will jump to the co-ordinates of the point with the selected number.

 

 

 

Force the cursor to either move vertically or horizontally to the new indicated point. Restriction on the movement is automatically lifted after a point is defined.

 

 

 

These endpoints relate to CAD elements.

 

anY

All the endpoints in the vicinity of the cursor will be inspected and the closest one selected. The cursor will jump to this endpoint. The endpoints include: the start and end of lines; text origin points; circle centres; and the start and end point of arcs. Additionally, the intersection of 2 crossing lines will also be considered.

 

Line

Only the start and end points of lines will be used.

 

Arc

Only the centre and start and end points of circles and arcs will be used.

 

teXt

The cursor will jump to the closest origin point of a text element.

 

arrH

The cursor will jump to the closest arrow head end point.

 

 

 

 

These definition points relate to CAD lines only.

 

mHid

 

The cursor will jump to the center point (midway between the start and end) of the line closest to the cursor position.

 

peRp

 

The cursor will jump to the closest CAD line, but will meet that line at 90 degrees based on the position of the previous fixed point. This fixing format is unique because the position of the new point is related to the position of the previous point that the cursor jumped to.

 

tracK

 

This tracks the polyline selected.  Useful when doing areas, tracking lines for strings and horizontal alignment.

 

diSt

 

The closest CAD line will be picked. You will be asked to enter the distance that the cursor must travel along the line in order to get the fixed co-ordinate.

 

Near

 

The cursor will jump to the closest CAD line and hook onto that line by traveling the shortest distance between the line and the indicated point.

Note:

The distance is always measured from the endpoint of the CAD line closest to the cursor.

 

 

CAD circles/arcs are used for this process.

 

Tangent

The closest circle/arc will be picked. The cursor will jump to the arc by traveling along the border to the first tangent point. A straight line will thus be drawn from the previous cursor position to where it touches the arc at one point only.

 

nEar

The cursor will travel the shortest distance to the closest arc.

 

cIrcent

The cursor will jump to the center point only of the closest arc/circle.

 

 

 

 

 

 

 

The intersection between 2 elements will be found.

 

B

Between 2 lines. Indicate the 2 lines. The lines will be extrapolated if necessary to calculate the intersection. Another quick way to indicate line intersections is to use the anY endpoint option.

 

Q

Between a line and an arc/circle.

 

C

Between 2 arc/circles.

 

V

This method will define a point based on the intersection of 2 lines, offset from 2 base lines.

 

The 2 white lines are the base lines.

 

The user will point to the offset positions and enter the offset distances. This will define the green dotted lines (not actual lines, just shown as explanation)

 

The intersection of the offsets (yellow dot) is the determined cursor position.

 

 

 

 

 

 

 

These are general methods to position the cursor.

World

 

Enter the desired E&N (Northern hemisphere users) or Y&X (Southern hemisphere users) co-ordinates of the required position.

 

The following window (depending on hemisphere) will be displayed.

 

or  

DYDX

 

Enter the desired relative Y&X co-ordinates that the cursor must move from the current position. Useful to generate a rectangle with defined lengths.

If southern hemisphere is selected, the movement will be : Dy (to the left) Dx (to the bottom).

If northern hemisphere is selected, the movement will be : Dn (to the top) De (to the right).

 

 

pOlar

 

A polar (distance and direction) will be calculated from the previous position of the cursor.

FREE

 

Indicate the cursor position by freehand. The position of the cursor will be excepted without moving or jumping

J

This cursor mode  allows you to determine a position between 2 indicated points.

 

EXAMPLE:

In this example, draw a line from point F to E, where E is midway between C & D while C is midway between A & B

1)                Start your line at point F with the Linepoint cursor mode.

2)                Activate the 2pointJ option (By pressing CTRL J) You will be prompted for the first of the 2 points.

3)                Use the Linepoint mode and snap to point D.

4)                You can use the 2pointJ mode again while in the same sequence.

5)                Activate the 2pointJ option again. This allows you to jump to points A and B as Linepoints.

6)                The second 2pointJ function determines point C and the first one point E between the calculated C and D points.

 

 

 

 

Defining a sequence of snapmode settings that must be repeated

 

Use this option when you are repeating a certain task that requires different snapmode formats between the activities.

 

A good example would be when you are adding cad lines where the start point is ENDPOINT mode and the end point is MIDPOINT mode.

Adding one such a cad line would not be a problem and you can easily change the snapmode between the actions. When you have to add 20 lines like this it would be much easier if the program could automatically set the desired snapmode.

 

The Repeat option is designed to change the snapmode formats based on the text string entered in the edit box.

 

How does it work?

1) All the snapmodes uses an associated shortcut identification letter and the appropriate letters must be used to define the sequence. If you want to repeat the Linepoint, MidPoint options, enter LM in the edit box.

To repeat the snapmode options TeXtpoint, cIrclecentre, enter the letters XI.

 

2) Once the letters are defined, click the Repeat button. Model Maker will exit the dialog box and set the first snap mode.

 

3) After you have pressed the mouse button and the first action has been completed, the snapmode will be set to the format as defined by the second letter in the sequence.

 

4) When the last letter has set it's snapmode and you click the mouse again the sequence will be repeated with the first letter as snapmode.

 

5) To stop the repeating action of the snapmode sequence, just activate the snapmode dialog box again by pressing the Spacebar of by clicking on the snapmode indicator.
