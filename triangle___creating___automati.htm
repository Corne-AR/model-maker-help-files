MENU : Tri - Csearch

 

Model Maker has designed algorithms which are extremely powerful and are used to quickly create triangles. Triangles can only be formed if 3 or more points exist in the active project.

Select Tri in the left hand side menu.

 

Select Csearch in the bottom menu. The Generate Trianglesdialogue will appear in the working screen.

 

Select the option to be used.

 

Model Maker will go through the standard procedure of sorting the points and checking for duplicated points. The program will inform you of any duplicate point numbers. If selected, MM will deactivate the duplicate points and continue to search for triangles.

 

If duplicate points do exist and you have not selected the automatic skip option, the following must be considered. Note the point numbers and delete the points manually via the Pnt Del  menu.

 

If triangles already exist in the active project, use the Continue search, or retain the existing ones and continue the  search, or manually add new triangles.

 

The continue search option automatically uses the Normal search process.

 

To generate new triangles, two search methods are available, QUICK and NORMAL SEARCH.
