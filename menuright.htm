MENU : Right side of main screen

 

 

The menu on the right of the screen has the following components.

 

Project indicator

 

The project buttons represent each DTM project

 

 

       The active project will have a green indicator on the side.

       The other inactive projects will have a red light.

 

To select the active project, click on the selected button numbered 1,2 or 3.

The active project may also be changed by pressing ALT (1,2 or 3) on the keyboard.

The project colours are assigned in the colours menu.

 

Note the importance of the active project on the following functions.

               On the top menu, the triangles, contours and quick contours are all drawn from the data of the active project.

               Editing points, triangles, volumes and analysis are also done on the data of the active project.

 

The display buttons.

 

 

The buttons just below the project indicator control the display of the projects.

In the example, projects # 1 & 3 will be displayed.

Project # 2 remains on the database but will not be shown in the plotting output.

 

 

Please also see Projects for more info

 

 

The monitor button opens a panel with check boxes that can be used to select the items to be plotted on the graphics screen.

 

These items can also be selected on the plot parameter menu.
