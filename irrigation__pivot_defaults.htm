Default Pivot short-list:

In the Default Pivot short-list, you can click the “Add current pivot to short-list button”. This will add span sizes and properties of the current pivot to the short list for easy use next time you need a similar pivot model. Left click the Default 1 name and type in your own Pivot Description for each template.

To activate the properties of a pivot in the short list, simply double-click a shortlist entry and the spans details will become the current pivot details.
