PROJECTS : Use of projects

 Top  Previous  Next

 

Certain tasks can be accomplished with only one project, while others make use of different projects.

 

The basic concept of projects is that Model Maker loads the data of each project into different databases. The databases are manipulated individually. Data can be exchanged between them and information like elevations and volumes can be calculated between them.

 

Up to 21 projects can be used.

 

Project 1 is generally used to manipulate contour plans.

Project 2 holds any design terraces done on the ground model.

Project 3 holds data extracted from project 1 or project 2 or of any combination of the other projects.

 

Use the different projects to display surveys of completely different areas or same area but with different survey dates.
