MENU : Co-ords Edit - Points

 

This menu is similar to the NEW projects option.  Specify your station configuration before moving onto the actual editor. The points editor is basically a line editor, meaning that you need to complete the entry of a full line before you moving up or down the list.

 

Note that for a KOORD station, the database is simplified to only the final co-ordinates, while normal tachy stations hold the full survey information like VERTICAL ANGLES, DIRECTIONS and DISTANCES.
