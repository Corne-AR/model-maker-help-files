MENU : Min - Optimize

 

This function will determine the largest balanced platform level within a perimeter string given the desired cut & fill embankment slopes.

Optionally, a dtm design surface may be used as a lid in place of a theoretical flat surface.

 

The cut and fill slopes refer to the side slope that will be used to tie the design surface to the ground surface.

 

If there is no comapction enter a zero value. If there is a compaction to be calculated, enter the value.

Example. A compaction of 90% must be entered as 0.10.

 

The grid interval. The process is done on a regular grid interval and it should be sufficiently small enough to achieve an accurate result.

It is suggested that the grid interval should not be greater than 0.05 times the length of the shortest segment of the original string.

In the example (below) the shortest segment length is 30 meters so the grid interval should be set to approximately 1.5 meters to get a good result.

 

Convert Boundary to cad. If this option is checked a cad line will be drawn around the biggest flat area that can fit within the boundary string.

compaction

 

Balanced Cut & Fill

 

If this option is not checked, the optimize function will determine the biggest flat surface (with a balanced cut and fill volume) to fit within the boundary string.

 

If this option is checked, a design surface (dtm points and triangles) must be present in the indicated project.

The design surface will act as a lid on top the ground and the elevation change that must be applied to the design surface (to place it in a balanced cut and fill situation) will be reported.

 

·The design surface is not affected.
·The user must apply the elevation change to the design surface if the surface is to be placed at the determined height.
·Notice - The design model cannot be the same as the active project since the active project is used as the ground model.
·Also, project 2 is used internally during the calculations, the design project can therefore also not be project 2.
·The standard would be to have the ground model in project1 and the design model in project3.

 

 

In this example, 8 points were used to create a surface according to certain slope values.

 

The white outline indicates the area to be optimized. Notice that the design surface is much larger than the boundary.

 

The boundary will be used as the perimeter toeline and the toepoints on the design surface will be extend towards the inside of the area.

Fill Only

 

 

 

 

 

With this setting a model will be created with a flat top at the specified design surface elevation.

 

Use this option to quickly determine the volume of a proposed mine dump with toepoints inside the defined area, with the defined slope parameters and maximum top elevation.

 

 

 

Output resulting grid data.

 

If this option is checked, new survey points will be generated in the selected survey project on the defined grid interval. These points will represent the final optimized model.

 

·The output project must be empty (no survey points) and it cannot be the active project, project 2 or the design project.

 

Use a general string to indicate the total stand area (Original string).

 

Cut and fill volume calculations will be done inside of this original string and the toe points of the embankment slopes will fall within the string. Model Maker will create a temporary platform, balance it and indicate the largest horizontal area (calculated platform area) that can be excavated within the perimeter string.

 

If the design model option is selected, the elevation that the design model must move up or down will be reported.

Optimize One

 

A single area as defined by an indicated string will be optimized.

 

Optimize ID

 

All the areas of the identified strings will be optimized. On the optimize parameters, select store to file, so that all the results may be printed at a later stage.

In the ID format, only the largest flat surface will be calculated. The design model as a top surface is only available when optimizing a single indicated string area.
