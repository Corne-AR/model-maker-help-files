Instructions to load Markers into Model Maker from a KML file.

 

 

 

 

1) Open Google Earth.

 

2) Use the place mark function and define the 4 (or more) corner points of the area you want to place grid points on.

 

3) Save the markers as a KML file.

 

 

         

 

 

 

 

 

1) Run Model Maker.

 

2) Use the Import ASCII option and load the KML file.

 

3) If the Google KML option is not shown:

  a) Make sure you have the latest version of Model Maker.

  b) If the option is still not shown, reset the Main Menu.
