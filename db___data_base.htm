MENU : dB - data base

 

The General data base function in Model Maker allows you to access an existing or create a regular data base file (dbf) that can be plotted along with other CAD or survey projects.

 

The Shape data base option allows you to load a SHP file and plot the contents of the database on screen, along with all the other Model Maker elements.

 

The Line data base is designed to hold regular 2 point lines and must has a certain defined field structure.
