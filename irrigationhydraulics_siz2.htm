MENU : Irr

 

Flow in pipes

This box displays the relationship of the pipes operating in the designated shifts. Each row contains the information related to each pipe.

The first column is the pipe number, the second is the maximum flow in each pipe and the third column indicates in which shift the maximum flow is occurring.

Columns 4 and on, display the flow in each pipe during the relative shift.

To see which blocks operate together in a shift, double click on the selected shift column. The blocks active in the shift will flash. Press any key to return to the menu.

 

Pipe Sizes

This box displays the current pipe size along with the proposed size based on hydraulic and economic parameters.

The first column holds the pipe number. Double click on the line, Model Maker will change to the graphics screen and allow you to edit the pipe.

The second column holds the current pipe diameter.

The third column displays the proposed size based on hydraulic considerations.

The hydraulic sizes have been calculated by using the smallest pipe sizes that will supply the required pressures at the valves based on the input pressure at the pump.

 

Note:

The second line from the top holds the total cost of the current and proposed pipes. These costs are just a straight addition of all the pipe lengths multiplied by their unit costs. The system life span and interest rates are not used in this calculation.

 

Pressure at pump

Use this box to input the pressure at which the  pump will operate. Model Maker will, based on this pressure value, calculate a set of pipe diameters that will satisfy the pressure requirements at the valves.

 

Note:

If the pressure at the pump is too low so that no set of pipes can be found to provide sufficient pressure at the valves, the word ERROR will be displayed in the proposed pipe diameter column.

Also note that the update panel will be displayed before the calculations are performed.

 

The update button can also be pressed to start the proposed pipe sizing.

 

Cost of system

This panel holds the cost values of the system. Notice that these costs are calculated for the full life time of the system. The costs takes into account the interest rate, applied to the capital costs and the inflation rate, applied to the pumping cost.

 

The life time of the system is set in the dialog box.

 

Proposed button

Click this button to make the proposed pipe diameters the actual pipe diameters.

 

Report

Click this button to print a report of the economic analysis to the editor.

 

Velocity

Use this input to limit the maximum velocity in the proposed pipe sizes. Model Maker will increase the diameter of a pipe if the calculated flow velocity exceeds the maximum limit.

 

Maximum pipe diameter

The list of all the available pipe diameters within the selected group are displayed. If you do not want Model Maker to consider all the pipe sizes when determining the optimum set of pipes, select (click on) the biggest pipe diameter that may be used.

 

Click on this button to start the calculations. Note that the new pipe diameters will be listed in the PROPOSED column of the pipe sizes list box. To actually make use of these diameters, click the Proposed button to copy the diameters over to the actual values.
