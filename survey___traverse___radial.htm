MENU : sUr - Setout

 

Setup at a specific station, back sight to another point and then automatically do joins and polars to other points taking the bearing correction angle into account.

 

This function is not a traverse calculation in the regular sense. It does not automatically allow for closures and corrections and would normally only be used in a project that does not require complete traverse adjustments.

It is a tool that allows you to setup at a point, calculate a correction angle, and then do polars (setting out) to new points. Using the data base functions of adding a station and then entering survey points from that station accomplishes the same result. However, this option is more interactive and graphical.

 

Station setup

This is the coordinates and name of the known point of the setup. Use the PICK button to graphically select a point on the screen.

 

Back Sight setup

This is the co-ordinates and name of the known point of the point of reference. Use the PICK button to graphically select a point on the screen.

 

Orientation correction

The program will automatically do a join between the station and the back sight point to calculate the direction. The SURVEYED BEARING will be related to that and the correction angle calculated and displayed. This correction angle will be added to the bearings that are entered to produce the correct azimuth value.

 

Setout to (Joins)

Enter the co-ordinates of the setout point or use the PICK button the select the point. The IDENTIFY button will include all the points that are identified in the list of joins.

Model Maker will calculate the Azimuth (direction), the bearing and the distance.

 

Polars (New points)

Generate new points. Enter the point's code, tag and either the Bearing or Azimuth from the setup station. Enter the distance. The new co-ordinates will be calculated.

Use the ADD button to copy the new points to the active project.
