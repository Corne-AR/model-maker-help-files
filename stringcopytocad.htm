MENU : gXt - Utility - str_Cad

 

Generate a copy of the string as a CAD line. The lines of a string exist independently of any CAD lines.

It is sometimes useful to have the component of a string as CAD lines as well. The original string is not altered.

The identified string may be copied to CAD as well.
