Enter the irrigation menu by clicking the Irr button on the left menu.

 

On the Project page, enter the project description and check the correct units.

Flow=m3/h, Pressure=psi, Diameter=mm and length=m.

The system consists of 4 blocks and one main line. The blocks will be defined first.

 

Go to the Block page and select Add new.

Click Define irregular area. Jump to points 1,2,5,4 to define the first block.

Two different snap modes may be used. We may either jump to the DTM points or to CAD lines. Remember, the DTM

points and CAD lines actually define the same co-ordinates.

 

Snapping to DTM points Ctrl P or spacebar and select Point or snap to CAD lines Ctrl Y or spacebar and select anY.

Define the 4 points and Press ESC. The area should be 1.69ha in size.

Now we have the option to fill the block with laterals. Before that is done, the micro jet type and lateral pipe type must be

defined.

Filling the Laterals

 

 

Now the block can be filled by using the Fill lateral option in the Blocks page.

Select the Block page.

Set the spacing for the emitters and rows. = 10 and 10.

Select the Sprinkler option and set Stagger to None. Development angle 90

Select the Border option.

 

 

Now click the Fill laterals button and select your Sprinkler Type.

 

 

Choose Lateral and the Pipe Name.

 

 

Click OK when done.

 

A ghost image of the layout would appear. Move the cursor to inside the block (close to side 1-2) and define the

position of the first lateral. The layout will be developed. Confirm the result.

 

Define manifold by cut pipes or Define the valve?

 

Select (Define the valve) and select the valve type from the list.

 

Change the snap mode to Free (spacebar or Ctrl F) and indicate the position of the valve just above point 1. Press

escape when done.

 

Note:

The correct elevation will automatically be added to the valve co-ordinates unless the point is defined outside of the

triangles in the dtm part of Model Maker. If the valve falls outside the block, Irri Maker will inform you and a zero elevation

will be allocated to the node.

 

 

If the valve falls off the DTM, manually enter the correct elevation value by:

Select the Nodes page and Edit single option.

 

Point to the valve and enter the correct Z value. You will have to extrapolate an elevation from the survey points. The

value should be at about 107.1 meters.

Now the manifold pipes are added to connect the laterals with the valve.

 

Define the manifold

In the Pipes Database define the following set of manifold pipes.

 

Pipes page and Database. (Use the same sequence as described when the lateral pipes were defined in the database.)

Because the top ends of the laterals are all in straight line, the Add Lines functions can be used.

 

Pipes page, Add lines.

 

Select the correct pipe parameters. MAKE SURE that the Manifold option is selected as the APPLICATION type.

Choose the Test Manifold.

To recap

The defined block has been filled with laterals at the specified row and emitter spacing.

The valve is defined and all the laterals connected by the manifold pipe.

Select the Global page.

If a yellow warning box is displayed at the bottom, the system is not connected in total.

Go through the design and check that all the laterals are connected to the valve with the manifold pipe.

Use the flow path option to check.

 

Click the design and calculate option.

Activate the Size Pipes page. Set the desired percentages.

 

 

Settings for the laterals pipes.

In this case only 2 different pipe sizes will be used in each lateral.

 

 

Settings for the manifold pipes.

Up to 4 different sizes may be used in the manifold.

Click the and then the button to resize the pipes.

Note: This need not be done if you want to work with the sizes as defined in the layout.

Activate the RESULTS page. Set the available pressure at the valve.

Press the calc flow along pipes button.

Irri Maker will calculated the head losses in the system and determine the maximum flow that the block will operate at.

(With a block of this size and the laterals being so densely spaced, the calculations may take a while)

The hydraulic profile of the system will be displayed.

 

Click on the laterals or manifold check boxes to view the pipe lines

individually.

 

To recap

The first block is defined and calculated. The flow and pressure to the block is recorded in the valve and can be used in the

main line.

The other 3 blocks must now be defined. If the other blocks each had a different shape, the above process must be repeated

for each. In this example, block 1 can be copied to produce the other three blocks.

 

Use the info page to get printed results.

Click the close button to return to the main irrigation menu.

To copy block 1 to blocks 2,3 and 4 activate the Blocks page and click the Copy button.

 

Click close to the top left corner of block 1. (Point 1)

The ghost rectangle is displayed.

Move the cursor to the top left corner of block 2 (Point 2)

Set the cursor mode to POINT mode. Press spacebar and select Point.

Snap to point 2. Block 2 is defined.

Do the same for block 3 (snap to point 4) and block 4 (snap to point 5)

 

Note:

If the valve node of block 3 falls off the dtm model and has a zero elevation set the correct elevation.

 

Activate block 3.

Select the Nodes page and Edit single option.

Point to the valve and enter the correct Z value. You will have to extrapolate an elevation from the survey points. The

value should be about 105.1 meters.

Sizing the pipes of blocks 2,3 and 4

For blocks 2,3 and 4 do the following.

Select the correct block number from the block list.

Go through the steps as described in the paragraph above, labeled HYDRAULICS

 

The blocks with different lateral pipe sizes and proposed pipe position.

The main line

The main line must be defined by first placing the pump and then connecting the pump and valves by means of the main

pipeline.

 

On the right - the Block number list - Select 0 (zero=main line)

Select nodes page and Add Single. Choose a pump type.

Set the snap mode to Free. $$ Press spacebar and select FREE.

Indicate the position of the pump just above point 1.

 

Note :

If the pump falls off the DTM, use edit SINGLE and enter the correct elevation. In this case it could be 107.1.0 meters.

The pump can be connected to the valves directly but we will introduce 2 Fitting nodes that will branch to the valves.

Nodes Page. Add single - Select fitting and define the position of fitting 1 (F1) and fitting 2 (F2)

Note:

If these fitting fall off the model. Use the Node Edit single function to change their elevations to: F1=107.0 and

F2=104.0

Now add the main line pipe.

Layout with positions of fittings 1 & 2.

 

                                                                                                  Note :

The position of valve 1 was manually defined while that of vales 2 to 4 were automatically defined when the blocks were

copied. We will use the same group of pipes that defined the manifolds in each block, to define the main line.

 

 

Pipes - Add Single .

Select the mainline pipe and start at the pump. Then point to F1, F2, V2 and V4. Press escape.

Start at V3 then point to V1 and then F1.

Press Escape to return to the main menu.

Shifts

 

The blocks can be allocated to operate in any combination.

Click the Shifts page - and double click in columns S1 to 4 to activate the blocks in each shift.

Each block allocated to a shift number.

 

 

To calculate the hydraulics in the main line. Global - Design and calculate.

Set the pressure at the pump to 14 psi.

Activate the SIZE page and click the button. The best pipe sizes that will supply sufficient pressure

at the nodes, will be allocated to the main line.

 

 

Activate the results page and click the Calc Flow button.

Use the shift number list box to calculate and inspect the flows in each shift.

Use the INFO page to get printed results. Save the irrigation network

Main irrigation menu - Click on the Projects page and then Save file.
