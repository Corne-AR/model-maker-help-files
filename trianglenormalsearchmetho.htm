TRIANGLE : Normal search method

MENU : Tri - Par

 

This finds triangles  the opposite way as the  QUICK SEARCH  method. First the triangle is formed between the 3 left most points. The other triangles will be added by standing in the middle of the open sides of existing triangles and then form the new triangle to the closest point within the maximum radius and search angle as set in the SEARCH PARAMETERS  menu.

 

Aspects on  QUICK  and  NORMAL SEARCH methods :

 

1)        The search radius and angle apply to NORMAL SEARCH only.

 QUICK SEARCH  works without these 2 parameters.

 

2)        QUICK SEARCH  can only be used as a method to search for triangles from scratch. NORMAL SEARCH  works from scratch or can        be used in the CONTINUE SEARCH option to find triangles from those already in memory.

 

3)        Both methods will adhere to the option that you may instruct it to search to only certain points, not to certain points or only the  ID'd        points as specified in the SEARCH PARAMETER  menu.

 

4)        Using the  ID'd Cadline function where the triangles are in the initial creation process formed in such a way that their legs will not        inspect an  ID'd CAD line, works only with the  NORMAL SEARCH.

 

5)        QUICK SEARCH  finds triangles about 3 times faster than  NORMAL SEARCH.

 

6)        If you have a project with, for example, two mine dumps 200 meters apart, NORMAL SEARCH  will find the triangles of the first dump        but be unable to continue on the second dump if the search radius is set to 30 meters.

 

It could be that even if you increase the radius to 300 meters  NORMAL SEARCH  will still be unable to find triangles on the second dump. This is caused by the restriction of the search angle being placed on the process. Remember the search angle is there to prevent  Model Maker from creating long thin triangles. It cannot jump from one dump to the other without long thin triangles. Two different methods to overcome this problem :-

 

 

a)        After using  NORMAL SEARCH  and all the triangles of the first dump are found, use the  ADD TRIANGLE  option and manually add at least 2 triangles with open sides in dump 1 to points in dump 2.

 

Use the  CONTINUE SEARCH  option. Triangles on dump 2 will be formed because Model Maker forms the triangle added manually to dump 2, search to the points within the set radius.

 

Use the  WIPEOUT  function to delete the long legged triangles that you added manually to create the bridge between the two dumps.

b)        Use  QUICK SEARCH  as the searching method from the outset. Triangles will be formed between all the points. Use the  WIPEOUT LINE  function to delete unwanted triangles between the 2 models.

 

7)                The final triangle model for  Model Maker is no different no matter which search process was used.
