The caption of the dialog box, eg., IDENTIFY DTM POINTS (0)  gives an indication of the amount of points that is currently identified, e.g. 0

 

Click radial button to activate option Include / Exclude

The whole identification process has a different result depending on this setting. If the include option is active, Model Maker will identify the necessary points that meet the required parameters. Setting this option to exclude will result in a point that has already identified, becoming unidentified (not  marked or included in the list) if it meets the identification parameters. It is often easier and quicker to identify everything and then use the exclude option.

 

EXAMPLE: Identify (include) all the points in a survey and then un-identify (exclude) all the points with a code of MHIL (maybe ManHole Invert Level)

 

If this box is checked you will be returned to the regular points menu after selecting an action from the this dialog box. If it is not checked, you will be returned to this dialog box to select other menu options. You would then have to press CLOSE to exit the dialog box when done.

These operations only work on the current or active project. The active project is indicated by the highlighted number displayed on the bottom left hand corner of the menu.

                You can activate a project by clicking on the required project number.

 

 

 

Check whether you are including or excluding from the list

Previously identified points remain identified until they are unidentified. Use the Show option if you are not sure. This will display all the identified points.

 

Use the NEW function to un-identify everything if required. You may wish to keep points identified and just add to the list. This means that the process works on an inclusive basis. Identifying points by code TOP and then identifying points by elevation above 1000m, means that all the points coded TOP irrelevant of their elevation plus all the points above 1000m, irrelevant of their code will be identified.

 

Here is another example of how include and exclude work together to identify only the points coded TOP which are above 1000m:

 

Identify all the points coded TOP. Change the IDENTIFY STATUS (include) to exclude. This has the effect that the identify function now works on an exclusive basis, meaning that any point complying with the identify parameters is in fact un-identified.

 

Now identify all the points between elevation 0 and 1000m. Because of the EXCLUDE function, all the points falling within the specified elevation limits will be un-identified.

 

This results in all the TOP codes below 1000m which were identified in the previous command, being un-identified.

 

 

The identify function can be compared to a flag being set or reset for each point. Identifying a point will set the flag. If the same point is identified again, using perhaps a different id function, the result is still the same. the point remains identified. The NEW function resets all the flags.

 

The  MANIPULATE ID  (Manid) functions search the whole database and each point with a set flag will be manipulated according to the option selected from this menu.

 

Most of the MANIPULATE id functions are basically intended to work on points of a KOORD station. Changing codes of points may be done on all data.

 

Functions in  MODEL MAKER  where you may use the identified points apart from those found in the MAN_ID  menu are :

 

a)                WIPEOUT                TRIANGLES                :  Delete all the triangles connected to identified points.

b)                DELETE                POINTS                :  Delete all the identified points.

c)                SAVE                        TOTAL FILE                :  The total file saves all the identified points plus all the triangles                                                                                                                                                                                                                                                                                                                                                           connected to these points in an ASCII file.

 

d)                PROJECT                COPY                        :  Copy the identified points of the active project to a different                                                                                                                                                                                                                                       project.

 

e)                DATA - GENERAL - CONSTANT                                :  Apply the functions on this menu to only the identified points.

 

 

Exits and goes to Manid menu.
