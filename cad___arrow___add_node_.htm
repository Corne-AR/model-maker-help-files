MENU : Cad - aRrow - Add                                

 

Define arrows in the same way that you add poly lines to CAD. Always start the arrow from the tail to the head (pointer).

If the arrow is a single line, indicate the tail and then the head.  An arrow pointer will automatically be added to the second point.

 

Up to 10 nodes can be added from the tail to the head. Once the last node is indicated, press Escape. Model Maker will automatically assume it is the head and add the arrow pointer.  Press '+'  if the last added arrow must be deleted.

 

The arrow will be drawn in the current Pen parameters.
