MENU : pAper

 

Paper windows providesa method of placing the paper area onto a section of data to plot.

 

 

Note:

If the Size option is set to physical, and a serial plotter is connected, switched on, and the paper is loaded, the paper size will be supplied by the plotter.

 

New P

The paper size set with the size option will be shown on the screen. This rectangle can be moved to any position using the arrow keys, or the mouse. Press the Enter key, or click on the mouse to indicate the new position. Up to 50 plot windows can be set in advance. The page up/down keys rotate the window to fit the data.

 

The snap mode will apply to the windows Eg.  Setting a new window to the endpoint of a line.

 

Show

Select this option to show the current window.

 

Clear

This option will delete all the paper windows which have been set.

 

Delete

This option will delete the indicated window. Point to the inside of the selected window.

 

Size

Keyplan

Input the key plan window by indicating the bottom left (first) point and top right (second) point. This key plan may then be plotted on plans with an indication of the current paper window on them. Before defining the window, make sure that the correct plotting scale is set.

 

Scale

This option is used to change the plotting scale. The dialogue below will appear. Enter the new scale size. The paper size relative to the plotting area depends on the specified plotting scale. The scale may also be changed in the plot parameters.

 

Maximum

The dialogue below displays the maximum scale sizes for each paper size (A4 - A0). This is the maximum scale at which the project can be plotted on a single sheet.  It is displayed only to help plan the layout.

 

Move

Move an existing paper window to a new position. Enter the number of the window you want to move. Use the SHOW option to display the window layout. Move the defined window to a new position. All the cursor modes (eg rotating) may be applied.

 

Files

Model Maker allows you to load and save the window settings. Use this option if you have a complicated sheet layout that you want to use again.

View Ports
