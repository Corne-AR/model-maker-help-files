MENU : gXt - Vert - View

 

Put the indicated strings into the vertical view mode. One or more strings may be viewed simultaneously. The co-ordinate system will then work, not as Y&X co-ordinates, but as chainage distance and elevations (Z value of nodes).

 

The chainage system will be dictated by the first indicated string. The vertical distortion, that is the ratio between the horizontal and vertical axes, may be set.

 

Another option is to extract and draw the long section as taken from the active DTM project, along with the strings. This section will always be drawn in white and may not be edited. Any of the vertical grading options may be used while in the vertical view mode.  

 

STRING : Hor

 

Set the graphics back to the plan view mode. The strings displayed in vertical mode are placed back into the plan format.
