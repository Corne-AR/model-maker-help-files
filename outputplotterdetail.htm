MENU : Output - Plotter / printer

 

 

A) This dialog has 2 tabs. Detail of the output and selection of North Arrow patterns.

 

Output Detail :

 

Plot & Label all points

  This will label all the survey points independent of their distances apart. No check is made to see if any labels might overwrite each other if they are too close together.

Do not label those closer than specified distance

  Only the points further than the specified distance will be labeled. The rest will only be marked by the positional dot.

Label only identified points

  Only the points identified with the Pnt - Id options on the graphic screen will be plotted.

Rotate the points that are within 10 millimeters of each other ?

  Model Maker has the facility to rotate the detail labeled along a DTM point if 2 points are closer than 10mm from each other.

 

North Arrow : Select if a North Arrow must be drawn and where on the paper it should be.

  You may also select different patterns to represent the north arrow. The tab - North Arrow Patterns - has the options to specify the list of patterns to be used for north arrows.

 

You may also select to display the filename, keyplan, viewports, units and contour intervals in the title block.

 

 

This tab manages the quick select list for the north arrow pattern.

 

The list will always have the North.pat pattern as the first item.

To add a pattern to the quick list, click the Add Pattern and it will be added to the first open slot.

To remove a pattern, select it and click Remove pattern.

 

Remember the patterns in the list will be available on the Output Detail tab as a drop down so you can select the desired north arrow.
