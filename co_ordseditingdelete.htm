MENU : Co-ords - Editing - Delete

 

The delete function on the graphics screen does not remove the points from the database, instead it changes the codes of the points to " *.* ". This code indicates to MODEL MAKER that the points must be treated as if being deleted.

 

The point will not be plotted or edited. Be sure that triangles connected to a point are deleted before the point is deleted. Use the WIPEOUT SPOT  function.

 

To remove the point from the database, use the  DATA - GENERAL - DELETE function from the top menu. Model Maker will check for any triangles still connected to deleted points.

 

These triangles will be removed from the database and the rest will be updated to fit the revised database after the points are removed.

 

Often, it is easier to copy the points to a different project, leaving the unwanted points behind.

 

Use the POINT ID  function and then copy the identified points to a different project using the PROJECT - COPY  command. This process is discussed in detail under the PROJECT part.
