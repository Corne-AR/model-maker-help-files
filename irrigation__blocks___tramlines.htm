Creating a tram line layout in a drip block.

 

 

When doing a simple double line layout, the option to specify 2 lateral spacings can be enough.

 

For a more complicated tram line layout, click the Tram Lines tab to access more complete features.

 

0) The main lateral spacing. The laterals will be spaced by this distance and then the tram lines (see below) arranged around it.

 

 

 

1) Enter the number of tram lines in addition to the main lateral spacing (0) defined in the layout tab.

 

2) Enter the offset spacing between the lines. The main lateral will always be at offset = 0 and the tramlines at offsets in relation to the lateral.

 

In this example, 5 lines will be created.  Main laterals are 10m apart and parallel to each main lateral, 4 additional ones are created and all will be 1m apart.

Negative offsets are to the left, positive offset value is on the right of the lateral.

 

3) Check this box if each lateral group is first connected to a cluster and the cluster then connected with a single pipe to the manifold.

 

4) Press the button to start the fill action. Two dialog boxes will be shown. The first one is to select the dripline pipe and emitter and the second to select the end node type. If checkbox (3) is checked, a third dialog box (see below) will be displayed.

 

See description below if the manifold is in the

middle of the block.

1) The node used to make the cluster connection. This parts that make up the physical cluster will be associated with this node.

2) The node connecting the cluster to the main manifold. The parts used for this connection (ie a saddle) will be link to this node.

3) The pipe connecting the cluster to the manifold. This pipe will always be included in the layout as a manifold pipe (even if you selected it as a lateral type).

The pipe's diameter will also be fixed so the correct size must be selected from the pipe database. Since the diameters is fixed, it will not be changed when pipe resizing is done on the hydraulics dialog box.

4) The pipe type used to build the manifold.

5) These nodes will automatically be defined as 'dummy' nodes since their only purpose is to clarify the layout and do not exist in reality.

 

The image above shows the connection when the manifold is on the side of the block.

Please note: Before the layout is developed, a ghost image of the proposed pipes is shown allowing you to rotate and place it. The mouse cursor is connected  to the ghost image lines on one side. (The side that controls the movement). The manifold (side of block option) will automatically be created on the side of the lines connected to the cursor.

 

 

Click OK and indicate the cadline. If the line is not yet defined, click Cancel and use the cad function to create the line.

 

For the program to know where the manifold will be in the middle of the block,

a regular cad line must exist which you need to select. This line should be a single cadline and not a polyline.

 

 

 

Just like the manifold on the side of the block, the middle manifold uses the 4 defined components to create the sub manifold.

 

1) The cluster node

 

2) The node on the main manifold.

 

3) The cluster to manifold pipe.

 

4) The actual manifold pipe.

 

 

The advantage of this automatic layout function is:

 

1) A cluster node exists for every group of driplines. The correct part number can be added to these node (in the BOM) and the quantities will automatically be correct.

2) A unique node exists on the manifold for each group of driplines. This node knows what the manifold pipe diameter is so that the correct part number can be added.

3) The correct pipe diameter can be defined for the cluster to manifold connection so it works hydraulically correct.

 

4) The manifold will automatically be developed between all the nodes (3).

 

The user then just have to define the valve position.
