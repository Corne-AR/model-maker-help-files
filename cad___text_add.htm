MENU : Cad - Text - Add

 

To add text, define the position of the text . When the position has been defined the dialogue below will appear.

 

Set the default text parameters in this menu. Activate the menu by clicking the button on the right screen border. The menu will also be presented automatically whenever a text element is being edited.

 

How to add text.

 

1.        Select the CAD button on the left hand side menu.

2.        In the bottom menu, select Text.

3.        A sub menu will appear.  Select Add.

 

RTF

This new feature allows you to enter text in a ritch text editor and place it as a single entity on the cad drawing surface.

 

How to select windows fonts.

 

               Height and Width:-

 

These values are set in millimeters. Use the drop down arrow to select from the predefined sizes. Text is drawn on the plan in relation to other elements based on their sizes. For example, if the scale is set at 1/1000 and the text size at 2mm height, this text will fit easily between 2 parallel lines being 3 meters apart. This is because 3 meters on a 1/1000 scale is physically on plan 3mm apart. If the scale is changed to 1/2000, the text (if not changed) appears to be twice the original size.

This is not so. What happened is that the 2 lines being 3 meters apart are now physically 1.5  mm apart on plan, making the text seem too large. Therefore, text size should be decreased to fit in the available space. For this reason, when the scale value is changed in Model Maker the program will display the text update option in order to deal with the text.

 

The text sizes of all the elements may also be changed in the text menu.(Rescale).  The CAD identify menu also holds options to change text sizes. When the height or width input box has focus you may use the + and - keys to dynamically expand or shrink the text size. This feature is useful to precisely control the text size in a confined area. Double clicking on the height or width input boxes will activate the dialog box. Select a predefined text ratio to be applied when the size is changed.

 

               Direction:-  Set the direction in degrees. Use the drop down list to select from default values. Whenever text is moved or copied, use the short cut keys to change the angle, if  necessary.

 

Styles

 

Line and text styles can be pre defined. Click the button to access the styles dialog box.

 

Get/Set element

               Element:-  Retrieve the attributes of any current text element on the drawing and use in the text parameters. Check the boxes of the attributes to grab an element. Click the button to point to the text entity of the attributes. The menu parameters will change to the values taken from the indicated element.

 

Use the set option to apply the selected attributes to other indicated text.

If the checkbox is checked, the set parameters will be applied to all the identified text.

 

               Reset:-  Use this button to click all the check boxes in this group to a default state.

 

Navigate

               Nav:-  When text is entered using the attributes dialog box, the actual picture of the text string, as it is being typed, is displayed on the graphics. The Nav box can be clicked on at any time. This will allow the position of the text string to be        moved on the graphics. The following keys are used.

 

               Page up and down:-  Rotate the text around the base point. Keep the Ctrl key down to rotate in larger increments.

 

               Arrow keys:-  Moves the text vertical and horizontally. Holding down the Shift while using the arrow keys will move the text in larger increments. Holding down the Ctrl key while using the arrow keys will move the text at right angles to the text direction.

 

               Home key:-  Place the text horizontally.

 

               Ctrl Home:-  Aligns the text with the closest CAD line.

 

               Move:-This button is active when new text is being entered or text is being edited. It allows you to move the text around on the drawing while changing the parameters. Use the short cut keys while the text is displayed as the white rectangle.
