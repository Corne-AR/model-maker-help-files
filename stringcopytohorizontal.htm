MENU : gXt - Utility - str_Hor

 

The horizontal component of a string is very similar to the horizontal alignment of a road or other feature. Model Maker has the main function Hor on the left side of the graphics menu that features all the horizontal alignment options. This option, str_Hor takes an indicated string and places its horizontal component in the horizontal alignment part of Model Maker. This horizontal alignment can then be manipulated and used to extract cross-sections via the Section menu.
