MENU : Cad - Id - do_Id                                

 

Identify CAD elements to edit their global functions.

All the functions that identify elements are here. Identified elements will be drawn in red. Unidentified elements are drawn in green. The red and green colours can be changed to any user selected colour.

 

Reasons to identify CAD elements include:

 

1.  Using the Manid functions to copy, move, delete, rotate, scale and change all the identification elements.

2.   Setting the search parameters  to identify CAD line to function as a break-line and prevent the triangles from crossing it.

3.   Automatically updating the triangular model through the auto-edit function.  

4.  Saving identified elements to disk.

 

 

 

Identify on the fly.

 

If this option is selected, the mouse cursor (when in neutral mode ) has 3 new functions.

 

1) Just by pointing to a cad element the element will be identified/un-identified.

2) By clicking and dragging the mouse over an area, all the elements in the are will be identified.

3) By clicking the RIGHT mouse key while keeping the SHIFT key down, brings up a abbreviated Identify dialog box.
