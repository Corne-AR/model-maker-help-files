MENU : gXt - Vert - Special - Add node

 

Add a new node to the string. Remember that the nodes of general strings consist of the vertical and horizontal component in one. Therefore, adding vertical nodes to a general string will automatically add a horizontal node on the line joining the two original nodes. Alignment strings the vertical and horizontal components are treated separately. Adding vertical nodes will not influence the position of the horizontal component.
