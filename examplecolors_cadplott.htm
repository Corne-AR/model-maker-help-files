EXAMPLE : Colors - CAD & plotter

 Top  Previous  Next

 

This example is  an overview of the relation between the colour of CAD lines and the plotted output.

 

Model Maker has 255 different colours that you may define when working in CAD. These colours are used when plotting to the screen and plotter and are set and controlled in the color menu.

In the color dialog box you can map screen pen colors to the desired plotter colors and line thickness.

 

 

 

In the color dialog box, use the 3 columns on the Pen Palette tab to set the relationship between screen and plotter pens.

 

A) The screen pen color

B) The corresponding plotter color for each screen color.

C) The line thickness (on the plotter) for the pen color.
