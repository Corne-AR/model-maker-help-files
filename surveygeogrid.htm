MENU : sUr – Geo_Grid          

 

This function allows you to draw Geographical gridlines in a LO co-ordinate environment.

 

 

Grid Corners

Specify the top left and bottom right position as geographical latitude and longitude co-ordinates

Data Format

The data format of the latitude and longitude co-ordinates that will be used. This will also be the data values that will be drawn on the geographical gridlines.

 

Scale & LO

The scale of 1/10 0000 or 1/50 000 is used to define the interval of the major grid lines. For additional gridlines, make use of the Number of extra gridlines.

 

Lo System

Enter the longitudinal central meridian in this input box. This is the odd numbered central meridian in the vicinity of the work area.
