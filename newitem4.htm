MENU : 3D

 

This menu allows you to do the following:

               Create a gridded DTM file. This can be a binary data file, used to draw the wire frame 3D view,

         or a regular ASCII co-ordinate file.

               View the gridded binary file as a 3D wire frame.

               Use the Quick option to view the 3D as a bitmap image.
