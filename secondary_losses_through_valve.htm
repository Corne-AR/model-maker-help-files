Secondary losses through Valve

 Top  Previous  Next

 

Secondary losses in valves are calculated by using the following approach.

 

In the valve database you may enter a valve diameter value.

If the diameter in the database is zero then the diameter of the upstream pipe (when the valve is placed in the system) will be used to calculate the secondary loss in the valve.

If a value other than zero is entered for the valve, then the pipe connection diameters will be ignored and the specified diameter of the valve will be used in the loss calculations.

 

 

This is how the program will display the valve info when the database diameter is zero.

Secondary losses through block valves are not calculated when the block is being designed. Losses in block valves are calculated when the hydraulics of the mainline is done.
