Line and Text Styles

 Top  Previous  Next

 

Line and Text styles is a structured system used to store predefined drawing attributes.

 

When producing cad drawings, certain features (roads, hedges, telephone lines etc.) are normally drawn with the same selected pen colors, layers names, line types, text sizes and font.

Saving multiple combinations of these attributes for quick selection is made easy with the pen styles system.

 

Explaining the various elements of the pen styles dialog box is done by the referenced call out symbols.

 

 

 

G) Layer Name. New elements will be drawn on this layer when the style is selected.

 

H) Sets the pen number (color) for the style.

 

I) Specify a line width (mm). If the value is left at zero the default line width will be used.

 

J) Specify the line type.

 

K) New text height and width.

 

L) The text origin.

 

M) The text font name

 

N) The tree list of style names.

 

O) Expand - open all the sub categories of the main styles.

    Collapse - Shows only the main categories.

 

A) When adding a new pen style, use this text box to enter the new style name. After the name has been entered, click one of the ADD buttons.

 

Styles are ordered in categories.

B) To add a main category, click the category add button.

C) To add a sub category, click the sub add button.

 

D) To delete a main or sub style, select the style and then click the delete button.

 If you are deleting a main category style, all the sub styles of that category will be deleted as well.

 

E) Each style sets values for lines and text. You may however select that only lines or only text global attributes must be updated when a style is selected on the main page.
