MENU : Cad - Text - Block

 

Use the block text enter dialog box to copy multiple rows of text to CAD.                

 

 

The first line of text will normally start at the indicated base co-ordinate. The other lines will follow below the first line and at the text angle as specified by the current text parameters.

 

This option controls the way the lines are aligned below the first line.

Left sides        : All the text will start in a line below the base co-ordinate and continue to the right.

Right sides        : The text will all line up at the end of each line.

Centres                : The text will be spaced evenly below the base co-ordinate and will extend an equal distance to the left and right.                                                                                                                

 

This is the distance between each line relative to the size of the text.

EXAMPLE :

If the text is 2mm high and the spacing is 1, then the lines will be exactly 2mm apart with no space between them. If the spacing is changed to 2, there will be 4mm between the centers of the lines of text with 2mm  between them.
