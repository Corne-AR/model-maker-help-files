MENU : Cad - aRrow - Copy                                

 

This function allows you to copy an existing arrow to a new position.

 

Point to the arrow that must be copied. The cursor will hook onto the selected arrow at the closest node which will form the base co-ordinate to move the arrow.

 

If necessary, use the short cut keys (rotation - Page Up & Page Down, aligning - Shift Tab) to manipulate the arrow while copying .
