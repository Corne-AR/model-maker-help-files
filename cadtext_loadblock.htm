MENU : Cad - Text - Load - loadBlock

 

LoadBlock

 

Any ASCII files may be read from disk and drawn into CAD using the current text parameters. You will be prompted to indicate the base position. The first line in the text file will be drawn into the cad with its origin on the base point. The rest of the lines will be read from the file and placed in lines below the first. It is important to have the default text parameters set correctly before the operation. It is also good practice to set the drawing scale correctly beforehand.  

 

Userpoint

 

Copy user indicated points to CAD text in a pre defined block format. Town planners often use this format. It can be used to dimension stand boundaries.

 

DTM Stations

 

Copy the co-ordinates of all the DTM stations to CAD in block format. This includes only actual stations. Dummy Koord stations will be ignored.
