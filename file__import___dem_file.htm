FILE : Import - Dem file

1

 

Digital Elevation Model DEM file.

2

 

 

There are many sites available to get the files from but for illustration I use only one. You can read some more on this site: https://gisgeography.com/free-global-dem-data-sources/

You need to register on the sites to be able to download the available Free files, and for the one I want to use you register here. https://urs.earthdata.nasa.gov/users/new

Once registered go to this site to choose the correct file to download. http://dwtkns.com/srtm30m/

Zoom into the area to get the correct file.

Unzip the file and put it in to a folder on your computer, try to keep the files together so next time you need a file you might already have it.

3

 

Before you import the file set your Settings > Configuration to that of the area you want to Import.

4

 

Go to File > Import ASCII & Other > Extract Elevation data from DEMs.

Currently we support .hgt .bil .dem and .tif file formats.

5

You have 3 Options to choose from:

1.Load all the Dem points.

This will load the extends of the file.

2.Only load points falling in Current plot/screen area.

This will load the area of your current zoom level

3. Points inside a user defined area.

Here you need to select a area with the Model Maker area tool before the Ok button is available.

6

 

At the bottom you have a option to Import all or only some of the points.

This will allow you to check if the file you downloaded falls in the right area and speed up the loading process.
