MENU : Min - Rehabilitation - rehab-String

 

The DTM points of the active model will be remodeled based on the following:

 

1) A string must be indicated that will control the design elevations.

2) A string must be indicated that will define the boundary within which the remodeling will take place.

3) The slope that the dtm points must be elevated to, based on their position relative to the base string must be entered as a %.

4) All the dtm points of the active project will be inspected. Points falling within the area defined by the boundary string will be re-elevated  based on the shortest distance to the base string and the % slope that has been entered.
