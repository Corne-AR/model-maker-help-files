TRIANGLE : Cross-section

 Top  Previous  Next

 

MENU  : Tri - Xsect

 

This provides you with a visual cross-section of the model on a line through two indicated points. Use it to get a cross-section picture of the terrain to familiarize yourself with the model so you can  edit the triangles from a more informed viewpoint.

 

 

 

Triangle : Indicate CAD position

 

Check this box to make a hard copy of the section along the indicated route.

A dialogue will appear asking you to indicate a position on the screen where the section must be drawn. The section will be drawn from left to right and in the same direction as the section line.

If the box is left unchecked, the section will be drawn on the screen until a replot is done.

Indicate the start and end points along which the section must be extracted.

 

Triangle : Section between points

 

If this box is unchecked, a cross section will be drawn across the whole model. If the box is checked a cross section will only be drawn between the indicated points.

 

Vertical distortion

 

This value controls the relation between the vertical and horizontal scales. A value of 2 means that the vertical scale is going to be twice that of the horizontal (plotting) scale. This value is often increased when viewing sections of a very flat terrain. The small undulations would then be emphasized.

 

Triangle : Cross section project

 

The section will be taken through all the projects, giving a combined view or only the active project.

 

Note:

This section is not saved to disk, but is only temporarily displayed on the screen. If requested, it is recorded into the cad. Use the SECTIONS option for complete cross-section extraction.
