MENU : gXt - Id - Man-id - Combine

 

All the identified strings will be inspected and if the endpoints of any two strings fall on the same co-ordinate, Model Maker will combine the two strings.

The purpose is to have fewer individual strings with more nodes per string. This makes manipulation of the strings easier if they need to be re-elevated or moved.

This function can be used on strings generated by reading a .dxf contour file directly to strings.
