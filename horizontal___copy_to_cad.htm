The intersection point information of the alignment, eg. the curve length, deflection angle, etc., may also be copied to CAD.

Check the AT THE NODES option to place the PI information next to each intersection point.

Use the IN TABLE FORMAT to group the information in a user specified position.

 

 

Chainage @ PI

This option only effects an alignment of PI points with a zero radius value. If checked, Model Maker adds the chainage values of the PI points as text values to CAD.

 

Tangent to PI

Checked, to include the straight (tangent) lines from PI to PI in the plot.

 

 

Label PI data by using defined patterns.

 

       

 

The Pattern option allows you to use a specific pattern to display all the relevant information for the PI according to your layout

.

Use the regular cad functions and create a pattern based on the parameters as described below.

The pattern used for circular curves must be called HorPiCircle and for  transitional curves, HorPiTrans. These 2 patterns must exist in the regular pattern folder which by default is the :

\Program files\MMSystems\MM\Patdata folder.

 

This is an example of a pattern comprising of cad text and lines. The elements have specific meaning and all the different settings are explained based on this example.

 

There are 29 PI attributes that can be displayed on the PI by means of a pattern.

 

The 29 attributes are the following. Notice that the spelling must be exactly the same and in capital letters.

In the image below, the red underlined numbers are only used as a reference and should not be included in the actual definition of the pattern.

 

 

Four codes are used to display header attributes

 

0 HORNAMEVAL:Alignment name

1 PINOVAL: Pi number

2 PINAMEVAL: Pi name

3 CURVEDIRVAL: Curve direction

Five codes are used to display circular attributes

 

5 RADIUSVAL:Radius

6 DEFANGLEVAL:Deflection angle

7 TANLENVAL:Tangent Length

8 CURVELENVAL:Curve Length

Four codes are used to display the transitional curve data.

 

9 TRANS1VAL                : Transitional length

10 TANGENT1VAL        : Calculated tangent length

11 TRANS2VAL                : Transitional length

12 TANGENT2VAL        : Calculated tangent length

 

 

Fourteen codes are used to display the coordinates and chainages:

 

13 to 24        : Coordinates of the different positions of the Pi

25 to 28        : Chainages of the respective positions on the curve

In summary:

a) Use the pattern option when you want to define your own Pi description layout.

b) Draw the pattern in cad using the specific codes to identify the attributes.

c) The pattern to display a PI without any transition curves must be called HORPICIRCLE. This pattern will typically not include the codes numbered 9 to 12 since the information will not be relevant.

d) The pattern to display a PI with transition curves must be called HORPITRANS.

e) Only regular text and regular lines should be used in the construction of the pattern.
