MENU : Left of main screen

 

 

 

These are the main editing and information menus. When a option on the left is selected a new set of sub-menus are displayed at the bottom of the screen.

Activate an option on this menu by clicking the button or by pressing the Shift-key together with the capital letter of the required menu, e.g., Shift-E for the sEc menu, Shift T for the Triangle menu etc.

 

Notice that you can only select a new option from this menu if

thecursor is in the neutral mode.

 

While the cursor is in the shape of a cross the program is busy executing a command from the bottom menu and the left menu options cannot be used. To get the cursor back in the neutral mode, press Escape a few times. The action from the bottom menu will be terminated and the options on the left menu activated.
