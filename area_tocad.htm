MENU : sUr - Area                                

 

Use the Survey menu to calculate the area value of a user defined area . The define area function is used to define the selected area.

The result will be displayed in a dialog box, allowing you to export the information to the editor, CAD or strings.

The size of the area is shown.

The length of the area's perimeter is shown.

Copy the defined area to CAD as hatch. This option is only available if the defined area has fewer than 200 nodes which is the        maximum number of nodes that the hatch option can accommodate.

 

Use the PARAMETER button to set the hatch parameters before the hatch is generated.

Use the COPY button to activate the copy action. All the selected attributes will be copied to CAD.

 

Defaults

The DEFAULT 1 & 2 pages allows you to specify the area and perimeter pre & post fix attributes.

On the CONSIST page, set the number of decimals to use when doing the consistency check.

 

Strings

 

 

 

Draw the perimeter of the defined area as a string. Check the hatch option for the string to hatch itself. Use the COPYX button to initiate the copy string action.

 

Multiple areas

 

 

This option is used when multiple areas are defined in succession. After the first area is defined, click on the QUICK button. Model Maker will then allow you to define each area and without returning to this dialog box each time, execute all the commands (COPY TO CAD, HATCH STRING and/or INFO TO EDITOR) that are selected.

This allows you to define one area after another without clicking all the other buttons during the process.

Editor

 

1)                Clear                : clear the editor from any previous input.

2)                Header          : copy the standard header to the editor.

3)                Info                 : copy the actual information (area and/or perimeter) to the editor.

 

Check the combination of AREA, PERIMETER and/or ACTUAL CO-ORDINATES boxes to define the information to be copied. If the co-ordinate option is selected the additional box, CONSISTENCY is also shown. If this box is checked the co-ordinates will  be displayed as well as the join information between each leg of the area.

             

In the consistency output, each area node will have a name. This name will be the point code of the DTM point if the area was defined by using the Tacpoint option. If the area was defined by using the Nearline (follow CAD line option) mode and there are DTM points within a 0.1 radius of the defined node, the code of the closest point will be displayed. If the CAD line does not end close to any DTM point a blank code is displayed.

 

Use the File editor option to inspect the data.
