How to generate irrigation blocks form ID CAD lines

Step 1:

Draw real polylines on a unique layer. Start by creating the layer as follows:

Create a layer called “BLOCK BOUNDARIES”

 

Now draw the block boundaries on the plan using CAD > line > add

Draw your blocks using snap modes in any shape or size and as many as you like. You can also draw the blocks with segmented CAD lines, and convert them to polylines later if it helps you (not discussed here).

Step 2:

ID the lines as follows:

CAD > ID > do-Id > Layer, then select the Block Boundaries layer you used when you drew the CAD layout

 

Step 3:

Convert the ID CAD lines to Irrigation blocks as follows:

Open the irrigation Window, then go to Blocks > Auto :

Now select the option to create Blocks form CAD polylines…

 

Now click a line across the polygons in the sequence you want to number the new blocks. Irrimaker will create new blocks for the poly lines you cross with the mouse line.
