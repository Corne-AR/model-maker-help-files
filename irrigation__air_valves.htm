Irrimaker Airvalves.

How to add Airvalves in Irrimaker.

Do the Mainline design of the Irrigation.

 

 

1.To Drape the pipeline on to the surface use the “Global”  > “Divide segments” This will put the pipeline flat on the ground at the interval you specify. (based on a multiple of the pipe length)
2.Go to Global Edit Section and select the start and end Node you want to view the section of. (This might be from the pump to a Block Valve)

 

Once you selected the Start and End node of your section in plan view, choose to display both the Ground and String/Irrigation pipe on the section. (Image below)

The vertical distortion will only be applied on the screen. If the area is fairly flat increase the “Vert. distortion” value. A value of 1 has no distortion on the elevation.

 

NOTE: When the ground profile is derived from triangles, IrriMaker will determine high points based on the triangle legs. If there are no triangles and the profile is derived from a strip survey, the ground profile is generated from the dtm points within the strip width of the pipe line. When new air valves are created it would not replace any fitting nodes that are already on the high points but rather just add a new air valve node close to that existing node.

 

 

 

 

Use the “Special” > “Air_Valves” function at the bottom to add a Airvalve node.

 

 

 

You have 2 options to add the air valve:

1.Fixed Intervals. You specify the distance based on your design preference (typical length would be 350m to 500m)
2.At High points in the system. The program will add Airvalves at all the surface level high points not closer than the specified distance.

Please inspect the result is based on the surface and not the pipe level.

 

 

Typical section where you would manipulate the result.

 

 

 

 

 

 

 

 

 

To view the description on the section as above select “Display” and turn on “Description”






 

Change or delete the Airvalves to get the desired result with “Editz”; “n_mOve”; and “n_Del”

On the “Editz” popup you can change the Node elevation to be the elevation value you want, or the DTM elevation.

Select “Special” to view this menu.

 

On “n_mOve” move any of the existing airvalves to a different position between the previous and next node.

On “n_Del” Delete the airvalves you do not want to use.

 

Note: To go out of this vertical view Right click with the mouse to go back one menu.

Then select “Horiz” at the bottom to go back to plan view.

To view the plan position of the Airvalve, In the Irrigation menu, turn on the User name display of the node.

Note: Inspect the system like above to view if it placed the airvalves at the positions you want them.

Adding nodes manually on the pipe line:

 

 

.

Do the Mainline design of the Irrigation.

1. To Drape the pipeline on to the surface use the “Global”  > “Divide segments” This will put the pipeline flat on the ground at the interval you specify. (based on a multiple of the pipe length)

2. Go to Global Edit Section and select the start and end Node you want to view the section of. (This might be from the pump to a Block Valve)

 

Once you selected the Start and End node of your section in plan view, choose to display both the Ground and String/Irrigation pipe on the section. (Image below)

The vertical distortion will only be applied on the screen. If the area is fairly flat increase the Vert.distortion value. A value of 1 has no distortion on the elevation.

 

 

Use the “Special” > “n_Add” function at the bottom to add a Airvalve node.

 

Indicate the high point of the position of the Airvalve on the section.

 

The node database will be shown from where you can select the node type you want to add.

 

 

On the next popup you can also change the elevation to be the elevation value you want or the DTM elevation.

 

Use the “Display” option at the bottom to turn on the Elevations to view the high points in a area.

To go out of this vertical view Right click with the mouse to go back one menu.

 

 

Then select “Horiz” at the bottom to go back to plan view.

 

 

To view the plan position of the Airvalve turn on the User name display of the node

 

Use the following options to get a report on the air valves.

 

A) Do a global report of the nodes to see how many air valves are in the project.

 

B) To display the air valve distances, use the Long section option on the Hydraulics/Info page.

 

 

Note : The air valves are placed in the system for informational purposes only and has no hydraulic effect on the design.
