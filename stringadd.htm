MENU : gXt - Add

Use the String name input box to specify the name of the string to be defined.

 

 2 

Select a layer to place the string on.

 

 3 

Use the Pen button to set the default pen attributes of the new string. Change these attributes at any later stage by using the Edit Par function.

 

 4 

Three types of strings may be defined. You have a few different options for generating strings.

 

The general string consists of nodes, each having co-ordinates of the form Y,X,Z. The Y,X define the position of the node in plan and the Z value that of the elevation of the node.

 

Alignment strings are more complicated and are defined in the same way as a road horizontal alignment. Each node consists of the PI point and the radius of the arc at the node. The vertical component of an alignment string operates independent of the horizontal nodes and is defined in the vert menu.

 

Pipe stings are similar to general strings but also holds data for manhole diameter, pipe diameter and inlet and outlet elevations.

New string method

 7 

Indicate

Define general and alignment strings. You can point to the position of every new node. Any of the cursor snap modes may be used.

Use the Near Line mode to make the string follow the position of a previous defined CAD poly line.

 

Parallel

Add a new string parallel to an existing one. Only general strings may be defined this way.

 

Cad-par

Define a general string parallel to a series of CAD lines. You will be asked to indicate each CAD line. Model Maker will continually add new segments to the string based on the indicated CAD line, the user defined distance away from the line, and the side of the line you have picked. Use the + key remove the last segment if you are not satisfied with the result.

 

Editor

Enter the data for a general or alignment string via co-ordinates using the strings data editor.

 

Gro file

An existing GRO (Longsection file) will be read. The string will be a straight line starting from an indicated position and have the length of the maximum chainage distance in the gro file. The node elevations will be values taken from the file.

 

DTM

Use the special dialog box that allows you to add CAD lines automatically onto survey points.

 5 

The Z value new general strings will have the value as set in the Default elevation input box. If the new node is defined by pointing to a DTM point or CAD line endpoint, the Z co-ordinate of the node will be changed to the Z value of the indicated entity.

 6 

Check this option to allow Model Maker to display the node co-ordinates of the string while each one is defined. It gives you better control because you may edit and update the node information while the string is created. Unchecking the option allows you to quickly define any string without inspecting each node. The information of every node may be edited at a later stage with the node edit option.

 8 

Use this input box to specify the default node names of the new string. If the Increment name is checked and the default name ends in a numeric value, eg PI1, Model Maker will use the defined name for the first node after which the name will be incremented with one for each additional node. eg. PI2,PI3 etc.

 

This option instructs Model Maker to use the code of DTM points if the cursor snap mode is set to jumping to DTM points. The default node name will be replaced by the code of the DTM point that is used for the definition of the node co-ordinates.
