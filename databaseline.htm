Data base : Line

 Top  Previous  Next

 

This database function is specifically developed to handle regular 2 point lines and stores the line attributes (coordinates and pen) in a DBF file. This allows the user to modify the dbf record structure and add any additional fields that may be needed to associate any other information with each line.

 

 

 

DataBase specifications

 

The database used for this dialog box must be a DBF type and must at least have the following fields.

 

Y1          : Float type > holding the start Y coordinate

X1          : Float type > holding the start X coordinate

Z1          : Float type > holding the start Z coordinate

Code1  : String type > holding the start point's code

 

Y2          : Float type > holding the end Y coordinate

X2          : Float type > holding the end X coordinate

Z2          : Float type > holding the end Z coordinate

Code2  : String type > holding the end point's code

 

Pen          : Integer type > holding the pen number

Linetype   : Integer type > holding the line type number

 

To generate a default standard DBF file, use the CREATE, Standard DBF  option.

Once a standard DBF is created you may use the CREATE, Userdefined option to edit it's structure and to add additional information fields.
