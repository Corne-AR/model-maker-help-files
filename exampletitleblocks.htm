MENU : Example - Title blocks

 

This example shows how to create and use, user defined custom title blocks.

 

               You must have Module 6, CAD, active on your system to work through this example.

 

Step 1:

 

Use the necessary CAD functions and draw the title block that you will use. You may also use the DXF in function to import a title block if it is available in another CAD system.

 

Step 2:

 

Follow the steps described in the title block option to define and save the title block in the Model Maker format.

 

Step 3:

 

Activate the plot parameters menu and click on the TITLE button in the middle of the dialog box. Locate the name of the newly defined title block and select it. Exit the dialog box.

 

Step 4:

 

Click on the GEN option on the main menu. Select the Paper function and set the correct paper size by using the Psize option.

Use the Newp option and place the title block over the drawing in the desired position.

 

Step 5:

 

Output the drawing to the printer by using the Output function.

 

Step 6:

 

When the title block dialog box is displayed, edit any of the text items that were defined on layer starting with a question mark.

 

The drawing will be produced by automatically :

               Changing the text items defined on ? layers to that supplied by the user.

               Plotting the complete title block, ignoring the other elements at first.

               Clipping the active plotting area and drawing all the other elements, (survey, CAD, strings and irrigation).
