DEFINITION : Status

 Top  Previous  Next

 

 

NONE                                                : No data is loaded.

NEW                                                : The data has been entered from some source but not yet saved.

LOADED UNCHANGED                : The data was loaded from disk and no changes made since.

LOADED CHANGED                  : The data was loaded from disk and some changes were made to it, meaning that the data in computer and on the disk is no longer the same.

SAVED UNCHANGED                : The data was saved to disk and no changes made since.

SAVED CHANGED                  : The data was saved to disk and some changes were made to it, meaning that the data in computer and on disk is no longer the same.
