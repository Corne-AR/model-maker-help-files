SURVEY : Polar input

MENU : sUr - pOlar

 

 

This dialog box is  displayed when a distance and direction is requested.

 

If a regular polar is calculated, or when a line is copied and CTRL home is pressed, you need  to specify a distance and direction. This dialog box allows you to specify the values in a powerful way.

 

The dialog box is divided in two parts. The top half is used to enter the distance.

 

 

Type in the required distance  By default, the (a) input will hold the original value. Change that value and use input (b) to automatically add and subtract any other value from (a). To subtract (b) from (a), enter the value in (b) as a negative value.

Eg.    100  +   -50 = 50

 

 

To measure the distance between other points or lines use the measure menu button. You will be transferred to the measure functions and, on return, the measured value will be placed in (c).

 

To use the measured value as is, select the DIST 2 radio button. Use the button to copy the measured value up to the input box (a). You can use (b) again to add a constant.

 

Check the appropriate radio button so the correct distance is used in the final calculations.

 

 

The bottom half of the page is used to enter the direction. It can be done in one of three ways.

 

 

The first 2 methods work the same as the formats used to calculate the distance as described above. Note that the angle should be entered as degrees.minsec.

When the automatic addition is used, Model Maker will take the entered values (deg.minsec) and convert them to decimal degrees when doing the calculations.

 

 

The third option requires some explanation.

 

 

The full 360 Deg. circle system is divided in 4 quarters. In order to specify your direction, first specify the quadrant by pressing the appropriate button and enter the angle.

 

Example in the Southern Hemisphere:

NW                : The value will be subtracted from the north direction.        

                     eg. 45:                180-45=135.

 

NE                : The value will be added to the north direction.                        

                         eg. 45:                180+45=225.

 

SW                : The value will be added to the south direction.                        

                         eg. 45:                0+45=45.

 

SE                : The value will be subtracted from the south direction.        

                       eg. 45:                360-45=325.

 

Accepted values

Depending on the format (1 or 2) for the distance and (3,4 or 5) for the direction that is selected, the final value to be used will be displayed in the ACCEPTED lines.

 

If the auto pick option is checked, the Format (1..5) will automatically be updated when a change is made to the selected entry box.

 

For example, when the dialog box is entered, the DIST 1 format is selected. If you use the measure function, Model Maker assumes you want to use the measured distance and automatically changes the format to DIST 2.
