Extract of typical menu

 

Model Maker uses this type of menu to gather data. The input descriptions are on the left side of the dialog box. The input boxes and Yes/No options are on the right side.

The example above shows an input box. In this case, it will only accept numeric values. The Y/N button will toggle between the 2 options.

Use the tab key to move up and down the entries or point directly to an option with the mouse.

Click the OK button to close the dialog box and continue with your options. Use the Cancel button to abort the current process.
