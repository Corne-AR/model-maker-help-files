MENU : Pnt - Add - Poly cadline

 

Add DTM points to an indicated CAD polyline. The following dialog box defines the attributes of the new points.

 

The caption of the dialog box indicates the number of points on the polyline.

 

Points are often placed on poly CAD lines when importing .dxf files. These files could have been generated from contours and invariably have very short line segments. Putting DTM points on such short line lengths result in a cluster of undesirable points.

 

1.

Avoid clustering by specifying a skip distance between points. Model Maker will only add a new point on the polyline if the distance between it and the previous added point is larger than the skip distance.

 

2.

This is the default elevation of the first and last point to be added to the polyline. Model Maker will interpolate the elevation of the in between points if the default Z values are not the same. Interpolation is done on the basis of distance from starting point / total length of the polyline.

 

Alternative, the Z values can be obtained from the selected polyline itself.

 

3.

The new DTM points will each have a code equal to the new specified code. If the default code ends in a numeric number such as a1, and the Increment code box is checked, the codes of all the successive points will have the codes a2, a3, a4 etc.

 

4.

By default, points are not placed on the ends of the line. Here you can turn that option on and off.

Instead of placing dtm points on line connection points, dtm points can be placed on regular intervals. Set the interval spacing here.
