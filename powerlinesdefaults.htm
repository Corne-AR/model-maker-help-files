MENU : gXt - Power - deFaults

 

Most of the defaults that control the shape, attributes and characteristics of the a power line are in this menu.

 

The main elements used in designing power lines are the ground data, the poles (pylons) and the conductor.

 

The ground data is defined as regular surveyed points in the DTM section of Model Maker.

 

Data base

 

Access the database with the pole information with this button.

 

This button leads you to the dialog box where the SPT files and conductor types are defined.

 

Display defaults

 

This page sets some display features when the string is viewed in the vertical mode.

 

Enter the minimum height difference allowed between the ground and the lowest point on the conductor as per each span. If the is checked each time the string is drawn in vertical mode, Model Maker displays the position on each span where the conductor is the closest to the ground. The value is displayed in white except when the height difference is less than the minimum specified, then it is displayed in red as a warning.

 

If checked, the low points will be indicated on the vertical profile.

 

If checked, the elevation of the nodes of the design string will be re-elevated each time an adjustment is made.

 

When a pole is defined, enter a pattern name to be used as a symbol to display the node. If this box is checked, the patterns will be displayed at each node.

 

Even if pattern names are defined, the nodes will be drawn with regular lines. This may speed up the process of redrawing the design string. When displaying the string in vertical mode, the lines will be drawn the length of the pole. In plan mode the patterns will be ignored.

Copy to CAD

 

A power line is defined by creating a general string that follows the surveyed horizontal alignment. After the line is fully developed using the available functions in the GXT vertical menu, then use the picture of the power line when plotted along with the surveyed points in a detail layout plan. Looking at a string in plan format does not always provide sufficient information.

The CAD is then used to obtain information from the string and to present it on the detail plan using of dimensions and other related CAD text elements.

 

The following features may be added to the detail plan

 

 The length and direction of each span will be copied to CAD as text.

 

 The deflection angle at each turning point along the line will be labeled.

 

The string node names will be labeled. These names are the ones used when the poles were added to the string.

 

 The total length and Equivalent length between the strain points (poles with the ! before the names)

 

 When poles types are defined, you can specify a pattern name to produce the symbol when displaying the node. Define different patterns for displaying the node in plan and vertical view.  If the Node Pattern option above is checked, the pattern (if defined) will be added to the cad at each pole position.

 

After all the features to be added to CAD have been selected, press this button and then point to the string .
