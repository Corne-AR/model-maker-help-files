MENU : Irr - Blocks - Auto - Divide blocks

 

The auto function allows you to quickly divide a selected area into smaller irrigation blocks based on certain input parameters.

 

Before dividing an area into smaller blocks, first enter the necessary data which Irri Maker will use to calculate the block sizes.

 

 

Step 1:

Define the area A,B,C,D,E,F

 

Step 2:

Draw a CAD line between X-X. This line will be used as base for the block orientation. The blocks will be developed extending perpendicular across the base line.

 

Step 3:

After entering all the layout values, click on the divide button and indicate the line X-X as base. In the picture, the line was indicated at the yellow cross between the top X and the value 1.

 

Step 4:

Irri Maker will move down line X-X from top right to bottom left (from the endpoint closest to the indicated position) and place the border divisions at the green lines. Line L1-L1 being the first boundary line. Blocks I,II,III and IV are the result.

 

 

If the length of the blocks eg. L1-L1 is longer than acceptable, draw an additional line parallel to X-X at the desired distance. Line Y-Y.

 

Redefine a new sub area 1,2,3,4. The blocks will only be generated between lines X-X and Y-Y resulting in blocks being the correct size.

 

 

Layout

Spacing

Enter the row and emitter spacing. These values will be used to determine the area that each emitter occupies.

 

Emitter

Enter the discharge of each emitter. Use the Select button to choose a desired sprinkler type from the data base.

 

Flow

Enter the maximum flow available to each block. Based on the emitter spacing and flow, an approximate block size is calculated.

 

Select

Click this button to access the node data base from where a new sprinkler type can be selected.

 

Area To Be Divided In Blocks

Define area

Use the normal area definition functions to define the sub area that must be divided into smaller blocks. The sub area size will be displayed.

 

After the layout has been specified and the area defined, Irri Maker will calculate the results and display it in the results screen.

At this stage, no actual work has really been done other than showing the theoretical results.

 

Divide Sub Area

Use the DIVIDE in blocks button to proceed with the actual definition of the new blocks. In the example, the sub area will be divided in 8 blocks of approximately 20207 m2 in size. The ninth block will be 0.06*20207 square meters. These are the theoretical values and may be overwritten by the user. In the above case, you may either specify each block to be 20000m2 in size, resulting in 8.14 blocks (162843/20000) or by specifying exactly 8 blocks, each will be 20355m2 big.
